DSM mobile ger DSM admin (eller användare från gruppen administrators) möjlighet att hantera DSM-inställningar och kontrollera DSM-informationen med en iOS® (iPhone®, iPad®, eller iPod touch®) eller Android® enhet. Detta hjälpavsnitt om DSM mobile innehåller följande ämnen och resurser för att hjälpa dig att komma igång.

Logga in och logga ut

Efter att du anslutit till DSM med webbläsaren på din iOS- eller Android-enhet kommer du att se inloggningsfönstret för inmatning av dina DSM-kontouppgifter.

Så här gör du för att logga in på DSM mobile:

Skriv in dina användaruppgifter för DSM-användare och tryck på Logga in. Om du vill spara din inloggningsinformation och logga in automatiskt nästa gång, sätt på Kom ihåg mig innan du loggar in.

Så här gör du för att logga ut från DSM mobile:

Efter att du loggat in på DSM mobile kommer du att se startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) med en lista över alternativ och Synology-mobilapplikationer som finns installerade på din iOS- eller Android-enhet. Tryck på Logga ut för att logga ut och gå tillbaka till inloggningsfönstret.

Hantera DSM-användare

Tryck på Användare på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) för att se listan över användare.

Skapa en användare:

  1. Tryck på + och skriv sedan in användarnamnet och lösenordet och e-postadressen för den nya användaren.
  2. Aktivera Skicka meddelande för att skicka ett meddelande till användarens e-postadress när hans eller hennes DSM-användarkonton har skapats.
    Obs! Innan du skickar meddelandet, se till att meddelandeinställningarna har slutförts på din DiskStation. Logga in på DSM med din dators webbläsare för att aktivera denna funktion (på Huvudmeny > Kontrollpanel > Meddelande > E-posta för att ändra inställningarna.
  3. Tryck på Klar för att bekräfta eller tryck på Avbryt för att avbryta.

Så här gör du för att snabbt hitta en användare:

Skriv in användarens namn i sökfältet.

Så här gör du för att ändra en användares uppgifter:

  1. Tryck på användarens namn och gör något av följande:
    • Ändra användarens namn eller lösenord i fältet Namn eller Lösenord.
    • Välj Avaktivera för att avaktivera användaren. Du kan välja att aktivera användaren vid ett senare tillfälle utan att förlora användarens inställningar.
    • Aktivera Administrator för att lägga till användaren till gruppen admin. Användare som tillhör gruppen admin tilldelas administrativa privilegier för att ändra DSM-inställningarna med DSM mobile eller datorns webbläsare.
  2. Gör något av följande under Privilegier och tryck sedan på Bakåt:
    • Tryck på Delad mapp och tryck på den delade mappens namn för att ändra användarens åtkomstprivilegier: Läsa/Skriva: Användaren kan göra ändringar i filerna och undermapparna i den delade mappen; Endast läsbar: Användaren kan komma åt filerna och undermapparna i den delade mappen, men kan inte göra ändringar i dem; Ingen åtkomst: Användaren kan inte komma åt filerna och undermapparna i de delade mapparna.
      Obs! Om det uppstår en konflikt mellan privilegier kommer prioriteten att vara som följande: Ingen åtkomst > Läsa/skriva > Endast läsbar.
    • Tryck på Applikation för att ändra användarens åtkomstprivilegier till DSM-applikationer.
  3. Tryck på Klar för att bekräfta eller tryck på Avbryt för att avbryta.

Hantera delade mappar

Tryck på Delad mapp på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) för att se listan över delade mappar på din DiskStation.

Så här gör du för att skapa en delad mapp:

  1. Tryck på + och skriv sedan in namnet för den nya mappen.
  2. Tryck på Klar för att bekräfta eller tryck på Avbryt för att avbryta.

Så här gör du för att ändra en delad mapp:

Tryck på den delade mappens namn och gör något av följande:

  • Ändra den delade mappens namn i fältet Namn.
  • Under Privilegier trycker du på Användare och trycker på en användare för att ändra hans eller hennes åtkomstprivilegier till den delade mappen: Läsa/Skriva: Användaren kan göra ändringar i filerna och undermapparna i den delade mappen; Endast läsbar: Användaren kan komma åt filerna och undermapparna i den delade mappen, men kan inte göra ändringar i dem; Ingen åtkomst: Användaren kan inte komma åt filerna och undermapparna i de delade mapparna.
    Obs! Om det uppstår en konflikt mellan privilegier kommer prioriteten att vara som följande: Ingen åtkomst > Läsa/skriva > Endast läsbar.

Aktivera tjänster

Tryck på Tjänst på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) för att aktivera eller avaktivera DiskStation-tjänster.

Så här gör du för att hantera DiskStations LAN-tjänster:

  1. Aktivera eller avaktivera något av följande alternativ:
    • Windows-filservice gör att din DiskStation blir en Windows-filserver (CIFS) inom LAN.
    • Mac-filservice gör att din DiskStation blir en Mac-filserver (AFP) inom LAN.
    • NFS-tjänst gör att Linux-klienter inom LAN kan komma åt DiskStation-data.
    • Bonjour-skrivare ger Mac-klienter möjlighet att skriva ut på en USB-skrivare som är ansluten till din DiskStation.
    • Media Server gör det möjligt för DLNA/UPnP-hemmaenheter inom LAN att söka och spela upp multimediainnehåll på din DiskStation.
    • iTunes ger iTunes-klienter inom LAN möjlighet att söka och spela upp musik eller videor på din DiskStation.
  2. Tryck på Klar för att bekräfta.

Så här gör du för att hantera DiskStations internettjänster:

  1. Aktivera eller avaktivera något av följande alternativ:
    • Med Auto Block kan du förhindra oauktoriserad inloggning.
    • FTP aktiverar din DiskStation så att den fungerar som en FTP-server.
    • Webbtjänster gör att du kan fungera som värd för webbsidor (med PHP+MySQL-stöd) på din DiskStation.
    • WebDAV gör att klientprogram kan hantera och redigera DiskStation-filer via HTTP-protokollet.
    • WebDAV (HTTPS) gör att klientprogram kan hantera och redigera DiskStation-filer via HTTPS-protokollet.
    • File Station är en webbaserad applikation som gör att du kan hantera DiskStation-filer genom en anpassad port utan att behöva logga in på DSM.
    • File Station (HTTPS) gör att du kan ansluta till File Station via HTTPS-protokollet.
    • Surveillance Station är en webbaserad applikation som kan hantera IP-kameror för att säkra din hem- eller kontorsmiljö.
    • Audio Station är en webbaserad applikation som ger dig åtkomst till musikbiblioteket på din DiskStation.
    • Photo Station är ett fotoalbum online som är integrerat med en blogg som sparas på din DiskStation.
    • Download Station (BT/HTTP/FTP/NZB/RSS) är en webbaserad applikation som hjälper dig att ladda ner filer från Internet till din DiskStation.
    • eMule aktiverar eMule-stödet för Download Station.
  2. Tryck på Klar för att bekräfta.

Obs! För mer information om de ovannämnda tjänsterna hänvisas du till följande:

  • DSM hjälp (tillgänglig när du loggar in på skrivbordsutgåvan av DSM med din dators webbläsare och går till Huvudmeny > DSM hjälp)
  • Synology DiskStations bruksanvisning (finns på Synologys Nerladdningscenter)

Hantera Auto Block

Tryck på Auto Block på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet) för att förhindra oauktoriserad inloggning.

Hantera auto block så här:

  1. Markera Aktiverad för att aktivera Auto Block.
  2. Ange ett antal för Antal försök och Inom (minuter) för att blockera en IP-adress efter det angivna antalet inloggningsförsök inom angivet antal minuter.
  3. Specificera Blockerade dagar för att häva blockering av en IP-adress efter angivet antal dagar. Om du anger "0" här kommer IP-adressen att blockeras på obestämt tid till dess att tu tar bort den från blocklistan.
  4. Om du vill hantera blocklistan eller ta bort en blockerad IP-adress från listan pekar du på Blockeringslista.
  5. Tryck på Klar.

Hantera Paketcenter

Synology har integrerat applikationer utvecklade av tredje part eller Synology till paket som kan installeras och aktiveras enkelt på DiskStation.

Så här gör du för att hantera Paketcenter:

Logga in på DSM med din dators webbläsare för att aktivera denna funktion (på Huvudmeny > Paketcenter för att ändra paketen. Detaljerade instruktioner finns i dess hjälpfil.

För att köra eller stoppa installerade paket:

  1. Kör eller stoppa paketen med hjälp av av- och påknapparna.
  2. Tryck på Klar.

Visa systeminformation

Så här gör du för att se din DiskStations systeminfo:

  1. Gör något av följande:
    • Tryck på Allmänt för att se den grundläggande informationen om din DiskStation, inklusive serienummer, modellnamn, mängden fysiskt minne, DSM-versionen, samt tidinformation och temperaturstatus.

  2. Tryck på knappen Uppdatera högst upp till höger för att hålla systeminformation uppdaterad.

Välj en åtgärd för DiskStation
(Starta om, Stäng av, eller Pip)

Med DSM mobile kan du starta om eller stänga av din DiskStation med din iOS- eller Android-enhet. Om det finns flera DiskStation i närheten kan du dessutom snabbt identifiera din DiskStation genom att be den spela upp pip-ljud.

Så här gör du för att välja en åtgärd för DiskStation:

  1. Tryck på åtgärdsknappen högst upp till höger på startsidan (iPhone/iPod touch eller Android-telefon) eller den vänstra panelen (iPad eller Android-tablet).
  2. Gör något av följande:
    • Tryck på Starta om och sedan på Ja för att starta om din DiskStation.
    • Tryck på Stäng av och sedan på Ja för att stänga av din DiskStation.
    • Tryck på Hitta mig för att DiskStation ska börja pipa. Tryck på OK för att stoppa pipsignalen.
