DSM mobile permet à l'admin DSM (ou à un utilisateur appartenant au groupe administrators) de gérer les paramètres de DSM et de vérifier les informations DSM à l'aide d'un appareil iOS® (iPhone®, iPad®, ou iPod touch®) ou d'un appareil Android®. Cette aide de DSM mobile met à votre disposition les rubriques et ressources suivantes permettant de vous assister au début de votre utilisation.

Connecter et Déconnecter

Une fois connecté au DSM à l'aide de votre navigateur web sur votre appareil iOS ou Android, vous verrez l'écran de connexion dans lequel vous pouvez saisir les informations de votre compte DSM.

Pour se connecter à DSM mobile :

Saisissez vos identifiants d'utilisateur de DSM et appuyez sur Connecter. Si vous souhaitez enregistrer vos informations de connexion et vous connecter automatiquement la prochaine fois, cochez Se souvenir de moi avant de vous connecter.

Pour vous déconnecter de DSM mobile :

Une fois connecté à DSM mobile, vous verrez la page d'accueil (iPhone/iPod touch ou téléphone Android) ou le panneau gauche (tablette iPad ou tablette Android) contenant une liste d'options et les applications mobiles Synology installées sur votre appareil iOS ou Android. Touchez Déco. pour vous déconnecter et revenir à l'écran de connexion.

Gérer les utilisateurs de DSM

Appuyez sur Utilisateur de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour afficher la liste d'utilisateurs.

Pour créer un utilisateur :

  1. Appuyez sur + puis saisissez le nom de l'utilisateur, son adresse électronique et le mot de passe du nouvel utilisateur.
  2. Activez Envoyer notification pour envoyer un e-mail de notification à l'adresse de l'utilisateur lorsque son compte utilisateur de DSM est créé.
    Remarque : Avant d'envoyer une notification, assurez-vous que les paramètres de notification ont été configurés sur votre DiskStation. Connectez-vous à l'édition desktop de DSM avec le navigateur web de votre ordinateur, puis sélectionnez Menu principal > Panneau de configuration > Notification > E-mail pour modifier les paramètres.
  3. Appuyez sur Terminé pour confirmer ou appuyez sur Annuler pour annuler.

Pour détecter rapidement un utilisateur :

Entrez le nom de l'utilisateur dans le champ Recherche.

Pour modifier les informations d'un utilisateur :

  1. Appuyez sur le nom de l'utilisateur et procédez comme suit :
    • Modifiez le nom de l'utilisateur et le mot de passe dans le champ Nom ou Mot de passe.
    • Appuyez sur Désactiver pour désactiver le compte de l'utilisateur. Vous pouvez choisir d'activer le compte de cet utilisateur plus tard et sans perdre ses paramètres de son compte.
    • Activez Administrateur pour ajouter l'utilisateur au groupe admin. Tous les utilisateurs appartenant au groupe admin bénéficieront de privilège administratif permettant de modifier les paramètres de DSM à l'aide de DSM mobile ou du navigateur web de leur ordinateur.
  2. Sous Privilèges, effectuez l'une des opérations suivantes puis appuyez sur Retour :
    • Appuyez sur Dossiers partagés et appuyez sur le nom du dossier partagé pour modifier le privilège du compte de l'utilisateur : Lecture/écriture : L'utilisateur peut effectuer des modifications sur les fichiers et sous-dossiers contenus dans le dossier partagé ; Lecture seule : L'utilisateur peut accéder aux fichiers et sous-dossiers contenus dans le dossier partagé, mais ne peut pas effectuer de modifications sur ces derniers. Pas d'accès : L'utilisateur ne peut pas accéder aux fichiers et sous-dossiers du dossier partagé.
      Remarque : Si des conflits de privilèges se produisent, la priorité du privilège sera : Pas d'accès > Lecture/écriture > lecture seule.
    • Appuyez sur Application pour modifier le privilège d'accès de l'utilisateur aux applications DSM.
  3. Appuyez sur Terminé pour confirmer ou appuyez sur Annuler pour annuler.

Gérer des dossiers partagés

Appuyez sur Dossier partagé de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour afficher la liste des dossiers partagés sur votre DiskStation.

Pour créer un dossier partagé :

  1. Appuyez sur + puis saisissez le nom du nouveau dossier.
  2. Appuyez sur Terminé pour confirmer ou appuyez sur Annuler pour annuler.

Pour modifier un dossier partagé :

Appuyez sur le nom du dossier partagé et procédez comme suit :

  • Modifiez le nom du dossier partagé dans le champ Nom.
  • Sous Privilèges, touchez Utilisateur et touchez un utilisateur pour modifier son privilège d'accès au dossier partagé : Lecture/écriture : L'utilisateur peut effectuer des modifications sur les fichiers et sous-dossiers contenus dans le dossier partagé ; Lecture seule : L'utilisateur peut accéder aux fichiers et sous-dossiers contenus dans le dossier partagé, mais ne peut pas effectuer de modifications sur ces derniers. Pas d'accès : L'utilisateur ne peut pas accéder aux fichiers et sous-dossiers du dossier partagé.
    Remarque : Si des conflits de privilèges se produisent, la priorité du privilège sera : Pas d'accès > Lecture/écriture > lecture seule.

Activer les services

Touchez Service de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour activer ou désactiver les services de DiskStation.

Pour gérer les services LAN de DiskStation :

  1. Activez ou désactiver l'un des services suivants :
    • Service de fichiers Windows permet à votre DiskStation de devenir un serveur de fichiers Windows (CIFS) sur le réseau local.
    • Service de fichiers Mac permet à votre DiskStation de devenir un serveur de fichiers Mac (AFP) sur le réseau local.
    • Service NFS permet aux clients Linux sur le réseau local d'accéder aux données de DiskStation.
    • Imprimante Bonjour permet aux clients Mac d'imprimer vers le(s) imprimante(s) connectée(s) à votre DiskStation.
    • Serveur multimédia permet aux périphériques domestiques DLNA/UPnP sur le réseau local de parcourir et de lire des contenus multimédia sur votre DiskStation.
    • iTunes permet aux clients iTunes sur le réseau local de parcourir et de lire de la musique ou des vidéos sur votre DiskStation.
  2. Touchez Terminé pour confirmer.

Pour gérer les services Internet du DiskStation :

  1. Activez ou désactiver l'un des services suivants :
    • Blocage auto vous permet d'empêcher une connexion non autorisée.
    • FTP permet à votre DiskStation de devenir un serveur FTP.
    • Services web vous permet d'héberger des sites web (à l'aide de la prise en charge PHP+MySQL) sur votre DiskStation.
    • WebDAV permet aux programmes client de gérer et d'éditer les fichiers du DiskStation via le protocole HTTP.
    • WebDAV (HTTPS) permet aux programmes client de gérer et d'éditer les fichiers du DiskStation via le protocole HTTPS.
    • File Station est une application web vous permettant de gérer les fichiers du DiskStation via un port personnalisé sans avoir à vous connecter au DSM.
    • File Station (HTTPS) vous permet e vous connecter à File Station via le protocole HTTPS.
    • Surveillance Station est une application Web qui peut gérer des caméras IP pour protéger l'environnement de votre domicile ou de votre bureau.
    • Audio Station est une application web vous permettant d'accéder à la bibliothèque musicale de votre DiskStation.
    • Photo Station est un album photos intégré à un blog en ligne enregistré sur votre DiskStation.
    • Download Station (BT/HTTP/FTP/NZB/RSS) est une application web vous permettant de télécharger des fichiers depuis Internet vers votre DiskStation.
    • eMule permet de prendre en charge eMule sur Download Station.
  2. Touchez Terminé pour confirmer.

Remarque : Pour plus d'informations sur les services mentionnés ci-dessus, consultez l'une des méthodes suivantes :

  • Aide DSM (disponible en vous connectant à l'Édition desktop du DSM à l'aide du navigateur web de votre ordinateur et allez sur Menu principal > Aide DSM)
  • Guide de l'utilisateur de Synology DiskStation (disponible sur le Centre de téléchargement de Synology)

Gérer le blocage auto

Appuyez sur Blocage auto de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour empêcher une connexion non autorisée.

Pour gérer le blocage automatique :

  1. Utilisez Activé pour activer le blocage automatique.
  2. Entrez un nombre pour les Nombre de tentatives et Sous (minutes) pour bloquer une adresse IP après un nombre spécifié d'échecs de tentatives de connexion sous le nombre de minutes spécifié.
  3. Spécifiez les Jours bloqués pour débloquer l'adresse IP après le nombre de jours spécifiés. Si vous entrez « 0 » ici, l'Adresse IP bloquée restera bloquée indéfiniment jusqu'à ce que vous la retiriez de la liste des blocages.
  4. Si vous voulez gérer la liste des blocages ou supprimer une adresse IP bloquée de la liste, appuyez sur Liste des blocages.
  5. Appuyez sur Terminé.

Gérer le Centre des paquets

Synology a intégré des applications tierces et conçues par Synology dans des paquets pouvant être installés et activés facilement sur DiskStation.

Pour gérer le Centre des paquets :

Connectez-vous à l'édition desktop de DSM avec le navigateur web de votre ordinateur, puis sélectionnez Menu principal > Centre des paquets pour modifier les paquets. Pour des instructions détaillées, consultez son Aide.

Pour exécuter ou arrêter les paquets installés :

  1. Activez ou désactivez les commutateurs pour exécuter ou arrêter les paquets.
  2. Appuyez sur Terminé.

Afficher les infos système

Appuyez sur Infos système de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour obtenir une vue d'ensemble du statut de votre DiskStation.

Pour afficher les infos système de votre DiskStation :

  1. Employez l’une des méthodes suivantes :
    • Appuyez sur Général pour afficher les informations de base de votre DiskStation, notamment le numéro de série, le nom du modèle, la quantité de mémoire physique, la version de DSM, ainsi que les informations sur la durée de fonctionnement et les températures.
    • Appuyez sur Réseau pour afficher la configuration réseau de votre DiskStation et le statut de chaque interface réseau.
    • Touchez Stockage pour afficher l'utilisation du volume du DiskStation.
  2. Touchez le bouton Actualiser situé dans le coin supérieur droit pour que les informations système soient mises à jour.

Contrôler les ressources du système

Appuyez sur Moniteur de ressources de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (tablette iPad ou Android) pour surveiller le processeur et l'utilisation de mémoire du DiskStation.

Afficher les journaux ou les connexions actuelles

Appuyez sur Journal sur la page d'accueil (iPhone/iPod touch ou téléphone Android) ou sur le panneau gauche (tablette iPad ou Android) pour surveiller les connexions actuelles ou afficher les journaux des services DiskStation.

Pour surveiller les connexions actuelles :

Appuyez sur Connexions actuelles dans la section Surveiller pour afficher la liste des utilisateurs qui accèdent actuellement aux services DiskStation.

Pour consulter les enregistrements du journal d'un service spécifique :

Appuyez sur un élément dans la section Journaux d'événements ou Journaux de transfert de fichiers pour voir la liste des journaux en rapport.

Lancer des applications mobiles

DSM mobile fournit un endroit facile d'accès vous permettant de lancer des applications mobiles Synology installées sur votre appareil iOS ou Android.

Pour lancer les applications mobiles Synology avec DSM mobile :

Appuyez sur une des applications suivantes situées dans la section inférieure de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (Tablette iPad ou Android) :

Configurer DiskStation avec le menu Action
(Redémarrer, éteindre ou émettre des bips)

DSM mobile vous permet de redémarrer ou d'éteindre votre DiskStation à l'aide d'un appareil iOS ou Android. De plus, lorsque plusieurs DiskStations sont disponibles, vous pouvez rapidement identifier votre DiskStation en lui demandant qu'il émette des bips.

Pour configurer votre DiskStation avec le menu Action :

  1. Touchez le bouton Action situé dans le coin supérieur droit de la page d'accueil (iPhone/iPod touch ou téléphone Android) ou du panneau gauche (Tablette iPad ou Android).
  2. Employez l’une des méthodes suivantes :
    • Touchez Redémarrer puis touchez Oui pour redémarrer votre DiskStation.
    • Touchez Éteindre puis touchez Oui pour éteindre votre DiskStation.
    • Appuyez sur Me trouver pour demander à votre DiskStation d'émettre des bips. Appuyez sur OK pour arrêter le signal sonore.

Basculer vers l'Édition Desktop de DSM

Touchez le bouton Action situé dans le coin supérieur droit du panneau gauche puis touchez Edition Desktop pour basculer vers l'édition desktop de DSM.
