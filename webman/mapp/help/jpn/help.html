DSM mobile では、DSM admin (または administrators グループに属しているユーザー) は iOS® (iPhone®、iPad®、または iPod touch®) や Android® デバイスを使用して DSM 設定を管理し、DSM 情報を確認することができます。この DSM mobile のヘルプでは、使用開始するのに役立つ以下のトピックやリソースが用意されています。

ログインとログアウト

iOS や Android デバイスで Web ブラウザを使用して DSM に接続すると、ログイン画面が表示されます。DSM アカウント情報を入力します。

DSM mobile にログインする：

DSM ユーザー資格情報を入力して [ログイン] をタップします。ログイン情報を保存して次回から自動的にログインするようにするには、ログインする前に [この情報を保存する] を有効にします。

DSM mobile からログアウトする：

DSM mobile にログインすると、ホームページ (iPhone/iPod touch または Android 携帯電話) が表示されるか、オプション リストと iOS や Android デバイスにインストールされている Synology モバイル アプリを含んでいる左パネル (iPad または Android タブレット) が表示されます。ログアウトしてログイン画面に戻るには、[ログアウト] をタップします。

DSM ユーザーの管理

[ユーザー] をホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップしてユーザーリストを表示します。

ユーザーを作成する：

  1. [+] をタップして新しいユーザーのユーザー名、パスワード、電子メールアドレスを入力します。
  2. [通知の送信] を有効にすると、DSM ユーザーアカウントを作成するとユーザーの電子メールアドレスに通知のメールが送信されます。
    注：通知を送信する前に、通知の設定が DiskStation で完了しているか確認してください。コンピュータの Web ブラウザで DSM のデスクトップエディションにログインし、[メイン メニュー] > [コントロール パネル] > [通知] > 電子メール] の順に選択して設定を変更します。
  3. [完了]をクリックして確定するか、[キャンセル] をクリックして中断します。

ユーザーをクイック検索する

ユーザー名を検索フィールドに入力します。

ユーザーの情報を変更する

  1. ユーザーの名前をタップして、以下のいずれかを行います。
    • [名前] または [パスワード] フィールドで、ユーザーの名前やパスワードを変更します。
    • [無効] をオンにしてユーザーを無効にします。設定はそのままで後でユーザーを有効にすることができます。
    • [管理者] をオンにして admin グループにユーザーを追加します。admin グループに属しているユーザーに管理者権限を与えて、DSM 設定を DSM mobile やコンピュータの Web ブラウザから変更できるようにします。
  2. [特権] から以下のいずれかを実行して、[戻る] をタップします。
    • [共有フォルダ] をタップし、共有フォルダの名前をタップしてユーザーのアクセス権限を変更します。読み込み/書き込み：ユーザーは、共有フォルダのファイルやサブフォルダに変更を行うことができます。読込み専用：ユーザーは、共有フォルダのファイルやサブフォルダにアクセスできますが、変更を加えることはできません。アクセスなし：ユーザーは、共有フォルダのファイルとサブフォルダにアクセスできません。
      注：権限の競合が発生した場合は、権限の優先順位は次のようになります。アクセスなし > 読み込み/書き込み > 読込み専用.
    • [アプリケーション] をタップして、DSM アプリケーションのユーザーのアクセス権限を変更します。
  3. [完了]をクリックして確定するか、[キャンセル] をクリックして中断します。

共有フォルダの管理

[共有フォルダ] をホームページでタップするか (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして DiskStation の共有フォルダのリストを表示します。

共有フォルダを作成する

  1. [+] をタップして新しいフォルダの名前を入力します。
  2. [完了]をクリックして確定するか、[キャンセル] をクリックして中断します。

共有フォルダを変更する

共有フォルダの名前をタップして、以下のいずれかを行います。

  • [名前]フィールドの共有フォルダの名前を変更します。
  • [特権] から [ユーザー] をタップし、ユーザーをタップして以下の共有フォルダのアクセス権限を変更します。読み込み/書き込み：ユーザーは、共有フォルダのファイルやサブフォルダに変更を行うことができます。読込み専用：ユーザーは、共有フォルダのファイルやサブフォルダにアクセスできますが、変更を加えることはできません。アクセスなし：ユーザーは、共有フォルダのファイルとサブフォルダにアクセスできません。
    注：権限の競合が発生した場合は、権限の優先順位は次のようになります。アクセスなし > 読み込み/書き込み > 読込み専用.

サービスを有効にする

[サービス] をホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして DiskStation サービスを有効または無効にします。

DiskStation の LAN サービスを管理する

  1. 以下のいずれかを有効または無効にします。
    • Windows のファイルサービス では、DiskStation を LAN 内の Windows (CIFS) ファイルサーバーにすることができます。
    • Mac のファイルサービス では、DiskStation を LAN 内の Mac (AFP) ファイルサーバーにすることができます。
    • NFS サービス は LAN 内の Linux クライアントが DiskStation データへアクセスできるようにします。
    • Bonjour プリンタは Mac クライアントが DiskStation へ接続した USB プリンタに印刷できるようにします。
    • メディアサーバーは LAN 内の DLNA/UPnP ホームデバイスが DiskStation のコンテンツを参照および再生できるようにします。
    • iTunes は LAN 内の iTunes クライアントが DiskStation の音楽やビデオを参照および再生できるようにします。
  2. [完了] をタップして確定します。

DiskStation のインターネット サービスを管理する

  1. 以下のいずれかを有効または無効にします。
    • 自動ブロック は不正なログインを防止します。
    • FTP は DiskStation を FTP サーバーにすることができます。
    • Web サービスは DiskStation で Web サイトをホストすることができます (PHP+MySQL のサポートで)。
    • WebDAV はクライアントのプログラムが HTTP プロトコルを介して DiskStation を管理および編集できるようにします。
    • WebDAV (HTTPS) はクライアント プログラムが HTTPS プロトコルを介して DiskStation ファイルを管理および編集できるようにします。
    • File Station は Web ベースのアプリケーションで、DSM にログインしないでカスタマイズポートからの DiskStation ファイルの管理を可能にします。
    • File Station (HTTPS) は File Station に HTTPS プロトコルを介して接続します。
    • Surveillance Station は、IP カメラを管理して自宅や会社の環境を守るウェブベースのアプリケーションです。
    • Audio Station は Web ベースのアプリケーションで、DiskStation の音楽ライブラリを利用することを可能にします。.
    • Photo Station は、DiskStation に保存されているブログと一体化したオンラインフォトアルバムです。
    • Download Station (BT/HTTP/FTP/NZB/RSS) は Web ベースのアプリケーションで、ファイルをインターネットから DiskStation にダウンロードすることができます。
    • eMule は、Download Station の eMule サポートを有効にします。
  2. [完了] をタップして確定します。

注：サービスの詳細については、以下のいずれかを参照してください。

  • DSM ヘルプ (コンピュータの Web ブラウザで DSM のデスクトップエディションにログインし、[メイン メニュー] > [DSM ヘルプ] の順に選択)
  • eCore OS ユーザーガイド (Synology のダウンロード センターから入手できます)

自動ブロックの管理

自動ブロック をホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして、不正なログインを防止します。

自動ブロックを管理する：

  1. 有効をオンにして、自動ブロックを有効にします。
  2. 試行回数および有効時間(分)の数を入力して、指定した時間(分)以内に指定した回数ログインに失敗した IP アドレスをブロックします。
  3. 指定した日数が経過した後、IP アドレスをブロック解除するように、ブロックした日数を指定します。ここで「0」を入力すると、ブロックされた IP アドレスはブロック リストから除去するまで継続してブロックされたままとなります。
  4. ブロック リストを管理する、またはリストからブロックした IP アドレスを削除するには、ブロック リストをタップします。
  5. [完了] をタップします。

パッケージ センター の管理

Synology は、サードパーティや Synology デザインのアプリケーションを簡単にインストールできるようにパッケージに統合し、DiskStation で簡単に有効にできるようにしました。

パッケージ センターを管理するには：

コンピュータの Web ブラウザで DSM のデスクトップエディションにログインし、メイン メニュー > パッケージ センター の順に選択してパッケージを変更します。詳しい手順については、ヘルプを参照してください。

インストールしたパッケージを起動または停止する：

  1. パッケージを起動または停止するには、スイッチをオン/オフにします。
  2. [完了] をタップします。

システム情報の表示

[システム情報] をホームページでタップするか (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして DiskStation の状態を表示します。

DiskStation のシステム情報を表示する

  1. 以下のいずれかを行います。
    • [全般] をタップして、シリアル番号、モデル名、実メモリの容量、DSM バージョン、時間情報、温度の状態など、DiskStation の基本情報を表示する。
    • [ネットワーク] をタップして DiskStation のネットワーク設定と各インターフェイスの状態を表示する。
    • [ストレージ] をタップして DiskStation のボリューム使用状況を表示する。
  2. 右上隅の [最新の情報に更新] ボタンをタップしてシステム情報を最新にする。

システムリソースの監視

[リソースモニタ] をホームページでタップするか (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして DiskStation の CPU とメモリの使用状況を監視します。

ログまたは現在の接続を表示する

ログ をホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) からタップして、現在の接続状況をモニターしたり、DiskStation サービスのログを見たりすることができます。

現在の接続状況をモニターする：

モニターセクションで現在の接続をタップして、DiskStation サービスに現在アクセスしているユーザーのリストを表示します。

特定のサービスのログ記録を見る：

イベント ログまたはファイル転送ログセクションのアイテムをタップすると、関連するログのリストが表示されます。

携帯アプリの起動

DSM mobile には、iOS や Android デバイスにインストールした Synology 携帯アプリを起動する便利な機能名装備されています。

Synology 携帯アプリを DSM mobile で起動する

ホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) の下部にある以下のいずれかのアプリをタップします。

  • DS photo+ (Photo Station サービスが有効時にのみ使用可能)
  • DS audio (Audio Station サービスが有効時にのみ使用可能)
  • DS cam (Surveillance Station サービスが有効時にのみ使用可能)
  • DS file (WebDAV サービスが有効時に使用可能)

注：これらのアプリの詳細については、アプリに付属のヘルプ ファイルを参照してください。

操作中に DiskStation を設定
(再起動、シャットダウン、または警告音)

DSM mobile を使用すると、iOS や Android モバイルデバイスを使って DiskStation の再起動やシャットダウンを行うことができます。さらに、複数の DiskStations がある場合は、警告音を鳴らすように要求することで DiskStation を簡単に識別することができます。

操作中に DiskStation を設定する

  1. ホームページ (iPhone/iPod touch または Android 携帯電話)、または左パネル (iPad または Android タブレット) の右上隅にある操作ボタンをタップします。
  2. 以下のいずれかを行います。
    • ［再起動］、［はい］の順にタップして DiskStation を再起動する。
    • ［シャットダウン］、［はい］の順にタップして DiskStation をシャットダウンする。
    • [サーバー検出] をタップして、DiskStation に警告音再生を要求する。[OK] をタップすると、警告音が停止します。

DSM のデスクトップ版に切り替える

左パネルの右上隅にある操作ボタンをタップして、[ディスクトップ版] をタップして DSM のデスクトップ版に切り替えます。
