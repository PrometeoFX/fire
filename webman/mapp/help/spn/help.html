DSM mobile permite al admin de DSM (o a un usuario que pertenezca al grupo de administrators) administrar la configuración de DSM y comprobar la información de DSM con un dispositivo iOS® (iPhone®, iPad® o iPod touch®) o Android®. Esta Ayuda eCore mobile le proporciona los siguientes temas y recursos para ayudarle a comenzar.

Conectarse y cerrar sesión

Después de conectarse a DSM con el navegador web en su dispositivo iOS o Android, verá la pantalla de conexión, donde puede especificar su información de cuenta de DSM.

Para iniciar sesión en DSM mobile:

Escriba sus credenciales de usuario de DSM y toque en Iniciar sesión. Si desea guardar su información de conexión y conectarse automáticamente la próxima vez, encienda el conmutador Recordarme antes de conectarse.

Para cerrar sesión en DSM mobile:

Después de iniciar sesión en DSM mobile, verá la página de inicio (iPhone/iPod touch o teléfono Android) o el panel izquierdo (iPad o tableta Android) con una lista de opciones y aplicaciones móviles de Synology instaladas en su dispositivo iOS o Android. Toque Cerrar sesión para cerrar sesión y volver a la pantalla de conexión.

Administrar usuarios de DSM

Toque en Usuario en la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android) para ver la lista de usuarios.

Para crear un usuario:

  1. Toque + y escriba el nombre de usuario y la contraseña, y la dirección de correo electrónico del nuevo usuario.
  2. Habilite Enviar notificación para enviar un correo de notificación a la dirección de correo electrónico del usuario cuando se creen sus cuentas de usuario de DSM.
    Nota: Antes de enviar la notificación, asegúrese de que la configuración de notificación se ha completado en su DiskStation. Conéctese a la edición de escritorio de DSM con el explorador web de su equipo y vaya a Menú principal > Panel de control > Notificación > Correo electrónico para modificar la configuración.
  3. Toque en Listo para confirmar o en Cancelar para cancelar.

Para buscar rápidamente un usuario:

Escriba el nombre del usuario en el campo de búsqueda.

Para modificar la información de un usuario:

  1. Toque el nombre del usuario y elija una de las siguientes opciones:
    • Modifique el nombre o la contraseña del usuario en el campo Nombre o Contraseña.
    • Active Deshabilitar para deshabilitar al usuario. Puede habilitar el usuario más tarde sin perder su configuración.
    • Active Administrador para añadir al usuario al grupo de admin. A cualquier usuario que pertenezca al grupo admin se le otorgarán privilegios administrativos para modificar la configuración de DSM con DSM mobile o con el navegador web de su equipo.
  2. En Privilegios, realice una de las siguientes opciones y toque Anterior:
    • Toque Carpeta compartida y toque el nombre de una carpeta compartida para modificar el privilegio de acceso del usuario: Lectura/Escritura: El usuario puede acceder a los archivos y subcarpetas de la carpeta compartida y hacerles cambios; Sólo lectura: El usuario puede acceder a los archivos y subcarpetas de la carpeta compartida, pero no puede hacerles cambios; Sin acceso: El usuario no puede acceder a los archivos y subcarpetas de la carpeta compartida.
      Nota: Si hubiera conflictos de privilegios, la prioridad será la siguiente: Sin acceso > Lectura/Escritura > Sólo lectura.
    • Toque Aplicación para modificar el privilegio de acceso del usuario a las aplicaciones DSM.
  3. Toque en Listo para confirmar o en Cancelar para cancelar.

Administrar carpetas compartidas

Toque en Carpeta compartida en la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android) para ver la lista de carpetas compartidas en DiskStation.

Para crear una carpeta compartida:

  1. Toque + y escriba el nombre de la nueva carpeta.
  2. Toque en Listo para confirmar o en Cancelar para cancelar.

Para modificar una carpeta compartida:

Toque el nombre de la carpeta compartida y elija una de las siguientes opciones:

  • Modifique el nombre de la carpeta compartida en el campo Nombre.
  • En Privilegios, toque Usuario y toque a un usuario para modificar su privilegio de acceso a la carpeta compartida: Lectura/Escritura: El usuario puede acceder a los archivos y subcarpetas de la carpeta compartida y hacerles cambios; Sólo lectura: El usuario puede acceder a los archivos y subcarpetas de la carpeta compartida, pero no puede hacerles cambios; Sin acceso: El usuario no puede acceder a los archivos y subcarpetas de la carpeta compartida.
    Nota: Si hubiera conflictos de privilegios, la prioridad será la siguiente: Sin acceso > Lectura/Escritura > Sólo lectura.

Habilitar servicios

Toque en Servicio en la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android) para habilitar o deshabilitar los servicios de DiskStation.

Para administrar los servicios LAN de DiskStation:

  1. Habilite o deshabilite cualquiera de las siguientes opciones:
    • Servicio de archivos Windows permite que su DiskStation se convierta en un servidor de archivos Windows (CIFS) dentro de una red LAN.
    • Servicio de archivos Mac permite que su DiskStation se convierta en un servidor de archivos Mac (AFP) dentro de una red LAN.
    • Servicio NFS permite que el cliente Linux dentro de la red LAN acceda a datos de DiskStation.
    • Impresora Bonjour permite que los clientes de Mac impriman en impresoras USB conectadas a DiskStation.
    • Servidor multimedia permite que los dispositivos domésticos DLNA/UPnP examinen y reproduzcan el contenido multimedia de su DiskStation.
    • iTunes permite que clientes de iTunes dentro de la red LAN examinen y reproduzcan música o vídeo en su DiskStation.
  2. Toque Listo para confirmar.

Para administrar los servicios de Internet de DiskStation:

  1. Habilite o deshabilite cualquiera de las siguientes opciones:
    • Bloqueo automático permite impedir conexiones no autorizadas.
    • FTP permite que su DiskStation se convierta en un servidor FTP.
    • Servicios Web le permite alojar sitios web (con compatibilidad PHP+MySQL) en su DiskStation.
    • WebDAV permite que los programas cliente administren y editen archivos de DiskStation mediante el protocolo HTTP.
    • WebDAV (HTTPS) permite que los programas cliente administren y editen archivos de DiskStation mediante el protocolo HTTPS.
    • File Station es una aplicación basada en web que le permite administrar archivos de DiskStation a través de un puerto personalizado sin iniciar sesión en DSM.
    • File Station (HTTPS) le permite conectarse a File Station mediante el protocolo HTTPS.
    • Surveillance Station es una aplicación basada en web que puede administrar cámaras IP para proteger su entorno doméstico o de oficina.
    • Audio Station es una aplicación basada en web que le permite acceder a la biblioteca de música en su DiskStation.
    • Photo Station es un álbum de fotos en línea integrado con un blog guardado en DiskStation.
    • Download Station (BT/HTTP/FTP/NZB/RSS) es una aplicación basada en web que le ayuda a descargar archivos desde Internet a DiskStation.
    • eMule habilita la compatibilidad con eMule de Download Station.
  2. Toque Listo para confirmar.

Nota: Para obtener más información acerca de los anteriores servicios, consulte cualquiera de lo siguiente:

  • Ayuda eCore (disponible si inicia sesión en la edición de escritorio de DSM con el navegador web del equipo y accede a Menú principal > Ayuda eCore)
  • Manual del usuario de eCore OS (disponible en el Centro de descargas de Synology)

Administrar Bloqueo automático

Toque en Bloqueo automático en la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android) para impedir conexiones no autorizadas.

Para administrar bloqueo automático:

  1. Encienda Habilitado para habilitar el bloqueo automático.
  2. Escriba un número para Número de intentos y En (minutos) para bloquear una dirección IP al cabo de un número definido de intentos de conexión fallidos en un número de minutos especificado.
  3. Especifique los Días bloqueados para desbloquear una dirección IP tras el número especificado de días. Si introduce aquí "0", la dirección IP bloqueada permanecerá indefinidamente hasta que la elimine de la lista de bloqueados.
  4. Si desea administrar la lista de bloqueados o eliminar una dirección IP bloqueada de la lista, pulse en Lista de bloqueo.
  5. Toque Listo.

Administrar Centro de paquetes

Synology ha integrado aplicaciones diseñadas por Synology o terceros en paquetes que pueden instalarse y habilitarse fácilmente en DiskStation.

Para administrar el Centro de paquetes:

Inicie sesión en la edición de escritorio de DSM con el explorador Web de su ordenador y vaya a Menú principal > Centro de paquetes para modificar los paquetes. Para instrucciones detalladas, consulte su Ayuda.

Para ejecutar o detener los paquetes instalados:

  1. Encienda o apague los conmutadores para ejecutar o detener los paquetes.
  2. Toque Listo.

Ver información del sistema

Toque en Inform sistema en la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android) para obtener una información general del estado de su DiskStation.

Para ver la información del sistema de DiskStation:

  1. Realice cualquiera de las siguientes operaciones:
    • Toque General para ver información básica acerca de su DiskStation, incluido el número de serie, el nombre del modelo, la cantidad de memoria física, la versión de DSM e información sobre hora y estado térmico.
    • Toque Red para ver la configuración de red de DiskStation y el estado de cada interfaz de red.
    • Toque Almacenamiento para ver el uso de volumen de DiskStation.
  2. Toque el botón Actualizar en el vértice superior derecho para mantener actualizada la información del sistema.

Supervisar recursos del sistema

Toque en Monitor de recursos en la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android) para supervisar el uso de CPU y de memoria de su DiskStation.

Ver registros o conexiones actuales

Toque en Registro en la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android) para supervisar las conexiones actuales o ver los registros de los servicios de DiskStation.

Para supervisar las conexiones actuales:

Toque en Conexiones actuales en el apartado Supervisar para ver la lista de usuarios que actualmente acceden a los servicios de DiskStation.

Para ver los registros de un servicio específico:

Toque en cualquier elemento en el apartado Registros de eventos o Registros de transferencia de archivos para ver la lista de los correspondientes registros.

Iniciar aplicaciones móviles

DSM mobile proporciona un práctico lugar para que inicie las aplicaciones móviles de Synology instaladas en su dispositivo iOS o Android.

Para iniciar las aplicaciones móviles de Synology con DSM mobile:

Toque cualquiera de las siguientes aplicaciones en la sección inferior de la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android):

  • DS photo+ (disponible si está habilitado el servicio Photo Station)
  • DS audio (disponible si está habilitado el servicio Audio Station)
  • DS cam (disponible si está habilitado el servicio Surveillance Station)
  • DS file (disponible si está habilitado el servicio WebDAV)

Nota: Para obtener más información acerca de estas aplicaciones, consulte los archivos de Ayuda incluidos con las aplicaciones.

Poner DiskStation en marcha
(Reiniciar, Apagar o Pitar)

DSM mobile le permite reiniciar o apagar su DiskStation con su iOS o dispositivo Android. Además, si tiene varios DiskStations cerca, podrá identificar rápidamente el suyo solicitando que reproduzca sonidos.

Para poner DiskStation en marcha:

  1. Toque el botón de acción situado en el vértice superior de la página de inicio (iPhone/iPod touch o teléfono Android) o en el panel izquierdo (iPad o tableta Android).
  2. Realice cualquiera de las siguientes operaciones:
    • Toque Reiniciar y, a continuación, toque Sí para reiniciar su DiskStation.
    • Toque Apagar y, a continuación, toque Sí para apagar su DiskStation.
    • Toque Buscarme para solicitar a DiskStation que emita pitidos. Toque OK para detener el pitido.

Cambiar a la Edición de escritorio de DSM

Toque el botón de acción situado en el vértice superior derecho del panel izquierdo y, a continuación, toque Edición de escritorio para cambiar a la edición de escritorio de DSM.
