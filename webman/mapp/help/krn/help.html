DSM mobile을 사용하면 DSM admin 또는 administrators 그룹에 속한 사용자가 iOS® (iPhone®, iPad® 또는 iPod touch®) 또는 Android® 장치를 통해 DSM 설정을 관리하고 DSM 정보를 확인할 수 있습니다. 이 DSM mobile 도움말은 시작하는 데 도움이 되는 다음의 항목과 리소스를 제공합니다.

로그인 및 로그아웃

iOS 또는 Android 장치에서 웹 브라우저를 통해 DSM에 연결하면 DSM 계정 정보를 입력할 수 있는 로그인 화면이 표시됩니다.

DSM mobile에 로그인하기:

DSM 사용자 자격 증명을 입력하고 로그인을 누릅니다. 로그인 정보를 저장해서 다음 번에 자동으로 로그인하려면 로그인 전에 저장 스위치를 설정합니다.

DSM mobile 로그아웃하기:

DSM mobile에 로그인하면 iOS 또는 Android 장치에 설치된 옵션과 Synology 모바일 앱 목록이 포함되어 있는 홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad/Android 태블릿)을 볼 수 있습니다. 로그아웃해서 로그인 하면으로 돌아가려면 로그아웃을 누릅니다.

DSM 사용자 관리

홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad/Android 태블릿)에서 사용자를 눌러 사용자 목록을 확인합니다.

사용자 생성하기:

  1. +를 누른 다음, 새 사용자의 사용자 이름, 패스워드 및 이메일 주소를 입력합니다.
  2. DSM 사용자 계정이 생성될 때 사용자의 이메일 주소로 알림 메일을 전송하려면 알림 전송을 활성화합니다.
    참고: 알림을 보내기 전에 DiskStation에 알림 설정이 완료되었는지 확인하십시오. 컴퓨터의 웹 브라우저를 사용하여 DSM의 데스크탑 버전에 로그인한 다음, 메인 메뉴 > 제어판 > 알림 > 이메일로 이동해서 설정을 수정합니다.
  3. 완료를 눌러 확인하거나 취소를 눌러 취소합니다.

사용자를 빠르게 찾기:

검색 필드에 사용자의 이름을 입력합니다.

사용자 정보 수정하기:

  1. 사용자의 이름을 누르고 다음 중 하나를 수행합니다.
    • 이름 또는 패스워드 필드에서 사용자의 이름이나 패스워드를 수정합니다.
    • 사용자를 비활성화하려면 비활성화을 선택합니다. 설정의 손실 없이 나중에 사용자를 활성화하도록 선택할 수 있습니다.
    • Administrator를 설정해서 사용자를 admin 그룹에 추가합니다. admin 그룹에 속하는 모든 사용자에게는 DSM mobile이나 컴퓨터의 웹 브라우저를 사용하여 DSM 설정을 수정할 수 있는 관리자 권한이 부여됩니다.
  2. 사용 권한에서 다음 중 하나를 수행한 다음, 뒤로를 누릅니다.
    • 공유 폴더를 누르고 공유 폴더의 이름을 눌러 사용자의 접근 권한을 수정합니다. 읽기/쓰기: 사용자가 공유 폴더에 있는 파일과 하위 폴더에 변경 내용을 적용할 수 있습니다. 읽기 전용: 사용자가 공유 폴더에 있는 파일과 하위 폴더에 접근할 수 있지만 변경 내용을 적용할 수는 없습니다. 접근 금지: 사용자가 공유 폴더에 있는 파일과 하위 폴더에 접근할 수 없습니다.
      참고: 사용 권한이 충돌하는 경우 권한의 우선 순위는 접근 금지 > 읽기/쓰기 > 읽기 전용 순입니다.
    • 응용 프로그램을 눌러 DSM 응용 프로그램에 대한 사용자의 접근 권한을 수정합니다.
  3. 완료를 눌러 확인하거나 취소를 눌러 취소합니다.

공유 폴더 관리

홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad/Android 태블릿)에서 공유 폴더를 눌러 DiskStation의 공유 폴더 목록을 확인합니다.

공유 폴더 생성하기:

  1. +를 누른 다음, 새 폴더의 이름을 입력합니다.
  2. 완료를 눌러 확인하거나 취소를 눌러 취소합니다.

공유 폴더 수정하기:

공유 폴더의 이름을 누르고 다음 중 하나를 수행합니다.

  • 이름 필드에서 공유 폴더의 이름을 수정합니다.
  • 사용 권한에서 사용자를 누르고 사용자를 눌러 공유 폴더에 대한 접근 권한을 수정합니다. 읽기/쓰기: 사용자가 공유 폴더에 있는 파일과 하위 폴더에 변경 내용을 적용할 수 있습니다. 읽기 전용: 사용자가 공유 폴더에 있는 파일과 하위 폴더에 접근할 수 있지만 변경 내용을 적용할 수는 없습니다. 접근 금지: 사용자가 공유 폴더에 있는 파일과 하위 폴더에 접근할 수 없습니다.
    참고: 사용 권한이 충돌하는 경우 권한의 우선 순위는 접근 금지 > 읽기/쓰기 > 읽기 전용 순입니다.

서비스 활성화

홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad/Android 태블릿)에서 서비스를 눌러 DiskStation 서비스를 활성화하거나 비활성화합니다.

DiskStation의 LAN 서비스 수정하기:

  1. 다음을 활성화하거나 비활성화합니다.
    • Windows 파일 서비스를 사용하면 LAN 내에서 DiskStation을 Windows (CIFS) 파일 서버로 설정할 수 있습니다.
    • Mac 파일 서비스를 사용하면 LAN 내에서 DiskStation을 Mac (AFP) 파일 서버로 설정할 수 있습니다.
    • NFS 서비스를 사용하면 LAN 내의 Linux 클라이언트가 DiskStation 데이터에 접근할 수 있습니다.
    • Bonjour 프린터를 사용하면 Mac 클라이언트가 DiskStation에 연결된 USB 프린터로 인쇄할 수 있습니다.
    • 미디어 서버를 사용하면 LAN 내의 DLNA/UPnP 홈 장치가 DiskStation에 있는 멀티미디어 컨텐트를 찾아 재생할 수 있습니다.
    • iTunes를 사용하면 LAN 내의 iTunes 클라이언트가 DiskStation에 있는 음악 또는 비디어를 찾아 재생할 수 있습니다.
  2. 완료를 눌러 확인합니다.

DiskStation의 인터넷 서비스 관리하기:

  1. 다음을 활성화하거나 비활성화합니다.
    • 자동 차단 기능을 사용하면 승인되지 않은 로그인 시도를 차단할 수 있습니다.
    • FTP를 사용하면 DiskStation을 FTP 서버로 설정할 수 있습니다.
    • 웹 서비스를 사용하면 DiskStation에서 웹사이트를 호스팅(PHP+MySQL 지원 포함)할 수 있습니다.
    • WebDAV를 사용하면 클라이언트 프로그램이 HTTP 프로토콜을 통해 DiskStation 파일을 관리하고 편집할 수 있습니다.
    • WebDAV(HTTPS)를 사용하면 클라이언트 프로그램이 HTTPS 프로토콜을 통해 DiskStation 파일을 관리하고 편집할 수 있습니다.
    • File Station은 DSM에 로그인하지 않고도 사용자 지정 포트를 통해 DiskStation 파일을 관리할 수 있는 웹 기반 응용 프로그램입니다.
    • File Station(HTTPS)을 사용하면 HTTPS 프로토콜을 통해 File Station에 연결할 수 있습니다.
    • Surveillance Station은 가정 또는 사무실 환경을 안전하게 보호하기 위한 IP 카메라를 관리할 수 있는 웹 기반 응용 프로그램입니다.
    • Audio Station은 DiskStation의 음악 라이브러리에 접근할 수 있는 웹 기반 응용 프로그램입니다.
    • Photo Station은 DiskStation에 저장되어 있는 블로그와 통합된 온라인 사진 앨범입니다.
    • Download Station(BT/HTTP/FTP/NZB/RSS)은 인터넷에서 DiskStation으로 파일 다운로드를 지원하는 웹 기반 응용 프로그램입니다.
    • eMule은 Download Station의 eMule 지원을 활성화합니다.
  2. 완료를 눌러 확인합니다.

참고: 상기 서비스에 대한 자세한 정보는 다음을 참조하십시오.

  • DSM 도움말(컴퓨터의 웹 브라우저에서 DSM의 데스크탑 버전에 로그인하고 메인 메뉴 > DSM 도움말로 이동)
  • eCore OS 사용자 가이드(Synology의 다운로드 센터에서 제공)

자동 차단 관리

홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad 또는 Android 태블릿)에서 자동 차단을 눌러 권한이 없는 로그인을 막습니다.

자동 차단 관리하기:

  1. 자동 차단을 활성화하려면 활성화됨을 켭니다.
  2. 지정된 시간(분) 이내에 실패한 로그인 시도 횟수에 도달했을 때 IP 주소를 차단하려면 시도 횟수와 다음 시간 이내(분)에 숫자를 입력합니다.
  3. 차단 날짜를 지정하여 지정된 날짜 이후에 IP 주소 차단을 해제합니다. 여기에서 "0"을 입력하면, 차단 목록에서 제거하기 전까지 차단된 IP 주소가 차단된 상태로 무기한 남아 있습니다.
  4. 차단 목록을 관리하거나 차단된 IP 주소를 목록에서 제거하려면 차단 목록을 누릅니다.
  5. 완료를 누릅니다.

패키지 센터 관리

Synology는 타사 또는 Synology가 설계한 응용 프로그램을 DiskStation에 설치해서 간편하게 사용할 수 있는 패키지에 통합하였습니다.

패키지 센터 관리하기:

컴퓨터의 웹 브라우저를 사용하여 DSM의 데스크탑 버전에 로그인한 다음, 메인 메뉴 > 패키지 센터로 이동하여 패키지를 수정합니다. 자세한 내용은 해당 도움말을 참조하십시오.

설치된 패키지를 실행 또는 중지하기:

  1. 패키지를 실행 또는 중지하려면 스위치를 켜거나 끕니다.
  2. 완료를 누릅니다.

시스템 정보 보기

홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad/Android 태블릿)에서 시스템 정보를 눌러 DiskStation의 상태 개요를 가져올 수 있습니다.

DiskStation의 시스템 정보 보기:

  1. 다음을 수행하십시오.
    • 시리얼 번호, 모델 이름, 물리적 메모리 크기, DSM 버전을 비롯하여 시간 정보와 열 상태를 포함한 DiskStation의 기본 정보를 보려면 일반을 누릅니다.
    • DiskStation의 네트워크 구성과 각 네트워크 인터페이스의 상태를 보려면 네트워크를 누릅니다.
    • DiskStation의 볼륨 사용을 보려면 저장소를 누릅니다.
  2. 시스템 정보를 최신 상태로 유지하려면 오른쪽 상단 모서리에 있는 새로 고침 단추를 누릅니다.

시스템 리소스 모니터링

홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad 또는 Android 태블릿)에서 리소스 모니터를 눌러 DiskStation의 CPU와 메모리 사용을 모니터링할 수 있습니다.

로그 또는 현재 연결 보기

홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad 또는 Android 태블릿)에서 로그를 눌러 현재 연결을 모니터링하거나 DiskStation 서비스의 로그를 살펴봅니다.

현재 연결 모니터링하기:

모니터 섹션에서 현재 연결을 눌러 현재 DiskStation 서비스를 액세스하는 사용자 목록을 살펴봅니다.

특정 서비스의 로그 기록 보기:

이벤트 로그 또는 파일 전송 로그 섹션에서 항목을 눌러 관련 로그 목록을 살펴봅니다.

모바일 앱 시작

DSM mobile은 iOS 또는 Android 장치에 설치된 Synology 모바일 앱을 시작할 수 있는 편리한 장소를 제공합니다.

DSM mobile을 사용하여 Synology 모바일 앱 시작하기:

홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad 또는 Android 태블릿)의 하단 섹션에서 다음 앱 중 하나를 누릅니다.

  • DS photo+(Photo Station 서비스가 활성화된 경우 사용 가능)
  • DS audio(Audio Station 서비스가 활성화된 경우 사용 가능)
  • DS cam(Surveillance Station 서비스가 활성화된 경우 사용 가능)
  • DS file(WebDAV 서비스가 활성화된 경우 사용 가능)

참고: 해당 앱에 대한 자세한 정보는 앱과 함께 제공된 도움말 파일을 참조하십시오.

DiskStation 작업 설정
(다시 시작, 종료 또는 신호음)

DSM mobile을 사용하면 iOS 또는 Android 장치를 사용하여 DiskStation을 다시 시작하거나 종료할 수 있습니다. 뿐만 아니라 사용자 근처에 여러 개의 DiskStation이 있는 경우 신호음 재생을 요청하여 사용자 자신의 DiskStation을 신속히 식별할 수 있습니다.

DiskStation 작업 설정하기:

  1. 홈 페이지(iPhone/iPod touch 또는 Android 휴대폰) 또는 왼쪽 패널(iPad 또는 Android 태블릿)의 오른쪽 상단 모서리에 있는 작업 단추를 누릅니다.
  2. 다음을 수행하십시오.
    • DiskStation을 다시 시작하려면 다시 시작을 누른 다음, 예를 누릅니다.
    • DiskStation을 종료하려면 종료를 누른 다음, 예를 누릅니다.
    • DiskStation이 신호음을 재생하도록 요청하려면 파인드 미를 누릅니다. 신호음을 정지하려면 확인을 누릅니다.

DSM의 데스크탑 버전으로 전환

DSM의 데스크탑 버전으로 전환하려면 왼쪽 패널의 오른쪽 상단 모서리에 있는 작업 단추를 누른 다음, 데스크탑 버전을 누릅니다.
