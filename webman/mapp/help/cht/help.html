DSM mobile 允許 DSM admin (或屬於 administrators 群組的使用者) 使用 iOS (iPhone、iPad 或 iPod touch) 或 Android 裝置管理 DSM 設定並檢查 DSM 資訊。此 DSM mobile 說明為您提供下列主題及資源，協助您快速上手。

登入及登出

使用 iOS 或 Android 裝置的網頁瀏覽器連線至 DSM 來查看登入畫面，您可在其中輸入 DSM 帳號資訊。

若要登入 DSM mobile：

輸入您的 DSM 使用者憑證資訊並點一下登入。如果您要儲存登入資訊並在下次自動登入，請在登入之前開啟記住我的登入資訊開關。

若要登出 DSM mobile：

登入 DSM mobile 之後，您會看到首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 列出各個選項及 iOS 或 Android 裝置上安裝的 Synology 行動應用程式。點一下登出進行登出並回到登入畫面。

管理 DSM 使用者

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下使用者帳號來查看使用者清單。

若要新增使用者：

  1. 點一下 +，然後輸入新使用者的使用者名稱、密碼及電子郵件位址。
  2. 啟動寄送通知來在 DSM 使用者的帳號建立時寄送通知信到他/她的電子郵件位址。
    注意：傳送通知之前，請確定您已完成 DiskStation 的通知設定。請使用電腦的網頁瀏覽器登入一般版 DSM，並前往主選單 > 控制台 > 通知 > 電子郵件來修改設定。
  3. 點一下完成來確認設定，或是點一下取消來取消設定。

若要快速尋找使用者：

在搜尋欄位中輸入使用者名稱。

若要修改使用者的資訊：

  1. 點一下使用者名稱並執行下列任何操作：
    • 在名稱或密碼欄位中修改使用者的名稱或密碼。
    • 開啟停用來停用使用者。您可以選擇稍後啟動使用者，這樣做並不會遺失該使用者的設定。
    • 開啟系統管理者來將使用者加到 admin 群組。屬於 admin 群組的任何使用者皆擁有管理權限，可以使用 DSM mobile 或電腦的網頁瀏覽器修改 DSM 設定。
  2. 在權限管理下方，請執行下列任何操作，然後點一下返回：
    • 點一下共用資料夾然後點一下共用資料夾的名稱來修改使用者的存取權限：可讀寫：使用者可以變更此共用資料夾中的檔案及子資料夾；唯讀：使用者可以存取共用資料夾中的檔案及子資料夾，但是無法變更它們；禁止存取：使用者無法存取此共用資料夾中的檔案及子資料夾。
      注意：發生權限衝突時，權限的優先順序為：禁止存取 > 可讀寫 > 唯讀。
    • 點一下應用程式來修改使用者對 DSM 應用程式的存取權限。
  3. 點一下完成來確認設定，或是點一下取消來取消設定。

管理共用資料夾

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下共用資料夾來查看 DiskStation 上的共用資料夾清單。

若要新增共用資料夾：

  1. 點一下 +，然後輸入新資料夾的名稱。
  2. 點一下完成來確認設定，或是點一下取消來取消設定。

若要修改共用資料夾：

點一下共用資料夾的名稱並執行下列任何操作：

  • 在名稱欄位中修改共用資料夾的名稱。
  • 在權限管理下方，請點一下使用者並點一下使用者來修改該使用者對此共用資料夾的存取權限：可讀寫：使用者可以變更此共用資料夾中的檔案及子資料夾；唯讀：使用者可以存取共用資料夾中的檔案及子資料夾，但是無法變更它們；禁止存取：使用者無法存取此共用資料夾中的檔案及子資料夾。
    注意：發生權限衝突時，權限的優先順序為：禁止存取 > 可讀寫 > 唯讀。

啟動服務

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下系統服務來啟動或停用 DiskStation 服務。

若要管理 DiskStation 的區域網路服務：

  1. 啟動或停用下列任何項目：
    • Windows 檔案服務可讓 DiskStation 成為區域網路中的 Windows (CIFS) 檔案伺服器。
    • Mac 檔案服務可讓 DiskStation 成為區域網路中的 Mac (AFP) 檔案伺服器。
    • NFS 服務可讓區域網路中的 Linux 用戶端存取 DiskStation 的資料。
    • Bonjour 印表機可讓 Mac 用戶端使用 DiskStation 上連接的 USB 印表機進行列印。
    • 媒體伺服器可讓區域網路中的 DLNA/UPnP 家用裝置瀏覽並播放 DiskStation 上的多媒體內容。
    • iTunes 可讓區域網路中的 iTunes 用戶端瀏覽並播放 DiskStation 上的音樂或影片。
  2. 點一下完成來進行確認。

若要管理 DiskStation 的網際網路服務：

  1. 啟動或停用下列任何項目：
    • 自動封鎖可讓您預防未經允許的登入嘗試。
    • FTP 可讓您的 DiskStation 成為 FTP 伺服器。
    • 網頁服務可讓您在 DiskStation 上架設網站 (支援 PHP+MySQL)。
    • WebDAV 可讓用戶端程式透過 HTTP 通訊協定管理並編輯 DiskStation 檔案。
    • WebDAV (HTTPS) 可讓用戶端程式透過 HTTPS 通訊協定管理並編輯 DiskStation 檔案。
    • File Station 是網頁介面的應用程式，可讓您無須登入 DSM 即可透過自訂埠管理 DiskStation 檔案。
    • File Station (HTTPS) 可讓您透過 HTTPS 通訊協定連線至 File Station。
    • Surveillance Station 是網頁介面的應用程式，其可管理網路攝影機來保衛家庭或辦公室環境的安全。
    • Audio Station 是網頁介面的應用程式，可讓您存取 DiskStation 上的音樂資料庫。
    • Photo Station 是整合了網誌功能的相簿，存放在您的 DiskStation 上。
    • Download Station (BT/HTTP/FTP/NZB/RSS) 是網頁介面的應用程式，可協助您將網際網路上的檔案下載到 DiskStation 上。
    • eMule 可啟動 Download Station 的 eMule 支援功能。
  2. 點一下完成來進行確認。

注意：如需更多上述服務的相關資訊，請參閱下列任何資料：

  • DSM 說明 (您可以使用電腦的網頁瀏覽器登入 DSM 的一般版，並前往主選單 > DSM 說明進行存取)
  • eCore OS 使用手冊 (位於 Synology 下載中心)

管理自動封鎖

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下自動封鎖來預防未經授權的登入嘗試。

若要管理自動封鎖：

  1. 開啟已啟動來啟動 [自動封鎖]。
  2. 在嘗試次數和幾分鐘內輸入數字。系統將會封鎖在設定分鐘內登入失敗超過嘗試登入次數的 IP 位址。
  3. 指定封鎖天數來在指定天數之後解除封鎖 IP 位址。如果您在此處輸入「0」，所封鎖的 IP 位址會保持封鎖狀態 (除非您將該位址從封鎖名單中移除)。
  4. 如果您要管理封鎖名單或從名單中移除封鎖的 IP 位址，請點一下封鎖名單。
  5. 按一下完成。

管理套件中心

Synology 已將協力廠商或是 Synology 設計的應用程式整合到套件中，您可以在 DiskStation 上輕鬆地安裝並啟動這些程式。

若要管理套件中心：

請使用電腦的網頁瀏覽器登入一般版 DSM，並前往主選單 > 套件中心來修改套件。如需詳細指示，請參考其說明。

若要執行或停止已安裝的套件：

  1. 開啟或關閉開關來執行或停止套件。
  2. 按一下完成。

檢視系統資訊

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下系統資訊來取得 DiskStation 的狀態總覽。

若要檢視 DiskStation 的系統資訊：

  1. 請執行下列任一項操作：
    • 點一下一般來檢視 DiskStation 的基本資訊，包含序號、型號、實體記憶體大小、DSM 版本、時間資訊以及散熱狀態。
    • 點一下網路來檢視 DiskStation 的網路設定以及各個網路介面的狀態。
    • 點一下儲存空間來檢視 DiskStation 的儲存空間用量。
  2. 點一下右上角的重新整理按鈕來讓系統資訊保持最新狀態。

監控系統資源

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下資源監控來監控 DiskStation 的 CPU 及記憶體用量。

檢視日誌或目前連線

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 上點一下日誌來監控目前的連線或是檢視 DiskStation 服務的日誌。

若要監控目前連線：

點一下監控區塊中的目前連線來檢視正在存取 DiskStation 服務的使用者。

若要查看特定服務的日誌記錄：

點一下事件日誌或檔案傳輸日誌區塊中的任何項目來查看相關日誌的清單。

開啟行動應用程式

DSM mobile 為您提供一個便利的位置來開啟 iOS 或 Android 裝置上安裝的 Synology 行動應用程式。

若要使用 DSM mobile 開啟 Synology 行動應用程式：

在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 的下半部點按下列任何應用程式：

  • DS photo+ (啟動 Photo Station 服務後才可存取)
  • DS audio (啟動 Audio Station 服務後才可存取)
  • DS cam (啟動 Surveillance Station 服務後才可存取)
  • DS file (啟動 WebDAV 服務後才可存取)

注意：如需更多使用這些應用程式的相關資訊，請參閱應用程式隨附的說明文件。

操控 DiskStation
(重新啟動、關機或嗶聲控制)

DSM mobile 可讓您使用 iOS 或 Android 裝置為 DiskStation 執行重新啟動或關機。此外，當您身邊有多部 DiskStation 時，您也可以要求您的 DiskStation 播放嗶聲以進行快速識別。

若要操控 DiskStation：

  1. 在首頁 (iPhone/iPod touch 或 Android 手機) 或左側面板 (iPad 或 Android 平板裝置) 的右上角點一下動作按鈕。
  2. 請執行下列任一項操作：
    • 點一下重新開機然後點一下是來為 DiskStation 重新開機。
    • 點一下關機然後點一下是來將 DiskStation 關機。
    • 點一下搜尋我來要求 DiskStation 發出嗶聲。點一下確定來停止嗶聲。

切換至 DSM 一般版

點一下左側面板右上角的動作按鈕，然後點一下一般版來切換至 DSM 一般版。
