DSM mobile erlaubt DSM admin (oder einem Benutzer der administrators-Gruppe), die DSM-Einstellungen zu verwalten und DSM-Daten mit einem iOS®-Gerät (iPhone®, iPad® oder iPod touch®) oder Android®-Gerät zu überprüfen. In dieser Hilfe zu DSM mobile finden Sie folgende Themen sowie Ressourcen für den Einstieg.

An- und Abmeldung

Nachdem Sie sich mit dem Webbrowser auf Ihrem iOS- oder Android-Gerät mit DSM verbunden haben, erscheint der Anmeldebildschirm, in dem Sie die Daten für Ihr DSM-Konto eintragen können.

So melden Sie sich bei DSM mobile an:

Geben Sie Ihre Benutzerdaten für DSM ein und tippen Sie auf Anmelden. Wenn Sie Ihre Anmeldedaten speichern und sich beim nächsten Mal automatisch anmelden möchten, aktivieren Sie Merken, ehe Sie sich anmelden.

So melden Sie sich bei DSM mobile ab:

Nach Ihrer Anmeldung bei DSM mobile sehen Sie auf der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) eine Liste mit Optionen und mobilen Apps von Synology, die auf Ihrem iOS- oder Android-Gerät installiert sind. Tippen Sie auf Abmelden, um sich abzumelden und zum Anmeldebildschirm zurückzugelangen.

DSM-Benutzer verwalten

Tippen Sie auf der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf Benutzer, um die Benutzerliste anzuzeigen.

So erstellen Sie einen Benutzer:

  1. Tippen Sie auf + und geben Sie dann den Benutzernamen, das Kennwort und die E-Mail-Adresse für den neuen Benutzer ein.
  2. Aktivieren Sie Nachricht senden, um eine Benachrichtungs-E-Mail an die E-Mail-Adresse des Nutzers zu schicken, wenn sein oder ihr DSM-Benutzerkonto erstellt worden ist.
    Hinweis: Ehe Sie eine Benachrichtigung senden, müssen die Benachrichtigungseinstellungen auf Ihrer DiskStation vollständig ausgefüllt sein. Melden Sie sich bei der Desktop Edition des DSM über den Webbrowser Ihres Computers an und wechseln Sie nach Hauptmenü > Bedienfeld > Benachrichtigung > E-Mail, um die Einstellungen zu ändern.
  3. Tippen Sie zur Bestätigung auf Fertig oder wählen Sie Abbrechen, um den Vorgang abzubrechen.

So finden Sie einen Benutzer schnell auf:

Geben Sie den Benutzernamen in das Feld Suche ein.

So ändern Sie die Benutzerinformationen:

  1. Tippen Sie den Namen des Benutzers ein und wählen Sie eine der folgenden Möglichkeiten:
    • Ändern Sie den Namen oder das Kennwort des Benutzers im Feld Name bzw. Kennwort.
    • Markieren Sie Deaktivieren, um den Benutzer zu deaktivieren. Sie können den Benutzer später wieder reaktivieren, ohne seine oder ihre Einstellungen zu verlieren.
    • Aktivieren Sie administrator, um den Benutzer zur admin-Gruppe hinzuzufügen. Benutzer, die der admin-Gruppe angehören, verfügen über Administratorprivilegien und dürfen DSM-Einstellungen mit DSM mobile oder im Webbrowser ihres Computers ändern.
  2. Führen Sie unter Berechtigungen Folgendes aus und tippen Sie anschließend auf Zurück:
    • Tippen Sie auf Gemeinsamer Ordner und tippen Sie den Namen eines gemeinsamen Ordners ein, um das Zugriffsprivileg des Benutzers zu ändern: Lesen/Schreiben: Der Benutzer kann Dateien und Unterordner im gemeinsamen Ordner ändern; Nur Lesen: Der Benutzer hat Zugriff auf Dateien und Unterordner im gemeinsamen Ordner und kann keine Änderungen vornehmen.Kein Zugriff: Der Benutzer hat keinen Zugriff auf die Dateien und Unterordner im gemeinsamen Ordner.
      Hinweis: Bei widersprüchlichen Privilegien gilt folgende Priorität: Kein Zugriff > Lesen/Schreiben > Nur lesen.
    • Tippen Sie auf Applikation, um das Zugriffsprivileg des Benutzers für DSM-Anwendungen zu ändern.
  3. Tippen Sie zur Bestätigung auf Fertig oder wählen Sie Abbrechen, um den Vorgang abzubrechen.

Gemeinsame Ordner verwalten

Tippen Sie auf der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf Gemeinsamer Ordner, um die gemeinsamen Ordner auf Ihrer DiskStation anzuzeigen.

So erstellen Sie einen gemeinsamen Ordner:

  1. Tippen Sie auf + und geben Sie dann den Namen für den neuen Ordner ein.
  2. Tippen Sie zur Bestätigung auf Fertig oder wählen Sie Abbrechen, um den Vorgang abzubrechen.

So ändern Sie einen gemeinsamen Ordner:

Tippen Sie den Namen des gemeinsamen Ordners ein und wählen Sie eine der folgenden Möglichkeiten:

  • Ändern Sie den Namen des gemeinsamen Ordners im Feld Name.
  • Tippen Sie unter Berechtigungen auf Benutzer und dann auf einen Benutzer, um dessen Zugriffsprivileg für den gemeinsamen Ordner zu ändern: Lesen/Schreiben: Der Benutzer kann Dateien und Unterordner im gemeinsamen Ordner ändern; Nur Lesen: Der Benutzer hat Zugriff auf Dateien und Unterordner im gemeinsamen Ordner und kann keine Änderungen vornehmen.Kein Zugriff: Der Benutzer hat keinen Zugriff auf die Dateien und Unterordner im gemeinsamen Ordner.
    Hinweis: Bei widersprüchlichen Privilegien gilt folgende Priorität: Kein Zugriff > Lesen/Schreiben > Nur lesen.

Dienste aktivieren

Tippen Sie auf der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf Dienst, um Dienste der DiskStation zu aktivieren oder zu deaktivieren.

So verwalten Sie LAN-Dienste der DiskStation:

  1. Aktivieren oder deaktivieren Sie eine der folgenden Möglichkeiten:
    • Windows-Dateidienst ermöglicht Ihrer DiskStation, als Windows (CIFS) Datei-Server im LAN zu dienen.
    • Mac Dateidienst ermöglicht Ihrer DiskStation, als Mac (AFP) Datei-Server im LAN zu dienen.
    • NFS-Dienst ermöglicht dem Linux-Clienten im LAN den Zugriff auf DiskStation-Daten.
    • Bonjour-Drucker ermöglicht Mac-Clienten, auf an Ihrer DiskStation angeschlossenen USB-Druckern auszudrucken.
    • Medienserver ermöglicht DLNA/UPnP-Heimgeräten im LAN, Multimedia-Inhalte auf Ihrer DiskStation zu durchsuchen und wiederzugeben.
    • iTunes ermöglicht iTunes-Clienten im LAN, Musik und Videos auf Ihrer DiskStation zu durchsuchen und wiederzugeben.
  2. Tippen Sie zur Bestätigung auf Fertig.

So verwalten Sie Internetdienste der DiskStation:

  1. Aktivieren oder deaktivieren Sie eine der folgenden Möglichkeiten:
    • Automatische Blockierung ermöglicht Ihnen, nicht autorisierte Anmeldungen zu verhindern.
    • FTP ermöglicht Ihrer DiskStation, als FTP-Server zu dienen.
    • Webdienste ermöglicht Ihnen, Websites (mit PHP+MySQL Support) auf Ihrer DiskStation zu hosten.
    • WebDAV ermöglicht Clientprogrammen, DiskStation-Dateien über das HTTP-Protokoll zu verwalten und zu bearbeiten.
    • WebDAV (HTTPS) ermöglicht Clientprogrammen, DiskStation-Dateien über das HTTPS-Protokoll zu verwalten und zu bearbeiten.
    • File Station ist eine webbasierte Anwendung, mit der Sie DiskStation-Dateien über einen benutzerdefinierten Port verwalten können, ohne sich bei DSM anzumelden.
    • File Station (HTTPS) ermöglicht Ihnen, sich mit dem HTTPS-Protokoll mit der File Station zu verbinden.
    • Surveillance Station ist eine webbasierte Anwendung, die IP-Kameras verwaltet, um Ihr Haus oder Ihr Büro zu schützen.
    • Audio Station ist eine webbasierte Anwendung, mit der Sie die Musikbibliothek auf Ihrer DiskStation öffnen können.
    • Photo Station ist ein Online-Fotoalbum mit einem Blog, das auf Ihrer DiskStation gespeichert wird.
    • Download Station (BT/HTTP/FTP/NZB/RSS) ist eine webbasierte Download-Anwendung, mit der Sie Dateien aus dem Internet auf Ihre DiskStation herunterladen können.
    • eMule aktiviert den eMule-Support der Download Station.
  2. Tippen Sie zur Bestätigung auf Fertig.

Hinweis: Weitere Informationen über die oben aufgeführten Dienste finden Sie in folgenden Quellen:

  • DSM-Hilfe (melden Sie sich dazu im Webbrowser Ihres Computers bei der Desktop Edition Ihres DSM an und wechseln Sie nach Hauptmenü > DSM-Hilfe)
  • Benutzerhandbuch eCore OS (verfügbar im Synology Download-Zentrum)

Automatische Blockierung verwalten

Tippen Sie auf der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf Automatische Blockierung, um nicht autorisierte Anmeldungen zu verhindern.

So verwalten Sie die automatische Blockierung:

  1. Markieren Sie Aktiviert, um die automatische Blockierung zu aktivieren.
  2. Geben Sie eine Zahl für die Anzahl von Versuchen und Innerhalb von (Minuten) ein, um eine IP-Adresse nach der eingestellten Anzahl Fehlversuche innerhalb der eingestellten Minutenzahl zu blockieren.
  3. Legen Sie Tage der Blockierung fest, um die Blockierung einer IP-Adresse nach einer bestimmten Anzahl von Tage aufzuheben. Wenn Sie hier eine „0“ eingeben, wird die IP-Adresse solange blockiert, bis Sie sie aus der Blockierungsliste entfernen.
  4. Wenn Sie die Blockierungsliste verwalten oder eine blockierte IP-Adresse aus der Liste entfernen möchten, tippen Sie auf Blockierungsliste.
  5. Tippen Sie auf Fertig.

Paketzentrum verwalten

Synology verfügt über eigene oder integrierte Anwendungen von Drittanbietern, die auf einfache Weise auf der DiskStation installiert und verwaltet werden können.

So verwalten Sie das Paketzentrum:

Melden Sie sich bei der Desktop Edition des DSM über den Webbrowser Ihres Computers an und wechseln Sie nach Hauptmenü > Paketzentrum, um die Pakete zu ändern. Detaillierte Anweisungen finden Sie in der Hilfe.

So führen Sie installierte Pakete aus oder beenden sie:

  1. Betätigen Sie die Schalter, um die Pakete auszuführen oder zu stoppen.
  2. Tippen Sie auf Fertig.

Systeminformationen anzeigen

Tippen Sie auf der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf Systeminformationen, um eine Statusübersicht für Ihre DiskStation anzuzeigen.

So zeigen Sie Systeminformationen für die DiskStation an:

  1. Verwenden Sie eine der folgenden Methoden:
    • Tippen Sie auf Allgemein, um grundlegende Informationen über Ihre DiskStation anzuzeigen, darunter Seriennummer, Modellname, Größe des physikalischen Speichers, DSM-Version sowie Uhrzeit und Thermal-Status.
    • Tippen Sie auf Netzwerk, um die Netzwerkkonfiguration Ihrer DiskStation und den Status der einzelnen Netzwerkschnittstellen anzuzeigen.
    • Tippen Sie auf Speicher, um die Volume-Auslastung Ihrer DiskStation anzuzeigen.
  2. Tippen Sie auf die Schaltfläche Aktualisieren (oben rechts), um die Systeminformationen auf aktuellem Stand zu halten.

Systemressourcen überwachen

Tippen Sie auf der Homepage (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf Ressourcen-Monitor, um die CPU- und Volume-Auslastung Ihrer DiskStation anzuzeigen.

Protokolle oder aktuelle Verbindungen anzeigen

Tippen Sie auf der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf Protokoll, um aktuelle Verbindungen zu überwachen oder Protokolle von DiskStation-Diensten anzuzeigen.

So überwachen Sie aktuelle Verbindungen:

Tippen Sie auf Aktuelle Verbindungen unter Überwachung, um die Liste mit Benutzern anzuzeigen, die derzeit auf DiskStation-Dienste zugreifen.

So zeigen Sie die Protokolleinträge eines bestimmten Dienstes an:

Tippen Sie auf ein Element unter Ereignisprotokolle oder Dateiübertragungs-Protokolle, um die Liste mit entsprechenden Protokollen anzuzeigen.

Mobile Anwendungen starten

DSM mobile ist eine bequeme Möglichkeit, um mobile Synology-Anwendungen, die auf Ihrem iOS- oder Android-Gerät gespeichert sind, zu starten.

So starten Sie mobile Synology-Anwendungen mit DSM mobile:

Tippen Sie im unteren Bereich der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf eine der folgenden Anwendungen:

  • DS photo+ (verfügbar, wenn der Photo Station-Dienst aktiviert ist)
  • DS audio (verfügbar, wenn der Audio Station-Dienst aktiviert ist)
  • DS cam (verfügbar, wenn der Surveillance Station-Dienst aktiviert ist)
  • DS file (verfügbar, wenn der WebDAV-Dienst aktiviert ist)

Hinweis: Weitere Informationen zu diesen Apps finden Sie in den Hilfe-Dateien, die Sie zu diesen Apps erhalten.

DiskStation in Aktion versetzen
(Neustart, Ausschalten oder Signalton)

DSM mobile ermöglicht Ihnen, Ihre DiskStation mit Ihrem iOS- oder Android-Gerät neu zu starten oder auszuschalten. Zudem können Sie, wenn sich mehrere DiskStations in Ihrer Nähe befinden, Ihre eigene DiskStation schnell identifizieren, indem Sie sie Signaltöne ausgeben lassen.

So setzen Sie die DiskStation in Aktion:

  1. Tippen Sie oben rechts auf der Startseite (iPhone/iPod touch oder Android-Telefon) oder im linken Feld (iPad oder Android-Tablet) auf die Schaltfläche Aktion:
  2. Verwenden Sie eine der folgenden Methoden:
    • Tippen Sie auf Neustart und dann auf Ja, um Ihre DiskStation neu zu starten.
    • Tippen Sie auf Herunterfahren und dann auf Ja, um Ihre DiskStation auszuschalten.
    • Tippen Sie auf Suchen, um die DiskStation Signaltöne ausgeben zu lassen. Tippen Sie auf OK, um den Signalton zu beenden.

Zur Desktop Edition von DSM wechseln

Tippen Sie oben rechts oder im linken Feld auf die Schaltfläche Aktion und dann auf Desktop Edition, um zur Desktop Edition von DSM zu wechseln.
