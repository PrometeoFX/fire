DSM mobile 允许 DSM admin（或属于 administrators 组的用户）管理 DSM 设置并用 iOS®（iPhone®、iPad® 或 iPod touch®）或 Android® 设备检查 DSM 信息。此 DSM mobile 帮助为您提供下列主题及资源，帮助您上手。

登录和注销

使用 iOS 或 Android 设备的网页浏览器连接到 DSM 后，您可以看到登录画面，可在其中输入 DSM 帐户信息。

若要登录 DSM mobile：

输入您的 DSM 用户证书信息并点按登录。如果您要保存登录信息并在下次自动登录，请在登录之前打开记住我开关。

若要注销 DSM mobile：

登录 DSM mobile 后，您会看到主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板），包含选项列表及 iOS 或 Android 设备上安装的 Synology 移动应用程序。点按注销进行注销并返回登录屏幕。

管理 DSM 用户

在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）上点按用户来查看用户列表。

若要创建用户：

  1. 点按 +，然后输入新用户的用户名称、密码及电子邮件地址。
  2. 启用发送通知以在 DSM 用户的帐户创建时发送通知到其电子邮件地址。
    注：发送通知前，请确定您已完成 DiskStation 的通知设置。用您计算机的网页浏览器登录至 DSM 桌面版，并进入主菜单 > 控制面板 > 通知 > 电子邮件修改设置。
  3. 点按完成进行确认或点按取消进行取消。

若要快速查找用户：

在搜索字段中输入用户名。

若要修改用户信息：

  1. 点按用户名并运行下列任何操作：
    • 在名称或密码字段中修改用户名和密码。
    • 打开禁用以禁用用户。您可以选择稍后启用用户，这样做并不会遗失该用户的设置。
    • 打开系统管理者，添加用户至 admin 组。属于 admin 组的任何用户皆拥有管理权限，可以使用 DSM mobile 或计算机的网页浏览器修改 DSM 设置。
  2. 在权限选项卡，进行以下任何操作并点按返回：
    • 点按共享文件夹并点按共享文件夹的名称，修改用户访问权限：读取/写入：用户可以访问并更改此共享文件夹中的文件及子文件夹；只读：用户可以访问此共享文件夹中的文件及子文件夹，但是无法进行更改；禁止访问：用户无法访问此共享文件夹中的文件及子文件夹。
      注：如果发生权限冲突，权限优先将为：禁止访问 > 读取/写入 > 只读。
    • 点按应用程序来修改用户对 DSM 应用程序的访问权限。
  3. 点按完成进行确认或点按取消进行取消。

管理共享文件夹

在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）上点按共享文件夹来查看 DiskStation 上的共享文件夹列表。

若要创建共享文件夹：

  1. 点按 +，然后输入新文件夹的名称。
  2. 点按完成进行确认或点按取消进行取消。

若要修改共享文件夹：

点按共享文件夹的名称并运行下列任何操作：

  • 在名称字段中修改共享文件夹的名称。
  • 在权限选项卡，点按用户并点按一个用户，修改其对共享文件夹的访问权限：读取/写入：用户可以访问并更改此共享文件夹中的文件及子文件夹；只读：用户可以访问此共享文件夹中的文件及子文件夹，但是无法进行更改；禁止访问：用户无法访问此共享文件夹中的文件及子文件夹。
    注：如果发生权限冲突，权限优先将为：禁止访问 > 读取/写入 > 只读。

启用服务

在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）上点按服务来启用或禁用 DiskStation 服务。

若要管理 DiskStation 的局域网服务：

  1. 启用或禁用下列任何项目：
    • Windows 文件服务可让 DiskStation 成为局域网中的 Windows (CIFS) 文件服务器。
    • Mac 文件服务可让 DiskStation 成为局域网中的 Mac（AFP） 文件服务器。
    • NFS 服务可让局域网中的 Linux 客户端访问 DiskStation 的数据。
    • Bonjour 打印机可让 Mac 客户端打印到与 DiskStation 连接的 USB 打印机。
    • 媒体服务器可让局域网中的 DLNA/UPnP 家用设备浏览并播放 DiskStation 上的多媒体内容。
    • iTunes 可让局域网中的 iTunes 客户端浏览并播放 DiskStation 上的音乐或视频。
  2. 点按完成确认。

若要管理 DiskStation 的 Internet 服务：

  1. 启用或禁用下列任何项目：
    • 自动锁定可以防止未授权登录尝试。
    • FTP 可让您的 DiskStation 成为 FTP 服务器。
    • 网页服务可让您在 DiskStation 上架设网站（支持 PHP+MySQL）。
    • WebDAV 可让客户端程序通过 HTTP 协议管理并编辑 DiskStation 文件。
    • WebDAV (HTTPS) 可让客户端程序通过 HTTP 协议管理并编辑 DiskStation 文件。
    • File Station 是网页类应用程序，可让您无须登录 DSM 即可通过自定义端口管理 DiskStation 文件。
    • File Station (HTTPS) 可让您通过 HTTPS 协议连接到 File Station。
    • Surveillance Station 是一个基于网络的应用程序，可以管理 IP 摄像机以保护您的家庭和办公环境。
    • Audio Station 是网页界面的应用程序，可让您访问 DiskStation 上的音乐库。
    • Photo Station 是整合保存在 DiskStation 上的博客的在线相册。
    • Download Station (BT/HTTP/FTP/NZB/RSS) 是基于 Web 的应用程序，可辅助您将 Internet 上的文件下载到 DiskStation 上。
    • eMule 提供 Download Station 的 eMule 支持。
  2. 点按完成确认。

注：以上服务详情参看以下：

  • DSM 帮助（可通过用计算机网页浏览器登录 DSM 桌面版并进入主菜单 > DSM 帮助打开）
  • Synology DiskStation 用户指南（位于 Synology 的下载中心）

管理自动锁定

在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）上点按自动锁定防止未授权登录。

若要管理自动锁定：

  1. 打开已启用启用自动锁定。
  2. 在尝试次数和在（分钟）内输入数字，锁定在设置分钟内登录失败超过尝试登录次数的 IP 地址。
  3. 指定锁定天数在指定的天数后解锁 IP 地址。如果您在此输入“0”，锁定的 IP 地址将永久锁定，直到您从锁定列表中将其删除。
  4. 如果您想要管理锁定列表或从列表中删除锁定 IP 地址，点按锁定列表。
  5. 点按完成。

管理软件包中心

Synology 已将第三方或是 Synology 自行设计的应用程序整合到软件包中，可在 DiskStation 上安装并轻易启用。

欲管理软件包中心：

用您计算机的网页浏览器登录至 DSM 桌面版，并进入主菜单 > 软件包中心修改设置。详情请参看帮助。

欲运行或停止安装的软件包：

  1. 打开或关闭开关以运行或停止软件包。
  2. 点按完成。

查看系统信息

在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）上点按系统信息来取得 DiskStation 的状态概况。

若要查看 DiskStation 的系统信息：

  1. 运行下列任何操作：
    • 点按常规来查看 DiskStation 的基本信息，包含序列号、型号、物理内存、DSM 版本、时间信息以及散热状态。
    • 点按网络来查看 DiskStation 的网络设置以及各个网络接口的状态。
    • 点按存储来查看 DiskStation 的存储空间。
  2. 点按右上角的刷新按钮来让系统信息保持最新状态。

监控系统资源

在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）上点按资源监控来监控 DiskStation 的 CPU 及内存使用。

查看日志或当前连接

在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）上点按日志来监控当前连接或查看 DiskStation 服务日志。

欲监控当前连接：

点按监控部分中的当前连接查看当前访问 DiskStation 服务的用户列表。

若要查看指定服务的日志记录：

点按事件日志或文件传输日志部分的任何项目，查看相关日志列表。

开通移动应用程序

DSM mobile 为您提供一个便利的位置来打开 iOS 或 Android 设备上安装的 Synology 移动应用程序。

若要使用 DSM mobile 开通 Synology 移动应用程序：

在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）的下半部点按下列任何应用程序：

  • DS photo+（当 Photo Station 服务启用时可用）
  • DS audio（当 Audio Station 服务启用时可用）
  • DS cam（当 Surveillance Station 服务启用时可用）
  • DS file（当 WebDAV 服务启用时可用）

注：如需更多使用这些应用程序的相关信息，请参阅应用程序随附的帮助文件。

设置 DiskStation 动作
（重新启动、关机或哔声）

DSM mobile 可让您使用 iOS 或 Android 设备为 DiskStation 运行重新启动或关机。此外，当您身边有多部 DiskStation 时，您也可以要求您的 DiskStation 播放哔声以进行快速识别。

若要设置 DiskStation 动作：

  1. 在主页（iPhone/iPod touch 或 Android 手机）或左侧面板（iPad 或 Android 平板）的右上角点按动作按钮。
  2. 运行下列任何操作：
    • 点按重新启动然后点按是重启 DiskStation。
    • 点按关机然后点按是关闭 DiskStation。
    • 点按找我，要求 DiskStation 播放哔声。点按确定停止发出哔声。

切换至 DSM 桌面版

点按左侧面板右上角的动作按钮，然后点按桌面版来切换至 DSM 桌面版。
