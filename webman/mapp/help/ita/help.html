DSM mobile consente all'admin DSM (o a un utente appartenente al gruppo administrators) di gestire le impostazioni DSM e di controllarne le informazioni DSM con un dispositivo iOS® (iPhone®, iPad® o iPod touch®) o Android®. Questa Guida al DSM mobile fornisce i seguenti argomenti e risorse.

Accesso e disconnessione

Dopo la connessione al DSM con il browser web sul dispositivo iOS o Android, sarà visibile la schermata di accesso dove inserire le informazioni dell'account DSM.

Per accedere a DSM mobile:

Inserire le credenziali utente DSM e toccare Accedi. Se si desidera salvare le informazioni di accesso e accedere automaticamente la volta successiva, attivare l'interruttore Memorizza prima di accedere.

Per uscire da DSM mobile:

Dopo la connessione a DSM mobile, sarà visibile la home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) contenente un elenco di opzioni e applicazioni mobile Synology installate sul dispositivo iOS o Android. Toccare Esci per uscire e tornare alla schermata di accesso.

Gestione utenti DSM

Toccare Utente sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per vedere l'elenco utenti.

Per creare un utente:

  1. Toccare + e inserire il nome utente e la password e l'indirizzo e-mail del nuovo utente.
  2. Attivare Invia notifica per inviare una mail di notifica all'indirizzo e-mail dell'utente quando è stato creato l'account dell'utente DSM.
    Nota: prima di inviare la notifica, accertarsi di aver completato le impostazioni di notifica sulla DiskStation. Accedere all'edizione desktop del DSM con il browser web del computer e selezionare Menu principale > Pannello di controllo > Notifica > E-mail per modificare le impostazioni.
  3. Toccare Fine per confermare o Annulla per annullare.

Per trovare velocemente un utente:

Inserire il nome utente nel campo di ricerca.

Per modificare le informazioni di un utente:

  1. Toccare il nome dell'utente e procedere come segue:
    • Modificare il nome o la password dell'utente nel campo Nome o Password.
    • Attivare Disabilita per disabilitare l'utente. È possibile scegliere di abilitare l'utente in un secondo momento senza perdere le sue impostazioni.
    • Attivare Amministratore per aggiungere l'utente al gruppo admin. Un utente qualsiasi appartenente al gruppo admin avrà i privilegi amministrativi per modificare le impostazioni DSM tramite DSM mobile o il browser web del computer.
  2. In Privilegi, eseguire quanto segue e toccare Indietro:
    • Toccare Cartella condivisa e toccare il nome della cartella condivisa per modificare il privilegio di accesso dell'utente: Lettura/Scrittura: l'utente può apportare modifiche a file e sottocartelle nella cartella condivisa. Sola lettura: l'utente può accedere a file e sottocartelle nella cartella condivisa, ma non apportarvi modifiche; Nessun accesso: l'utente non può accedere a file e sottocartelle nella cartella condivisa.
      Nota: se si verificano conflitti di privilegio, la priorità sarà: Nessun accesso > Lettura/Scrittura > Sola lettura.
    • Toccare Applicazione per modificare il privilegio di accesso dell'utente alle applicazioni DSM.
  3. Toccare Fine per confermare o Annulla per annullare.

Gestire le cartelle condivise

Toccare Cartella condivisa sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per vedere l'elenco delle cartelle condivise su DiskStation.

Per creare una cartella condivisa:

  1. Toccare + e inserire il nome della nuova cartella.
  2. Toccare Fine per confermare o Annulla per annullare.

Per modificare una cartella condivisa:

Toccare il nome della cartella condivisa e procedere come segue:

  • Modificare il nome della cartella condivisa nel campo Nome.
  • In Privilegi, toccare Utente e poi un utente per modificarne i privilegi di accesso alla cartella condivisa: Lettura/Scrittura: l'utente può apportare modifiche a file e sottocartelle nella cartella condivisa. Sola lettura: l'utente può accedere a file e sottocartelle nella cartella condivisa, ma non apportarvi modifiche; Nessun accesso: l'utente non può accedere a file e sottocartelle nella cartella condivisa.
    Nota: se si verificano conflitti di privilegio, la priorità sarà: Nessun accesso > Lettura/Scrittura > Sola lettura.

Abilitare i servizi

Toccare Servizio sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per abilitare o disabilitare i servizi DiskStation.

Per gestire i servizi LAN DiskStation:

  1. Abilitare o disabilitare quanto segue:
    • Servizio file Windows consente alla DiskStation di diventare un server file Windows (CIFS) nella LAN.
    • Servizio file Mac consente alla DiskStation di diventare un server file Mac (AFP) nella LAN.
    • Servizio NFS consente ai client Linux nella LAN di accedere ai dati DiskStation.
    • Stampante Bonjour permette ai client Mac di stampare sulle stampanti USB connesse a DiskStation.
    • Server media consente ai dispositivi domestici DLNA/UPnP nella LAN di cercare e riprodurre i contenuti multimediali su DiskStation.
    • iTunes consente ai client iTunes nella LAN di cercare e riprodurre musica e video su DiskStation.
  2. Toccare Fine per confermare.

Per gestire i servizi Internet DiskStation:

  1. Abilitare o disabilitare quanto segue:
    • Blocco automatico consente di evitare accessi non autorizzati.
    • FTP permette alla DiskStation di diventare un server FTP.
    • Servizi Web consente di ospitare siti (con supporto PHP+MySQL) sulla DiskStation.
    • WebDAV consente ai programmi client di gestire e modificare i file DiskStation tramite protocollo HTTP.
    • WebDAV (HTTPS) consente ai programmi client di gestire e modificare i file DiskStation tramite protocollo HTTPS.
    • File Station è un'applicazione su base web che permette la gestione dei file DiskStation tramite porta personalizzata senza connettersi al DSM.
    • File Station (HTTPS) consente la connessione a File Station tramite protocollo HTTPS.
    • Surveillance Station è un'applicazione su web che può gestire le telecamere IP per il controllo dell'ambiente di casa o lavorativo.
    • Audio Station è un'applicazione su web che permette di accedere alla libreria musicale su DiskStation.
    • Photo Station è un album fotografico online con blog integrato salvato sulla DiskStation.
    • Download Station (BT/HTTP/FTP/NZB/RSS) è un'applicazione web che aiuta a scaricare i file da Internet su DiskStation.
    • eMule abilita il supporto eMule di Download Station.
  2. Toccare Fine per confermare.

Nota: per maggiori informazioni sui servizi, consultare quanto segue:

  • Guida DSM (disponibile collegandosi all'edizione desktop del DSM con il browser del computer e andando su Menu principale > Guida DSM)
  • Guida dell'utente eCore OS (disponibile al Centro download Synology)

Gestire il blocco automatico

Toccare Blocco automatico sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per evitare l'accesso non autorizzato.

Per gestire il blocco automatico:

  1. Attivare Abilitato per abilitare il Blocco automatico.
  2. Inserire un numero per il Numero di tentativi e Entro (minuti) per bloccare un indirizzo IP dopo un determinato numero di accessi non riusciti entro il numero di minuti indicato.
  3. Specificare Giorni bloccati per sbloccare un indirizzo IP dopo un determinato numero di giorni. Se viene inserito "0", l'indirizzo IP bloccato resterà indefinitivamente bloccato fino a quando non viene rimosso dall'elenco bloccati.
  4. Per gestire l'elenco bloccati o rimuovere un indirizzo IP bloccato dall'elenco, toccare Elenco blocchi.
  5. Toccare Fine.

Gestire Centro pacchetti

Synology ha integrato applicazioni progettate da Synology o da terzi in pacchetti che possono essere installati e abilitati su DiskStation.

Per gestire Centro pacchetti:

Accedere all'edizione desktop del DSM con il browser web del computer e selezionare Menu principale > Centro pacchetti per modificare i pacchetti. Per istruzioni dettagliate, consultare la Guida.

Per avviare o arrestare i pacchetti installati:

  1. Portare il cursore su on o off per avviare o arrestare i pacchetti.
  2. Toccare Fine.

Visualizzazione informazioni di sistema

Toccare Informazioni di sistema sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per ottenere lo stato della DiskStation.

Per visualizzare le informazioni di sistema della DiskStation:

  1. Eseguire una delle seguenti azioni:
    • Toccare Generale per visualizzare le informazioni di base su DiskStation, incluso numero di serie, nome modello, quantità di memoria fisica, versione DSM, e informazioni sull'ora e lo stato termico.
    • Toccare Rete per visualizzare la configurazione di rete di DiskStation e lo stato di ogni interfaccia di rete.
    • Toccare Archiviazione per visualizzare l'utilizzo di volume della DiskStation.
  2. Toccare il pulsante Aggiorna nell'angolo a destra per mantenere il sistema aggiornato.

Controllo delle risorse di sistema

Toccare Controllo risorse sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per controllare utilizzo della memoria e della CPU di DiskStation.

Visualizzazione registri o connessioni concorrenti

Toccare Registro sulla home page (telefoni iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android) per monitorare le connessioni concorrenti o visualizzare i registri dei servizi DiskStation.

Per monitorare le connessioni concorrenti:

Toccare Connessioni concorrenti nella sezione Controlla per visualizzare un elenco di utenti che al momento stanno accedendo ai servizi DiskStation.

Per vedere le registrazioni di log di un servizio specifico:

Toccare un qualsiasi elemento nella sezione Registro eventi o Registro trasferimento file per visualizzare un elenco di registri.

Avviare applicazioni mobile

DSM mobile fornisce un luogo comodo dove avviare le applicazioni mobile Synology installate sul dispositivo iOS o Android.

Per avviare le apps mobile Synology con DSM mobile:

Toccare una delle seguenti applicazioni nella sezione in basso della home page (telefono iPhone/iPod touch o Android) o il pannello a sinistra (tablet iPad o Android):

  • DS photo+ (disponibile quando il servizio Photo Station è abilitato)
  • DS audio (disponibile quando il servizio Audio Station è abilitato)
  • DS cam (disponibile quando il servizio Surveillance Station è abilitato)
  • DS file (disponibile quando il servizio WebDAV è abilitato)

Nota: per informazioni su queste applicazioni mobile, consultare i file della Guida allegati alle applicazioni.

Azionare DiskStation
(Riavvio, Spegnimento o Avviso acustico)

DSM mobile consente di riavviare o spegnere la DiskStation con il dispositivo iOS o Android. Inoltre, quando sono presenti numerose DiskStation la propria è facilmente identificabile richiedendo che emetta un avviso acustico.

Per azionare la DiskStation:

  1. Toccare il pulsante Azione in alto a destra sulla home page (telefono iPhone/iPod touch o Android) o nel pannello a sinistra (tablet iPad o Android).
  2. Eseguire una delle seguenti azioni:
    • Toccare Riavvia e Sì per riavviare la DiskStation.
    • Toccare Spegnimento e Sì per spegnere la DiskStation.
    • Toccare Trovami per richiedere l'emissione di avvisi sonori da DiskStation. Toccare OK per interrompere l'avviso acustico.

Passare all'edizione desktop del DSM

Toccare il pulsante Azione in alto a destra del pannello sinistro e poi Edizione desktop per passare all'edizione desktop del DSM.
