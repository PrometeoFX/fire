응용 프로그램 포털

응용 프로그램 포털에서 다양한 응용 프로그램의 연결 설정을 구성하여 독립 브라우저 탭이나 창에서 이들 응용 프로그램을 액세스하고 실행할 수 있습니다.

예를 들어, http://:diskstation:5000/file/을 웹 브라우저에 입력하여 File Station을 열 수 있습니다. 사용자 지정 별칭, HTTP 및 HTTPS 포트를 포함한 연결 설정을 사용할 수 있습니다. 이와 같은 설정을 찾으려면 제어판 > 응용 프로그램 포털로 이동합니다.

참고:

응용 프로그램 포털에서 Audio Station, Download Station, Surveillance Station, Video Station, File Station 등과 같은 다양한 응용 프로그램의 액세스 설정을 편집할 수 있습니다.

사용자 지정 별칭

사용자 지정 별칭으로, 웹 브라우저에 http://:diskstation:5000/file/과 같은 주소를 입력하여 File Station 등의 응용 프로그램을 열 수 있습니다.

사용자 지정 별칭 활성화하기:

  1. 응용 프로그램을 선택합니다.
  2. 편집을 클릭합니다.
  3. 나타나는 창에서 사용자 지정 별칭 활성화를 선택합니다.
  4. 사용자 지정 별칭을 지정합니다.

참고:

  • 리디렉션 원본 선택 상자를 선택하고, 짧고 기억하기 쉬운 별칭을 사용하여 선택한 응용 프로그램을 액세스합니다.
  • 다음은 응용 프로그램 즉, Audio Station, Download Station, Surveillance Station, Video Station, File Station 각각의 기본 별칭입니다: audio, download, cam, video, file.
  • 별칭 이름은 시스템 또는 브라우저용으로 예약되어 있거나 다른 응용 프로그램에서 사용하고 있는 것과 동일한 이름이 될 수 없으며, 문자 수는 2 에서 20 자 이내여야 합니다.
  • 별칭 이름에는 영문 대소문자 혼합, 숫자 및 두 개의 특수 문자인 (-)와 (_)를 포함할 수 있지만, 이 두 문자는 첫 번째와 마지막 문자를 제외한 위치에만 사용할 수 있습니다.

사용자 지정 HTTP 또는 HTTPS 포트

사용자 지정 HTTP 또는 HTTPS 포트로, http://diskstation:7000 또는 http://diskstation:7001과 같은 주소를 입력하여 응용 프로그램을 열 수 있습니다.

사용자 지정 HTTP 또는 HTTPS 포트 활성화하기:

  1. 응용 프로그램을 선택합니다.
  2. 편집을 클릭합니다.
  3. 나타나는 창에서 사용자 지정 포트(HTTP) 활성화 또는 사용자 지정 포트(HTTPS) 활성화를 선택합니다.
  4. 사용자 지정 포트 번호를 지정합니다.

참고:

  1. 포트 번호의 범위는 1 - 65535가 될 수 있지만, 다음 포트는 시스템에서 사용하기 위해 예약된 것이므로 제외시켜야 합니다: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule 기본 포트: 4662 (TCP), 4672(UDP), 및 FTP 기본 포트 범위: 실제 범위는 모델에 따라 다를 수 있습니다.
