파일 색인 지정

공유 폴더의 내용에 색인을 지정하면 파일 검색 중 파일 또는 폴더를 보다 신속히 찾을 수 있습니다.

공유 폴더의 파일 색인 지정하기:

  1. 색인을 지정할 공유 폴더를 선택합니다.
  2. 편집을 클릭한 다음, 파일 색인 지정 탭을 클릭합니다.
  3. 파일 색인 지정 활성화를 선택합니다.
  4. 확인을 클릭합니다.

파일 색인 재설정

일반적인 상황에서는 파일 색인 재설정이 필요하지 않습니다. 그러나 폴너 내의 파일이 검색 기준에 일치하나 어떤 이유로 파일 검색 중 누락된 것으로 나타나는 경우, 색인 재설정을 수행하여 파일의 색인 구조를 다시 구성해야 할 수도 있습니다.

공유 폴더의 파일 색인 재설정하기:

  1. 색인을 재설정할 공유 폴더를 선택합니다.
  2. 편집을 클릭한 다음, 파일 색인 지정 탭을 클릭합니다.
  3. 색인 재설정을 클릭합니다.
  4. 작업이 완료되면 시스템에 색인이 지정된 파일과 하위 폴더 수가 표시됩니다.

참고: 원격 폴더 또는 외장 USB 드라이브의 파일 색인 지정은 File Station에서 지원하지 않습니다.
