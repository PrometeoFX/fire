DSM 설정

제어판 > 네트워크 > DSM 설정에서 클라이언트가 DSM에 액세스하는 데 사용하는 포트 번호를 변경할 수 있습니다. HTTPS를 활성화하면 SSL/TLS를 사용하여 연결이 암호화됩니다.

기본 포트 번호 변경하기:

  1. HTTP 또는 HTTPS 필드에 포트 번호를 입력합니다.
  2. 적용을 클릭합니다.

HTTPS 연결 활성화하기:

  1. HTTPS 연결 활성화를 선택합니다.
  2. HTTP 포트(5000)를 통해 DSM에 액세스할 때 클라이언트를 HTTPS 포트(5001)로 리디렉션하려면 HTTP 연결에서 HTTPS로 자동으로 리디렉션 상자를 선택합니다.
  3. 적용을 클릭합니다.

HTTPS 연결에 사용되는 포트:

  • DSM: 기본 포트 번호가 5001이므로 다음 주소를 입력하여 안전하게 DSM에 액세스할 수 있습니다: https://server-hostname:5001/
  • Web Station 및 Photo Station: 기본 포트 번호가 443이므로 다음 주소를 입력하여 안전하게 Web Station 및 Photo Station에 액세스할 수 있습니다: https://server-hostname/

DiskStation이(가) Windows 네트워크 검색 기능으로 검색되도록 하기:

  1. Windows 네트워크 검색 활성화를 선택합니다.
  2. 적용을 클릭합니다.
