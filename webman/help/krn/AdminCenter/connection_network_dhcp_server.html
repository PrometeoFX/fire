DHCP Server

DiskStation은(는) DHCP 서버 역활을 가정하고 동적 IP 주소를 로컬 영역 네트워크 내의 DHCP 클라이언트에 할당합니다.

DHCP 서버 활성화하기:

  1. 제어판 > 네트워크 > 네트워크 인터페이스로 이동합니다.
  2. 네트워크 인터페이스를 선택하고(예: LAN) 편집을 클릭합니다.
  3. DHCP Server 탭으로 이동한 다음 DHCP Server 활성화를 선택합니다.
  4. 주소 임대 시간 필드에 값을 입력하여 DHCP 클라이언트에 할당된 IP 주소에 대한 DHCP 임대 시간(분)을 설정합니다.
  5. 기본 DNS(필수) 및 보조 DNS(옵션) 필드에 DHCP 클라이언트의 DNS 주소를 입력합니다.
  6. 도메인 이름(옵션) 필드에 DHCP 서버의 도메인 이름을 설정합니다.
  7. 확인을 클릭합니다.

서브넷 추가하기:

  1. 추가를 클릭한 후 다음 열 아래에서 정보를 지정합니다:
    • 활성화: 서브넷을 활성화 또는 비활성화합니다.
    • 시작: 서브넷의 첫 번째 IP 주소입니다.
    • 종료: 서브넷의 마지막 IP 주소입니다.
    • 넷마스크: 서브넷 마스크.
    • 게이트웨이: 서브넷의 게이트웨이.
  2. 확인을 클릭합니다.

참고:

  • 최대 32개의 서브넷을 추가할 수 있습니다.

DHCP 클라이언트

DHCP 클라이언트 및 해당 네트워크 구성 목록을 보려면(예: MAC 주소, IP 주소, 호스트 이름 및 DHCP 임대 만료까지 남은 시간) DHCP 클라이언트 탭으로 이동합니다.

DHCP 예약

클라이언트가 DHCP 임대 갱신 동안 항상 동일한 IP 주소를 받도록 하려면 클라이언트를 주소 예약 목록에 추가합니다.

클라이언트를 주소 예약 목록에 추가하기:

  • 추가를 클릭합니다.
  • 클라이언트의 MAC 주소, 클라이언트에 할당할 IP 주소 및 클라이언트의 호스트 이름을 지정합니다.
  • 확인을 클릭하여 설정을 저장합니다.
