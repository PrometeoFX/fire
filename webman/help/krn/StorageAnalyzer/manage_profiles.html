보고서 프로파일 관리

볼륨 사용에 대한 보고서를 생성하도록 예약된 작업을 생성 및 수정할 수 있습니다(저장소 분석기 > 보고서 프로파일).

보고서 작업 생성하기:

  1. 생성을 클릭해서 마법사를 엽니다.
  2. 새 보고서 작업의 이름을 지정하고 이 보고서를 보낼 이메일 주소를 입력합니다.
  3. 생성할 보고서 유형을 하나 또는 여러 개 선택합니다.
  4. 보고서를 생성할 시간을 지정하려면 다음을 수행하십시오:
    • 시스템이 볼륨 사용을 정기적으로 분석하도록 하려면 일정에 따라 보고서 생성을 선택하고 시간을 지정합니다.
    • 지금 수동으로 보고서 작업을 시작하려면 지금 보고서 생성을 선택합니다.
  5. 필요하면, 보고서 범위를 좁히기 위해 특정 사용자 또는 공유 폴더가 포함되도록 지정합니다. (암호화된 공유 폴더에 대한 보고서를 지정 및 생성하려면 암호화된 공유를 마운트한 후 보고서 작업을 수행하십시오.)
  6. 적용을 클릭하여 설정을 저장합니다.

참고:

  • 여러 개의 이메일 주소를 구분하기 위해 각 이메일 주소 다음에 Enter/Return을 누릅니다.
  • 파일 그룹별로 파일을 나열할 사용자를 지정할 때 각 사용자 다음에 Enter/Return을 누릅니다.
  • 파일 그룹별 파일 보고서는 다음 파일 유형을 열거합니다:
    오디오, 디스크 이미지 파일, 문서, 실행 파일, 그림, 비디오, 웹 페이지, 압축된 파일 등
  • 보고서 목록의 큰 파일 섹션, 사용 보고서 페이지의 가장 큰 500개 파일 또는 전체 보고서 페이지의 가장 큰 200개 파일.

보고서 작업 편집하기:

  1. 목록에서 보고서 작업을 선택하고 편집을 클릭합니다.
  2. 개별 탭을 클릭하여 이 작업의 설정을 수정합니다.

보고서 작업 삭제하기:

목록에서 보고서 작업을 선택하고 삭제를 클릭합니다.

기록 보고서 보기:

  1. 목록에서 보고서 작업을 선택하고 보고서 > 기록 보고서 보기를 클릭합니다.
  2. 나타나는 창에서 항목을 클릭하여 지정된 시간에 생성된 보고서를 봅니다.

보고서 즉시 생성하기:

목록에서 보고서 작업을 선택하고 보고서 > 지금 보고서 생성을 클릭하여 필요에 따라 보고서를 만듭니다.

보고서를 저장할 위치 변경하기:

  1. 설정을 클릭합니다.
  2. 새 대상 폴더를 선택하여 보고서를 저장할 위치를 변경합니다.

정기 일정에 따라 볼륨 사용 기록 수집하기:

기본적으로 볼륨 사용 데이터는 시스템이 시작될 때와 볼륨 사용 보고서가 생성될 때마다 수집됩니다. 이와 같은 경우 이외에 추가로 새 일정에서 사용 기록을 수집하도록 선택할 수 있습니다. 이렇게 하려면 설정을 클릭한 다음 일정에 따라 볼륨 사용 기록 수집을 선택하고 시간을 예약합니다.

참고:

여기에서 예약한 시간에 수집된 기록은 주로 특정 용도 및 볼륨 사용 섹션에서 사용 동향의 소스로 이용됩니다. 이 옵션을 활성화하면 볼륨 사용 동향과 시간에 따른 변경 내용을 표시하는 선 그래프가 나타납니다. 그렇지 않으면 현재 사용 백분률을 표시하는 막대 차트가 대신 나타납니다.

보고서 작업 분석하기:

원하는 보고서 작업을 더블 클릭하거나 원하는 보고서 작업 옆에 있는 이 보고서 작업 분석(> 아이콘)을 클릭하여 관련 분석 페이지를 엽니다. 자세한 정보는 사용 및 보고서 보기를 참조하십시오.
