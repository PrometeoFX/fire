복원

백업 및 복제 > 복원으로 이동하여 백업 데이터를 복원합니다.

백업 작업 복원하기:

  1. 백업 작업을 선택한 다음 복원을 클릭합니다. 작업이 복수 버전 백업인 경우, 복원 > 전체 작업을 선택합니다.
  2. 시스템 구성, 서로 다른 버전의 백업 데이터 등을 선택하라는 메시지가 표시될 수 있습니다. 이는 데이터 백업 작업, LUN 백업 작업, 단일 버전 백업 대상(예: 공유 폴더) 또는 복수 버전 백업 대상(예: 볼륨)인지 여부에 따라 다릅니다.
  3. 마법사를 따라 설치를 완료하십시오.
  1. 백업 작업을 선택한 다음 복원을 클릭합니다.
  2. 시스템 구성, 서로 다른 버전의 백업 데이터 등을 선택하라는 메시지가 표시될 수 있습니다. 이것은 백업 작업의 종류에 따라 다릅니다.
  3. 마법사를 따라 설치를 완료하십시오.

참고:

  • 특정 응용 프로그램에 대한 복원이 지원됩니다. 특정 패키지가 복원을 지원하는지 및 그 제한에 대해서 알아보려면 각 응용 프로그램의 도움말을 참조하십시오.
  • 패키지를 복원할 때, 해당 패키지가 설치되어 있지 않으면 시스템이 같은 버전이거나 최신 버전인 응용 프로그램을 자동으로 설치합니다. 응용 프로그램이 설치되어 있지만 백업된 버전보다 오래된 경우 시스템이 자동으로 원래 백업된 버전으로 업그레이드합니다.
  • DSM 5.1 이상에서 백업한 응용 프로그램은 DSM 5.0 이하 버전에서 복원되지 않습니다.

참고:

  • 복원이 완료되면 백업 LUN이 선택한 대상 볼륨의 파일 기반 LUN에 복원됩니다.*
  • 특정 응용 프로그램에 대한 복원이 지원됩니다. 특정 패키지가 복원을 지원하는지 및 그 제한에 대해서 알아보려면 각 응용 프로그램의 도움말을 참조하십시오.
  • 패키지를 복원할 때, 해당 패키지가 설치되어 있지 않으면 시스템이 같은 버전이거나 최신 버전인 응용 프로그램을 자동으로 설치합니다. 응용 프로그램이 설치되어 있지만 백업된 버전보다 오래된 경우 시스템이 자동으로 원래 백업된 버전으로 업그레이드합니다.
  • DSM 5.1 이상에서 백업한 응용 프로그램은 DSM 5.0 이하 버전에서 복원되지 않습니다.

복수 버전 백업 작업에 대해 단일 파일 또는 폴더 복원하기:

  1. 파일 버전 브라우저를 실행합니다. 다음 방법 중 하나를 사용하여 이를 수행할 수 있습니다:
    • 복원을 클릭하고 드롭다운 메뉴에서 개별 파일/폴더를 선택합니다.
    • 백업 대상으로 이동하여 선택된 백업 대상을 확장합니다. 백업 버전 정보 아이콘을 클릭합니다. 나타나는 창에서, 복원하려는 백업 버전을 선택하고 파일 버전 브라우저 아이콘을 클릭합니다.
  2. 화면의 아래쪽에 있는 타임라인 또는 캘린더 아이콘을 사용하여 원하는 파일 또는 폴더의 버전을 탐색할 수 있습니다.
  3. 관리하려는 버전을 선택하고 복원을 클릭합니다.
  4. 또는, 복사 대상...이나 다운로드 단추를 사용하여 파일 또는 폴더를 다른 위치로 복사하거나 다운로드할 수 있습니다.
  5. 또한 파일이나 폴더를 마우스 오른쪽 단추로 클릭하고 드롭다운 메뉴에서 이러한 옵션을 선택할 수도 있습니다.

특정 버전 검색하기:

  1. 오른쪽 위 구석에 있는 필터 표시줄에서 화살표를 클릭합니다.
  2. 드롭다운 메뉴에서 다음을 기준으로 파일을 필터링할 수 있습니다:
    • 키워드: 파일 또는 폴더 이름을 입력합니다. 키워드에 와일드카드[*](1개 또는 그 이상의 문자를 나타냄) 또는 [?](1개의 임의 문자를 나타냄)를 사용할 수 있습니다. 예를 들어, [a*n]은 [action], [attention] 또는 [an]을 나타낼 수 있습니다.
    • 파일 유형: 드롭다운 메뉴에서 임의, 임의 폴더 또는 임의 파일을 선택하여 검색 대상을 좁힐 수 있습니다.
    • 크기: 임의를 선택하거나, 같음, 이상, 미만을 선택한 다음 값을 입력합니다.
    • 날짜 범위: 오늘, 어제, 지난 주, 지난 달 또는 사용자 지정 중에서 선택하여 시작 및 종료 날짜를 입력하고 특정 범위 안에서 버전을 찾습니다.
  3. 필터을 클릭하여 필터를 적용합니다.
  4. 필터링된 검색을 재설정하려면, 재설정 단추를 클릭합니다.

참고:

  • 복수 버전 백업 작업만이 파일 버전 브라우저 기능을 제공합니다.
  • DSM 5.1 이하에서는 파일 버전 브라우저가 일부 기능만 가집니다.
  • 시스템 구성 및 응용 프로그램은 복원 > 전체 작업을 통해 복원할 수 있습니다.
  • 메타데이터는 복사 대상... 및 다운로드 동작에 포함되지 않습니다.

다른 백업 대상에서 백업 작업 복원하기:

  1. 다음에서 복원...을 클릭합니다.
  2. 다음 중에서 선택하여 백업 작업을 검색하고 다음을 클릭합니다:
    • 로컬 복원: 로컬 공유 폴더 또는 외부 디스크에서 데이터를 복원하려면 이 옵션을 선택합니다.
    • 네트워크 복원: 다른 DiskStation 또는 rsync 호환 서버에서 데이터를 복원하려면 이 옵션을 선택합니다.
    • 공용 클라우드 서비스 복원: 공용 클라우드 서버에서 데이터를 복원하려면 이 옵션을 선택합니다.
  3. 메시지가 표시되면 백업 작업이 있는 소스를 선택한 다음 데이터를 복원할 대상을 선택합니다.
  4. 마법사를 따라 완료합니다.

참고:

  • Embedded DataStation은 공유 폴더를 ext3/ext4 외부 장치에만 복원할 수 있습니다.
  • Embedded DataStation은 ext3/ext4 외부 장치가 없으면 응용 프로그램 복원을 지원하지 않습니다.
