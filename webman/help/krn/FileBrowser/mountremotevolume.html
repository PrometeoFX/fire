원격 폴더 마운트

원격 서버(예: 네트워크 컴퓨터나 다른 DiskStation)에서 원격 폴더를 마운트하여 File Station으로 사용 가능한 네트워크 리소스를 쉽게 탐색할 수 있습니다.

원격 폴더를 마운트하려면 다음과 같이 하십시오:

  1. 도구 > 원격 폴더 마운트를 클릭합니다.
  2. 폴더 필드에 "\\Remote_Server\Remote_Folder"를 입력하여 원격 폴더의 경로를 지정하십시오. 예를 들면 "\\192.168.0.2\share" 또는 "\\chad\cheeseburger\pictures"와 같습니다.
  3. 원격 서버에서 원격 폴더에 대한 액세스 권한을 필요로 하면 계정 이름 과 패스워드 필드에 원격 서버의 사용자 자격증명을 입력하십시오.
  4. 원격 폴더를 마운트하기 위해 DiskStation에서 빈 대상 폴더를 선택하거나 만들려면 찾아보기를 클릭하십시오.
  5. 시스템을 시작하거나 재부팅할 때마다 DiskStation에서 이 원격 폴더를 마운트하도록 하려면 시작 시 자동 마운트를 선택하십시오.
  6. 마운트를 클릭하십시오. 이제 원격 폴더가 마운트된 대상 폴더를 탐색할 수 있고 해당 원격 폴더의 내용에 액세스할 수 있습니다.

원격 폴더의 마운트를 해제하려면 다음과 같이 하십시오:

  1. 마운트 해제할 원격 폴더를 선택한 다음 도구 > 마운트 해제를 클릭합니다.
  2. 또는, 단순히 원격 폴더를 마우스 오른쪽 단추로 클릭하고 마운트 해제를 선택합니다.

참고:

  • 동시에 사용 가능한 원격 폴더의 최대 수는 100입니다.
  • CIFS 프로토콜을 지원하는 원격 서버의 원격 폴더만 마운트할 수 있습니다.
