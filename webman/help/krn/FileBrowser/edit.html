사진 편집

File Station에서 Pixlr Editor나 Pixlr Express 같은 웹 기반 외부 사진 편집기를 사용하여 사진을 편집할 수 있습니다.

웹 기반 사진 편집기로 사진 편집하기:

  1. 사진이 위치한 폴더를 찾습니다.
  2. 사진을 마우스 오른쪽 단추로 클릭하고 Pixlr Editor에서 편집 또는 Pixlr Express에서 편집을 선택합니다.
      참고:
    • Pixlr Editor는 다음 형식을 지원합니다: jpg, gif, png, psd, pxd(Pixlr 레이어 이미지)
    • Pixlr Express는 다음 형식을 지원합니다: jpg, gif, png
  3. 선택한 사진 편집기가 새 브라우저 창에서 열립니다. 현재 사진을 확인한 다음, 제공된 도구를 사용하여 편집할 수 있습니다.
  4. 사진 편집이 끝나면 다음을 수행하여 사진을 저장하십시오.
    • Pixlr Express: 사진을 DiskStation의 원래 위치에 원래 형식으로 저장하려면 완료를 클릭합니다.
    • Pixlr Editor의 경우: 파일 > 저장을 선택하고 왼쪽 패널에서 사진을 저장할 위치(DiskStation에 있는 사진의 원래 위치, 컴퓨터 또는 친목도모 네트워크가 될 수 있음)를 선택한 다음, 원하는 대로 새 이름이나 형식을 결정하고 확인을 클릭합니다.

참고: 웹 기반 사진 편집기를 사용하여 사진을 편집하려면 DiskStation이 외부 IP 주소를 성공적으로 검색했는지 확인하십시오(네트워크 구성에 따라 메인 메뉴 > 제어판 > 외부 액세스 > DDNS 또는 메인 메뉴 > 제어판 > 네트워크 > 네트워크 인터페이스 > PPPoE). DiskStation이(가) 라우터 안에 있을 경우 메인 메뉴 > 제어판 > 외부 액세스 > 라우터 구성에서 DSM에 대한 포트 전달 규칙을 지정했는지 확인하십시오. 그래도 사진 편집기를 여는 데 문제가 있을 경우, 메인 메뉴 > 제어판 > 외부 액세스 > 고급에서 DSM의 라우터 포트를 지정해야 할 수도 있습니다.
