텍스트 편집기

텍스트 편집기는 매우 유용한 텍스트 및 코드 편집기로, 웹 브라우저에서 직접 DiskStation에 있는 텍스트 파일 또는 코드에 액세스하고 신속하게 편집할 수 있습니다.

파일 관리

새 파일 생성하기:

파일을 클릭하고 새로 만들기를 선택합니다. 새 문서.txt라는 제목의 새로운 빈 탭이 표시됩니다.

텍스트 편집기에서 파일 열기:

파일을 클릭하고 열기를 선택합니다. 대화 상자가 표시되고, 특정 문자 인코딩을 가진 파일을 선택하여 탭에서 열 수 있습니다. 파일이 이미 열려 있을 경우 텍스트 편집기는 해당 파일을 현재 상태로 보여줍니다.

File Station에서 파일 열기:

File Station에서 선택한 파일을 마우스 오른쪽 단추로 클릭한 다음 컨텍스트 메뉴에서 텍스트 편집기로 열기를 클릭합니다. 파일이 이미 열려 있을 경우 텍스트 편집기는 해당 파일을 현재 상태로 보여줍니다. 커서를 사용하여 하나 또는 여러 개의 파일을 File Station에서 텍스트 편집기로 끌어서 파일을 여는 방법도 있습니다.


참고:

텍스트 편집기는 10 MB를 초과하는 파일은 열 수 없습니다.

파일 저장하기:

파일을 클릭하고 저장을 선택합니다. 기존에 있던 파일인 경우에는 현재 내용으로 파일을 덮어씁니다. 열려 있는 파일을 모두 한꺼번에 저장하려면 파일 > 모두 저장으로 이동합니다. 아직 파일 이름이 없거나(새 파일) 파일 > 새 이름으로 저장을 선택한 경우, 파일 이름을 입력하라는 대화 상자가 팝업됩니다. 그러면 위치, 파일 이름 및 파일에 저장할 인코딩을 선택할 수 있습니다.

파일 닫기:

파일을 클릭하고 닫기를 선택합니다. 각 탭의 닫기 단추를 클릭하여 파일을 닫을 수도 있습니다. 파일에 저장하지 않은 데이터가 있으면 텍스트 편집기에서 해당 변경 내용을 저장하지 않고 닫을 것인지 묻는 메시지를 항상 표시합니다.

참고:

양방향 언어를 지원하는 특정 문자 인코딩으로 파일을 저장할 수 있습니다. 또한 텍스트 편집기가 파일을 제대로 표시할 수 있도록 파일을 열 때 인코딩을 지정할 수 있습니다.

편집

  • 표준 작업
    편집 컨트롤은 간단하게 실행 취소, 다시 실행, 모두 선택을 지원합니다. 이러한 작업은 대부분의 편집기에서 볼 수 있으며 작동하는 방식도 매우 유사합니다. 편집 메뉴에서 해당 메뉴 항목을 선택하여 이러한 작업을 수행할 수 있습니다. 또한 기본적으로 잘라내기, 복사 및 붙여넣기 옵션을 브라우저의 컨텍스트 메뉴를 통해 사용할 수 있습니다.
  • 들여쓰기
    텍스트 들여쓰기는 소스 코드를 편집할 때의 일반적인 작업입니다. 들여쓰기는 텍스트 블록을 구분하기 위해 행의 시작 부분에 여분의 공백을 추가하여 계층 구조로 만듭니다. 정상적으로는 편집할 때 Tab 키를 누르면 현재 위치에 탭 문자가 삽입됩니다. 그러나, 텍스트를 선택한 상태에서 Tab 키를 누르면 선택한 텍스트가 있는 각 행의 시작 부분에 공백이 추가되어 전체 텍스트 블록을 한꺼번에 들여쓸 수 있습니다. Shift와 Tab을 같이 누르면 들여쓰기가 제거됩니다.
  • 겹쳐쓰기
    Insert 키를 눌러 겹쳐쓰기를 활성화 또는 비활성화할 수 있습니다. 즉, 커서 위치에 문자를 삽입하지 않고 문자를 교체합니다. 표시된 커서는 그에 맞추어 겹쳐쓰기 모양으로 바뀝니다.
  • 열 모드 편집
    파일을 편집하는 특별한 방식으로 열 모드 편집이 있습니다. 이것은 텍스트를 행이 아니라 열에서 선택할 수 있음을 의미합니다. 열 모드 선택을 시작하려면 Alt 키를 누른 상태로 텍스트를 선택하고, 필요한 만큼 커서를 아래 또는 위로 이동합니다. 이러한 열 모드에서 텍스트가 선택되어 있으면, 입력할 때 각 선택된 문자의 열이 입력된 문자로 변경됩니다.

검색 중

  • 일반 검색
    대부분의 프로그램에서처럼 찾기 대화 상자를 사용하여 텍스트를 검색할 수 있습니다. 검색 > 찾기를 선택하여 이 대화 상자를 열 수 있습니다. 이 대화 상자는 항상 텍스트 위에 있으며, 이 대화 상자가 열려 있으면 텍스트를 편집할 수 없습니다. 검색 필드에 찾을 텍스트를 입력합니다.
  • 바꾸기
    대부분의 프로그램에서처럼 바꾸기 대화 상자를 사용하여 텍스트를 찾아서 바꿀 수 있습니다. 검색 > 바꾸기를 선택하여 이 대화 상자를 열 수 있습니다. 모두 바꾸기는 쿼리의 모든 어커런스를 교체 텍스트로 바꿉니다.
  • 대소문자 구분
    이것은 쿼리와 정확하게 일치하는 경우만 검색하게 합니다. 이를 선택하지 않으면 대소문자를 구분하지 않고 검색합니다.
  • 단어 단위로만
    이것은 단어의 일부가 아니라 전체 단어와 일치하는 경우만 검색하게 합니다. 예를 들어 "exam"을 단어 단위로만 검색하면, "example"이 아니라 "exam"만 검색합니다.
  • 정규식
    이 쿼리는 정규식처럼 처리됩니다.

정규식이란?

정규식은 검색 패턴을 형성하는 일련의 문자들로서 시간을 상당히 절약할 수 있는 기술입니다. 매우 간단한 정규식 사용의 예는 텍스트 편집기에서 M[ea]rry와 같이 두 가지 서로 다른 방식으로 철자가 쓰여진 동일한 단어를 찾는 것입니다.

스타일 구성자

스타일 구성자는 주로 텍스트 편집기에서 텍스트의 모양을 제어하는 데 사용됩니다.
오른쪽 아래 드롭다운 목록에는 적절한 코딩 환경을 구축하는 데 도움이 되도록 스타일 지정이 가능한 지원되는 모든 프로그램밍 언어가 들어 있습니다.

기본 설정

  • 보이지 않는 문자 표시
    이를 선택하면 공백, 탭, 줄 바꿈 또는 리턴 문자가 표시됩니다.
  • 들여쓰기 가이드 표시
    이를 선택하면 들여쓰기 가이드가 표시됩니다. 이는 코딩 언어에서 코드를 정렬하는 데 유용합니다.
  • 현재 행 하이라이트
    이를 선택하면, 커서가 있는 행이 특별한 배경 색상으로 표시됩니다.
  • 선택한 단어 하이라이트
    이를 선택하면, 동일한 텍스트 파일 안에서 선택한 단어와 일치하는 모든 단어(일부분 아님)가 자동으로 강조 표시됩니다.
  • 행 번호 표시
    이를 선택하면 행 번호가 표시됩니다.
  • 코드 폴딩 활성화
    이를 선택하면, 특정 특징을 기준으로 코드의 일부 행들을 숨길 수 있습니다. 이 기능은 C++ 또는 XML 같은 구조화된 언어에서 잘 작동합니다. 표시 위젯을 자동으로 숨김을 선택하면 코드의 특정 행이 자동으로 숨겨집니다. 또한 위젯과 함께 여는 대괄호만 표시하거나 여는 대괄호와 닫는 대괄호를 모두 표시하도록 선택할 수 있습니다.
  • 단어 줄 바꿈
    하드 리턴을 삽입하지 않아도 창 가장자리 또는 특정 열 번호에서 텍스트가 줄 바꿈되도록 할지 여부를 지정합니다.
  • 닫는 문자를 자동으로 첨부
    이를 선택하면, 코드에서 닫는 대괄호(문자)가 자동으로 추가됩니다.
  • 탭 대신 공백 삽입
    이를 선택하면, Tab 키를 누를 때 탭 문자 대신 공백 문자를 삽입합니다. 탭 너비는 기본 설정 > 글꼴 및 색상에서 문자 수로 정의될 수 있습니다.
  • 기본 인코딩
    기본 인코딩은 양방향 언어를 지원하는 특정 문자 인코딩을 사용하여 파일을 저장하는 데 사용됩니다. 또한 텍스트 편집기가 파일을 제대로 표시할 수 있도록 파일을 열 때 인코딩을 지정할 수 있습니다.
  • 글꼴 및 색상
    탭 너비, 글꼴 크기 및 테마는 개인적인 기호에 따라 사용자 지정될 수 있습니다. 탭 너비는 공백 문자 수로 정의됩니다.

지원되는 파일 형식

텍스트 편집기는 일반 텍스트 파일만 지원합니다. *.doc 및 *.pdf 같은 리치 포맷팅 텍스트 파일은 지원되지 않습니다.
