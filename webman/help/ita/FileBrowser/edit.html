Modifica delle foto

Con File Station, è possibile modificare le foto utilizzando editor basati sul web come Pixlr Editor o Pixlr Express.

Per modificare una foto con un editor di foto basato sul Web:

  1. Accedere alla cartella dove è posizionata la foto.
  2. Fare clic con il tasto destro e scegliere Modifica in Pixlr Editor o Modifica in Pixlr Express.
      Nota.
    • Pixlr Editor supporta i seguenti formati: jpg, gif, png, psd, pxd (LIFF Pixlr)
    • Pixlr Express supporta i seguenti formati: jpg, gif, png
  3. L'editor di foto selezionato verrà aperto in una nuova finestra del browser. È possibile vedere le foto in un momento e poi modificarle con gli strumenti forniti.
  4. Al termine delle modifiche, procedere come segue per salvare la foto.
    • Per Pixlr Express: Fare clic su Fine per salvare la foto nel formato e nella posizione originale su DiskStation.
    • Per Pixlr Editor: Scegliere File > Salva, scegliere la posizione (che potrebbe essere la posizione originale della foto su DiskStation, il computer o altri social network) di salvataggio nel pannello a sinistra, decidere un nuovo nome o formato secondo le preferenze e fare clic su OK.

Nota. Per modificare le foto con uno degli editor su base web, assicurarsi che la DiskStation abbia recuperato un indirizzo IP esterno (in Menu principale > Pannello di controllo > Accesso esterno > DDNS o Menu principale > Pannello di controllo > Rete > Interfaccia di rete > PPPoE, in base alla propria configurazione di rete). Se DiskStation è posizionato dietro un router, accertarsi di aver specificato la regola di invio porta per il DSM in Menu principale > Pannello di controllo > Accesso esterno > Configurazione router. Se ancora si hanno problemi nell'aprire gli editor di foto, potrebbe essere necessario specificare la porta del router per il DSM in Menu principale > Pannello di controllo > Accesso esterno > Avanzate.
