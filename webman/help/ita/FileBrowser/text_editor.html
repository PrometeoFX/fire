Text Editor

Text Editor è un editor di codice e testo altamente versatile che consente di accedere e modificare rapidamente codici e file di testo su DiskStation proprio nel browser Web.

Gestione dei file

Per creare nuovi file:

Fare clic su File e scegliere Nuovo. Appare una nuova scheda vuota con il titolo Nuovo Documento.txt.

Per aprire file da Text Editor:

Basta fare clic su File e scegliere Apri. Viene visualizzata una finestra di dialogo ed è possibile selezionare un file con codifica dei caratteri specifica che deve essere aperta in una scheda. Se il file è già stato aperto, Text Editor mostrerà quel file nel suo stato corrente.

Per aprire file da File Station:

In File Station, basta fare clic con il tasto destro sul file selezionato e quindi fare clic su Apri con Text Editor nel menu contestuale. Se il file è già stato aperto, Text Editor mostrerà quel file nel suo stato corrente. Un altro modo per aprire un file è trascinare uno o più file da File Station in Text Editor usando il cursore.


Nota.

Con Text Editor non è possibile aprire un file di dimensioni maggiori a 10 MB.

Per salvare file:

Basta fare clic su File e scegliere Salva. Se è un file esistente, sarà sovrascritto con i contenuti correnti. Se si vogliono salvare tutti i file aperti in una volta, è possibile andare su File > Salva tutto. Se il file non ha ancora un nome (un nuovo file) o si è selezionato File > Salva come, viene visualizzata una finestra di dialogo, che chiede il nome del file. È possibile scegliere quindi la posizione, il nome del file e la codifica per salvare il file.

Per chiudere i file:

Basta fare clic su File e scegliere Chiudi. Un altro modo per chiudere un file è fare clic sul pulsante Chiudi su ogni scheda. Se nel file ci sono dati non salvati, Text Editor chiederà sempre se si desidera chiudere senza salvare queste modifiche.

Nota.

È possibile salvare i file con una codifica dei caratteri specifica per supportare lingue bidirezionali. È possibile anche specificare una codifica quando si apre un file, in modo che Text Editor visualizzi il file correttamente.

Modifica

  • Operazioni standard
    Il controllo modifica supporta Annulla, Ripeti e Seleziona tutto. Queste operazioni si possono trovare nella maggior parte degli editor e vengono eseguite in modo molto simile. È possibile eseguire queste operazioni selezionando la corrispondente voce del menu in Modifica. Inoltre, come impostazione predefinita le opzioni Taglia, Copia e Incolla sono disponibili nel menu contestuale del browser.
  • Capoverso rientrato
    Far rientrare il testo è un'operazione comune quando si modifica il codice sorgente. Il rientro è in aggiunta allo spazio bianco extra all'inizio di una riga per separare i blocchi di testo, aggiungendo una gerarchia. Normalmente quando si modifica, premendo il tasto Tab si inserirà un carattere di tabulazione nella posizione corrente. Tuttavia, se è selezionato del testo, premendo il tasto Tab si aggiungerà spazio bianco extra all'inizio di ciascun riga che ha del testo selezionato, consentendo di rientrare l'intero blocco di testo selezionato in un'unica volta. Premendo Shift e Tab insieme si rimuoverà il rientro.
  • Sovrascrivere
    È possibile attivare e disattivare la sovrascrittura premendo il tasto Ins. Quindi, invece di inserire i caratteri con il cursore, sostituirli. Il cursore visualizzato cambierà forma in base allo stato di sovrascrittura.
  • Modifica modalità colonna
    Un modo speciale per modificare i file è la modifica della modalità colonna. Ciò significa che il testo può essere selezionato nelle colonne, piuttosto che nelle righe. Per avviare una selezione della modalità colonna, premere il tasto Alt quando si seleziona il testo, spostando il cursore verso l'alto o il basso a seconda delle necessità. Quando il testo è selezionato in questa cosiddetta modalità colonna, digitando cambierà ogni colonna di caratteri selezionata nel carattere digitato.

Ricerca

  • Ricerca normale
    È possibile ricercare il testo come nella maggior parte dei programmi usando la finestra Trova. È possibile aprire questa finestra selezionando Cerca > Trova. Questa finestra si trova sempre sulla parte alta del testo e non può continuare la modifica del testo se è aperta. Nel campo di ricerca, digitare il testo che si desidera trovare.
  • Sostituzione
    È possibile trovare e sostituire il testo come nella maggior parte dei programmi usando la finestra Sostituisci. È possibile aprire questa finestra selezionando Cerca > Sostituisci. Sostituisci tutto sostituirà tutte le occorrenze della query con il testo di sostituzione.
  • Maiuscole/minuscole
    Per rispettare esattamente lo stesso utilizzo di maiuscole/minuscole della query. Se non ha il segno di spunta, la ricerca non fa distinzione tra lettere minuscole e maiuscole.
  • Solo parola intera
    Porterà a un risultato solo se la corrispondenza consiste di un'intera parola, non parte di essa. Ad esempio, se si cerca la parola "esame" con Solo parola intera, si troverà solo "esame", invece di "esempio".
  • Espressione regolare
    La query sarà gestita come un'espressione regolare.

Cos'è un'espressione regolare?

Un'espressione regolare è una sequenza di caratteri che forma un modello di ricerca per una tecnica con forte risparmio energetico. Un utilizzo molto semplice di un'espressione regolare individuerà la stessa parola pronunciata in due modi diversi in un editor di testo, ad esempio M[ea]rry.

Configuratore stile

Il configuratore stile è usato innanzitutto per controllare l'aspetto del testo in Text Editor.
L'elenco a discesa in basso a destra contiene tutte le lingue di programmazione supportate che possono essere modificare lo stile per aiutare a creare un ambiente di codifica idoneo.

Preferenze

  • Mostra caratteri invisibili
    Se con il segno di spunta spazi, tabulazioni, righe nuove o caratteri di ritorno saranno visualizzati.
  • Mostra guide capoverso rientrato
    Se con il segno di spunta, le guide del capoverso rientrato saranno visualizzate. È utile per allineare codici nella lingua di codifica.
  • Evidenzia linea attuale
    Se con il segno di spunta, la linea in cui è presente il segno di omissione sarà tracciata con un colore di sfondo speciale.
  • Evidenzia parola selezionata
    Se con il segno di spunta, tutte le parole (non parziali) in uno stesso file di testo che corrispondono alla parola selezionata saranno automaticamente selezionate.
  • Mostra numeri di linea
    Se con il segno di spunta, i numeri di linea saranno visualizzati.
  • Abilita codice piegatura
    Se con il segno di spunta, alcune linee del codice possono essere nascoste in base a determinati tratti. Questa funzione è adatta ai linguaggi strutturati come C++ o XML. Se Nascondi automaticamente widget aperti ha il segno di spunta, alcune linee dei codici saranno piegate automaticamente. È possibile anche di contrassegnare solo la parentesi di apertura o la parentesi di apertura e chiusura con widget aperti.
  • Word Wrap
    Specifica se il testo è sovrapposto al bordo della finestra o su un numero di colonna specifico senza inserire un rientro forzato.
  • Aggiungi chiusura caratteri automatica
    Se con il segno di spunta, le parentesi di chiusura (caratteri) saranno aggiunte automaticamente nei codici.
  • Inserisci spazi invece di tabulazioni
    Se con i segni di spunta, premendo il tasto Tab si inseriranno i caratteri di spazio invece di un carattere di tabulazione. Larghezza tabulazione può essere definita dal numero di caratteri in Preferenze > Font e colore.
  • Codifica predefinita
    La codifica predefinita è utilizzata per salvare i file con una codifica dei caratteri specifica per supportare lingue bidirezionali. È possibile anche specificare una codifica quando si apre un file, in modo che Text Editor visualizzi il file correttamente.
  • Font e Colore
    Larghezza tabulazione, Dimensioni font e Tema possono essere personalizzati in base alle proprie preferenze. Larghezza tabulazione è definita dal numero di caratteri di spazio.

Formati file supportati

Text Editor supporta solo file di testo semplici. I file di testo con formattazione ricca come *.doc e *.pdf non sono supportati.
