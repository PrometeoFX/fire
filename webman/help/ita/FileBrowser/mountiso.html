Montare unità virtuale

È possibile montare le unità virtuali per accedere ai contenuti delle immagini disco con il File Station.

Per montare una unità virtuale:

  1. Selezionare un'immagine del disco e fare clic su Strumenti > Monta unità virtuale.
  2. Fare clic su Sfoglia per selezionare o creare una cartella di destinazione vuota sulla DiskStation per il montaggio dell'unità virtuale.
  3. Selezionare Montaggio automatico all'avvio per consentire a DiskStation di montare l'unità virtuale tutte le volte che il sistema viene avviato o riavviato. Fare clic su Monta per continuare. A questo punto è possibile passare alla cartella di destinazione, dove è montata l'unità virtuale, e accedere ai contenuti dell'immagine disco.

Per smontare una unità virtuale:

  1. Fare clic con il tasto destro del mouse sulla cartella di destinazione in cui è montata l'unità virtuale, quindi selezionare Smonta.
    Nota:
  1. Formati immagine disco supportati: file ".iso" che utilizza il file system ISO 9660 e UDF.
  2. Il numero massimo di unità virtuali simultanee è 16.
