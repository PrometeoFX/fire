Montare le cartelle remote

È possibile montare le cartelle remote dai server remoti (ad esempio computer di rete o un altra DiskStation) per scorrere in modo semplice le risorse di rete disponibili utilizzando il File Station.

Per montare una cartella remota:

  1. Fare clic su Strumenti > Monta cartella remota.
  2. Inserire "\\Remote_Server\Remote_Folder" nel campo Cartella per specificare il percorso della cartella remota. Ad esempio, "\\192.168.0.2\share" o "\\chad\cheeseburger\pictures".
  3. Se il server remoto richiede i privilegi di accesso alla cartella remota, inserire le credenziali utente del server remoto nei campi Nome account e Password.
  4. Fare clic su Sfoglia per selezionare o creare una cartella di destinazione vuota sulla DiskStation per il montaggio della cartella remota.
  5. Selezionare Montaggio automatico all'avvio per consentire a DiskStation di montare la cartella remota tutte le volte che il sistema viene avviato o riavviato.
  6. Fare clic su Monta. A questo punto è possibile passare alla cartella di destinazione dove è montata la cartella remota, e gestire i contenuti delle cartelle remote.

Per smontare una cartella remota:

  1. Selezionare la cartella remota da smontare e fare clic su Strumenti > Smonta.
  2. In alternativa, è sufficiente fare clic con il tasto destro del mouse sulla cartella remota e selezionare Smonta.

Nota:

  • il numero massimo di cartelle remote simultanee è 100.
  • È consentito montare solo cartelle remote da un server remoto che supporta il protocollo CIFS.
