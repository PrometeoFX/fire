Come effettuare il backup dei dati sul PC/Mac utilizzando Cloud Station

Panoramica

Cloud Station non è solo un sofisticato software di sincronizzazione con piattaforma trasversale ma può essere utilizzato anche come strumento intuitivo per il backup dei personal computer. Ecco alcuni vantaggi dell'utilizzo di Cloud Station come strumento di backup. Innanzi tutto, la replica in tempo reale fornisce un miglior RPO, un minor periodo in cui i dati possono essere persi (non ripristinabile) dal PC a causa di problemi maggiori. Non solo è istantaneo, ma aiuta anche a risparmiare banda larga in modo considerevole rispetto al backup tradizionale che richiede una maggiore quantità di banda larga in un dato periodo. È possibile filtrare in modo semplice i file da escludere dal backup nella stessa cartella impostando la dimensione del file o il tipo di file sul client del PC. Cloud Station può anche conservare fino a 32 versioni della cronologia di un singolo file che con Cloud Station sarà possibile ripristinare in un PC client da parte del server (NAS).

1. Configurare Cloud Station su Synology NAS

  1. Accedere al DSM usando un account che appartiene a administrator.
  2. Avviare Cloud Station.
  3. Se la funzione home utente non è stata abilità, verrà richiesto di abilitarla. Fare clic su OK.
  4. L'utente sarà reindirizzato alla pagina delle impostazioni dell'utente. Abilitare homes utente e fare clic su Applica.
  5. A questo punto, tornare a Cloud Station e verrà richiesto di abilitarla. Fare clic su Sì.
    6. Nota: QuickConnect è un servizio di trasmissione che consente di collegarsi facilmente alla propria Synology NAS dall'esterno della propria rete locale. Per sincronizzare i propri file quando si è lontani da casa o ufficio, si consiglia di configurare QuickConnect andando su Pannello di controllo > QuickConnect.
  6. Andare su Privilegi e specificare gli utenti che potranno usare Cloud Station. Fare clic su Salva.
  7. Configurare una cartella condivisa su Synology NAS per il backup. A tal fine, andare su Pannello di controllo > Cartella condivisa e fare clic su Crea.
  8. Indicare un nome della cartella e inserire i dettagli in modo corretto. Fare clic su OK. Per esempio, nominare la cartella Backup.
  9. Nella scheda Permessi, specificare quali utenti potranno effettuare l'accesso alla cartella. Per esempio, dare i permessi di lettura/scrittura ad admin nella cartella. Fare clic su OK.
  10. Andare su Cloud Station > Impostazioni > Condivisione, selezionare la cartella Backup e fare clic su Abilita.

2. Configurare Cloud Station sul PC Windows

Seguire la procedura per impostare Cloud Station su Mac o PC

2.1 Per PC

  1. Andare su Cloud Station > Panoramica e fare clic sul pulsante Scarica.
  2. Fare clic sul pulsante per scaricare Cloud Station per il PC Windows (è anche possibile scaricare Cloud Station dal Centro download Synology).
  3. Avviare l'installer e seguire le istruzioni per installare Cloud Station.
  4. Dopo l'installazione, eseguire Cloud Station sul computer. Fare clic su Avvia adesso.
  5. Inserire l'indirizzo per il Synology NAS che esegue Cloud Station (o ID QuickConnect), nome utente e password. È possibile anche fare clic sull'icona di ricerca a destra per consentire a Cloud Station di ricercare automaticamente un altro Synology NAS nella LAN in uso. Fare clic su Avanti.
    6. Nota: Assicurarsi che l'account inserito disponga dei permessi in Cloud Station > Privilegi sulla Synology NAS.
  6. Selezione come procedere alla configurazione dell'attività (qui verrà selezionata la Configurazione avanzata per la configurazione di una cartella specificamente per il backup). Fare clic su Avanti.
  7. Selezionare una cartella da tenere sincronizzata con la cartella Backup creata in precedenza su Synology NAS. Fare clic su Avanti.
  8. Fare clic su Fine per completare la configurazione.

Per creare un'altra attività:

  1. Andare su Cloud Station > Impostazioni e abilitare una cartella condivisa per questa nuova attività. In questo esempio è foto.​Assicurarsi di disporre dei privilegi di lettura/scrittura per foto.
  2. Tornare a Cloud Station nel PC, fare clic su Crea.
  3. ​Selezionare un Synology NAS per la nuova attività. In questo esempio, verrà selezionato Synology NAS connesso poiché viene trattata la creazione di un'altra attività per lo stesso Synology NAS. Fare clic su Avanti.
  4. Selezionare Configurazione avanzata e seguire le istruzioni della procedura guidata per continuare la configurazione.

3. Recuperare i dati da Cloud Station

Se si rimuove o si sovrascrive in modo accidentale un file, con Cloud Station è facilmente possibile recuperarlo.

4.1 Per scaricare una versione precedente di un file sincronizzato sul PC

  1. Fare clic sull'icona Cloud Station nel vano sistema, quindi fare clic sull'icona della cartella.
  2. Fare clic con il tasto destro sul file e scegliere Synology Cloud Station > Sfoglia le versioni precedenti.
  3. Trovare la versione che si desidera scaricare e fare clic sull'icona del download.
  4. Inserire il nome del file e scegliere la destinazione per salvare il file.

4.1 Per scaricare una versione precedente di un file sincronizzato sul Mac

  1. Fare clic sull'icona Cloud Station nel vano sistema, quindi fare clic sull'icona della cartella.
  2. Fare clic con il tasto destro sul file e scegliere Synology Cloud Station > Sfoglia le versioni precedenti.
  3. Trovare la versione che si desidera scaricare e fare clic sull'icona del download.
  4. Inserire il nome del file e scegliere la destinazione per salvare il file.

4.3 Per scaricare o ripristinare una versione precedente di un file sincronizzato da Cloud Station

  1. Andare su Cloud Station > Cronologia versione e selezionare la cartella Backup creata in precedenza.
  2. Scegliere il file da recuperare e fare clic su Azione > Sfoglia le versioni precedenti.
  3. Scegliere la versione del file che si vuole scaricare e fare clic su Download. Oppure, se si è sicuri di voler ripristinare la versione scelta, fare clic su Ripristina e la versione corrente verrà sovrascritta.

4. Note importanti

  1. Conservare sempre le versioni della cronologia qualora un file venga eliminato o sovrascritto per sbaglio.
  2. In Impostazioni globali è possibile configurare se un file debba essere rilevato da NAS o eliminato su NAS quando si elimina un file sul PC.
