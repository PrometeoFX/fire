Sincronizzazione dei file tra eCore Server e il computer mediante Cloud Station

Panoramica

Cloud Station per eCore Server è un'applicazione per la sincronizzazione di file che permette di sincronizzare i file facilmente sulla propria eCore Server con altri dispositivi, come computer o dispositivi mobile (come DS cloud). Una volta che la Cloud Station è stata installata sulla eCore Server e il computer, i file archiviati in una cartella specifica del proprio computer saranno automaticamente sincronizzati alla propria eCore Server.

1. Installare Cloud Station

  1. Andare su Centro pacchetti, individuare Cloud Station e fare clic su Installa.

2. Abilita Cloud Station

  1. Avviare Cloud Station.
  2. Se la funzione home utente non è stata abilità, verrà richiesto di abilitarla. Fare clic su OK.
  3. L'utente sarà reindirizzato alla pagina delle impostazioni dell'utente. Abilitare homes utente e fare clic su Applica.
  4. A questo punto, tornare a Cloud Station e verrà richiesto di abilitarla. Fare clic su Sì.
  5. Andare su Privilegi e specificare gli utenti che potranno usare Cloud Station. Fare clic su Salva.

3. Installare e abilitare il client

  1. Andare su Cloud Station > Panoramica e fare clic sul pulsante Computer.
  2. Fare clic sul pulsante per scaricare Cloud Station per il proprio sistema operativo. (è anche possibile scaricare Cloud Station dal Centro download Synology).
  3. Lanciare il programma di installazione nel computer e seguire le istruzioni sulla schermata.
  4. Dopo l'installazione, eseguire Cloud Station sul computer. Fare clic su Avvia ora.
  5. Inserire l'indirizzo per il eCore Server che esegue Cloud Station (o ID QuickConnect), nome utente e password. È possibile fare clic sull'icona di ricerca a destra per cercare gli IP nella LAN. Fare clic su Avanti.
  6. Selezionare la modalità di configurazione dell'attività (nell'esempio, si selezionerà Configurazione rapida), quindi fare clic su Avanti.
  7. Per completare la configurazione della cartella Cloud Station, fare clic su Fine.
  8. Fare clic su Aprire adesso per aprire la cartella Cloud Station.
  9. L'icona Cloud Station si trova nell'area di notifica.
  10. Fare doppio clic sull'icona di Cloud Station per aprire l'area di notifica. Qui, è possibile visualizzare l'avanzamento e lo stato dei file. Fare clic su App principale per avviare Cloud Station.
  11. All'avvio, verranno mostrati alcuni suggerimenti su come usare Cloud Station. Fare clic sulla freccia a destra per vedere altri suggerimenti oppure fare clic su Vai ai tutorial online per maggiori informazioni. Per non visualizzare i suggerimenti, fare clic su Non mostrare più.
  12. A questo punto, sarà possibile gestire le attività di sincronizzazione in Cloud Station.
