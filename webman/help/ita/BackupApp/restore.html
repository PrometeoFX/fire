Ripristina

Andare su Backup e Replica > Ripristina per ripristinare i dati di backup.

Per ripristinare un'attività di backup:

  1. Selezionare un'attività di backup e fare clic su Ripristina. Se l'attività è un backup multi-versione, selezionare Ripristina > Attività completa.
  2. Può essere chiesto di selezionare la configurazione del sistema, versioni diverse di dati di backup o altro. Dipende se l'attività è un'attività di backup di dati, attività di backup LUN*, destinazione di backup versione singola (ad es. cartella condivisa) o destinazione di backup a più versioni (ad es. volume).
  3. Seguire la procedura guidata per completare la configurazione.
  1. Selezionare un'attività di backup e fare clic su Ripristina.
  2. Può essere chiesto di selezionare la configurazione del sistema, versioni diverse di dati di backup o altro. Dipende dal tipo di attività di backup in questione.
  3. Seguire la procedura guidata per completare la configurazione.

Nota:

  • Il ripristino di certe applicazioni è supportato. Per visualizzare se un pacchetto specifico supporti il ripristino e per visualizzarne i limiti, fare riferimento alla guida di ciascuna applicazione.
  • Quando si ripristina un pacchetto, se non è installato, verrà installata automaticamente la stessa o l'ultima versione dell'applicazione. Se l'applicazione viene installata ma in un versione più vecchia di quella di cui si è eseguito il backup, verrà aggiornato automaticamente alla versione di cui si dispone il backup.
  • Le applicazioni di cui è stato effettuato il backup in DSM 5.1 o superiore non possono essere ripristinate in DSM 5.0 o inferiore.

Per ripristinare un singolo file o cartella per l'attività di backup multi-versione:

  1. Launch Browser versione file. È possibile procedere scegliendo uno dei metodi seguenti:
    • Fare clic su Ripristina e selezionare File/cartelle singole dal menu a discesa.
    • Andare su Destinazione di backup ed espandere la destinazione di backup selezionata. Fare clic sull'icona Informazioni versione di backup. Nella finestra visualizzata, selezionare la versione di backup da ripristinare e fare clic sull'icona Browser versione file.
  2. È possibile sfogliare le versioni di un file o di una cartella desiderata utilizzando l'icona della timeline o del calendario alla base della schermata.
  3. Selezionare la versione che si desidera gestire, quindi fare clic su Ripristina.
  4. In alternativa, è possibile copiare un file o una cartella in un'altra posizione, oppure scaricarla utilizzando il pulsante Copia in... o Download.
  5. È possibile anche fare clic con il tasto destro del mouse sul file o la cartella e selezionare queste opzioni dal menu a discesa.

Per cercare una versione specifica:

  1. Fare clic sulla freccia nella barra del filtro nell'angolo in alto a destra.
  2. Nel menu a discesa, è possibile filtrare i file per:
    • Parola chiave: inserire il nome di un qualsiasi file o cartella Nelle parole chiave, è possibile utilizzare i caratteri jolly [*] (per indicare uno o più caratteri) e il simbolo [?] (per indicare un carattere casuale). Ad esempio, [a*n] può rappresentare [action], [attention] o [an].
    • Tipo di file: nel menu a discesa è possibile scegliere fra Qualsiasi, Qualsiasi cartella o Qualsiasi file per restringere la ricerca.
    • Dimensione: scegliere Qualsiasi o selezionare è uguale a, è maggiore di, è minore di, quindi inserire un valore.
    • Intervallo data: scegliere tra Oggi, Ieri, Settimana scorsa o Mese scorso o Personalizza per inserire una data di inizio e di fine per trovare le versioni che rientrano in uno specifico periodo.
  3. Per applicare i filtri, fare clic su Filtro.
  4. Per reimpostare la ricerca filtrata, fare clic sul pulsante Reset.

Nota:

  • Solo le attività di backup multi-versione forniscono la funzione Browser versione file.
  • La funzione Browser versione file presenta soltanto funzioni parziali in DSM 5.1 o inferiori.
  • La configurazione di sistema e le applicazioni possono essere ripristinate mediante Ripristina > Attività completa.
  • I metadati non saranno inclusi nelle azioni Copia in... e Download.

Per ripristinare attività di backup da un'altra destinazione di backup:

  1. Fare clic su Ripristina da....
  2. Scegliere uno dei seguenti per individuare il backup dei LUN e fare clic su Avanti:
    • Ripristino locale: scegliere questa opzione per ripristinare dati da una cartella condivisa locale o un disco esterno.
    • Ripristino rete: scegliere questa opzione per ripristinare dati da un'altra DiskStation o server compatibile rsync.
    • Ripristino servizio cloud pubblico: scegliere questa opzione per ripristinare i dati da un server cloud pubblico.
  3. Quando chiesto, selezionare l'origine dove è conservato il backup, quindi selezionare la destinazione per ripristinare i dati.
  4. Seguire la procedura guidata per completare.

Nota:

  • Con Embedded DataStation è possibile solo ripristinare le cartelle condivise nei dispositivi esterni EXT3/EXT4.
  • Embedded DataStation non supporta il ripristino di applicazioni senza un dispositivo esterno EXT3/EXT4.
