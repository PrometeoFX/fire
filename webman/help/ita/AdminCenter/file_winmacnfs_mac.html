Servizio file Mac

In Pannello di controllo > Servizi file > Win/Mac/NFS, è possibile impostare il servizio file Mac per rendere DiskStation un server file Mac (server AFP).

Per abilitare il servizio file Mac:

  1. Mettere il segno di spunta sulla casella Abilita il servizio file Mac per rendere DiskStation un server Mac file (server AFP).
  2. Impostare una cartella di backup Time Machine, se necessario (consultare la sezione che segue per maggiori informazioni).
  3. Fare clic su Applica.

Sulla cartella di backup Time Machine:

Per eseguire il backup dei dati da Time Machine a DiskStation, selezionare una cartella condivisa nel menu a discesa.

Nota: Le cartelle condivise in uso per altri servizi non saranno disponibili per Time Machine.

Impostazioni avanzate

Applica autorizzazioni UNIX predefinite:

abilitando questa opzione si applicano le autorizzazioni UNIX predefinite quando si caricano o creano file e cartelle. Le autorizzazioni applicate sono le stesse applicate dal comando UNIX umask. Quando questa opzione è abilitata, le autorizzazioni UNIX sono 644 per i file e 755 per le cartelle. Quando questa opzione è disabilitata, le autorizzazioni UNIX sono 666 per i file e 777 per le cartelle. Il valore unmask predefinito è 022.

Nota: L'abilitazione di questa funzione può causare problemi di autorizzazioni non coerenti tra diversi protocolli. Per evitare incoerenze, consigliamo di lasciare questa opzione disabilitata.

Rilasciare subito le risorse dopo la disconnessione:

L'attivazione di quest'opzione comporta il rilascio immediato delle risorse in caso di disconnessione imprevista del server AFP. Se, durante l'utilizzo di Time Machine, viene visualizzato il messaggio di errore "Volume in uso", abilitare l'opzione.

Per abilitare Bonjour printer broadcast:

Mettere un segno di spunta su Abilita Bonjour Printer Broadcast per permettere ai clienti Mac di cercare e localizzare la stampante connessa a DiskStation.
