DHCP Server

DiskStation può assumere il ruolo di un server DHCP e assegna indirizzi IP dinamici ai clienti DHCP entro la LAN.

Per abilitare il server DHCP:

  1. Andare su Pannello di controllo > Rete > Interfaccia di rete.
  2. Selezionare un'interfaccia di rete (ad es. LAN) e fare clic su Modifica.
  3. Andare sulla scheda Server DHCP e fare clic su Abilita DHCP Server.
  4. Inserire un valore nel campo Durata lease indirizzo per impostare la durata lease DHCP (in minuti) per gli indirizzi IP assegnati ai client DHCP.
  5. Inserire l'indirizzo DNS per i client DHCP nei campi DNS principale (obbligatorio) e DNS secondario (opzionale).
  6. Impostare il nome dominio per il server DHCP nel campo Nome dominio (opzionale).
  7. Fare clic su OK.

Per aggiungere una subnet:

  1. Fare clic su Aggiungi e specificare le informazioni nelle seguenti colonne:
    • Abilita: per abilitare o disabilitare la subnet.
    • Avvia: il primo indirizzo IP della subnet.
    • Ferma: l'ultimo indirizzo IP della subnet.
    • Subnetmask: la subnetmask.
    • Gateway: il gateway della subnet.
  2. Fare clic su OK.

Nota.

  • Si possono aggiungere solo fino a 32 subnet.

Client DHCP

Per visualizzare un elenco di client DHCP e loro configurazioni di rete (ad es. indirizzo MAC, indirizzo IP, nome host e il tempo rimasto prima della scadenza della lease DHCP), è possibile andare sulla scheda Client DHCP.

Prenotazione DHCP

Per consentire al client di ricevere sempre lo stesso indirizzo IP durante il rinnovo del lease DHCP, è possibile aggiungere il client all'elenco prenotazioni indirizzi.

Per aggiungere un client all'elenco prenotazione indirizzo:

  • Fare clic su Aggiungi.
  • Specificare l'indirizzo MAC del client, l'indirizzo IP da assegnare al client e il nome host del client.
  • Fare clic su OK per salvare le impostazioni.
