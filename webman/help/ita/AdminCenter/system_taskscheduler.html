Utilità di pianificazione

Utilità di pianificazione consente di pianificare e eseguire i servizi o script in momenti predefiniti.

Attività supportate

È possibile creare le attività nei seguenti modi:

  • Script definito dall’utente: eseguire gli script o comandi definiti dall'utente.
  • Cestino: vuotare il cestino delle cartelle condivise specificate. Inoltre, le impostazioni di Politica di conservazione consentono di controllare quali file sono cancellati in modo permanente quando si svuota il Cestino. Ad esempio, i file possono essere cancellati o conservati in base a dimensione, tempo di cancellazione o formato file.
  • Controllo promemoria acustico: consente a DiskStation di emettere un bip.
  • Servizio: avviare o interrompere i servizi di rete, ad esempio il Servizio file Windows, FTP, Blocco automatico, ecc.

Gestione delle attività

La pagina principale dell'Utilità di pianificazione visualizza tutte le attività esistenti e consente di creare, modificare o eseguire le attività.

Per creare un'attività:

  1. Fare clic su Crea e selezionare il tipo di attività da pianificare.
  2. Quando viene visualizzata la finestra Crea attività, modificare le seguenti impostazioni nella scheda Generale:
    • Impostazioni generali: immettere un nome per l'attività, selezionare un utente e abilitare o disabilitare l'attività.
    • Altre Impostazioni: in questa scheda possono essere modificate altre impostazioni specifiche dell'attività (solo per alcune attività). Ad esempio, quando viene creata un'attività per vuotare il cestino, è necessario selezionare il cestino di una cartella condivisa.
  3. Nella scheda Programmazione, modificare le seguenti impostazioni:
    • Data: specificare quando eseguire l'attività. È possibile pianificare l'esecuzione in dati giorni della settimana (ad esempio ogni giorno, nei fine settimana o ogni domenica, ecc.) o in date specifiche (ad esempio 21/10/2015).
    • Ora: specificare il numero di volte che l'attività deve essere eseguita ogni giorno, oltre alla prima e ultima ora di esecuzione.
  4. Modificare le impostazioni specifiche dell'attività nella scheda Impostazioni attività (solo per alcune attività).
  5. Fare clic su OK per terminare

Per modificare le impostazioni dell'attività:

  1. Selezionare un'attività esistente.
  2. Fare clic su Modifica.
  3. Viene visualizzata una finestra dove è possibile modificare l'attività.

Per eliminare un'attività:

  1. Selezionare un'attività esistente.
  2. Fare clic su Elimina.

Per abilitare o disabilitare l'attività:

  1. Nella pagina principale di Utilità di pianificazione, selezionare o deselezionare le caselle di controllo appropriate in Abilitato per abilitare o disabilitare l'attività.
  2. Fare clic su Salva.

Per eseguire manualmente l'attività:

  1. Selezionare una o più attività esistenti.
  2. Fare clic su Esegui.

Limitazioni dell'Utilità di pianificazione:

  • I nomi delle attività possono includere 512 caratteri visualizzabili.
  • Numero massimo di attività: 1024.
  • Lunghezza massima del percorso script o linea di comando: 4095.
  • La durata bip può essere impostata da 1 a 600 secondi.
