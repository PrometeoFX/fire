Indici file

L'indicizzazione dei contenuti di una cartella condivisa può permettere di trovare i file o le cartelle più velocemente.

Per indicizzare i file in una cartella condivisa:

  1. Selezionare la cartella condivisa che si desidera indicizzare.
  2. Fare clic su Modifica e poi sulla scheda Indicizzazione file.
  3. Selezionare Abilita indicizzazione file.
  4. Fare clic su OK.

Reindicizza i File

La reindicizzazione di file non è necessaria in circostanze normali. Tuttavia, se i file in una cartella non corrispondono ai criteri di ricerca ma paiono mancanti durante la ricerca di file, occorre eseguire la reindicizzazione per ricostruire la struttura di indice dei file.

Per reindicizzare i file in una cartella condivisa:

  1. Selezionare la cartella condivisa che si desidera reindicizzare.
  2. Fare clic su Modifica e poi sulla scheda Indicizzazione file.
  3. Fare clic su Reindicizza.
  4. Al termine del processo, il sistema mostrerà quanti file e sottocartelle sono stati indicizzati.

Nota: L'indicizzazione dei file nelle cartelle remote o nelle unità USB esterne non è supportata da File Station.
