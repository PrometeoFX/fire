TFTP/PXE

Le opzioni Pannello di controllo > Servizi file > TFTP/PXE consentono di configurare le impostazioni TFTP e PXE.

TFTP

TFTP (Trivial File Transfer Protocol) è un semplice protocollo di trasferimento dei file utilizzato in genere per il trasferimento dei file di configurazione e di avvio, in situazioni dove non è necessaria l'autenticazione. L'abilitazione del servizio TFTP consente alla DiskStation di trasferire i caricatori di avvio ai dispositivi client.

Per modificare le impostazioni TFTP:

  1. Selezionare la casella indicata con Abilita il servizio TFTP.
  2. Nel campo Cartella radice TFTP, specificare a quale cartella nella DiskStation è possibile accedere tramite i client TFTP.
  3. È possibile fare clic sul pulsante Impostazioni avanzate per visualizzare e modificare le seguenti impostazioni:
    • Abilitare Registro di trasferimento file TFTP: questa opzione consente di tenere traccia dei dati trasferiti tramite TFTP.
    • Autorizzazione client TFTP: selezionare le autorizzazioni di lettura e scrittura per i client TFTP. Quando viene selezionato Sola lettura, i client TFTP possono solo visualizzare i contenuti della cartella radice. Quando viene selezionato Scrivibile, i client TFTP possono modificare i contenuti della cartella radice.
    • Client consentiti: scegliere quali client possono connettersi tramite TFTP. È possibile consentire ai client di connettersi selezionando Tutte le connessioni oppure è possibile scegliere quali client possono connettersi specificando un intervallo di indirizzi IP.
  4. Una volta modificate le impostazioni, fare clic su Applica per salvare.

Configurare il servizio DHCP per PXE con DiskStation

Oltre al TFTP, DiskStation offre anche il servizio DHCP per poter implementare tutte le configurazioni PXE in una sola. Pre-boot Execution Environment (PXE) è un ambiente di rete che consente ai dispositivi client di avviarsi utilizzando un'interfaccia di rete al posto di un dispositivo di archiviazione dei dati (ad esempio i dischi rigidi). Durante l'implementazione PXE, un caricatore di avvio può essere archiviato nella DiskStation in modo che i dispositivi client possano essere avviati tramite la rete utilizzando il caricatore di avvio.

Per fornire i servizi DHCP con la DiskStation, assicurarsi di aver installato il pacchetto DHCP Server di Synology.

Per configurare il servizio DHCP per PXE:

  1. Selezionare la casella indicata con Configura servizio DHCP su questo server per PXE. Sarà chiesto di installare il pacchetto DHCP Server nel caso non sia già installato su DiskStation.
  2. Selezionare un file del caricatore di avvio dalla cartella radice TFTP e specificare le impostazioni relative al DHCP.
  3. Fare clic su Applica per salvare le impostazioni.
