Portale applicazione

Con Portale applicazione, è possibile configurare le impostazioni di collegamento delle varie applicazioni, consentendo di accedere ed eseguire queste applicazioni in schede o finestre del browser indipendenti.

Ad esempio, è possibile apire File Station inserendo http://:diskstation:5000/file/ nel proprio browser web. Le impostazioni di connessione disponibili comprendono la personalizzazione degli alias, porte HTTP e HTTPS. Per trovare queste impostazioni, andare su Pannello di controllo > Portale applicazione.

Nota:

Portale applicazione consente di modificare le impostazioni di accesso per le varie applicazioni come Audio Station, Download Station, Surveillance Station, Video Station, File Station e altre.

Personalizzare alias

Con un alias personalizzato, un'applicazione come File Station può essere aperta inserendo un indirizzo come http://:diskstation:5000/file/ nel proprio browser web.

Per abilitare un alias personalizzato:

  1. Selezionare un'applicazione.
  2. Fare clic su Modifica.
  3. Nella finestra visualizzata, selezionare Abilita alias personalizzato.
  4. Specificare un alias personalizzato.

Nota:

  • è possibile selezionare la casella Reindirizza da e utilizzare un alias più breve semplice da ricordare per accedere all'applicazione selezionata.
  • Di seguito sono descritti gli alias predefiniti delle rispettive applicazioni Audio Station, Download Station, Surveillance Station, Video Station, File Station: audio, download, cam, video, file.
  • I nomi alias non possono essere gli stessi di quelli riservati per il sistema, il browser o utilizzati da altre applicazioni, e il numero di caratteri deve essere compreso tra 2 e 20.
  • I nomi alias possono avere lettere maiuscole e minuscole, caratteri numerici e due caratteri speciali (-), (_), i due caratteri speciali possono essere utilizzati per tutti i caratteri tranne il primo e l'ultimo.

Personalizzare porte HTTP o HTTPS

Con le porte HTTP o HTTPS personalizzate, un'applicazione può essere aperta inserendo un indirizzo come http://diskstation:7000 o http://diskstation:7001.

Per abilitare una porta HTTP o HTTPS personalizzata:

  1. Selezionare un'applicazione.
  2. Fare clic su Modifica.
  3. Nella finestra visualizzata, selezionare Abilita porta personalizzata (HTTP) o Abilita porta personalizzata (HTTPS).
  4. Specificare il numero della porta personalizzata.

Nota:

  1. l'intervallo del numero di porta può andare da 1 a 65535; le seguenti porte devono esser escluse perché sono riservate al sistema: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, porte predefinite eMule: 4662 (TCP), 4672(UDP) e intervallo predefinito della porta FTP: l'intervallo effettivo può variare in relazione al modello.
