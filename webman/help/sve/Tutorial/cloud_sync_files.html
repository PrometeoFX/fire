Synka filer mellan Synology NAS och din dator med Cloud Station

Översikt

Cloud Station för Synology NAS är en applikation för att synka filer som låter dig enkelt synkronisera filer på din Synology NAS med andra enheter såsom datorer eller mobila enheter (med DS cloud). När Cloud Station har installerats på både din Synology NAS och dator kommer filer som lagrats i en specificerad mapp på din dator kommer automatiskt att synkas till din Synology NAS.

1. Installera Cloud Station

  1. Gå till Paketcenter och hitta Cloud Station, klicka sedan på Installera.

2. Aktivera Cloud Station

  1. Starta Cloud Station.
  2. Om du inte aktiverat funktionen användarhem kommer du att uppmanas att göras så. Klicka på OK.
  3. Du kommer att omdirigeras till sidan för användarinställningar. Aktivera användarhem och klicka på Tillämpa.
  4. Gå nu tillbaka till Cloud Station och du kommer att uppmanas att aktivera den. Klicka på Ja.
  5. Gå till Privilegier och specificera vilka användare som kommer att kunna använda Cloud Station. Klicka på Spara.

3. Installera och aktivera klient

  1. Gå till Cloud Station > Översikt och klicka på knappen Datorer.
  2. Klicka på knappen för att ladda ned Cloud Station för ditt operativsystem. (Du kan också ladda ner Cloud Station från Synology Nerladdningscenter.)
  3. Starta installationsprogrammet på din dator och följ instruktionerna på skärmen.
  4. Efter installation, kör Cloud Station på din dator. Klicka på Starta nu.
  5. Ange din adressen till den Synology NAS som kör Cloud Station (eller QuickConnect ID), användarnamn och lösenord. Du kan klicka på sökikonen till höger för att söka efter IP-adresser inom ditt LAN. Klicka på Nästa.
  6. Välj hur du vill konfigurera din uppgift (här väljer vi Snabbinställning), klicka sedan på Nästa.
  7. Klicka på Slutför för att avsluta installationen av din mapp för Cloud Station.
  8. Klicka på Öppna den nu för att öppna din mapp i Cloud Station.
  9. Du finner ikonen för Cloud Station i ditt systemverktygsfält.
  10. Dubbelklicka på ikonen Cloud Station för att öppna verktygsfältsmenyn. Här kan du visa förloppet och status för dina filer. Klicka på Huvudapp för att starta Cloud Station.
  11. Vid start kommer du att få se några tips om att använda Cloud Station. Klicka på högerpilen för att se fler tips eller klicka på Gå till självstudier online för mer information. Om du inte behöver tipsen, klicka på Visa inte det här igen.
  12. Nu kan du hantera dina synkuppgifter i Cloud Station.

4. Synka data till dina mobila enheter

DS cloud finns tillgänglig gratis i Apple App Store och Google Play Store. Du kan även skanna QR-koden här nedanför, för att ladda ner direkt.
