Hur data på min dator/Mac säkerhetskopieras med Cloud Station

Översikt

Cloud Station är inte bara en sofistikerad mjukvara för synkning mellan plattformar, användare kan också använda den som ett smart sätt att säkerhetskopiera sina datorer. Det finns några fördelar med att använda Cloud Station som ditt verktyg för säkerhetskopiering. Först och främst tillhandahåller replikering en bättre RPO, en kortare period för vilken data kan förloras (inte gå att återställa) från din dator efter en större incident. Det är inte bara omedelbart utan hjälper dig även att drastiskt spara bandbredd, jämfört med traditionell säkerhetskopiering som alltid använder stora mängder bandbredd. Du kan enkelt filtrera ut filerna i samma mapp, som du inte vill säkerhetskopiera, genom att ställa in filstorleken eller filtypen på din datorklient. Cloud Station kan även bevara upp till 32 historiska versioner av en enda fil som Cloud Station låter dig återställa till en datorklient från serversidan (NAS).

1. Installera Cloud Station på Synology NAS

  1. Logga in på DSM med ett konto som hör till administrator.
  2. Starta Cloud Station.
  3. Om du inte aktiverat funktionen användarhem kommer du att uppmanas att göras så. Klicka på OK.
  4. Du kommer att omdirigeras till sidan för användarinställningar. Aktivera användarhem och klicka på Tillämpa.
  5. Gå nu tillbaka till Cloud Station och du kommer att uppmanas att aktivera den. Klicka på Ja.
    6. Obs: QuickConnect är en tjänst för överföring som enkelt låter dig ansluta till din Synology NAS från utanför ditt lokala nätverk. Om du vill synka dina filer när du inte är hemma eller på kontoret rekommenderar vi att du installerar QuickConnect genom att gå till Kontrollpanel > QuickConnect.
  6. Gå till Privilegier och specificera vilka användare som kommer att kunna använda Cloud Station. Klicka på Spara.
  7. Konfigurera en delad mapp på din Synology NAS i syfte att säkerhetskopiera. För att göra detta, gå till Kontrollpanel > Delad mapp och klicka på Skapa.
  8. Ge din mapp ett namn och fyll därefter i detaljerna. Klicka på OK. Som ett exempel ger vi mappen namnet Säkerhetskopiering.
  9. På fliken Behörigheter, specificera vilka användare som kan få åtkomst till mappen. För exemplet ger vi admin behörigheterna läs-/skrivbar till mappen. Klicka på OK.
  10. Gå till Cloud Station > Inställningar > Delning och välj mappen Säkerhetskopiering och klicka på Aktivera.

2. Installera Cloud Station på din Windowsdator

Följ stegen för att installera Cloud Station på din Mac eller dator

2.1 För dator

  1. Gå till Cloud Station > Översikt och klicka på knappen Ladda ner.
  2. Klicka på knappen för att ladda ner Cloud Station för Windowsdatorer. (Du kan också ladda ner Cloud Station från Synology Nerladdningscenter.)
  3. Starta installationsprogrammet på din dator och följ instruktionerna för att installera Cloud Station.
  4. Efter installation, kör Cloud Station på din dator. Klicka på Starta nu.
  5. Ange din adressen till den Synology NAS som kör Cloud Station (eller QuickConnect ID), användarnamn och lösenord. (Du kan även klicka på sökikonen till höger så kommer Cloud Station automatiskt att söka efter andra Synology NAS inom ditt LAN). Klicka på Nästa.
    6. Obs: Säkerställ att kontot som angivits har behörigheter tilldelade i Cloud Station > Privilegier på Synology NAS.
  6. Välj hur du vill konfigurera din uppgift (här väljer vi Avancerad installation eftersom vi konfigurera en mapp specifikt för säkerhetskopiering). Klicka på Nästa.
  7. Välj en mapp på din dator för att fortsätta vara synkad med mappen Säkerhetskopiering som du skapade tidigare på din Synology NAS. Klicka på Nästa.
  8. Klicka på Klart för att slutföra installationen.

Skapa en till uppgift:

  1. Gå till Cloud Station > Inställningar och aktivera en delad mapp för den nya uppgiften. Mitt exempel är photo. Se till att du har rättigheterna läsa/skriva för photo.
  2. Tillbaka på Cloud Station på din PC, klicka på Skapa.
  3. Välj en Synology NAS för din nya uppgift. Här väljer vi Ansluten Synology NAS eftersom vi skapar en till uppgift för samma Synology NAS. Klicka på Nästa.
  4. Välj Avancerad installation och följ resten av guidens instruktioner för att fortsätta med installationen.

3. Hämta data från Cloud Station

Om du oavsiktligt tar bort eller skriver över en fil låter Cloud Station dig enkelt hämta den.

4.1 Att ladda ner en tidigare version av en synkad fil på datorn

  1. Klicka på ikonen Cloud Station i ditt systemverktygsfält och klicka på mappens ikon.
  2. Högerklicka på din valda fil och välj Synology Cloud Station > Bläddra i tidigare versioner.
  3. Hitta den version du vill ladda ner och klicka på ikonen för nerladdning.
  4. Ange filnamn och välj målet för att spara filen.

4.1 Att ladda ner en tidigare version av en synkad fil på Mac

  1. Klicka på ikonen Cloud Station i ditt systemverktygsfält och klicka på mappens ikon.
  2. Högerklicka på din valda fil och välj Synology Cloud Station > Bläddra i tidigare versioner.
  3. Hitta den version du vill ladda ner och klicka på ikonen för nerladdning.
  4. Ange filnamn och välj målet för att spara filen.

4.3 Att ladda ner eller återställa en tidigare version av en synkad fil från Cloud Station

  1. Gå till Cloud Station > Versionshistorik och välj mappen Säkerhetskopiering som du tidigare skapade.
  2. Välj den fil du vill hämta och klicka på Åtgärd > Bläddra i tidigare versioner.
  3. Välj den version av filen som du vill ladda ner och klicka på Ladda ner. Eller, om du är säker på att du vill ha den valda versionen återställd, klicka på Återställ och den skriver över din aktuella version.

4. Viktiga noteringar

  1. Behåll alltid historiska versioner för det fall en fil tas bort eller skrivs över av misstag.
  2. I Globala inställningar kan du konfigurera om en fil ska hämtas igen från din NAS eller raderas på din NAS när du raderar en fil på din PC.
