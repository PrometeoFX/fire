Datasäkerhetskopiering

På fliken Säkerhetskopiering i Säkerhetskopiera & Replikering kan du säkerhetskopiera specifika data på DiskStation manuellt eller enligt ett anpassat schema. Med denna funktion kan systemkonfiguration, innehåll i delade mappar och data för filpaket säkerhetskopieras till ett mål lokalt, på nätverket eller i molnet.

Skapa en säkerhetskopieringsuppgift:

  1. Välj Skapa > Datasäkerhetskopieringsuppgift.
  2. Följ guiden för att slutföra installationen. För mer information, besök Säkerhetskopieringsmål.

Så här gör du för att hantera säkerhetskopieringsuppgifter:

Välj en befintlig uppgift och klicka på någon knapparna här nedanför:

  • Redigera: Ändra mål, källa, schema och andra inställningar för den valda uppgiften.
  • Radera: Radera den valda uppgiften.
  • Avbryt: Avbryt den valda uppgiften som körs för närvarande.
  • Säkerhetskopiera nu: Utför omedelbart säkerhetskopiering av den valda uppgiften.

Rotation av säkerhetskopior

Rotation av säkerhetskopior är en mekanism för versionshantering. Den hjälper till att hantera versioner av säkerhetskopiering genom att automatiskt radera äldre, onödiga versioner och potentiellt frigöra lagringsutrymme. Du kan aktivera den här funktionen i Inställningar för uppgifter eller konfigurera den när du skapar en säkerhetskopieringsuppgift.

Rotera versioner:

  1. Välj den säkerhetskopieringsuppgift som du vill tillämpa rotering av versioner på och klicka sedan på Redigera.
  2. Gå till fliken Inställningar för uppgifter och markera Aktivera rotation av säkerhetskopior.
  3. Här kan du välja en av två metoder för rotation:
    • Från den tidigaste versionen: Det här är en grundläggande metod för rotation av versioner. När rotationsvillkoret blir utlöst tas den tidigaste version bort först.
    • Smart Recycle: Det här är en intelligent metod för rotation av versioner. När rotationsvillkoret blir utlöst tar systemet bort de versioner som är valda av mekanismen för bibehållande. Mekanismen för rotation fungerar på följande sätt:
      • Versioner som gjorts varje timme från de senaste i 24 timmarna: Systemet sparar den tidigaste versionen som skapas varje timme.
      • Dagliga versioner från senaste 1 dag till 1 månad: Systemet sparar den tidigaste versionen som skapas varje dag.
      • Versioner som gjorts 1 gång per vecka: Systemet sparar den tidigaste versionen som skapas varje vecka.
  4. Specificera det maximala antalet versioner. När versionsantalet överskrider det antalet börjar rotation av versioner. Antalet inkluderar inte låsta versioner.
  5. Om du har konfigurerat ett återkommande schema för säkerhetskopiering visas en förhandsgranskning över hur dina versioner av säkerhetskopior kommer att bibehållas och den tidigaste tidpunkten för återställning baserat på dina inställningar för rotation.   
  6. Klicka på OK för att spara dina inställningar.

Noteringar:

  • Låsta versioner tas aldrig bort. För mer information, besök Säkerhetskopieringsmål.
  • Endast säkerhetskopieringsuppgifter i flera versioner tillhandahåller funktioner för rotation av versioner.
  • Rotation av versioner har inte stöd i DSM 5.1 eller lägre.
  • Rotation kommer att bli utlöst direkt efter att inställningarna ändras eller säkerhetskopieringsuppgiften är slutförd.
  • Låsta versioner är exkluderade vid beräkning av maximalt antal versioner.

Allmänna noteringar:

  • Namn på uppgifter kan innehålla 1 till 32 unicode-tecken men kan inte innehålla följande tecken:
    ! " # $ % & ' ( ) * + , / : ; < = > ? @ [ ] \ ^ ` { } | ~
  • Maximal längd för hela mappens sökväg för säkerhetskopiering är 2048 tecken.
  • Om mer än 32000 filer eller mappar laddas upp till en EXT3 delad mapp via AFP kommer alla säkerhetskopieringsuppgifter inklusive den delade mappen att misslyckas.
  • En krypterad mapp kan säkerhetskopieras antingen monterad eller omonterad. Den blir dock alltid återställd som omonterad och monteras inte automatiskt vid start. Du kommer att behöva ange eller importera krypteringsnyckeln till den delade mappen för att montera den.
  • Det finns inte stöd för att säkerhetskopiera en krypterad mapp till en annan krypterad mapp.
  • Det finns inte stöd för att använda fjärrmappar som källa eller mål för säkerhetskopieringen.
  • Om säkerhetskopieringsmålet är en volym kan endast den som äger uppgifter utföra säkerhetskopieringsuppgiften. För mer information, besök Säkerhetskopieringsmål.
  • Säkerhetskopiering för vissa applikationer har stöd. För att se huruvida ett specifikt filpaket har stöd för säkerhetskopiering och säkerhetskopieringens innehåll samt dess begränsningar, vänligen se hjälpen för vardera applikation.
  • DiskStation har inte stöd för säkerhetskopiering av applikationer utan en skrivbar extern enhet med EXT3/EXT4.
  • DiskStation har inte stöd för säkerhetskopiering till en volym utan en skrivbar extern enhet med EXT3/EXT4.
  • DiskStation kan bara säkerhetskopiera delade mappar som skapats på externa enheter med EXT3/EXT4.
  • Systemet kan inte säkerhetskopiera applikationer utan en skrivbar volym.

Säkerhetskopiera inställningar för Windows ACL:

Inställningar för Windows ACL för filerna på DiskStation kan också säkerhetskopieras, beroende på filsystemet på målet:

  • Lokalt mål för säkerhetskopiering, nätverksmål för säkerhetskopiering och Säkerhetskopieringsmål Gemensamt moln kan säkerhetskopiera Windows ACL-inställningar oavsett filsystem på målet
  • Synkronisera delade mappar kan säkerhetskopiera inställningar för Windows ACL när filsystemet på målet är EXT4/BTRFS.
