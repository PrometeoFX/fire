Återställ

Gå till Säkerhetskopiering & Replikering > Återställ för att återställa säkerhetskopieringsdata.

Att återställa en säkerhetskopieringsuppgift:

  1. Välj en säkerhetskopieringsuppgift och klicka sedan på Återställ. Om uppgiften är för flera versioner av säkerhetskopior, vänligen välj Återställ > Hela uppgiften.
  2. Du kan bli uppmanad att välja systemkonfiguration, olika versioner av säkerhetskopierad data, eller annat. Det beror på om det är en säkerhetskopieringsuppgift för data, en säkerhetskopieringsuppgift för LUN*, målet för säkerhetskopieringen är i en version (dvs. delad mapp) eller flera versioner (dvs. volym).
  3. Följ guiden för att slutföra installationen.
  1. Välj en säkerhetskopieringsuppgift och klicka sedan på Återställ.
  2. Du kan bli uppmanad att välja systemkonfiguration, olika versioner av säkerhetskopierad data, eller annat. Det beror på vilken typ av säkerhetskopieringsuppgift det är.
  3. Följ guiden för att slutföra installationen.

Obs:

  • Återställning för vissa applikationer har stöd. För att se huruvida ett specifikt filpaket har stöd för återställning och dess begränsningar, vänligen se hjälpen för vardera applikation.
  • Vid återställning av ett filpaket när ett filpaket inte är installerat kommer systemet automatiskt att installera samma eller senaste versionen av applikationen. Om applikationen är installerad men dess version är äldre än den säkerhetskopierade kommer systemet automatiskt att uppgradera den till den ursprungliga säkerhetskopierade versionen.
  • Applikationer som säkerhetskopierats i DSM 5.1 eller över kan inte återställas i DSM 5.0 eller lägre.

För att återställa en enstaka fil eller mapp för flera versioner av säkerhetskopior:

  1. Starta Filversionsbläddrare. Du kan göra det med någon av metoderna här nedanför:
    • Klicka på Återställ och välj Enstaka filer/mappar från rullgardinsmenyn.
    • Gå till Säkerhetskopieringsmål och expandera ditt valda Säkerhetskopieringsmål. Klicka på ikonen Versionsinformation för säkerhetskopiering. I fönstret som visas, välj den version av säkerhetskopia som du vill återställa och klicka på ikonen Filversionsbläddrare.
  2. Du kan bläddra bland versionerna för din valda fil eller mapp genom att använda tidslinjen eller ikonen kalender längst ner på skärmen.
  3. Välj den version som du vill hantera och klicka på ikonen Återställ.
  4. Alternativt kan du kopiera en fil eller mapp till en annan plats eller ladda ner den genom att använda Kopiera till... eller knappen Ladda ner.
  5. Du kan också högerklicka på filen eller mappen och välja dessa alternativ från rullgardinsmenyn.

Söka efter en specifik version:

  1. Klicka på pilen i filterfältet i det övre högra hörnet.
  2. I rullgardinsmenyn kan du filtrera filer efter:
    • Nyckelord: Ange namnet på valfri fil eller mapp. Du kan använda jokertecken [*] (vilket representerar ett eller flera tecken) och [?] (som representerar ett tecken, vilket som helst) i dina nyckelord. Till exempel kan [a*n] representera [aktion], [attention] eller [an].
    • Filtyp: i rullgardinsmenyn kan du välja från Alla, Alla mappar eller Alla filer för att begränsa din sökning.
    • Storlek: välj Alla eller välj är lika med, är större än, är mindre än och sedan ange ett värde.
    • Datumintervall: välj från Idag, Igår, Förra veckan eller Förra månaden eller Anpassa för att ange ett startdatum och ett slutdatum för att hitta versioner inom ett specifikt intervall.
  3. För att tillämpa dina filter, klicka på Filtrera.
  4. För att återställa din filtrerade sökning, klicka på knappen Återställ.

Obs:

  • Endast uppgifter med flera versioner av säkerhetskopior tillhandahåller funktionen Filversionsbläddrare.
  • Filversionsbläddrare fungerar endast till viss del i DSM 5.1 eller lägre.
  • Systemkonfiguration och applikationer kan endast återställas via Återställ > Hela uppgiften.
  • Metadata kommer inte att inkluderas i åtgärderna Kopiera till... och Ladda ner.

Att återställa säkerhetskopieringsuppgifter från andra säkerhetskopieringsmål:

  1. Klicka på Återställ från....
  2. Välj något av följande för att lokalisera säkerhetskopieringsuppgiften och klicka på Nästa:
    • Lokal återställning: Välj detta alternativ för att återställa data från en lokal delad mapp eller en extern disk.
    • Nätverksåterställning: Välj detta alternativ för att återställa data från en annan DiskStation eller rsync-kompatibel server.
    • Återställning av gemensam molntjänst: Välj det här alternativet för att återställa data från en server för gemensam molntjänst.
  3. Vid uppmaning, välj källan där säkerhetskopieringsuppgiften finns och välj sedan målet för att återställa data.
  4. Följ guiden för att slutföra.

Obs:

  • Inbäddad DataStation kan bara återställa delade mappar till externa enheter med ext3/ext4.
  • Inbäddad DataStation har inte stöd för återställning av applikationer utan en extern enhet med ext3/ext4.
