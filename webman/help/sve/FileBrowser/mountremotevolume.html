Montera fjärrmappar

Du kan montera fjärrmappar från fjärrservrar (t.ex. nätverksdatorer eller andra DiskStation) för att enkelt navigera i tillgängliga nätverksresurser med File Station.

Montera en fjärrmapp:

  1. Klicka på Verktyg > Montera fjärrmapp.
  2. Ange ”\\Remote_Server\Remote_Folder” i fältet Mapp för att specificera sökvägen för fjärrmappen. Till exempel ”\\192.168.0.2\share” eller ”\\chad\cheeseburger\pictures”.
  3. Om åtkomstprivilegium till fjärrmappen krävs av fjärrservern anger du dina användaruppgifter för fjärrservern i fälten Kontonamn och Lösenord.
  4. Klicka på Bläddra för att välja eller skapa en tom målmapp på din DiskStation för montering av fjärrmappen.
  5. Markera Montera automatiskt vid start om du vill att DiskStation ska montera den här fjärrmappen vid varje systemstart eller omstart.
  6. Klicka på Montera. Du kan nu navigera till destinationsmappen där fjärrmappen är monterad och hantera innehållet i fjärrmapparna.

Avmontera en fjärrmapp:

  1. Välj den fjärrmapp du vill avmontera och klicka på Verktyg > Avmontera.
  2. Alternativt, högerklicka helt enkelt på fjärrmappen och välj Avmontera.

Obs:

  • Max antal samtidiga fjärrmappar är 100.
  • Du får bara montera fjärrmappar från en fjärrserver som stöder CIFS-protokollet.
