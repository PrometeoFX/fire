Textredigerare

Textredigeraren är en mycket mångsidig text- och kodredigerare som låter dig få åtkomst till och snabbt redigera filer eller kod på din DiskStation direkt från en webbläsare.

Filhantering

Att skapa nya filer:

Klicka på Fil och välj Ny. En ny tom flik kommer att visas och ha titeln Nytt Dokument.txt.

För att öppna filer från textredigeraren:

Klicka helt enkelt på Fil och välj Öppna. En dialog visas och du kan välja en fil med specifik teckentabell som måste öppnas i en flik. Om filen redan var öppen; Textredigeraren visar dig filen i dess nuvarande skick.

För att öppna filer från File Station:

I File Station, högerklicka helt enkelt på den valda filen och klicka sedan på Öppna med Textredigerare från snabbmenyn. Om filen redan var öppen; Textredigeraren visar dig filen i dess nuvarande skick. Ett annat sätt att öppna en fil är att dra en eller flera filer från File Station till Textredigeraren genom att använda muspekaren.


Obs:

Textredigeraren kan inte öppna en fil som överstiger 10 MB.

För att spara filer:

Klicka helt enkelt på Fil och välj Spara. Om det är en befintlig fil kommer den att skrivas över med det nuvarande innehållet. Om du vill spara alla de filer du har öppna på samma gång, kan du gå till Fil > Spara alla. Om filen ännu inte har något namn (en ny fil) eller om du valt Fil > Spara som, kommer en dialog att dyka upp och fråga dig efter filens namn. Du kan sedan välja plats, filnamnet och teckentabell att spara med för filen.

Att stänga filer:

Klicka helt enkelt på Fil och välj Stäng. Ett annat sätt att stänga en fil är att klicka på knappen stäng på varje flik. Om du har något data som inte sparats i en fil kommer Textredigeraren alltid att fråga dig om du vill stänga den utan utan att spara dessa ändringar.

Obs:

Du kan spara filer med en specifik teckentabell för att få stöd för dubbelriktade språk. Du kan också specificera kodning vid öppning av fil så att Textredigeraren visar filen korrekt.

Redigering

  • Standardåtgärder
    Kontrollen för att redigera stöder enkla Ångra, Gör om och Välj alla. Dessa åtgärder finns i de flesta redigerare och fungerar i stort sett likadant. Du kan utföra dessa åtgärder genom att välja det motsvarande menyobjektet i menyn Redigera. Dessutom är alternativet Klipp ut, Kopiera och Klistra in tillgänglig via webbläsarens snabbmeny.
  • Indrag
    Indrag av text är en vanligt åtgärd vid redigering av källkod. Indrag är tillägg av extra mellanrum vid början av en rad för att separera textblock och lägga till en hierarki. I normalfallet, vid redigering, sätter ett tryck på Tabb-tangenten in ett tabb-tecken på aktuell plats. Om någon text är markerad kommer dock ett tryck på Tabb-tangenten att lägga till ytterligare mellanrum i början av varje rad som har någon markerad text, vilket gör det möjligt att göra indrag för hela textblock samtidigt. Att samtidigt trycka på Skift- och Tabb-tangenterna tar bort indraget.
  • Skriva över
    Du kan aktivera och inaktivera skriva över genom att trycka på tangenten Insert. Det betyder att du i stället för att infoga tecken vid markören, byter ut dem. Visningsläge för markören byter form i samband med status skriva över.
  • Redigering i kolumnläge
    Ett speciellt sätt att redigera filer är redigering i kolumnläge. Det betyder att texten kan väljas i kolumner, i stället för rader. För att initiera en markering i kolumnläge, tryck på Alt-tangenten vid markering av text och rör markören uppåt eller nedåt efter behov. När text är vald i så kallat kolumnläge kommer skrivning att ändra varje vald kolumn med tecken till det skrivna tecknet.

Söker

  • Normal sökning
    Du kan söka text såsom i de flesta program genom att använda dialogrutan Sök. Du kan öppna denna dialogruta genom att välja Sök > Hitta. Denna dialogruta kommer alltid att vara ovanpå texten och du kan inte fortsätta redigera text om den är öppnad. I sökfältet, skriv den text du vill hitta.
  • Ersätta
    Du kan söka och ersätta text såsom i de flesta program genom att använda dialogrutan Ersätt. Du kan öppna denna dialogruta genom att välja Sök > Ersätt. Ersätt alla kommer att byta ut alla förekomster av frågan med ersättningstexten.
  • Skiftlägeskänsligt
    Om den inte är markerad är sökningen inte skiftlägeskänslig.
  • Endast hela ord
    Får som resultat att det blir träff endast om överensstämmelsen består av ett helt ord, inte en del av det. Till exempel om du söker efter "extra" med Endast hela ord kommer du bara att hitta "extra", i stället för "extrahera".
  • Reguljärt uttryck
    Frågan kommer att behandlas som ett reguljärt uttryck.

Vad är ett reguljärt uttryck?

Ett reguljärt uttryck är en sekvens av tecken som utgör ett sökmönster för en kraftfull och tidsbesparande teknik. Ett mycket enkelt sätt att använda ett reguljärt uttryck skulle vara att hitta samma ord som är stavade på två olika sätt i en textredigerare, till exempel M[ear]rry.

Stilkonfigurator

Stilkonfiguratorn används i huvudsak för att kontrollera utseendet på texten i Textredigeraren.
Rullgardinen längst ned i högra hörnet innehåller alla programmeringsspråk som kan ha en stil, för att hjälpa dig att bygga en lämplig kodningsmiljö.

Preferenser

  • Visa osynliga tecken
    Om markerad kommer tecken för mellanslag, tabb-tecken, ny rad och nytt stycke att visas.
  • Visa guider för indrag
    Om markerad kommer guider för indrag att visas. Det är användbart för att justera koder i kodspråk.
  • Markera aktuell rad
    Om markerad kommer raden där inskjutningstecknet finns att fyllas med en speciell bakgrundsfärg.
  • Markera valt ord
    Om markerad, kommer alla ord (inte delar av) inom samma textfil som överrenstämmer med det valda ordet automatiskt att markeras.
  • Visa radnummer
    Om markerad kommer radnumren att visas.
  • Aktivera dölj kod
    Om markerad kan speciella rader av din kod döljas, baserat på vissa egenskaper. Denna funktion fungerar bra för strukturerade språk såsom C++ eller XML. När Dölj expansionswidget automatiskt är markerad kommer vissa rader av kod att döljas automatiskt. Du kan också välja att markera endast startparentes eller både start- och avslutningsparentes med expansionswidgets.
  • Radbrytning
    Specificerar huruvida din text ska brytas vid fönstrets kant eller vid ett specifikt kolumnnummer utan att infoga en stycketecken.
  • Lägg till avslutningstecken automatiskt
    Om markerad kommer avslutningsparentes (tecken) att automatiskt läggas till i din kod.
  • Infoga mellanslag i stället för tabb-tecken
    Om markerad kommer tryck på Tabb-tangent att infoga hårda mellanslag i stället för ett tabb-tecken. Tabb-bredd kan definieras genom antal tecken i Preferenser > Teckensnitt och Färg.
  • Standardteckentabell
    Standardteckentabellen används för att spara filer med specifik teckentabell för stöd för dubbelriktade språk. Du kan också specificera kodning vid öppning av fil så att Textredigeraren visar filen korrekt.
  • Teckensnitt och Färg
    Tabb-bredd, Teckenstorlek och Tema kan anpassas i enlighet med dina personliga preferenser. Tabb-bredd definieras som antalet mellanslagstecken.

Filformat som stöds

Textredigeraren har endast stöd för enkla textfiler. Textfiler med formatering, såsom *.doc och *.pdf har inte stöd.
