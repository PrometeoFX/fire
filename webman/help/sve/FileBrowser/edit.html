Redigera bilder

Med File Station kan du redigera bilder med webbaserade externa redigerare såsom Pixlr Editor eller Pixlr Express.

Redigera ett fot med en webbaserad fotoredigerare:

  1. Bläddra till den mapp där bilden finns.
  2. Högerklicka på bilden och välj Redigera i Pixlr Editor eller Redigera i Pixlr Express.
      Obs:
    • Pixlr Editor har stöd för de här formaten: jpg, gif, png, psd, pxd (Pixlr layered image)
    • Pixlr Express har stöd för de här formaten: jpg, gif, png
  3. Den valda bildredigeraren kommer att öppnas i ett nytt läsarfönster. Du kan se din bild efter en kort stund och sedan kan du börja redigera den med verktygen.
  4. När du är klar med bildredigeringen gör du som följande för att spara din bild.
    • För Pixlr Express: Klicka på Klar för att spara din bild i originalformatet på dess ursprungliga plats på DiskStation.
    • För Pixlr Editor: Välj Arkiv > Spara, välj plats (kan vara den ursprungliga platsen på DiskStation, din dator, eller andra sociala nätverk) för att spara den från den vänstra panelen, bestäm ett nytt namn eller format efter dina önskemål och klicka sedan på OK.

Obs: För att redigera dina bilder med någon av de webbaserade bildredigerarna, kontrollera att din DiskStation har hämtat en extern IP-adress (på Huvudmeny > Kontrollpanel > Extern åtkomst > DDNS eller Huvudmeny > Kontrollpanel > Nätverk > Nätverksgränssnitt > PPPoE, beroende på din nätverkskonfiguration). Om din DiskStation finns bakom en router, se till att du har angett regel för vidarebefordring av port för DSM i Huvudmeny > Kontrollpanel > Extern åtkomst > Konfiguration av router. Om du fortfarande har problem med att öppna bildredigerarna kan du behöva specificera routerns port för DSM i Huvudmeny > Kontrollpanel > Extern åtkomst > Avancerad.
