Sök efter filer eller mappar

File Stations Filter tillhandahåller alternativ för att söka och visa filer och mappar som uppfyller dina kriterier. Du kan filtrera genom filer eller mappar i den aktuella mappen eller göra en avancerad sökning i alla undermappar för ett mer förfinat sökresultat.

Om du vill söka efter filer eller mappar i den aktuella mappen:

  1. Använd File Station för att gå till den mapp som innehåller det du vill söka efter.
  2. I fältet Filter i det övre högra hörnet kan du ange ett sökord och se det filtrerade resultatet i fönstrets huvudområde.

Utföra en avancerad sökning:

  1. Använd File Station för att gå till den mapp som innehåller det du vill söka efter.
  2. Klicka på förstoringsglaset bredvid fältet Filter i det övre högra hörnet i File Station för att ta fram den avancerade menyn för sökning.
  3. Ange information eller välj från rullgardinsmenyerna bland följande sökkriterier:
    • Nyckelord: Ange ett sökord för att söka efter filer eller mappar med matchande namn.

      Obs:

      • Du kan använda jokertecken [*] (vilket betyder noll eller fler tecken) och [?] (som representerar ett tecken, vilket som helst) i sökordet. Till exempel kan [a*n] representera [aktion], [attention] eller [an].
      • Du kan använda kommatecken (,) för att separera flera sökord. Till exempel kan [a,tion] representera [aktion],[andra portion], [sektion avser], eller [nation].
      • Du kan lägga till dubbla citattecken ["] i början och slutet av sökorden för att få exakta träffar. Till exempel kan ["a typ"] representera [alla typer] men inte [typiska].
    • Plats: Välj från rullgardinsmenyn för att ställa in område och djup för avancerad sökning. Du kan också anpassa sökomfattningen genom att välja valfri annan listad delad mapp.
    • Filtyp: Välj Filändelse för att begränsa sökningen till vissa typer av filer, eller välj Valfri mapp för att endast söka efter mappar.
    • Datum: Välj Ändringsdatum, Skapad den eller Tid för senaste åtkomst och välj sedan ett datum från kalendern för att söka efter filer med matchande datum.
    • Storlek (MB): Välj mellan är lika med, är större än och är mindre än och ange därefter en storlek för att söka efter filer med matchande storlek. (Vid val av är lika med kommer sökresultatet att inkludera filer av ungefärligen samma storlek som den som specificerats.)
    • Ägare/grupp: Välj Ägare eller Grupp och välj sedan i rullgardinsmenyn för att söka efter filer eller mappar som hör till den valda ägaren eller gruppen.

      Obs:

      • För att utföra avancerad sökning måste du specificera Plats och minst ytterligare ett sökkriteria.
      • Om en ISO-mapp eller en fjärrmapp är monterad i en indexerad delad mapp kan du inte hitta filerna i dem med avancerad sökning.
    • Klicka på Sök för att starta sökningen eller klicka på Återställ och ange nya sökvillkor. När sökningen är klar visas de matchande filerna i File Stations huvudområde.
