Lagringsanalyserare

Lagringsanalyserare låter dig få en snabb översikt över tendensen för den övergripande användningen på din DiskStation, skapa och hantera uppgifter för att analysera lagringsutrymmen och generera detaljerade rapporter över volymanvändning. Du kan välja rapporttyper, ställa in ett schema för att ta emot periodiska lagringsrapporter och enkelt visa tidigare rapporter som överensstämmer med den specificerade tiden eller det specificerade kriteriet. Med Lagringsanalyserare blir det mycket enklare och mer effektiv att hantera ditt system och ändra dina inställningar i enlighet med analysen; till exempel, baserat på dess analysrapporter, kan du radera filer som sällan används för att frigöra utrymme för annan användning, öka kvot för användare med större behov eller installera mer lagringsutrymme för att motsvara behoven och tendenser för användning.
