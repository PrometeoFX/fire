Hantera rapportprofiler

Du kan skapa och ändra schemalagda uppgifter för att generera rapporter för volymanvändning i Lagringsanalyserare. > Rapportprofil.

Skapa en rapportuppgift:

  1. Klicka på Skapa för att öppna guiden.
  2. Specificera ett namn för den nya rapportuppgiften och ange e-postadressen till vilken du vill skicka rapporten.
  3. Välj en eller flera typer av rapporter som du vill generera.
  4. Gör något av följande för att specificera en tid för att generera rapporter:
    • Markera Generera rapporter enligt schema och specificera tiden om du vill att systemet ska analysera volymanvändning periodiskt.
    • Markera Generera rapporter nu om du vill starta en rapportuppgift manuellt nu.
  5. Om nödvändigt specificera vissa användare eller delade mappar som ska inkluderas för att avgränsa rapporterna. (Observera att om du vill specificera och generera rapporter för delade mappar som är krypterade, vänligen säkerställ att de krypterade utdelningarna har monterats innan rapportuppgiften utförs.)
  6. Klicka på Tillämpa för att spara inställningarna.

Obs:

  • Använd Enter/Return efter varje e-postadress för att separera flera e-postadresser.
  • Använd Enter/Return efter varje användare, när användare vars filer ska listas en och en av filgrupper specificeras.
  • Rapporten Filer per filgrupp listar följande filtyper:
    Ljudfiler, diskavbilder, dokument, körbara filer, bilder, videor, webbsidor, zip-filer, övriga
  • Avsnittet Stora filer i rapportlistan listar det största 500 filerna på rapportsidan över användning eller de största 200 filerna på den kompletta rapportsidan.

Redigera en rapportuppgift:

  1. Välj en rapportuppgift från listan och klicka på Redigera.
  2. Klicka på de enskilda flikarna för att redigera inställningarna för den här uppgiften.

Radera en rapportuppgift:

Välj en rapportuppgift från listan och klicka på Radera.

För att se historiska rapporter:

  1. Välj en rapportuppgift från listan och klicka på Rapport > Visa rapporthistorik.
  2. I fönstret som visas, klicka på en post för att visa rapporten som genererades vid den specificerade tiden.

För att generera rapporter genast:

Välj en rapportuppgift, klicka på Rapport och välj Generera rapporter nu för att generera rapporter vid behov.

För att ändra var rapporter sparas:

  1. Klicka på Inställningar.
  2. Välj den nya målmappen för att ändra platsen där rapporter sparas.

För att regelbundet samla in användningshistorik för volym:

Som standard samlas data för volymanvändning in varje gång en rapport för volymanvändning genereras och när systemet startar. Som tillägg till ovan exempel kan du även välja att skapa ett nytt schema för att samla in användningshistorik. För att göra det, klicka på Inställningar och markera Samla in volymanvändningshistorik enligt schema och schemalägg en tidpunkt.

Obs:

Historiken som samlats in vid den tidpunkt som schemalagts kommer huvudsakligen att användas för det syftet och som källa för användningstendenser i avsnittet Volymanvändning. När det här alternativet är aktiverat ser du ett linjediagram som visar tendensen för volymanvändningen och ändringar över tid; om inte visas ett stapeldiagram som procentuellt visar nuvarande användning.

För att analysera en rapportuppgift:

Dubbelklicka den önskade rapportuppgiften eller klicka helt enkelt på Analysera den här rapporten (ikonen >) bredvid den önskade rapportuppgiften för att öppna dessa analyssida. Vänligen se Visa användning och rapporter för mer information.
