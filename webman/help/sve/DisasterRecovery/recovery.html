Återställning

Den här sidan gör att systemadministrators kan utföra återställning. För att möjliggöra för användare att få åtkomst till ögonblicksbilder för delade mappar, vänligen aktivera Göra ögonblicksbilder synliga för varje delad mapp så att användare kan få åtkomst till ögonblicksbilderna via File Station eller Utforskaren.

Delad mapp

Att återställa individuella filer/mappar i en delad mapp:

  1. Välj en delad mapp som du vill återställa.
  2. Klicka på Återställ.
  3. Välj en version av ögonblicksbild som du vill återställa från.
  4. Klicka på Bläddra och kopiera filerna/mapparna som du vill återställa till målet.

Att återställa en delad mapp:

  1. Välj en delad mapp som du vill återställa.
  2. Klicka på Återställ.
  3. Välj en version av ögonblicksbild som du vill återställa från.
  4. Klicka på Åtgärd och välj Återställ på plats från rullgardinslistan.
  5. Du kommer att se ett bekräftelsemeddelande. Klicka på Ja för att bekräfta åtgärden.

För att ta en ögonblicksbild till en ny delad mapp:

  1. Välj en delad mapp som du vill klona.
  2. Klicka på Återställ.
  3. Välj en version av ögonblicksbild som du vill klona till en ny delad mapp.
  4. Klicka på Åtgärd och välj Klona med ett nytt namn från rullgardinslistan.
  5. Du blir omdirigerad till sidan Klon i Kontrollpanel > Delad mapp > Skapa. Ange namnet på den klonade delade mappen och klicka på OK för att slutföra.

Obs:

  • Den delade mappen "homes" kan inte återställas.
  • Den klonade delade mappen kan inte krypteras.
  • Om en ögonblicksbild tas när överföring av några filer fortfarande pågår garanteras inte dataöverensstämmelse.
  • Begränsningar på krypterade delade mappar:
    • Ingen typ av återställning är tillgänglig för monterade krypterade delade mappar.
    • Klona med ett nytt namn är inte tillgängligt för krypterade delade mappar.
    • Omonterade krypterade delade mappar kan återställas från tidigare tagna ögonblicksbilder genom att använda Återställ på plats.

iSCSI LUN

Återställa ett iSCSI LUN:

  1. Välj den iSCSI LUN som du vill återställa.
  2. Klicka på Återställ.
  3. Välj en version av ögonblicksbild som du vill återställa från.
  4. Klicka på Åtgärd och välj Återställ på plats från rullgardinslistan.
  5. Du kommer att se ett bekräftelsemeddelande. Klicka på Ja för att bekräfta åtgärden.

För att klona en ögonblicksbild till en ny iSCSI LUN:

  1. Välj den iSCSI LUN som du vill klona.
  2. Klicka på Återställ.
  3. Välj en version av ögonblicksbild som du vill klona till en ny iSCSI LUN.
  4. Klicka på Åtgärd och välj Klona med ett nytt namn från rullgardinslistan.
  5. Du blir omdirigerad till sidan Klon i Lagringshanterare > iSCSI LUN. Ange namnet på den klonade iSCSI LUN och klicka på OK för att slutföra.

Obs:

  • Om en ögonblicksbild tas när överföring av några filer fortfarande pågår garanteras inte dataöverensstämmelse.
