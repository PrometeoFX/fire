DHCP Server

Din DiskStation kan anta rollen som en DHCP-server och tilldela dynamiska IP-adresser till DHCP-klienter inom ditt lokala nätverk.

Att aktivera DHCP Server:

  1. Gå till Kontrollpanel > Nätverk > Nätverksgränssnitt.
  2. Välj ett nätverksgränssnitt (t.ex. LAN) och klicka på Redigera.
  3. Gå till fliken DHCP Server och markera Aktivera DHCP Server.
  4. Ange ett värde i fältet Adresslånetid för att ange DHCP-lånetid (i minuter) för IP-adresser som tilldelats DHCP-klienter.
  5. Ange DNS-adress för DHCP-klienter i fälten Primär DNS (obligatoriskt) och Sekundär DNS (valfritt).
  6. Ställ in domännamnet för DHCP-server i fältet Domännamn (valfritt).
  7. Klicka på OK.

Lägga till subnät:

  1. Klicka på Lägg till och specificera sedan informationen under följande kolumner:
    • Aktivera: Aktivera eller inaktivera subnät.
    • Start: Subnätets första IP-adress.
    • Slut: Subnätets sista IP-adress.
    • Nätmask: Subnätmasken.
    • Gateway: Subnätets gateway.
  2. Klicka på OK.

Obs:

  • Du kan bara lägga till upp till 32 subnät.

DHCP-klienter

För att se en lista över DHCP-klienter och deras nätverkskonfigurationer (dvs. MAC-adress, IP-adress, värdnamn och den tid som behövs innan DHCP-lån går ut), kan du gå till fliken DHCP-klienter.

DHCP-reservation

Om du vill att en klient alltid ska få samma IP-adress vid förnyelse av DHCP-lån kan du lägga till klienten i adressreservationslistan.

Lägga till en klient i adressreservationslistan:

  • Klicka på Lägg till.
  • Specificera klientens MAC-adress, IP-adresserna du vill tilldela till klienten och klientens värdnamn.
  • Klicka på OK för att spara dina inställningar.
