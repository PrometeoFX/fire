TFTP/PXE

Alternativen i Kontrollpanel > Filtjänster > TFTP/PXE låter dig konfigurera inställningar för TFTP och PXE.

TFTP

TFTP (Mindre avancerat filöverföringsprotokoll) är ett enkelt filöverföringsprotokoll som allmänt används för att konfigurationer eller bootfiler, i situationer där autentisering inte krävs. Att aktivera tjänsten TFTP tillåter DiskStation att överföra bootloaderfiler till klientenheter.

Ändra inställningar för TFTP:

  1. Markera rutan med texten Aktivera TFTP-tjänst.
  2. I fältet TFTP rootmapp, specificera till vilken mapp på DiskStation TFTP-klienter har åtkomst.
  3. Du kan klicka på knappen Avancerade inställningar för att visa och ändra följande inställningar:
    • Aktivera loggfil för TFTP-filöverföring: Detta alternativ låter dig hålla reda på data som överförs med TFTP.
    • TFTP-klientbehörigheter: Välj läs- och skrivbehörigheter för TFTP-klienter. När Endast läsbar är valt kan TFTP-klienter bara visa innehållet i rotmappen. När Skrivbar är valt kan TFTP-klienter ändra innehållet i rotmappen.
    • Tillåtna klienter: Välj vilka klienter som är tillåtna att ansluta via TFTP. Du kan tillåta alla klienter att ansluta genom att välja Alla anslutningar; eller så kan du begränsa vilka klienter som kan ansluta genom att specificera ett intervall av IP-adresser.
  4. Efter ändring av inställningarna, klicka på Tillämpa för att spara.

Konfigurera DHCP-tjänsten för PXE med din DiskStation

Förutom TFTP kan DiskStation också tillhandahålla tjänster för DHCP, för att implementera en allt-i-ett-installation av PXE. Pre-boot Execution Environment (PXE) är en nätverksmiljö som låter klientenheterna starta genom att använda ett nätverkskort istället för en datalagringsenhet (dvs. hårddisk). När PXE implementeras kan en bootloader lagras på DiskStation och klientenheter kan starta över nätverket genom att använda bootloadern.

För att tillhandahålla DHCP-tjänster med DiskStation, vänligen tillse att du har installerat filpaketet Synology DHCP Server.

Konfigurera DHCP-tjänsten för PXE:

  1. Markera rutan som är med texten Konfigurera DHCP-tjänst för PXE på denna server. Du blir uppmanad att installera filpaketet DHCP Server om det inte redan finns installerat på din DiskStation.
  2. Välj en boatloaderfil från TFTP rootmapp och specificera andra DHCP-relaterade inställningar.
  3. Klicka på Tillämpa för att spara inställningarna.
