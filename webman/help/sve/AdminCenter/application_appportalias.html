Applikationsportal

Med Applikationsportal kan du konfigurera anslutningsinställningarna för olika applikationer vilket låter dig få åtkomst till och köra dessa applikationer i oberoende bläddringsflikar eller fönster.

Du kan till exempel öppna File Station genom att öppna http://:diskstation:5000/file/ i din webbläsare. Tillgängliga anslutningsinställningar inkluderar anpassade alias, HTTP- och HTTPS-portar. För att hitta dessa inställningar, gå till Kontrollpanel > Applikationsportal.

Obs!

Applikationsportal låter dig redigera åtkomstinställningar för olika applikationer, såsom Audio Station, Download Station, Surveillance Station, Video Station, File Station och även andra.

Anpassa alias

Med ett anpassat alias kan en applikation, såsom File Station, öppnas genom att en adress såsom http://:diskstation:5000/file/ i din webbläsare.

Att aktivera anpassade alias:

  1. Välj en applikation.
  2. Klicka på Redigera.
  3. Markera Aktivera anpassade alias, i fönstret som visas.
  4. Specificera ett anpassat alias.

Obs!

  • Du kan markera kryssrutan Omdirigera från och använda ett kortare och enklare-att-minnas alias för att få åtkomst till den valda applikationen.
  • Följande är respektive standardalias för de fyra applikationerna Audio Station, Download Station, Surveillance Station, Video Station, File Station: audio, download, cam, video, file.
  • Aliasnamn kan inte vara desamma som är reserverade för systemet, webbläsare eller används av andra applikationer och antal tecken för dem måste begränsas till att vara mellan 2 och 20.
  • Aliasnamn kan bestå av blandade gemener och versaler, numeriska tecken och två specialtecken (-), (_), dessa två tecken kan dock bara används för andra tecknen utom som första eller sista tecken.

Anpassade HTTP- eller HTTPS-portar

Med anpassade HTTP- eller HTTPS-portar kan en applikation öppnas genom att ange en adress såsom http://diskstation:7000 eller http://diskstation:7001.

Att aktivera en anpassad HTTP- eller HTTPS-port:

  1. Välj en applikation.
  2. Klicka på Redigera.
  3. I fönstret som visas, markera Aktivera anpassad port (HTTP) eller Aktivera anpassad port (HTTPS).
  4. Specificera ett anpassat portnummer.

Obs!

  1. Portnumret kan vara från 1 till 65535, men följande portar ska inte användas eftersom de är reserverade för systembruk: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule-standardportar: 4662 (TCP), 4672(UDP), och standardportintervall för FTP: Det faktiska intervallet kan variera beroende på modell.
