Schemaläggare för uppgifter

Schemaläggaren för uppgifter låter dig schemalägga och köra tjänster eller skript på fördefinierade tider.

Uppgifter som stöds

Uppgifter kan skapas för att utföra följande:

  • Användardefinierat skript: Köra användardefinierade skript eller kommandon.
  • Papperskorg: Töm Papperskorgen för den specificerade delade mappen. Dessutom låter inställningarna för Bevarandeprincip dig kontrollera vilka filer som raderas permanent vid tömning av papperskorgen. Till exempel kan filer raderas eller behållas beroende på storlek, borttagningstid eller filformat.
  • Ljudkontroll: Gör så att DiskStation gör pipljud.
  • Tjänst: Startar eller stoppar nätverkstjänster såsom Windows File Service, FTP, Auto Block, etc.

Hantera uppgifter

Huvudsidan för schemaläggare för uppgifter visar alla befintliga uppgifter och tillhandahåller alternativ för att skapa, redigera eller köra uppgifter.

Skapa en uppgift:

  1. Klicka på Skapa och välj den typ av uppgift du vill schemalägga.
  2. När fönstret för Skapa uppgift visas, ändra följande inställningarna på fliken Allmänt:
    • Allmänna inställningar: Ange ett namn för uppgiften, välj en användare och aktivera eller avaktivera uppgiften.
    • Andra inställningar: Ytterligare uppgiftsspecifika inställningar kan ändras på denna flik (endast för vissa uppgifter). Till exempel när en uppgift att tömma papperskorgen skapas, kan du behöva välja papperskorgen för en särskild delad mapp.
  3. På fliken Schema, ändra följande inställningar:
    • Datum: Specificera när du vill köra uppgiften. Uppgifter kan schemaläggas att köras vissa dagar i veckan (t.ex. dagligen, veckoslut, varje söndag, etc.) eller på specifika datum (t.ex. 2015-10-21).
    • Tid: Specificera hur många gånger uppgiften ska köras varje dag, så väl som den första och sista tiden för körning.
  4. Ändra uppgiftsspecifika inställningar på fliken Inställningar för uppgifter (endast för vissa uppgifter).
  5. Klicka på OK för att slutföra

Redigera inställningar för uppgifter:

  1. Välj en befintlig uppgift.
  2. Klicka på Redigera.
  3. Ett fönster där du kan ändra inställningarna för uppgifter visas.

Radera en uppgift:

  1. Välj en befintlig uppgift.
  2. Klicka på Ta bort.

Aktivera eller avaktivera en uppgift:

  1. På huvudsidan för Schemaläggare för uppgifter, markera eller avmarkera den lämpliga kryssrutan under Aktiverad för att aktivera eller avaktivera en uppgift.
  2. Klicka på Spara.

Manuellt köra en uppgift:

  1. Välj en eller flera befintliga uppgift(er).
  2. Klicka på Kör.

Begränsningar för schemaläggaren för uppgifter:

  • Namn på uppgifter kan inte vara längre än 512 visningsbara tecken.
  • Maximalt antal uppgifter: 1024.
  • Maximal längd på sökväg för skript eller kommandorad: 4095.
  • Längd på pipljud kan ställas in från 1 till 600 sekunder.
