DSM-Inställningar

I Kontrollpanel > Nätverk > DSM-inställningar kan du ändra portnumret som klienter använder för att få åtkomst till DSM. När HTTPS är aktiverat kommer anslutningarna att krypteras med SSL/TLS.

Ändra standardportnumret:

  1. Ange portnumret i fältet HTTP eller HTTPS.
  2. Klicka på Tillämpa.

Aktivera HTTPS-anslutningar:

  1. Markera Aktivera HTTPS-anslutning.
  2. Om du vill omdirigera klienter till HTTP-porten (5001) vid försök att få åtkomst till DSM via HTTP-porten (5000), markera rutan Omdirigera HTTP-anslutningar direkt till HTTPS.
  3. Klicka på Tillämpa.

Portar som används för HTTPS-anslutning:

  • DSM: Standardportnumret är 5001 så att du kan använda säker åtkomst till DSM genom att ange följande adress: https://server-hostname:5001/
  • Web Station och Photo Station: Standardportnumret är 443 så du kan använda säker åtkomst till åt Web Station och Photo Station genom att ange följande adress: https://server-hostname/

För att göra så att det går att upptäcka DiskStation via Windows nätverksidentfiiering:

  1. Markera Aktivera Windows nätverksidentifiering.
  2. Klicka på Tillämpa.
