Loggsökning

På sidan Loggcentral > Loggsökning kan du titta på, söka efter och exportera loggfiler.

För att söka loggfiler:

  1. Välj att visa lokala systemloggar eller loggfiler från andra enheter genom att välja flik Lokal eller Från andra servrar.
  2. Om nödvändigt, välj att visa Senaste loggfilerna eller öppna ett befintligt loggarkiv genom att välja Öppna ett loggarkiv.
  3. I rullgardinsmenyn för sökning kan du specificera följande situationsbundna kriteria:
    • Nyckelord: Ange nyckelord för att söka efter loggmeddelanden som innehåller överensstämmande nyckelord.

      Obs!

      Du kan använda reguljära uttryck såsom [*] (dvs. okänt antal av tecken, vilka som helst) eller [?] (dvs. ett tecken, vilket som helst) i fältet nyckelord, vid sökning. Till exempel kan [a*n] representera [aktion], [attention] eller [an].

    • Datumintervall: Sök efter loggfiler efter det specificerade datumintervallet.
    • Nivå: Sök efter loggfiler efter den specificerade nivån.
    • Värdnamn: Sök efter loggfiler efter det specificerade värdnamnet.
    • Program: Sök efter loggfiler efter det specificerade programnamnet.
    • Kategori: Sök efter loggfiler efter den specificerade programkategorin.
  4. Klicka på Sök för att söka efter loggfiler som överensstämmer med specificerat kriteria.

För att exportera loggfiler:

  1. Klicka på pilen bredvid knappen Exportera.
  2. Välj HTML eller CSV för att ladda ned loggfilerna i ditt önskade format.

Obs!

  • Teckenuppsättning för exporterad loggfiler i CSV är UTF-8. Du kommer att behöva öppna filen med ett program som har stöd för teckentabell UTF-8.
