Учетная запись

На вкладке Учетная запись содержатся параметры для редактирования основных настроек учетной записи, например описания, пароля, электронной почты и 2-этапной проверки. Для просмотра данной вкладки нажмите значок в виде человека в правом верхнем углу рабочего стола и выберите Параметры.

Изменение настроек учетной записи

Здесь можно изменить следующие настройки учетной записи:

  • Описание: Введите описание учетной записи. Описание пользователя может содержать не более 64 отображаемых символов Unicode.
  • Новый пароль: Чтобы изменить пароль учетной записи, введите новый пароль и подтвердите его в поле Подтвердите пароль ниже. Он может быть длиной до 127 знаков и включать буквы, цифры, символы и пробелы с учетом регистра.
  • Эл. почта: Введите адрес электронной почты (только для обычных пользователей, не администраторов).
  • Язык отображения: Укажите язык отображения для учетной записи. Указанные здесь язык отображения будет иметь более высокий приоритет, чем язык, выбранный в пункте Панель управления > Региональные параметры > Язык.

После редактирования настроек учетной записи нажмите OK, чтобы сохранить изменения.

2-этапная проверка

Проверка в 2 этапа обеспечивает повышенную безопасность для вашей учетной записи DSM. После активации 2-этапной проверки помимо одноразового проверочного кода вам потребуется пароль для выполнения входа в DSM. Проверочный код можно получить через приложения проверки подлинности, установленные на мобильном устройстве. Поэтому, если злоумышленник захочет получить доступ к вашей учетной записи, ему потребуется не только ваше имя пользователя и пароль, но и мобильное устройство.

Требования:

2-этапная проверка требует наличия мобильного телефона и приложения проверки подлинности, которое поддерживает протокол Time-based One-Time Password (TOTP). Приложения проверки подлинности включают Google Authenticator (Android/iPhone/BlackBerry) и Authenticator (Windows Phone).

Включение 2-этапной проверки:

  1. Выберите Параметры > Учетная запись и установите флажок Включить 2-этапную проверку для запуска мастера настройки 2-этапной проверки.
  2. В первом окне введите адрес электронной почты. Если мобильное устройство утеряно, экстренный проверочный код может быть отправлен на указанный здесь адрес электронной почты.
  3. Загрузите и установите приложения проверки подлинности, такие как Google Authenticator (Android/iPhone/BlackBerry) и Authenticator (Windows Phone).
  4. В следующем окне откройте приложение проверки подлинности и выполните поиск кода QR. Либо нажмите ссылку, чтобы вручную ввести секретный ключ.
  5. Затем приложение проверки подлинности создаст 6-значный проверочный код. Введите его в текстовое поле мастера, чтобы подтвердить правильность конфигураций. При возникновении ошибки убедитесь, что системное время на мобильном устройстве синхронизировано с системным временем DSM. Кроме того, проверочные коды периодически обновляются, поэтому убедитесь, что вы ввели действительный код.
  6. После завершения работы мастера нажмите OK, чтобы сохранить изменения.

Выполнение входа в DSM с помощью 2-этапной проверки:

После активации 2-этапной проверки вам потребуется ввести 6-значный проверочный код при выполнении входа в DSM.

  1. На экране входа в DSM введите имя пользователя и пароль, как обычно.
  2. При запросе ввода проверочного кода откройте приложение проверки подлинности на мобильном устройстве.
  3. Найдите и введите 6-значный проверочный код для учетной записи. Если мобильное устройство утеряно, нажмите ссылку Потеряли телефон?, и экстренный код будет отправлен на ваш адрес электронной почты.

Примечание.

  • Настройки SMTP: Чтобы получить проверочный код по электронной почте, необходимо правильно настроить параметры SMTP сервера в пункте Панель управления > Уведомление.
  • Ограничение на экстренный код: Каждый пользователь может получить не более 5 экстренных кодов. При превышении этого лимита вам потребуется отключить и заново включить 2-этапную проверку перед получением дополнительных экстренных кодов.

Сброс пользовательских настроек 2-этапной проверки:

Пользователи, принадлежащие к группе administrators, могут выполнять сброс настроек 2-этапной проверки других пользователей.

  1. Выберите Панель управления > Пользователь.
  2. Дважды нажмите пользователя, 2-этапную проверку которого необходимо сбросить.
  3. На вкладке Информация о пользователе нажмите кнопку Сброс 2-этапной проверки.

Примечание.

Если пользователь, принадлежащий к группе administrators, недоступен, нажмите кнопку сброса на панели устройства DiskStation для сброса настроек DSM и отключения 2-этапной проверки.

Активность учетной записи

Активность учетной записи отображает последние сведения о входе в учетную запись DSM, например текущие соединения или журнал входов.

Проверка активности учетной записи:

Нажмите кнопку Активность учетной записи. В отобразившемся окне будут приведены следующие вкладки:

  • Подключенные пользователи: На данной вкладке отображается следующая информация:
    • Дата и время: Время доступа к учетной записи.
    • Название клиента и IP-адрес: Название и IP-адрес устройства, с которого осуществляется доступ к учетной записи. Название клиента отображается только тогда, когда доступ к учетной записи осуществляется с помощью протокола CIFS.
    • Служба: Протокол, используемый для доступа к учетной записи.
    • Источник: Системные ресурсы, используемые для доступа к учетной записи.
  • Журнал входов: На данной вкладке отображаются последние сеансы входа и следующая информация:
    • Дата и время: Время доступа к учетной записи.
    • Событие: Отображает имя пользователя и IP-адрес.
