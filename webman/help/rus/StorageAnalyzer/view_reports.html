Просмотреть отчеты об использовании

С помощью анализатора системы хранения можно не только отслеживать общие тенденции использования разделов, но и посмотреть подробный анализ системы хранения, сгенерированный из запланированных задач, для получения дополнительной информации о пространстве для хранения данных и более рационального использования DiskStation.

Использование раздела

В разделе Использование раздела находится линейная диаграмма, отображающая тенденции использования раздела и изменения во времени, или столбиковая диаграмма, определяющая процент текущего использования, если параметр ведения журнала использования раздела выключен (Профили отчета > Настройки > Сохранение истории использования раздела по расписанию.).

Чтобы посмотреть информацию об использовании раздела, выполните одно из следующих действий:

  • Наведите курсор на строку в линейной диаграмме для просмотра использованной, доступной и общей емкости раздела для указанной даты.
  • Нажмите кнопки 1 и 2 в правом верхнем углу для переключения между двумя режимами отображения (линейная диаграмма и список) в соответствии со своими предпочтениями. Линейная диаграмма позволяет быстро просматривать общие тенденции использования за короткий период, а список содержит самые последние собранные сведения.

Примечание.

Если на вашем DiskStation есть более 10 разделов, здесь отображаются только первые десять разделов с наиболее высоким уровнем использования.

Отчеты об использовании

Помимо информации о текущем использовании разделов можно также посмотреть различные типы отчетов об использовании, созданные из предыдущих задач, или легко найти нужный отчет по определенному критерию или дате всего за несколько щелчков мыши. Аналитические отчеты делятся на разные типы, включая использование квоты каждого типа папки или файла, а также использование файлов в соответствии со временем изменения или доступа.

Чтобы посмотреть отчеты об использовании, выполните следующие действия:

  1. В разделе Профиль отчета выберите задачу из списка профилей, затем дважды щелкните задачу или выберите Анализировать эту задачу отчета (значок >) рядом с нужной задачей для просмотра собранных данных и анализа этой задачи. (Обратите внимание, что задачи, которые ранее не выполнялись, не могут быть выбраны.)
  2. Щелкните отдельные вкладки на левой панели страницы отчета для просмотра соответствующего анализа:
    • Использование квоты: Отображение квоты, занятой каждой папкой общего доступа.
    • Квота пользователей: Отображение количества файлов, принадлежащих каждому пользователю в папке общего доступа, и их размера.
    • Тип: Отображение квоты каждого типа файлов в каждой папке общего доступа.
    • Копии: Отображение дублирующихся файлов, найденных в системе, для удобного упорядочивания или удаления ненужных копий. Обратите внимание, что чем больше заданное максимальное число дублирующихся файлов, тем больше времени занимает обработка.
    • Размер: Отображение до 500 файлов по убыванию размера.
    • Последние измененные файлы: Отображение до 500 файлов по убыванию даты последнего изменения.
    • Редко используемые файлы: Отображение до 500 файлов по возрастанию даты последнего доступа.
  3. При необходимости выберите критерий из раскрывающегося меню для более эффективной фильтрации отчетов и поиска необходимой информации.
  4. Нажмите на любую дату на временной шкале внизу для просмотра отчетов по выбранной дате.
  5. В качестве альтернативы можно нажать Просмотреть полный отчет в правом верхнем углу, чтобы посмотреть полный анализ на одной странице.
