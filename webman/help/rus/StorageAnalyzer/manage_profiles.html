Управление профилями отчетов

Вы можете создавать и управлять запланированными задачами для создания отчетов по использованию разделов в пункте Анализатор системы хранения > Профиль отчетов.

Создание отчета:

  1. Нажмите Создать, чтобы открыть мастер.
  2. Укажите имя нового отчета и введите адрес электронной почты, на который вы хотите отправить отчет.
  3. Выберите один или несколько типов отчетов для создания.
  4. Выполните одно из следующих действий, чтобы указать время создания отчетов:
    • Установите флажок Создавать отчеты по расписанию и укажите время, если вы хотите, чтобы система периодически выполняла анализ использования раздела.
    • Установите флажок Создать отчеты сейчас, если вы хотите вручную запустить отчет сейчас.
  5. При необходимости укажите пользователей или папки общего доступа, чтобы уменьшить число отчетов. (Обратите внимание, что если вы хотите указать и создать отчеты по защищенным папкам общего доступа, убедитесь, что защищенные папки были созданы до выполнения отчета.)
  6. Нажмите Применить, чтобы сохранить настройки.

Примечание.

  • Нажимайте Ввод/Возврат после каждого адреса электронной почты, чтобы разделить несколько адресов.
  • Нажимайте Ввод/Возврат после каждого пользователя при задании пользователей, файлы которых будут перечислены отдельно по группам файлов.
  • Отчет Файлы по группе файлов будет содержать следующие типы файлов:
    Аудио, файлы образов дисков, документы, исполняемые файлы, изображения, видео, веб-страницы, архивы и другое
  • Раздел Большие файлы отчетов содержит самые большие 500 файлов на странице отчета об использовании или 200 самых больших файлов на странице завершенных отчетов.

Редактирование отчета:

  1. Выберите отчет из списка и нажмите Редактировать.
  2. Нажмите отдельные вкладки, чтобы изменить настройки отчета.

Удаление отчета:

Выберите отчет из списка и нажмите Удалить.

Просмотр отчетов из истории:

  1. Выберите отчет из списка и нажмите Отчет > Просмотреть отчеты из истории.
  2. В отобразившемся окне выберите время, в которое были созданы необходимые вам отчеты.

Немедленное создание отчетов:

Выберите отчет из списка и нажмите Отчет > Создать отчеты сейчас, чтобы создать отчеты по требованию.

Изменение места сохранения отчетов:

  1. Нажмите Настройки.
  2. Выберите новую папку места назначения, чтобы изменить место сохранения отчетов.

Регулярный обзор истории использования раздела:

По умолчанию сведения об использовании раздела собираются при каждом создании отчета об использовании раздела и запуске системы. Кроме того, вы можете создать новое расписание для сбора данных об использовании. Для этого нажмите Настройки и установите флажок Собирать историю об использовании раздела в соответствии с расписанием и запланируйте время.

Примечание.

История, собираемая в запланированное время, будет в основном использоваться для отслеживания тенденций использования в разделе Использование раздела. При включении данного параметра вы увидите линейный график, отображающий тенденции использования раздела и изменения; либо вместо него может отображаться диаграмма текущего процента использования.

Анализ отчета:

Дважды щелкните задачу резервирования или просто нажмите Анализировать эту задачу отчета (значок >) рядом с необходимым отчетом для открытия страницы анализа. См. Просмотреть отчеты об использовании для получения дополнительной информации.
