Синхронизация файлов между NAS-сервером Synology и вашим компьютером с помощью Cloud Station

Обзор

Cloud Station для NAS-сервера Synology — это приложение для синхронизации файлов NAS-сервера Synology с другими устройствами: компьютерами или мобильными устройствами (с приложением DS cloud). После установки Cloud Station на компьютер и ваш NAS-сервер Synology файлы, хранящиеся в специальной папке на вашем компьютере, будут автоматически синхронизированы на NAS-сервер.

1. Установка Cloud Station

  1. Откройте Центр пакетов, найдите Cloud Station и нажмите Установить.

2. Включение Cloud Station

  1. Запустите Cloud Station.
  2. Если вы не активировали функцию главной папки пользователя, вам будет предложено это сделать. Нажмите OK.
  3. Вы будете перенаправлены на страницу настроек пользователя. Активируйте homes пользователей и нажмите Применить.
  4. Вернитесь в Cloud Station, вам будет предложено активировать приложение. Нажмите Да.
  5. Выберите Права доступа и укажите пользователей, которые будут иметь доступ к Cloud Station. Нажмите Сохранить.

3. Установка и активация клиента

  1. Выберите Cloud Station > Обзор и нажмите кнопку Компьютеры.
  2. Нажмите кнопку, чтобы загрузить Cloud Station для вашей ОС. (Вы также можете загрузить Cloud Station из Центра загрузок Synology.)
  3. Запустите на компьютере средство установки и следуйте инструкциям на экране.
  4. После установки запустите Cloud Station на компьютере. Нажмите Запустить сейчас.
  5. Введите адрес NAS-устройства Synology, на котором установлено приложение Cloud Station (или QuickConnect ID), имя пользователя и пароль. Чтобы найти IP-адреса в локальной сети, воспользуйтесь значком поиска справа. Нажмите Далее.
  6. Выберите настройки задачи (здесь мы выбираем Быстрая настройка) и нажмите Далее.
  7. Нажмите Готово для завершения настройки вашей папки Cloud Station.
  8. Чтобы открыть папку Cloud Station, нажмите Открыть сейчас.
  9. На панели задач компьютера появится значок Cloud Station.
  10. Чтобы открыть меню, дважды нажмите на значок Cloud Station. Здесь отображается состояние синхронизации и статус ваших файлов. Для запуска Cloud Station нажмите Главное приложение.
  11. После запуска вы увидите рекомендации по работе с Cloud Station. Нажмите на стрелку вправо, чтобы просмотреть дополнительные рекомендации, либо Читать руководства в Интернете для получения более подробной информации. Если вы не нуждаетесь в рекомендациях, нажмите Больше не показывать.
  12. Теперь вы можете управлять задачами синхронизации в Cloud Station.

4. Синхронизация данных с мобильными устройствами

DS cloud можно бесплатно скачать в Apple App Store и Google Play Store. Также можно просканировать QR-код для мгновенной загрузки.
