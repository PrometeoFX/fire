Как резервировать данные на ПК/Mac с помощью Cloud Station

Обзор

Cloud Station – современная кроссплатформенная программа синхронизации – может также использоваться как гибкий инструмент резервирования пользовательских данных. Использование Cloud Station в таком качестве дает ряд преимуществ. Прежде всего, репликация данных в режиме реального времени улучшает показатели целевых точек восстановления и уменьшает период времени, в течение которого в результате чрезвычайной ситуации данные компьютера могут быть потеряны без возможности восстановления. Cloud Station не только осуществляет мгновенное резервирование данных, но и существенно снижает использование пропускной способности за единицу времени по сравнению с традиционным методом резервирования. На клиентском компьютере в папке для резервирования можно выбрать файлы, которые резервировать не нужно, отфильтровав их по размеру или типу файла. Кроме того, Cloud Station может хранить до 32 версий файла и восстанавливать их с NAS-сервера на компьютер.

1. Настройка Cloud Station на NAS Synology

  1. Выполните вход в DSM с учетной записью пользователя, принадлежащего группе administrator.
  2. Запустите Cloud Station.
  3. Если вы не активировали функцию главной папки пользователя, вам будет предложено это сделать. Нажмите OK.
  4. Вы будете перенаправлены на страницу настроек пользователя. Активируйте homes пользователей и нажмите Применить.
  5. Вернитесь в Cloud Station, вам будет предложено активировать приложение. Нажмите Да.
    6. Примечание. QuickConnect — это служба ретрансляции, которая позволяет легко подключаться к NAS-серверу Synology вне локальной сети. Чтобы синхронизировать файлы, находясь вдали от дома или офиса, рекомендуем установить QuickConnect, выбрав Панель управления > QuickConnect.
  6. Выберите Права доступа и укажите пользователей, которые будут иметь доступ к Cloud Station. Нажмите Сохранить.
  7. Настройте папку общего доступа на NAS-устройстве Synology для целей резервирования. Выберите Панель управления > Папка общего доступа и нажмите Создать.
  8. Задайте имя папки и заполните соответствующие поля. Нажмите OK. Например, назовем папку Резервирование.
  9. На вкладке Разрешения укажите пользователей, которые будут иметь право доступа к папке. Например, зададим пользователю admin права на чтение/запись. Нажмите OK.
  10. Выберите Cloud Station > Настройки > Общий доступ, выберите папку Резервирование и нажмите Включить.

2. Настройка Cloud Station на ПК под управлением Windows

Для настройки Cloud Station на ПК или Mac выполните следующие действия

2.1 Для ПК

  1. Выберите Cloud Station > Обзор и нажмите кнопку Загрузить.
  2. Нажмите кнопку, чтобы загрузить Cloud Station для ПК под управлением Windows. (Вы также можете загрузить Cloud Station из Центра загрузок Synology.)
  3. Запустите на компьютере средство установки и следуйте инструкциям по установке Cloud Station.
  4. После установки запустите Cloud Station на компьютере. Нажмите Запустить сейчас.
  5. Введите адрес NAS-устройства Synology, на котором установлено приложение Cloud Station (или QuickConnect ID), имя пользователя и пароль. (Можно также нажать на значок поиска справа, и Cloud Station автоматически начнет искать другие NAS Synology в вашей локальной сети.) Нажмите Далее.
    6. Примечание. Для указанной учетной записи должны быть заданы права доступа в меню Cloud Station > Права доступа NAS Synology.
  6. Укажите параметры настройки задачи (здесь мы выберем Расширенная настройка, так как настраиваем специальную папку для резервирования). Нажмите Далее.
  7. Выберите папку на компьютере, чтобы синхронизировать ее с папкой "Резервирование", созданной на NAS-устройстве Synology. Нажмите Далее.
  8. Нажмите Готово для завершения настройки.

Создание новой задачи:

  1. Выберите Cloud Station > Настройки и включите папку общего доступа для новой задачи. В нашем примере это папка photo.​ Убедитесь, что вы имеете права на чтение/запись для папки photo.
  2. В программе Cloud Station на своем компьютере нажмите Создать.
  3. Выберите NAS Synology для новой задачи. Здесь мы выберем Подключенное NAS-устройство Synology, так как создаем новую задачу для того же NAS-устройства Synology. Нажмите Далее.
  4. Нажмите Расширенная настройка и следуйте инструкциям мастера для завершения настройки.

2.2 Для Mac

  2. Нажмите кнопку, чтобы загрузить Cloud Station для Mac. (Вы также можете загрузить Cloud Station из Центра загрузок Synology.)

    6. Примечание. Для указанной учетной записи должны быть заданы права доступа в меню Cloud Station > Права доступа NAS Synology.
  6. Укажите параметры настройки задачи (здесь мы выберем Расширенная настройка, так как настраиваем специальную папку для резервирования). Нажмите Далее.
  7. Выберите папку на компьютере, чтобы синхронизировать ее с папкой "Резервирование", созданной на NAS-устройстве Synology. Нажмите Далее.
  8. Нажмите Готово для завершения настройки.

3. Извлечение данных из Cloud Station

Если вы случайно удалили или перезаписали файл, его можно восстановить с помощью Cloud Station.

4.1 Загрузка предыдущих версий синхронизированного файла на ПК

  1. Нажмите значок Cloud Station на панели задач компьютера и нажмите значок папки.
  2. Нажмите нужный файл правой кнопкой мыши и выберите Synology Cloud Station > Обзор предыдущих версий.
  3. Выберите версию для загрузки и нажмите значок загрузки.
  4. Введите имя файла и выберите место для сохранения файла.

4.1 Загрузка предыдущих версий синхронизированного файла на Mac

  1. Нажмите значок Cloud Station на панели задач компьютера и нажмите значок папки.
  2. Нажмите нужный файл правой кнопкой мыши и выберите Synology Cloud Station > Обзор предыдущих версий.
  3. Выберите версию для загрузки и нажмите значок загрузки.
  4. Введите имя файла и выберите место для сохранения файла.

4.3 Загрузка и восстановление предыдущей версии синхронизированного файла из Cloud Station

  1. Выберите Cloud Station > Журнал версий и выберите созданную ранее папку Резервирование.
  2. Выберите файл для извлечения и нажмите Действие > Обзор предыдущих версий.
  3. Выберите версию файла, которую вы хотите загрузить, и нажмите Загрузить. Если вы хотите восстановить выбранную версию файла, нажмите Восстановить, чтобы перезаписать текущую версию файла.

4. Важно!

  1. Обязательно сохраняйте предыдущие версии файла – так вы сможете его восстановить в случае случайного удаления или перезаписи.
  2. В меню "Глобальные настройки" вы можете настроить действия, которые будут выполняться на NAS при удалении файла с компьютера: извлечение версии файла или полное его удаление.
