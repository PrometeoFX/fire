Индексные файлы

Индексирование содержимого папки общего доступа позволяет быстрее находить необходимые файлы или папки при поиске.

Индексирование файлов в папке общего доступа:

  1. Выберите папку общего доступа, которую необходимо проиндексировать.
  2. Нажмите Редактировать и выберите вкладку Индексирование файла.
  3. Установите флажок в поле Включить индексирование файла.
  4. Нажмите OK.

Переиндексирование файлов

Переиндексирование файлов не требуется в нормальных условиях работы. Однако, если файлы в папке соответствуют критериям, но не находятся при поиске файлов, необходимо выполнить переиндексирование для изменения структуры индекса файлов.

Индексирование файлов в папке общего доступа:

  1. Выберите папку общего доступа, которую необходимо переиндексировать.
  2. Нажмите Редактировать и выберите вкладку Индексирование файла.
  3. Нажмите Переиндексировать.
  4. После завершения процесса система отобразит количество проиндексированных файлов и подпапок.

Примечание. Индексирование файла в удаленных папках или на внешних дисках USB не поддерживаются File Station.
