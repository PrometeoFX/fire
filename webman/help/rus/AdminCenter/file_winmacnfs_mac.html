Файловая служба Mac

В меню Панель управления > Файловые службы > Win/Mac/NFS вы можете включить файловую службу Mac, чтобы сделать DiskStation файловым сервером Mac (AFP-сервером).

Включение файловой службы Mac:

  1. Установите флажок Включить файловую службу Mac, чтобы сделать DiskStation файловым сервером Mac (AFP-сервером).
  2. При необходимости укажите папку для резервного копирования Time Machine. (Для получения дополнительной информации см. раздел ниже.)
  3. Нажмите Применить.

О папке резервирования Time Machine:

Чтобы выполнить резервное копирование данных из Time Machine на DiskStation, выберите папку общего доступа из раскрывающегося меню.

Примечание. Папки общего доступа, используемые другими службами, будут недоступны для Time Machine.

Дополнительные настройки

Применение разрешений UNIX по умолчанию.

Включение данного параметра позволяет применять разрешения UNIX по умолчанию при выгрузке или создании папок и файлов. Те же разрешения можно применить с помощью команды UNIX umask. Если этот параметр включен, для файлов используется разрешение UNIX 644, а для папок — разрешение 755. Если этот параметр отключен, для файлов используется разрешение UNIX 666, а для папок — разрешение 777. По умолчанию параметр unmask имеет значение 022.

Примечание. Включение данного параметра может привести к несогласованности разрешений между различными протоколами. Во избежание несогласованности рекомендуется не включать данный параметр.

Немедленное освобождение ресурсов после отключения.

Включение данного параметра позволяет немедленно освобождать ресурсы при непредвиденном отключении сервера AFP. Включите этот параметр, если вы получаете сообщение об ошибке "Том используется" при использовании Time Machine.

Включение вещания принтера Bonjour:

Установите флажок Включить вещание принтера Bonjour, чтобы разрешить клиентам Mac просматривать и находить принтер, подключенный к DiskStation.
