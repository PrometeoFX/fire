Редактирование пользователя

Найдите учетную запись пользователя (введите имя пользователя в поле Поиск), дважды нажмите на нее или нажмите кнопку Редактировать, после этого вы сможете редактировать данные учетной записи.

    Примечание:
  1. Если отключить учетную запись, она перестанет функционировать.
  2. Значение Квоты должно быть целым числом, быть больше или равно 0 ГБ и быть меньше емкости диска/тома. Если значение квоты равно 0, то Эффективная квота будет определяться Лимитом группы.
  3. Если на сервере несколько томов/общих папок, можно установить квоту для пользователя по каждому тому/общей папке.
  4. Адрес электронной почты на странице «Уведомления» должен быть указан правильно, чтобы система могла уведомлять пользователя по электронной почте, после создания или изменения учетной записи.
  5. При установке ограничения скорости для пользователя в пункте Результат отобразится уже установленное ограничение скорости для данного пользователя, если применимо.
  6. Ограничение скорости, назначенное пользователю, не будет применяться к группе, к которой принадлежит данный пользователь.
