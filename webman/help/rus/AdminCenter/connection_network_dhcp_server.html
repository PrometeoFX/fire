DHCP Server

Данный параметр позволяет DiskStation выполнять роль сервера DHCP и назначать динамические IP-адреса для клиентов DHCP в локальной сети.

Включение DHCP-сервера.

  1. Выберите Панель управления > Сеть > Сетевой интерфейс.
  2. Выберите сетевой интерфейс (например, ЛВС) и нажмите Редактировать.
  3. Перейдите на вкладку DHCP Server и установите флажок Включить DHCP Server.
  4. Введите значение в поле Время аренды адреса для установки срока аренды DHCP (в минутах) для IP-адресов, назначенных клиентам DHCP.
  5. Введите адрес DNS для клиентов DHCP в полях Основной DNS (обязательно) и Дополнительный DNS (дополнительно).
  6. Установите имя домена для DHCP-сервера в поле Имя домена (дополнительно).
  7. Нажмите OK.

Добавление подсети.

  1. Нажмите Добавить, а затем укажите информацию в следующих столбцах.
    • Включить: Включите или отключите подсеть.
    • Пуск: Первый IP-адрес подсети.
    • Окончание: Последний IP-адрес подсети.
    • Маска подсети: Маска подсети.
    • Шлюз: Шлюз подсети.
  2. Нажмите OK.

Примечание.

  • Вы можете добавлять до 32 подсетей.

Клиенты DHCP

Чтобы просмотреть список клиентов DHCP и их сетевые конфигурации (например, MAC-адрес, IP-адрес, имя хоста и оставшееся количество времени перед окончанием аренды DHCP), перейдите на вкладку Клиенты DHCP.

Резервирование DHCP

Чтобы клиент всегда получал тот же самый IP-адрес во время обновления аренды DHCP, можно добавить клиента в список зарезервированных адресов.

Добавление клиента в список зарезервированных адресов.

  • Нажмите Добавить.
  • Укажите MAC-адрес клиента, IP-адрес, который необходимо назначить клиенту и имя хоста клиента.
  • Нажмите OK, чтобы сохранить изменения.
