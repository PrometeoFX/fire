Резервирование конфигурации

Чтобы создать резервную копию конфигурации на локальном компьютере, откройте Панель управления > Обновить и восстановить. Вы сможете сохранить различные настройки (пользователей, группы, настройки файловой службы и т.д.) для последующего восстановления. Системная конфигурация сохраняется в файл формата .dss.

Резервирование системных конфигураций:

Восстановление системных конфигураций:

    Будет выполнено резервирование следующих настроек:

    • Пользователи, группы и папки общего доступа
    • Рабочая группа, домен и LDAP
    • Службы файлового обмена (например, Windows File Service, Mac File Service, NFS Service, FTP и WebDAV) и резервирования (например, Network Backup, Shared Folder Sync и Time Backup Service)
    • Другие настройки (например, настройки главной папки пользователя, пароля, планировщика задач, отчета об использовании диска)
    • Пользователи, группы и папки общего доступа
    • Службы файлового обмена (например, Windows File Service, Mac File Service, NFS Service, FTP и WebDAV) и резервирования (например, Network Backup, Shared Folder Sync и Time Backup Service)
    • Другие настройки (например, настройки главной папки пользователя, пароля, планировщика задач, отчета об использовании диска)

    Примечание.

    • Службы резервирования содержат базовые настройки служб и не включают в себя сведения о задачах резервирования, местах назначения и т.д.
    • Главная папка пользователя содержит базовые настройки служб, но не включает в себя общую папку homes. Выберите пункт Резервирование и репликация для резервирования и восстановления общей папки homes и ее содержимого.
    • Планировщик задач содержит задачи, которые можно создать в Планировщике задач, но не включает в себя задачи, созданные другими приложениями (например, Резервирование и репликация и Отчет об использовании диска)
    • Отчет об использовании диска содержит задачи и график отчетов, но не включает в себя созданные ранее отчеты.
    • Если имена восстановленных пользователей или групп зарезервированы для системного использования, при восстановлении они будут изменены.
