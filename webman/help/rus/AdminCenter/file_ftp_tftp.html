TFTP/PXE

Параметры в пункте Панель управления > Файловые службы > TFTP/PXE позволяют настраивать параметры TFTP и PXE.

TFTP

TFTP (Trivial File Transfer Protocol) — простой протокол передачи файлов, который обычно используется для передачи конфигураций или загрузки файлов, когда проверка подлинности не требуется. Включение службы TFTP позволяет DiskStation передавать загрузчики на клиентские устройства.

Изменение настроек TFTP:

  1. Установите флажок Включить службу TFTP.
  2. В поле Корневая папка TFTP укажите папку на DiskStation, к которой будут иметь доступ клиенты TFTP.
  3. Нажмите кнопку Дополнительные настройки, чтобы просмотреть и изменить следующие настройки:
    • Включить журнал передачи по TFTP: Данный параметр позволяет отслеживать передачу данных по TFTP.
    • Права клиента TFTP: Выберите для клиента TFTP права на чтение и запись. Если выбран параметр Только для чтения, клиенты TFTP смогут только просматривать содержимое корневой папки. Если выбран параметр Для записи, клиенты TFTP смогут изменять содержимое корневой папки.
    • Разрешенные клиенты: Выберите клиентов, которые смогут подключаться с помощью TFTP. Вы можете разрешить подключение для всех клиентов, выбрав Все подключения; либо ограничить подключение клиентов, указав определенный диапазон IP-адресов.
  4. Нажмите Применить, чтобы сохранить настройки.

Настройка службы DHCP для PXE с помощью DiskStation

Помимо TFTP устройство DiskStation также предоставляет службу DHCP для выполнения комплексной установки PXE. PXE (Pre-boot Execution Environment) – это сетевая среда, которая позволяет клиентским устройствам выполнять загрузку с помощью сетевого интерфейса, а не устройства хранения данных (например, жестких дисков). При внедрении PXE загрузчик можно хранить на DiskStation и загружать клиентские устройства по сети с помощью этого загрузчика.

Для предоставления служб DHCP с помощью DiskStation установите пакет Synology DHCP Server.

Настройка службы DHCP для PXE:

  1. Установите флажок Настроить службу DHCP на данном сервере для PXE. Вам будет предложено установить пакет DHCP Server, если он еще не установлен на DiskStation.
  2. Выберите файл загрузчика из корневой папки TFTP и задайте другие настройки DHCP.
  3. Нажмите Применить, чтобы сохранить настройки.
