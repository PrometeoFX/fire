Безопасность

Вы можете изменить следующие настройки безопасности в пункте Панель управления > Безопасность > Безопасность для сеансов работы в DSM:

  • Таймер выхода (минут): Будет осуществлен автоматический выход пользователей из DSM, если они будут неактивны в течение определенного периода времени. Укажите значение в диапазоне от 1 до 65535.
  • Улучшить совместимость обозревателя, пропустив проверку IP-адреса: При доступе к DiskStation через прокси-сервер HTTP и возникновении произвольных завершений сеанса вы можете активировать данный параметр для пропуска проверки IP-адресов.
  • Повысить защиту от подделки межсайтовых запросов: Данный параметр повышает защиту системы от проведения атаки на основе межсайтовых сценариев. Данный параметр вступит в силу при следующем входе в DSM.
  • Запретить на DSM встраивание iFrame: Вы можете включить данный параметр, чтобы запретить другим веб-сайтам встраивание DSM в другие веб-страницы с помощью iFrame, предотвращая таким образом определенные типы атак от вредоносных веб-сайтов. Определенным веб-сайтам можно разрешить встраивание iFrame в DSM, нажав кнопку Разрешенные веб-сайты.
