Планировщик задач

Планировщик задач позволяет планировать и выполнять службы или скрипты в предварительно установленное время.

Поддерживаемые задачи

Задачи можно создать для выполнения следующих действий:

  • Скрипт, заданный пользователем: Запуск скриптов или команд, заданных пользователем.
  • Корзина: Очистка корзины заданных папок общего доступа. Кроме того, настройки функции Политика хранения позволяют управлять файлами для их полного удаления при очистке корзины. Например, файлы можно удалить или сохранить согласно их размеру, времени удаления или формату.
  • Управление звуковым сигналом: Использование устройством DiskStation звуковых сигналов.
  • Служба: Запуск или остановка сетевых служб, таких как файловая служба Windows, FTP, автоматическая блокировка и др.

Управление задачами

На главной странице планировщика задач отображаются все существующие задачи и параметры для создания, редактирования и выполнения задач.

Создание задачи:

  1. Нажмите Создать и выберите тип задачи, который вы хотите запланировать.
  2. Когда отобразится окно Создать задачу, измените следующие настройки на вкладке Общие:
    • Общие настройки: Введите название задачи, выберите пользователя и активируйте либо отключите задачу.
    • Другие настройки: Дополнительные настройки задачи можно изменить на данной вкладке (только для определенных задач). Например, при создании задачи по очистке корзины необходимо выбрать корзину определенной папки общего доступа.
  3. Нажмите вкладку Расписание, чтобы изменить следующие настройки:
    • Дата: Дата выполнения задачи. Задачи можно выполнять в определенные дни недели (например, ежедневно, по выходным, каждое воскресенье и др.) либо по определенным датам (например 21/10/2015).
    • Время: Укажите количество выполнений задачи каждый день, а также время первого и последнего выполнения.
  4. Измените настройки задачи на вкладке Настройки задачи (только для определенных задач).
  5. Для завершения нажмите OK

Редактирование настроек задачи:

  1. Выберите существующую задачу:
  2. Нажмите Редактировать.
  3. Отобразится окно, в котором можно изменять настройки задачи.

Удаление задачи:

  1. Выберите существующую задачу:
  2. Затем нажмите кнопку Удалить.

Включение и отключение задачи:

  1. На главной странице Планировщик задач установите или снимите соответствующие флажки в пункте Включено, чтобы включить или отключить задачу.
  2. Нажмите Сохранить.

Выполнение задачи вручную:

  1. Выберите одну или несколько задач.
  2. Нажмите Выполнить.

Ограничения планировщика задач:

  • Названия задач могут содержать до 512 отображаемых символов.
  • Максимальное число задач: 1024.
  • Максимальная длина пути скрипта или командной строки: 4095.
  • Продолжительность звукового сигнала может быть от 1 до 600 секунд.
