Настройки DSM

В пункте Панель управления > Сеть > Настройки DSM можно изменить номер порта, используемый клиентами для доступа к DSM. При включении HTTPS подключение будет защищено с помощью SSL/TLS.

Изменение номера порта по умолчанию:

  1. Введите номер порта в поле HTTP или HTTPS.
  2. Нажмите Применить.

Установка подключения HTTPS:

  1. Установите флажок Включить соединения HTTPS.
  2. Чтобы перенаправить клиентов на порт HTTPS (5001) при попытке доступа к DSM через порт HTTP (5000), установите флажок Автоматически перенаправлять подключения HTTP на HTTPS.
  3. Нажмите Применить.

Порты, используемые при подключении HTTPS:

  • DSM: Номер порта по умолчанию — 5001, поэтому вы можете безопасно осуществлять доступ к DSM путем ввода следующего адреса: https://server-hostname:5001/
  • Web Station и Photo Station: Номер порта по умолчанию — 443, поэтому вы можете безопасно осуществлять доступ к Web Station и Photo Station путем ввода следующего адреса: https://server-hostname/

Чтобы DiskStation обнаруживалось с помощью службы обнаружения сети Windows, выполните следующие действия:

  1. Установите флажок Включить обнаружение сети Windows.
  2. Нажмите Применить.
