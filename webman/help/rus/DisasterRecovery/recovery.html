Восстановление

На этой странице системные администраторы могут выполнять восстановление. Чтобы разрешить пользователям доступ к моментальным снимкам общих папок, включите опцию Сделать моментальный снимок видимым для каждой общей папки. Пользователи получат доступ к снимкам через приложение File Station или Проводник Windows.

Общая папка

Восстановление отдельных файлов/папок в общей папке:

  1. Выберите общую папку, которую необходимо восстановить.
  2. Нажмите Восстановить.
  3. Выберите версию моментального снимка, из которого необходимо выполнить восстановление.
  4. Нажмите Обзор и скопируйте файлы/папки, которые необходимо восстановить.

Восстановление общей папки:

  1. Выберите общую папку, которую необходимо восстановить.
  2. Нажмите Восстановить.
  3. Выберите версию моментального снимка, из которого необходимо выполнить восстановление.
  4. Нажмите Действие и выберите в раскрывающемся списке Восстановить на месте.
  5. Появится сообщение с подтверждением. Нажмите Да для подтверждения действия.

Клонирование снимка в новую общую папку:

  1. Выберите общую папку, которую необходимо клонировать.
  2. Нажмите Восстановить.
  3. Выберите версию моментального снимка, которую вы хотите клонировать в новую общую папку.
  4. Нажмите Действие и выберите в раскрывающемся списке Клонировать с новым именем.
  5. Вы будете перенаправлены на страницу Клонирование в меню Панель управления > Общая папка > Создать. Введите имя клонированной общей папки и нажмите OK для завершения.

Примечание.

  • Общую папку homes восстановить нельзя.
  • Клонированные общие папки не могут быть зашифрованы.
  • Если моментальный снимок создан во время передачи файлов, целостность данных не гарантируется.
  • Ограничения на зашифрованные общие папки:
    • Для подключенных зашифрованных общих папок недоступен ни один тип восстановления данных.
    • Параметр Клонировать с новым именем для зашифрованных общих папок недоступен.
    • Отключенные защищенные общие папки могут быть восстановлены по сделанным ранее моментальным снимкам при помощи опции Восстановить на месте.

iSCSI LUN

Восстановление iSCSI LUN:

  1. Выберите iSCSI LUN для восстановления.
  2. Нажмите Восстановить.
  3. Выберите версию моментального снимка, из которого необходимо выполнить восстановление.
  4. Нажмите Действие и выберите в раскрывающемся списке Восстановить на месте.
  5. Появится сообщение с подтверждением. Нажмите Да для подтверждения действия.

Клонирование моментального снимка в новый iSCSI LUN:

  1. Выберите iSCSI LUN, который необходимо клонировать.
  2. Нажмите Восстановить.
  3. Выберите версию моментального снимка, которую вы хотите клонировать в новый iSCSI LUN.
  4. Нажмите Действие и выберите в раскрывающемся списке Клонировать с новым именем.
  5. Вы будете перенаправлены на страницу Клонирование в меню Диспетчер хранилища > iSCSI LUN. Введите имя клонированного iSCSI LUN и нажмите OK для завершения.

Примечание.

  • Если моментальный снимок создан во время передачи файлов, целостность данных не гарантируется.
