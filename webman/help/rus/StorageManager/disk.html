HDD/SSD

Данный раздел позволяет осуществлять мониторинг состояния жесткого диска/твердотельного накопителя (HDD/SSD), установленного на DiskStation, предоставляя параметры для управления и анализа производительности и состояния жесткого диска/твердотельного накопителя (HDD/SSD). Выберите Диспетчер хранения > HDD/SSD для управления этими параметрами.

Просмотр информации о жестком диске/твердотельном накопителе (HDD/SSD):

  1. Перейдите на вкладку HDD/SSD.
  2. Выберите диск из списка и раскройте его. Здесь будут приведены такие сведения, как состояние после тестирования S.M.A.R.T., информация о разделе, название модели, серийный номер, версия микропрограммы и температура.

Просмотр информации о состоянии диске:

  1. Перейдите на вкладку HDD/SSD.
  2. Выберите диск.
  3. Нажмите Информация о состоянии.
  4. Откройте вкладку Обзор. Здесь можно узнать сколько часов работает диск, текущую температуру, количество переподключений/повторных идентификаций, число неисправных секторов и общее состояние S.M.A.R.T.
  5. Перейдите на вкладку Информация о S.M.A.R.T. Выведенные на экран цифры помогут определить состояние диска.

Примечание.

Диски SAS не поддерживают информацию о S.M.A.R.T.

Запуск тестирования S.M.A.R.T.:

Тестирование S.M.A.R.T. помогает выяснить состояние дисков и создать отчет для предупреждения потенциальных сбоев на диске. При обнаружении ошибок рекомендуется заменить диск.

  1. Перейдите на вкладку HDD/SSD.
  2. Выберите диск.
  3. Нажмите Тестирование S.M.A.R.T.
  4. Выберите тип тестирования и нажмите кнопку Пуск, чтобы начать тестирование.
  5. Результаты последнего тестирования отображаются в разделе Результат тестирования.

Примечание.

Время расчета может отличаться в зависимости от среды.

Планирование тестирования S.M.A.R.T.:

Тестирование S.M.A.R.T можно настроить для автоматического выполнения в заданное время путем создания задач.

  1. Перейдите на вкладку Планировщик тестирования.
  2. Нажмите кнопку Создать, чтобы настроить новую задачу.
  3. Нажмите вкладку Общие, чтобы изменить следующие настройки:
    • Задача: Введите имя для задачи.
    • Тип тестирования S.M.A.R.T.: Используется для выполнения быстрого или расширенного тестирования.
    • Диапазон тестирования S.M.A.R.T.: Выберите жесткие диски/твердотельные накопители (HDD/SSD) для тестирования при каждом выполнении тестирования S.M.A.R.T.
  4. Нажмите вкладку Расписание, чтобы изменить следующие настройки:
    • Дата: Выбор времени выполнения тестирования S.M.A.R.T. Тестирование S.M.A.R.T. можно выполнять в определенные дни недели (например, ежедневно, по выходным, каждое воскресенье и др.) либо по определенным датам (например 21/10/2015).
    • Время: Укажите количество выполнений тестирования S.M.A.R.T., а также день первого и последнего выполнения.
  5. Для завершения нажмите OK.

Выполнение Secure Erase:

  1. Перейдите на вкладку HDD/SSD.
  2. Выберите диск и нажмите Secure Erase.

Примечание.

  • Secure Erase невозможно выполнить на дисках, установленных на устройствах расширения.
  • Secure Erase полностью удалит все данные с вашего диска. Убедитесь, что выбранный диск не используется и что все необходимые данные были зарезервированы.
  • При выполнении Secure Erase выбранный диск будет временно заблокирован. После завершения процесса он будет разблокирован автоматически.
  • Если процесс Secure Erase неверно остановлен, вы можете перезапустить Secure Erase непосредственно из пользовательского интерфейса. Если диски удалены или используются еще где-то, когда они заблокированы, введите пароль "synology" для разблокировки дисков.

Включение поддержки записи в кэш для жесткого диска (HDD)

Включение записи в кэш повышает производительность DiskStation. Только определенные модели жестких дисков поддерживают данную функцию. Для обеспечения защиты данных при использовании записи в кэш настоятельно рекомендуется использовать ИБП. Также рекомендуется правильно выключать систему после каждого использования. Отключение записи в кэш снизит вероятность потери данных в случае сбоя питания, но производительность системы снизится.

Включение поддержки записи в кэш:

  1. Перейдите на вкладку Управление кэшем.
  2. Установите флажок Включить запись в кэш. По умолчанию данная функция включена.
  3. Нажмите Применить.
