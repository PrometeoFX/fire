Создать раздел

В данной статье описывается создание раздела на DiskStation. При создании нового раздела можно выбрать один из следующих способов:

  • Быстрый: Это является наилучшим параметром для новичков. Выберите это способ, если вы хотите автоматически оптимизировать емкость системы хранения и производительность, не беспокоясь о технических аспектах, например уровнях RAID.
  • Пользовательский: Для продвинутых пользователей, которые хотят управлять расширенными настройками создания раздела, например типами RAID и распределением емкости. Данный параметр поддерживает другие типы RAID, позволяя создавать один или несколько разделов, и предоставляет возможности для распределения определенной части емкости для каждого раздела (только при создании нескольких разделов на RAID).

Один или несколько разделов на RAID?

При создании раздела в пользовательском режиме можно выбрать один из следующих параметров:

  • Один раздел на RAID
    • Позволяет создать один раздел с помощью всего доступного объема хранения выбранных жесктих дисков.
    • Обеспечивает улучшенную скорость и производительность.
    • Поддерживает различные типы RAID.
  • Несколько разделов на RAID
    • Позволяет создавать несколько разделов на группе дисков.
    • Поддерживает распределение определенного объема для каждого раздела.
    • Обеспечивает улучшенную гибкость управления хранением.
    • Поддерживает различные типы RAID.

Перед началом работы:

  • Убедитесь, что на жестких дисках нет важных данных. Во время выполнения создания все данные будут удалены.
  • Перейдите на страницу HDD/SSD и убедитесь, что для каждого жесткого диска установленно состояние Нормальный или Не инициализировано.

Создание раздела:

  1. Откройте Диспетчер хранения.
  2. Выберите Раздел и нажмите Создать.
  3. Отобразится мастер создания разделов. Выберите Быстрый или Пользовательский в зависимости от ваших потребностей. Следуйте инструкциям мастера для завершения создания раздела.
