Настройки хранения

На странице Настройки хранения содержатся параметры по настройке получения журналов и мест их архивирования Центром журналов. Используйте приведенные ниже параметры для выбора места хранения и правил для автоматического создания архивов журналов.

Местоположение хранения для архивов

Местоположением называется папка, в которой хранятся и архивируются журналы, полученные Центром журналов. Нажмите Выбрать местоположение для выбора папки.

Если вы хотите выполнить архивирование локальных журналов DiskStation в одном местоположении, установите флажок Создать архив локальных журналов в указанном месте хранения выше. В большинстве случаев это обеспечивает дополнительный объем хранения для локальных журналов, позволяя использовать более полные системные журналы.

Примечание.

Локальные журналы (системные журналы, журналы подключений и журналы передачи файлов) сохраняются в системном разделе с ограниченным объемом хранения. При включении пункта Создать архив локальных журналов в указанном месте хранения выше все журналы будут заархивированы в указанном местоположении, согласно Правилам архивирования. Для просмотра заархиварованных локальных журналов нажмите Открыть архив журнала и выберите SYNOSYSLOGDB__LOCALARCH.DB.

Правила архива

Журналы автоматически архивируются согласно критериям, указанным в разделе Правила архива. Журналы можно автоматически архивировать, когда размер базы данных журналов достигает установленного размера, количество индивидуальных журналов достигает указанного числа или журналы достигают определенного срока давности.

Формат архива

В данном разделе содержатся следующие параметры.

  • Архивировать журналы в текстовом формате помимо формата по умолчанию: Сохранение архивов в качестве обычных текстовых файлов помимо файлов SQLite по умолчанию.
  • Сжимать архивы журналов: Сохранение архивов журналов в формате ZIP.
  • Архивировать журналы отдельно согласно устройству: Сохранение различных архивов для отдельных клиентских устройств, которые отправляют журналы на DiskStation.
