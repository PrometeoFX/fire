Поиск журнала

На странице Центр журналов > Поиск журналов можно просматривать, выполнять поиск и экспортировать журналы.

Поиск журналов:

  1. Выберите отображение локальных системных журналов или журналов с других устройств, выбрав вкладки Локальные или С других серверов.
  2. При необходимости выберите пункт Последние журналы или откройте архив журналов, выбрав Открыть архив журналов.
  3. В раскрывающемся меню поиска можно указать следующие критерии в зависимости от ситуации:
    • Ключевое слово: Введите ключевое слово для поиска совпадающих журнальных сообщений.

      Примечание.

      Во время поиска в поле ключевого слова можно использовать обычные выражения типа [*] (произвольное количество случайных символов) или [?] (один случайный символ). Например, [а*с] может означать [атмосфера], [ассоциация] или [ас].

    • Диапазон дат: поиск журналов по заданному диапазону даты.
    • Уровень: поиск журналов по заданному уровню.
    • Имя хост-сервера: поиск журналов по заданному имени хоста.
    • Программа: поиск журналов по заданному названию программы.
    • Категория: поиск журналов по заданной категории программы.
  4. Нажмите Поиск для поиска журналов, совпадающих с указанными критериями.

Экспорт журналов:

  1. Нажмите стрелку рядом с кнопкой Экспорт.
  2. Выберите HTML или CSV для загрузки журналов в необходимом формате.

Примечание.

  • Файлы экспортированного журнала CSV имеют кодировку UTF-8. Их необходимо открывать с помощью программы, которая поддерживает кодировку UTF-8.
