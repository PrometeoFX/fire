Редактирование фотографий

С помощью File Station вы можете редактировать фотографии с помощью веб-редакторов Pixlr Editor или Pixlr Express.

Редактирование фотографий с помощью веб-редактора:

  1. Откройте папку, в которой расположено фото.
  2. Нажмите фото правой кнопкой мыши и выберите Редактировать в Pixlr Editor или Редактировать в Pixlr Express.
      Примечание.
    • Pixlr Editor поддерживает следующие форматы: jpg, gif, png, psd, pxd (многослойное изображение Pixlr)
    • Pixlr Express поддерживает следующие форматы: jpg, gif, png
  3. Выбранный фоторедактор будет открыт в новом окне обозревателя. Вы сразу же можете просмотреть фотографию и отредактировать ее с помощью предоставленных инструментов.
  4. После завершения редактирования фото выполните следующие действия для сохранения результата.
    • Для Pixlr Express: Нажмите Выполнено, чтобы сохранить фотографию в исходном формате и в исходном местоположении на DiskStation.
    • Для Pixlr Editor: Выберите Файл > Сохранить, выберите местоположение (например, исходное местоположение фото на DiskStation, ваш компьютер или социальную сеть), чтобы сохранить фото из левой панели, укажите новое имя, формат и нажмите OK.

Примечание. Чтобы отредактировать фотографию с помощью любого веб-редактора, убедитесь, что DiskStation успешно получил внешний IP-адрес (для этого откройте Главное меню > Панель управления > Внешний доступ > DDNS или Главное меню > Панель управления > Сеть > Сетевой интерфейс > PPPoE в зависимости от конфигурации сети). Если устройство DiskStation расположено позади маршрутизатора, убедитесь, что указано правило переадресации порта для DSM в Главное меню > Панель управления > Внешний доступ > Конфигурация маршрутизатора. Если открыть фоторедакторы по-прежнему не удалось, возможно, необходимо указать порт маршрутизатора DSM в Главное меню > Панель управления > Внешний доступ > Дополнительно.
