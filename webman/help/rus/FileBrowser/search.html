Поиск файлов и папок

Используя Фильтр File Station, можно искать и отображать файлы и папки по заданным критериям. Можно фильтровать файлы и папки в текущей папке и выполнять расширенный поиск в подпапках для получения более точных результатов поиска.

Поиск файлов и папок в текущей папке:

  1. С помощью File Station перейдите в папку, в которой находится искомое содержимое.
  2. В поле Фильтр в правом верхнем углу введите ключевое слово и просмотрите результаты фильтрации в основной области окна.

Расширенный поиск:

  1. С помощью File Station перейдите в папку, в которой находится искомое содержимое.
  2. Нажмите значок лупы, расположенный рядом с полем Фильтр в правом верхнем углу, для открытия расширенного меню поиска.
  3. Введите информацию или выберите в раскрывающихся меню следующие критерии поиска:
    • Ключевое слово: Введите ключевое слово для поиска файлов или папок, в названиях которых есть данное слово.

      Примечание:

      • Вы можете использовать специальные символы [*] (ноль или несколько символов) и [?] (один произвольный символ) в ключевых словах. Например, [a*n] может означать [action], [attention] или [an].
      • Вы можете использовать запятые (,) для разделения нескольких ключевых слов. Например, [a,tion] может означать [action], [a function], [function added] или [nation].
      • Вы можете добавить кавычки ["] в начале и в конце ключевого слова для поиска точных совпадений. Например, ["a typ"] может означать [a type], но не [typical].
    • Местоположение: Выберите в раскрывающемся меню, чтобы установить диапазон и глубину расширенного поиска. Также можно настроить масштаб поиска, выбрав другую общую папку из списка.
    • Тип файла: Выберите Расширение, чтобы сузить область поиска до определенных типов файлов, либо выберите Любая папка для поиска только папок.
    • Дата: Выберите параметр Дата изменения, Дата создания или Последние открытые, а затем выберите дату для поиска файлов по указанным датам.
    • Размер (MБ): Выберите равен, более чем или менее чем, а затем введите размер для поиска файлов по заданному размеру. (При выборе равен результаты поиска будут включать файлы примерно одного указанного размера.)
    • Владелец/Группа: Выберите Владелец или Группа, а затем выберите в раскрывающемся меню для поиска файлов или папок, принадлежащих выбранному владельцу или группе.

      Примечание:

      • Чтобы выполнить расширенный поиск, необходимо указать Расположение и по меньшей мере еще один критерий поиска.
      • Если папка ISO или удаленная папка создана в индексированной общей папке, вы не сможете найти находящиеся в ней файлы с помощью расширенного поиска.
    • Нажмите Поиск, чтобы начать поиск, либо Сброс и укажите новые критерии поиска. После завершения поиска файлы, удовлетворяющие критериям, будут перечислены в основной области File Station.
