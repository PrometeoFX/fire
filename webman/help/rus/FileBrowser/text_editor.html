Текстовый редактор

Текстовый редактор — это многофункциональный редактор текстов и кодов, позволяющий быстро и эффективно редактировать текстовые файлы и коды на DiskStation непосредственно из веб-браузера.

Управление файлами

Создание новый файлов:

Нажмите Файл и выберите Создать. Отобразится пустая вкладка с заголовком Создание документа.txt.

Открытие файлов из текстового редактора:

Нажмите Файл и выберите Открыть. Отобразится диалоговое окно, в котором можно выбрать файл со специальной кодировкой символов для открытия во вкладке. Если файл уже открыт, текстовый редактор отобразит файл в текущем состоянии.

Открытие файлов из File Station:

В File Station просто нажмите правой кнопкой мыши выбранный файл и в контекстном меню выберите Открыть с помощью текстового редактора. Если файл уже открыт, текстовый редактор отобразит файл в текущем состоянии. Файл можно также открыть, перетащив его из File Station в текстовый редактор с помощью курсора.


Примечание.

Текстовый редактор не позволяет открывать файлы размером более 10 МБ.

Сохранение файлов:

Нажмите Файл и выберите Сохранить. Если такой файл уже существует, он будет перезаписан с текущим содержимым. Если вы хотите одновременно сохранить все открытые файлы, выберите Файл > Сохранить все. Если файл еще не имеет имени (новый файл) или выбран пункт Файл > Сохранить как, отобразится диалоговое окно с запросом на ввод имени файла. После этого можно выбрать место расположения, имя файла и кодировку.

Закрытие файлов:

Нажмите Файл и выберите Закрыть. Другим способом закрытия файла является нажатие кнопки закрытия на каждой вкладке. Если в файле есть несохраненные данные, текстовый редактор всегда предложит сохранить изменения перед закрытием файла.

Примечание.

Вы можете сохранить файлы со специальной кодировкой символов для поддержки двусторонних языков. Вы также можете указать кодировку при открытии файла, чтобы текстовый редактор правильно отображал файл.

Редактирование

  • Стандартные операции
    Редактирование поддерживает функции Отменить, Повторить и Выбрать все. Эти операции имеются в большинстве редакторов и работают аналогичным образом. Вы можете выполнить эти операции, выбрав соответствующий элемент меню в пункте Редактирование. Кроме того, параметры Вырезать, Копировать и Вставить доступны в контекстном меню браузера.
  • Отступ
    Текстовые отступы — это наиболее частая операция при работе с кодами источника. Отступ — это добавление пробела в начале строки для отделения блоков текста, создавая иерархию. Обычно во время редактирования при нажатии клавиши Tab в месте расположения курсора добавляется знак табуляции. Однако если выделен участок текста, нажатие клавиши Tab приведет к добавлению дополнительного пробела в начале каждой строки выделенного текста, позволяя создать отступ для всего блока одновременно. При нажатии клавиши Shift и Tab одновременно отступ будет удален.
  • Замена
    Вы можете включить или отключить режим замены, нажав клавишу Insert. При этом вместо вставки символов в месте расположения курсора будет осуществляться их замена. Отображаемый курсор изменит форму, если включен режим замены.
  • Редактирование в режиме столбца
    Особый способ редактирования файлов в режиме столбца. Это значит, что текст может быть выбран в столбцах, а не в строках. Для активации выбора в режиме столбца во время выбора текста нажмите клавишу Alt, перемещая курсор вниз или вверх при необходимости. Когда текст выбран в этом, так называемом, режиме столбца, при вводе будет выполнена замена каждого выбранного столбца символов на вводимые символы.

Выполняется поиск

  • Обычный поиск
    Вы можете выполнять поиск текста точно так же, как и в большинстве других программ, с помощью окна Найти. Чтобы открыть это окно, выберите Поиск > Найти. Данное окно всегда будет располагаться поверх текста и вы не сможете продолжить редактирование текста, если оно открыто. В поле поиска введите текст, который необходимо найти.
  • Найти и заменить
    Вы можете найти и заменить текст точно так же, как и в большинстве других программ, с помощью окна "Найти и заменить". Чтобы открыть это окно выберите Поиск > Найти и заменить. Использование параметра Заменить все приведет к замене всех найденных элементов на текст замены.
  • С учетом регистра
    При выборе данного параметра найденные совпадения будут иметь такой же регистр, как у запроса. Если данный флажок не установлен, поиск выполняется без учета регистра.
  • Только слово целиком
    При выборе данного параметра будет выполнен поиск только целого слова, а не его части. Например, если вы ищите слово "кот" и у вас установлен параметр Только слово целиком, будет выполнен поиск только слова "кот"; слова типа "который" найдены не будут.
  • Регулярное выражение
    Запрос будет интерпретироваться как регулярное выражение.

Что такое регулярное выражение?

Регулярное выражение — это последовательность символов, которая создает шаблон поиска для экономии времени. Очень простым примером использования регулярного выражения является поиск одного слова, написанного двумя различными способами в текстовом редакторе, например M[ea]rry.

Конфигуратор стиля

Конфигуратор стиля в основном используется для управления оформлением текста в текстовом редакторе.
Раскрывающийся список в нижнем правом углу содержит все поддерживаемые языки программирования, которые можно оформить для создания необходимой среды кодирования.

Предпочтения

  • Отображать скрытые символы
    При выборе данного параметра будут отображаться пробелы, знаки табуляции и символы возврата.
  • Отобразить направляющие отступа
    Если выбран данный параметр, будут отображены направляющие отступа. Это удобно для выравнивания кодов в языке кодирования.
  • Выделить текущую строку
    Если выбран данный параметр, строка, на которой расположена точка вставки, будет отмечена специальным фоновым цветом.
  • Выделить выбранное слово
    Если выбран данный параметр, все слова (полностью) в одном текстовом файле, совпадающие с выбранным словом, будут автоматически подчеркнуты.
  • Отображать номера строк
    Если выбран данный параметр, будут отображены номера строк.
  • Включить сворачивание кода
    Если выбран данный параметр, определенные строки кодов могут быть скрыты на основе определенных признаков. Данная функция эффективна для структурированных языков, например C++ или XML. Если выбран параметр Скрывать виджеты раскрытия автоматически, некоторые строки кодов будут свернуты автоматически. Вы также можете отметить только открывающую скобку или открывающую и закрывающую скобку в виджетах раскрытия.
  • Перенос по словам
    Укажите перенос текста у края окна или в определенном номере столбца без установки перевода строки.
  • Автоматически добавлять символы закрытия
    Если выбран данный параметр, закрывающие скобки (символы) будут добавлены к кодам автоматически.
  • Вставлять пробелы вместо знаков табуляции
    Если выбран данный параметр при нажатии клавиши Tab будет вставлен пробел, а не знак табуляции. Интервал табуляции можно задать числом символов в пункте Предпочтения > Шрифт и цвет.
  • Кодировка по умолчанию
    Кодировка по умолчанию используется для сохранения файлов с определенной кодировкой символов для поддержки двусторонних языков. Вы также можете указать кодировку при открытии файла, чтобы текстовый редактор правильно отображал файл.
  • Параметры Шрифт и цвет
    Интервал табуляции, Размер шрифта и Тема можно настроить согласно личным предпочтениям. Интервал табуляции определяется количеством символов пробела.

Поддерживаемые форматы файлов

Текстовый редактор поддерживает только обычные текстовые файлы. Текстовые файлы с широкими возможностями форматирования, такие как *.doc и *.pdf не поддерживаются.
