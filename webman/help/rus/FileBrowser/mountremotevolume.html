Подключить удаленные папки

Можно подключать удаленные папки на удаленных серверах (таких как сетевые компьютеры или другие устройства DiskStation) для удобной навигации по доступным сетевым ресурсам при помощи File Station.

Подключение удаленной папки:

  1. Выберите Инструменты > Подключить удаленную папку.
  2. Введите "\\Remote_Server\Remote_Folder" в поле Папка, чтобы указать путь к удаленной папке. Например, "\\192.168.0.2\share" или "\\chad\cheeseburger\pictures".
  3. Если для доступа к удаленной папке удаленный сервер требует прав доступа, то введите свои учетные данные для удаленного сервера в полях Имя учетной записи и Пароль.
  4. Нажмите Обзор, чтобы выбрать или создать пустую целевую папку в вашем устройстве DiskStation для подключения удаленной папки.
  5. Установите флажок Подключать автоматически при запуске, если хотите, чтобы устройство DiskStation подключало эту удаленную папку при каждом запуске или перезагрузке системы.
  6. Нажмите Подключить. Теперь можно открыть целевую папку, в которой подключена удаленная папка, и управлять содержимым удаленных папок.

Отключить удаленную папку:

  1. Выберите удаленную папку, которую необходимо отключить, и нажмите Инструменты > Отключить.
  2. Можно также щелкнуть правой кнопкой мыши удаленную папку и выбрать Отключить.

Примечание.

  • Одновременно можно подключить максимум 100 удаленных папок.
  • Подключать удаленные папки можно только с удаленного сервера, который поддерживает протокол CIFS.
