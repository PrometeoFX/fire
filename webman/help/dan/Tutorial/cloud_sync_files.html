Synkroniser filer mellem eCore Server og din computer med Cloud Station

Oversigt

Cloud Station for eCore Server er et filsynkroniseringsprogram, der sætter dig i stand til nemt at synkronisere filer på din eCore Server med andre enheder, f.eks. computere eller mobile enheder (med DS cloud). Når Cloud Station er installeret på både din eCore Server og computer, bliver filer, der er gemt i en angivet mappe, automatisk synkroniseret med din eCore Server.

1. Installer Cloud Station

  1. Gå til Pakkecenter, søg efter Cloud Station, og klik så på Installer.

2. Aktiver Cloud Station

  1. Start Cloud Station.
  2. Hvis du ikke har aktiveret brugerstartfunktionen, bliver du bedt om at gøre det. Klik på OK.
  3. Du bliver omdirigeret til brugerindstillingssiden. Aktiver brugerstart, og klik på Anvend.
  4. Gå så tilbage til Cloud Station, hvor du bedes om at aktivere den. Klik på Ja.
  5. Gå til Privilegier, og angiv de brugere, der kan bruge Cloud Station. Klik på Gem.

3. Installer og aktiver klient.

  1. Gå til Cloud Station > Oversigt, og klik på knappen Computere.
  2. Klik på knappen for at downloade Cloud Station til dit operativsystem. (Du kan også downloade Cloud Station fra Synologys Downloadcenter.)
  3. Start installationsprogrammet på computeren, og følg instruktionerne på skærmen.
  4. Efter installationen kører du Cloud Station på computeren. Klik på Start nu
  5. Indtast adressen på den eCore Server, der kører Cloud Station (eller QuickConnect ID), brugernavn og adgangskode. Du kan klikke på søgeikonet til højre for at søge efter IP'er på dit LAN. Klik på Næste.
  6. Vælg, hvordan vil du konfigurere din opgave (her vælger vi Hurtig opsætning), og klik på Næste.
  7. Klik på Udfør for at fuldføre konfigurationen af din Cloud Station-mappe.
  8. Klik på Åbn den nu for at åbne din Cloud Station-mappe.
  9. Du kan finde Cloud Station-mappen i systembakken.
  10. Dobbeltklik på Cloud Station-ikonet for at åbne bakkemenuen. Her kan du se fremskridt og status for dine filer. Klik på Hovedprogram for at starte Cloud Station.
  11. Ved start vises du nogle tip om, hvordan du bruger Cloud Station. Klik på højrepil for at se flere tip, eller klik på Gå til onlineselvstudier for at få flere oplysninger. Hvis du ikke har brug for tip, klikker du på Vis ikke dette igen.
  12. Nu kan du administrere dine synkroniseringsopgaver i Cloud Station.

4. Synkroniser data til mobilenheder

DS cloud findes gratis på Apple App Store og Google Play Store. Du kan også scanne QR-koden nedenfor for at downloade direkte.
