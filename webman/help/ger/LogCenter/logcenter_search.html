Protokollsuche

Auf der Seite Protokoll-Center > Protokollsuche können Protokolle angezeigt, gesucht und exportiert werden.

So werden Protokolle gesucht:

  1. Wählen Sie die Anzeige der lokalen Systemprotokolle oder der Protokolle von anderen Geräten, indem Sie die Registerkarte Lokal oder Von anderen Servern auswählen.
  2. Bei Bedarf wählen Sie die Anzeige von Letzte Protokolle oder öffnen Sie ein vorhandenes Protokollarchiv durch Protokollarchiv öffnen.
  3. Im Suche-Drop-down-Menü können Sie die folgenden Kriterien abhängig von der Situation angeben:
    • Schlüsselwort: Geben Sie ein Schlüsselwort ein, um Protokollmeldungen zu suchen, die das entsprechende Schlüsselwort enthalten.

      Hinweis:

      Sie können reguläre Ausdrücke verwenden, wie [*] (d. h. eine beliebige Anzahl von zufälligen Zeichen) oder [?] (d. h. 1 zufälliges Zeichen) im Schlüsselwortfeld bei der Suche. Beispielsweise kann [a*n] für [aktion], [alben] oder [an] stehen.

    • Datumsbereich: Suchen Sie Protokolle nach dem angegebenen Datumsbereich.
    • Stufe: Suchen Sie Protokolle nach der angegebenen Stufe.
    • Hostname: Suchen Sie Protokolle nach dem angegebenen Hostnamen.
    • Programm: Suchen Sie Protokolle nach dem angegebenen Programmnamen.
    • Kategorie: Suchen Sie Protokolle nach der angegebenen Programmkategorie.
  4. Klicken Sie auf Suche, um Protokolle zu suchen, die den angegebenen Kriterien entsprechen.

So werden Protokolle exportiert:

  1. Klicken Sie auf den Pfeil neben der Schaltfläche Exportieren.
  2. Wählen Sie HTML oder CSV, um die Protokolle im gewünschten Format herunterzuladen.

Hinweis:

  • Die Zeichenkodierung der exportierten CSV-Protokolldateien ist UTF-8. Die Datei muss mit einem Programm geöffnet werden, das die UTF-8-Zeichenkodierung unterstützt.
