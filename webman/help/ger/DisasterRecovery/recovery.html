Wiederherstellung

Auf dieser Seite können Systemadministratoren Wiederherstellungen durchführen. Damit auf Schnappschüsse von freigegebenen Ordnern zugegriffen werden kann, muss für jeden freigegebenen Ordner die Option Schnappschuss sichtbar machen aktiviert sein, damit über File Station oder Windows File Explorer auf die Schnappschüsse zugegriffen werden kann.

Freigegebener Ordner

Einzelne Dateien/Ordner in einem freigegebenen Ordner wiederherstellen:

  1. Wählen Sie einen freigegebenen Ordner aus, den Sie wiederherstellen möchten.
  2. Klicken Sie auf Wiederherstellen.
  3. Wählen Sie eine Schnappschussversion aus, aus der Sie wiederherstellen möchten.
  4. Klicken Sie auf Durchsuchen und kopieren Sie die Dateien/Ordner, die Sie wiederherstellen möchten, zum Ziel.

Freigegebenen Ordner wiederherstellen:

  1. Wählen Sie einen freigegebenen Ordner aus, den Sie wiederherstellen möchten.
  2. Klicken Sie auf Wiederherstellen.
  3. Wählen Sie eine Schnappschussversion aus, aus der Sie wiederherstellen möchten.
  4. Klicken Sie auf Aktion und wählen Sie Am Speicherort wiederherstellen aus der Dropdown-Liste aus.
  5. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie auf Ja, um die Aktion zu bestätigen.

Schnappschuss in einen neuen freigegebenen Ordner klonen:

  1. Wählen Sie einen freigegebenen Ordner aus, den Sie klonen möchten.
  2. Klicken Sie auf Wiederherstellen.
  3. Wählen Sie eine Schnappschussversion aus, die Sie in einen neuen freigegebenen Ordner klonen möchten.
  4. Klicken Sie auf Aktion und wählen Sie Mit neuem Namen klonen aus der Dropdown-Liste aus.
  5. Sie werden zur Seite Klonen unter Systemsteuerung > Freigegebener Ordner > Erstellen weitergeleitet. Geben Sie den Namen des geklonten freigegebenen Ordners ein und klicken Sie auf OK, um den Vorgang abzuschließen.

Hinweis:

  • Die „Homes“ eines freigegebenen Ordners können nicht wiederhergestellt werden.
  • Der geklonte freigegebene Ordner kann nicht verschlüsselt werden.
  • Wird ein Schnappschuss aufgenommen, während Dateien übertragen werden, kann die Datenkonsistenz nicht garantiert werden.
  • Einschränkungen für verschlüsselte freigegebene Ordner:
    • Für bereitgestellte verschlüsselte freigegebene Ordner ist keine Art der Wiederherstellung verfügbar.
    • Mit neuem Namen klonen ist für verschlüsselte freigegebene Ordner nicht verfügbar.
    • Getrennte, verschlüsselte freigegebene Ordner können mithilfe von Am Speicherort wiederherstellen aus zuvor aufgenommenen Schnappschüssen wiederhergestellt werden.

iSCSI LUN

iSCSI LUN wiederherstellen:

  1. Wählen Sie eine wiederherzustellende iSCSI LUN aus.
  2. Klicken Sie auf Wiederherstellen.
  3. Wählen Sie eine Schnappschussversion aus, aus der Sie wiederherstellen möchten.
  4. Klicken Sie auf Aktion und wählen Sie Am Speicherort wiederherstellen aus der Dropdown-Liste aus.
  5. Eine Bestätigungsmeldung wird angezeigt. Klicken Sie auf Ja, um die Aktion zu bestätigen.

Schnappschuss in einer neuen iSCSI LUN klonen:

  1. Wählen Sie die zu klonende iSCSI LUN aus.
  2. Klicken Sie auf Wiederherstellen.
  3. Wählen Sie eine Schnappschussversion aus, die Sie in eine neue iSCSI LUN klonen möchten.
  4. Klicken Sie auf Aktion und wählen Sie Mit neuem Namen klonen aus der Dropdown-Liste aus.
  5. Sie werden zur Seite Klonen unter Speicher-Manager > iSCSI LUN weitergeleitet. Geben Sie den Namen der geklonten iSCSI LUN ein und klicken Sie auf OK, um den Vorgang abzuschließen.

Hinweis:

  • Wird ein Schnappschuss aufgenommen, während Dateien übertragen werden, kann die Datenkonsistenz nicht garantiert werden.
