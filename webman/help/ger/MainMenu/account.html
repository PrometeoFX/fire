Konto

Die Konto-Registerkarte bietet Optionen, um grundlegende Kontoeinstellungen wie Beschreibung, Passwort, E-Mail und 2-Schritt-Verifizierung zu bearbeiten. Zum Anzeigen dieser Registerkarte klicken Sie auf das Figur-förmige Symbol in der oberen rechten Desktop-Ecke und wählen Sie Optionen.

Ändern von Kontoeinstellungen

Die folgenden Kontoeinstellungen können bearbeitet werden:

  • Beschreibung: Geben Sie eine Beschreibung Ihres Kontos ein. Benutzerbeschreibungen dürfen bis zu 64 anzeigbare Unicode-Zeichen enthalten.
  • Neues Passwort: Wenn Sie das Passwort Ihres Kontos ändern möchten, geben Sie hier ein neues ein und bestätigen Sie es im unteren Feld Passwort bestätigen. Bei Passwörtern wird die Groß-/Kleinschreibung berücksichtigt und sie können aus bis zu 127 Zeichen einschließlich Buchstaben, Zahlen, Symbolen und Leerzeichen bestehen.
  • E-Mail: Geben Sie eine E-Mail-Adresse ein (nur für Nicht-Admin-Benutzer).
  • Anzeigesprache: Legen Sie die Anzeigesprache für Ihr Konto fest. Die Einstellungen der Anzeigesprache, die Sie hier festlegen, haben Priorität über die unter Systemsteuerung > Regionale Optionen > Sprache festgelegten Sprachen.

Klicken Sie nach dem Ändern der Kontoeinstellungen auf OK, um die Änderungen zu speichern.

2-Stufen-Verifizierung

Die 2-Stufen-Verifizierung bietet eine gesteigerte Sicherheit für Ihr DSM-Konto. Wenn die 2-Stufen-Verifizierung aktiviert ist, müssen Sie Ihr Passwort sowie einen einmaligen Verifizierungscode bei der Anmeldung in DSM eingeben. Verifizierungscodes erhalten Sie von Authentifizierungs-Apps, die auf Ihrem Mobilgerät installiert sind. Wenn jemand auf Ihr Konto zugreifen möchte, wird er nicht nur Ihren Benutzernamen und das Passwort benötigen, sondern auch Ihr Mobilgerät.

Anforderungen:

Die 2-Stufen-Verifizierung erfordert ein Mobilgerät und eine Authentifizierungs-App, die das Protokoll „Time-based One-Time Password (TOTP)“ unterstützt. Authentifizierungs-Apps sind Google Authenticator (Android/iPhone/BlackBerry) oder Authenticator (Windows Phone).

2-Stufen-Verifizierung aktivieren:

  1. Wechseln Sie zur Registerkarte Optionen > Konto, und markieren Sie die Option 2-Stufen-Verifizierung aktivieren, um den Einrichtungsassistenten für die 2-Stufen-Verifizierung zu starten.
  2. Geben Sie im ersten Fenster eine E-Mail-Adresse ein. Notfall-Verifizierungscodes können an diese E-Mail-Adresse gesendet werden, wenn Ihr Mobilgerät verloren geht.
  3. Laden Sie eine Authentifizierungs-App z. B. Google Authenticator (Android/iPhone/BlackBerry) oder Authenticator (Windows Phone) herunter und installieren Sie sie.
  4. Öffnen Sie die Authentifizierungs-App im nächsten Fenster und scannen Sie den QR-Code. Alternativ können Sie auch auf den Link klicken, um manuell einen geheimen Schlüssel oder „Secret Key“ einzugeben.
  5. Die Authentifizierungs-App erstellt daraufhin einen 6-stelligen Verifizierungscode. Geben Sie diesen Code in das Textfeld im Assistenten ein, um zu bestätigen, dass die Konfigurationen korrekt sind. Wenn ein Fehler auftritt, stellen Sie bitte sicher, dass die Systemzeit Ihres Mobilgerätes mit der Systemzeit des DSM synchronisiert ist. Verifizierungscodes werden zudem von Zeit zu Zeit aktualisiert. Stellen Sie daher sicher, dass der von Ihnen eingegebene Code nicht abgelaufen ist.
  6. Klicken Sie nach dem Abschluss des Einrichtungsassistenten auf OK, um die Einstellungen zu speichern.

Anmeldung bei DSM mit 2-Stufen-Verifizierung:

Wenn die 2-Stufen-Verifizierung aktiviert ist, werden Sie zur Eingabe eines 6-stelligen Verifizierungscodes aufgefordert, wenn Sie sich bei DSM anmelden möchten.

  1. Geben Sie Ihren Benutzernamen und das Passwort wie gewohnt im DSM-Anmeldefenster ein.
  2. Wenn Sie zur Eingabe eines Verifizierungscodes aufgefordert werden, öffnen Sie die Authentifizierungs-App auf Ihrem Mobilgerät.
  3. Suchen und geben Sie den 6-stelligen Verifizierungscode für Ihr Konto ein. Wenn Sie Ihr Mobilgerät verloren haben, können Sie auf den Link Telefon verloren? klicken, dann wird ein Notfallcode an Ihre E-Mail-Adresse gesendet.

Hinweis:

  • SMTP-Einstellungen: Um Notfallcodes per E-Mail erhalten zu können, müssen die SMTP-Servereinstellungen unter Systemsteuerung > Benachrichtigung korrekt eingerichtet sein.
  • Notfallcode-Limit: Jeder Benutzer hat ein Limit von 5 Notfallcodes. Wenn Sie dieses Limit überschreiten, müssen Sie die 2-Stufen-Verifizierung deaktivieren und aktivieren, bevor Sie weitere Notfallcodes erhalten können.

2-Stufen-Verifizierungseinstellungen zurücksetzen:

Benutzer, die zur Gruppe administrators gehören, können die 2-Stufen-Verifizierung anderer Benutzer zurücksetzen.

  1. Gehen Sie zu Systemsteuerung > Benutzer.
  2. Doppelklicken Sie auf den Benutzer, dessen 2-Stufen-Verifizierung Sie zurücksetzen möchten.
  3. Klicken Sie auf der Registerkarte Benutzerinformationen auf die Schaltfläche 2-Stufen-Verifizierung zurücksetzen.

Hinweis:

Wenn kein Benutzer der Gruppe administrators verfügbar ist, können Sie die physische Reset-Taste am DiskStation drücken, um die DSM-Einstellungen zurückzusetzen und die 2-Stufen Verifizierung zu deaktivieren.

Kontoaktivität

Die Kontoaktivität zeigt die Anmeldeaktivität Ihres DSM-Kontos an, z. B. aktuelle Verbindungen oder den Anmeldeverlauf.

Kontoaktivität überprüfen:

Klicken Sie auf die Schaltfläche Kontoaktivität. Ein Popup-Fenster mit den folgenden Registerkarten wird angezeigt:

  • Verbundene Benutzer: Diese Registerkarte zeigt die unten stehenden Informationen:
    • Datum & Zeit: Wann auf das Konto zugegriffen wurde.
    • Client-Name und IP: Name und IP-Adresse des Gerätes, das auf das Konto zugegriffen hat. Der Client-Name wird nur angezeigt, wenn über das CIFS-Protokoll auf das Konto zugegriffen wurde.
    • Dienst: Das für den Zugriff auf das Konto verwendete Protokoll.
    • Ressource: Die vom Konto verwendeten Systemressourcen.
  • Anmeldeverlauf: Diese Registerkarte zeigt letzte Anmeldesitzungen und die unten stehenden Informationen an:
    • Datum & Zeit: Wann auf das Konto zugegriffen wurde.
    • Ereignis: Zeigt Benutzername und IP-Adresse.
