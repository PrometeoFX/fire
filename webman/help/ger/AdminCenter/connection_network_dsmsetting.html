DSM-Einstellungen

Unter Systemsteuerung > Netzwerk > DSM-Einstellungen können die von den Clients verwendeten Portnummern für den Zugriff auf DSM geändert werden. Wenn HTTPS aktiviert ist, werden Verbindungen mit SSL/TLS verschlüsselt.

So ändern Sie die Standardportnummer:

  1. Geben Sie die Portnummer im Feld HTTP oder HTTPS ein.
  2. Klicken Sie auf Übernehmen.

So aktivieren Sie HTTPS-Verbindungen:

  1. Markieren Sie HTTPS-Verbindung aktivieren.
  2. Wenn Sie Clients zum HTTPS-Port (5001) umleiten möchten, wenn sie versuchen, über den HTTP-Port (5000) auf den DSM zuzugreifen, markieren Sie das Kontrollkästchen Automatische Umleitung von HTTP-Verbindungen zu HTTPS.
  3. Klicken Sie auf Übernehmen.

Ports, die für die HTTPS Verbindung verwendet werden:

  • DSM: Die Standardportnummer ist 5001, damit Sie über die folgende Adresse sicher auf den DSM zugreifen können: https://server-hostname:5001/
  • Web Station und Photo Station: Die Standardportnummer ist 443, damit Sie über die folgende Adresse sicher auf Web Station und Photo Station zugreifen können: https://server-hostname/

So wird die DiskStation über die Windows-Netzwerkerkennung auffindbar:

  1. Markieren Sie Windows-Netzwerkerkennung aktivieren.
  2. Klicken Sie auf Übernehmen.
