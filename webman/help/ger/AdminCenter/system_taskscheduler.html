Aufgabenplaner

Mit dem Aufgabenplaner können Sie Dienste oder Skripte planen und zu bestimmten Zeiten ausführen.

Unterstützte Aufgaben

Aufgaben können für Folgendes erstellt werden:

  • Benutzer-definiertes Skript: Führen Sie benutzerdefinierte Skripte oder Befehle aus.
  • Papierkorb: Leeren Sie den Papierkorb von bestimmten gemeinsamen Ordnern. Aufgaben verwalten

Die Hauptseite des Aufgabenplaners zeigt sämtliche Aufgaben an und bietet Optionen zum Erstellen, Bearbeiten oder Ausführen von Aufgaben.

Erstellen einer Aufgabe:

  1. Klicken Sie auf Erstellen und wählen Sie den Aufgabentyp, den Sie planen möchten.
  2. Wenn das Fenster Aufgabe erstellen angezeigt wird, ändern Sie die folgenden Einstellungen auf der Registerkarte Allgemein:
    • Allgemeine Einstellungen: Geben Sie einen Aufgabennamen ein, wählen Sie einen Benutzernamen und aktivieren oder deaktivieren Sie eine Aufgabe.
    • Andere Einstellungen: Auf dieser Registerkarte können zusätzliche aufgabenspezifische Einstellungen geändert werden (nur für bestimmte Aufgaben). Wenn Sie zum Beispiel eine Aufgabe zum Leeres eines Papierkorbs erstellen, müssen Sie den Papierkorb eines bestimmten gemeinsamen Ordners auswählen.
  3. Auf der Registerkarte Zeitplan können Sie die folgenden Einstellungen ändern:
    • Datum: Geben Sie an, wann die Aufgabe ausgeführt werden soll. Aufgaben können an bestimmten Tagen der Woche (z. B. täglich, am Wochenende, jeden Sonntag usw.) oder an bestimmten Daten (z. B. 21.10.2015) ausgeführt werden.
    • Zeit: Geben Sie an, wie oft die Aufgabe pro Tag laufen soll, sowie die erste und letzte Ausführungszeit.
  4. Auf der Registerkarte Aufgabeneinstellungen können aufgabenspezifische Einstellungen geändert werden (nur für bestimmte Aufgaben).
  5. Klicken Sie zum Beenden auf OK.

So bearbeiten Sie Aufgabeneinstellungen:

  1. Wählen Sie eine Aufgabe aus.
  2. Klicken Sie auf Bearbeiten.
  3. Ein Fenster zum Bearbeiten der Aufgabeneinstellungen wird angezeigt.

Löschen einer Aufgabe:

  1. Wählen Sie eine Aufgabe aus.
  2. Klicken Sie auf Löschen.

So aktivieren oder deaktivieren Sie eine Aufgabe:

  1. Aktivieren oder deaktivieren Sie auf der Hauptseite von Aufgabenplaner das entsprechende Kontrollkästchen unter Aktiviert, um eine Aufgabe zu aktivieren oder zu deaktivieren.
  2. Klicken Sie auf Speichern.

Eine Aufgabe manuell ausführen:

  1. Wählen Sie eine oder mehrere Aufgaben aus.
  2. Klicken Sie auf Ausführen.

Aufgabenplaner-Einschränkungen:

  • Aufgabennamen können bis zu 512 anzeigbare Zeichen enthalten.
  • Maximale Anzahl an Aufgaben: 1024.
  • Maximallänge des Skript-Pfades oder der Befehlszeile: 4095.
