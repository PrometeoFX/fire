Applikationsportal

Mit dem Applikationsportal können Sie die Verbindungseinstellungen der verschiedenen Anwendungen konfigurieren, sodass diese in unabhängigen Browser-Tabs oder Fenstern auf diese Anwendungen zugreifen und sie ausführen können.

Sie können zum Beispiel File Station durch die Eingabe von http://:diskstation:5000/file/ in Ihren Webbrowser öffnen. Verfügbare Verbindungseinstellungen umfassen auch benutzerdefinierte Aliase, HTTP- und HTTPS-Ports. Gehen Sie zu Systemsteuerung > Applikationsportal, um diese Einstellungen zu finden.

Hinweis:

Mit dem Applikationsportal können Sie die Zugriffseinstellungen für verschiedene Anwendungen, wie Audio Station, Download Station, Surveillance Station, Video Station, File Station und auch andere bearbeiten.

Alias anpassen

Mit einem benutzerdefinierten Alias kann eine Anwendung wie File Station durch Eingabe einer Adresse wie http://:diskstation:5000/file/ in Ihrem Webbrowser geöffnet werden.

So aktivieren Sie einen benutzerdefinierten Alias:

  1. Wählen Sie eine Anwendung.
  2. Klicken Sie auf Bearbeiten.
  3. Im dann angezeigten Fenster markieren Sie Benutzerdefinierte Aliasnamen aktivieren.
  4. Geben Sie einen benutzerdefinierten Aliasnamen an.

Hinweis:

  • Sie können das Kontrollkästchen Umleiten von markieren und einen kürzeren, leicht zu merkenden Alias verwenden, um auf die ausgewählte Anwendung zuzugreifen.
  • Es gibt folgende Standard-Aliase für die Anwendungen Audio Station, Download Station, Surveillance Station, Video Station und File Station: audio, download, cam, video, file.
  • Aliasnamen dürfen nicht mit Namen identisch sein, welche für das System oder für Browser reserviert sind oder von anderen Applikationen genutzt werden. Sie müssen aus 2 bis 20 Zeichen bestehen.
  • Aliasnamen können Groß- und Kleinbuchstaben, Ziffern und zwei Sonderzeichen (-), (_) enthalten. Diese beiden Sonderzeichen dürfen aber nicht an erster oder letzter Stelle stehen.

Benutzerdefinierte HTTP- oder HTTPS-Ports

Mit benutzerdefinierten HTTP- oder HTTPS-Ports kann eine Anwendung durch Eingabe einer Adresse wie http://diskstation:7000 oder http://diskstation:7001 geöffnet werden.

So aktivieren Sie einen benutzerdefinierten HTTP- oder HTTPS-Port:

  1. Wählen Sie eine Anwendung.
  2. Klicken Sie auf Bearbeiten.
  3. Markieren Sie im angezeigten Fenster Benutzerdefinierten Port (HTTP) aktivieren oder Benutzerdefinierten Port (HTTPS) aktivieren.
  4. Geben Sie eine benutzerdefinierte Portnummer an.

Hinweis:

  1. Es kann eine Portnummer zwischen 1 und 65535 ausgewählt werden, aber die folgenden Ports sind bereits für das System reserviert: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule Standard-Ports: 4662 (TCP), 4672 (UDP) und FTP-Standard-Portbereich: Der tatsächliche Bereich kann je nach Modell abweichen.
