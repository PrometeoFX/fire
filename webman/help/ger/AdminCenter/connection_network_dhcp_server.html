DHCP Server

Ihr DiskStation kann die Funktion eines DHCP Servers übernehmen und weist den DHCP-Clients innerhalb Ihres lokalen Netzwerks dynamische IP-Adressen zu.

So aktivieren Sie den DHCP Server:

  1. Gehen Sie zu Systemsteuerung > Netzwerk > Netzwerk-Schnittstelle.
  2. Wählen Sie eine Netzwerk-Schnittstelle (z. B. LAN) aus und klicken Sie dann auf Bearbeiten.
  3. Gehen Sie auf die Registerkarte DHCP Server, und markieren Sie DHCP Server aktivieren.
  4. Geben Sie einen Wert in das Feld Lease-Time von Adresse ein, um die DHCP-Lease-Time (in Minuten) für Adressen einzustellen, die DHCP-Clients zugewiesen wurden.
  5. Geben Sie die DNS-Adresse für DHCP-Clients in die Felder Primäres DNS (obligatorisch) und Sekundäres DNS (optional) ein.
  6. Stellen Sie den Domainnamen für DHCP Server im Feld Domainname (optional) ein.
  7. Klicken Sie auf OK.

So fügen Sie ein Teilnetz hinzu:

  1. Klicken Sie auf Hinzufügen, und geben Sie anschließend die Informationen in die folgenden Spalten ein:
    • Aktivieren: Aktivieren oder deaktivieren Sie das Teilnetz.
    • Start: Die erste IP-Adresse des Teilnetzes.
    • Ende: Die letzte IP-Adresse des Teilnetzes.
    • Netzmaske: Die Teilnetz-Maske.
    • Gateway: Der Gateway des Teilnetzes.
  2. Klicken Sie auf OK.

Hinweis:

  • Sie können nur bis zu 32 Subnetze hinzufügen.

DHCP Clients

Gehen Sie zur Registerkarte DHCP Clients, um eine Liste der DHCP Clients und deren Netzwerkkonfigurationen (z. B. MAC-Adresse, IP-Adresse, Hostnamen und verbleibende Zeit bis zum Ablauf des DHCP-Leasing) anzuzeigen.

DHCP-Reservierung

Falls ein Client bei jeder Erneuerung des DHCP-Leasings dieselbe IP-Adresse erhalten soll, können Sie den Client zur Liste für die Adressenreservierung hinzufügen.

So fügen Sie einen Client der Liste für die Adressenreservierung hinzu:

  • Klicken Sie auf Hinzufügen.
  • Geben Sie die MAC-Adresse des Clients an, die IP-Adresse, welche Sie diesem zuweisen möchten, und den Hostnamen des Clients.
  • Klicken Sie auf OK, um Ihre Einstellungen zu speichern.
