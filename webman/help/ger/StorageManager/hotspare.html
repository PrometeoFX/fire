Hot Spare*

Hot Spare Festplatten sind Standby Festplatten, die ein fehlerhaftes Volume/Diskgruppe/iSCSI LUN durch den automatischen Austausch einer defekten Festplatte reparieren können. Hot Spare Disks können übergreifend zum Schutz jedes Volume/Diskgruppe/iSCSI LUN auf DiskStation zugewiesen werden, solange das Volume/Diskgruppe/iSCSI LUN den folgenden Kriterien entspricht:

  • Der RAID Typ des Volume/Diskgruppe/iSCSI LUN muss Datenschutz bieten (d.h. RAID 1, RAID 5, RAID 6, RAID 10).
  • Die Größe der Hot Spare Festplatte muss größer oder genauso groß sein wie die Größe der kleinsten Disk des Volume/Diskgruppe/iSCSI LUN sein.

Hot Spare Festplatten einrichten:

  1. Klicken Sie auf Verwalten.
  2. Ein Dialog wird den Hot Spare Status jedes Volume/Diskgruppe/iSCSI LUN sowie die verfügbaren Festplatten anzeigen. Volume/Diskgruppe/iSCSI LUN Status umfasst Folgendes:
    • Zugewiesen: Das Volume/Diskgruppe/iSCSI LUN wird von mindestens einer zugewiesenen Hot Spare Festplatte geschützt.
    • Verfügbar: Es gibt verfügbare Festplatten, die das Volume/Diskgruppe/iSCSI LUN schützen können, aber es wurde bislang noch kein Hot Spare zugewiesen.
    • Nicht verfügbar: Es sind keine verfügbaren Hot Spare Festplatten für den Schutz des Volume/Diskgruppe/iSCSI LUN vorhanden.
  3. Die Abschnitte Verfügbare Festplatten zeigen Festplatten an, die zu Hot Spare Festplatten zugewiesen werden können. Um eine Hot Spare Festplatte zuzuweisen, markieren Sie das Kästchen neben der Festplatte, die Sie als Hot Spare zuweisen möchten.
  4. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

Hinweis: Wenn eine Festplatte als eine Hot Spare Festplatte zugewiesen wird, werden sämtliche Daten auf der Festplatte gelöscht. Stellen Sie vor dem Zuweisen einer Hot Spare Festplatte sicher, dass keine wichtigen Daten darauf gespeichert sind.

*Nur bei bestimmten Modellen verfügbar.
