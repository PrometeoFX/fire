Dateilinks freigeben

Dateien oder Ordner, die auf Ihrer DiskStation gespeichert sind, können schnell und einfach mit anderen geteilt werden. Einfach mit der rechten Maustaste auf eine Datei oder einen Ordner klicken und Dateilinks teilen auswählen. Dadurch werden ein Link und ein QR-Code erstellt, die an Freunde oder andere Benutzer gesendet werden können. Sie können dem Link folgen, um die ausgewählte Datei oder den ausgewählten Ordner herunterzuladen, unabhängig davon, ob Sie ein DSM-Konto besitzen.

So teilen Sie Datei-Links:

  1. Wählen Sie die Dateien oder die Ordner aus, die Sie teilen möchten.
  2. Rechtsklicken Sie auf eine der ausgewählten Dateien oder Ordner oder klicken Sie auf das Menü Aktion. Wählen Sie dann Dateilinks teilen aus.
  3. Geteilte Links und QR-Codes werden im Fenster, das daraufhin erscheint, angezeigt.
  4. Die folgenden Optionen können ebenfalls geändert werden:
    • Gültigkeitszeitraum: Sie können gültige Zugriffszeiträume und Zugriffszeiten für geteilte Links anpassen.
      • Gültigkeitszeitraum festlegen: Sie können festlegen, wie lange ein geteilter Link nach der Erstellung aktiv bleiben soll. Der Link wird ungültig, sowie die angegebene Zeit überschritten ist.
      • Gültige Zugriffszeiten anpassen: Sie können festlegen, wie häufig auf einen geteilten Link zugegriffen werden darf. Der Link wird ungültig, sowie die angegebene Häufigkeit überschritten ist.
    • Mit DSM-Benutzern teilen: Sie können den Link für DSM-Benutzer oder Gruppen freigeben, indem Sie ihn aus der Dropdown-Liste auswählen. Beachten Sie, dass Sie den Link nicht für das Guest-Konto und Benutzergruppen freigeben können.
    • Mit anderen teilen: Sie können kennwortgeschützte Links für andere Personen, einschließlich Nicht-DSM-Benutzer freigeben.

Gofile.me-Links aktivieren:

  1. Gehen Sie zu Systemsteuerung > QuickConnect und überprüfen Sie, ob QuickConnect aktivieren markiert ist. Falls Sie QuickConnect nicht eingerichtet haben, können Sie es auf dieser Seite tun.
  2. Klicken Sie dann auf Erweitert und stellen Sie sicher, dass Dateifreigabe aktiviert ist.
  3. Gehen Sie zu Systemsteuerung > Netzwerk > DSM-Einstellungen und überprüfen Sie, ob HTTPS-Verbindung aktivieren aktiviert ist. Geteilte Dateien können heruntergeladen werden, ohne dass Sie ein DSM-Konto besitzen, indem Sie auf den Austausch-Link „gofile.me“ zugreifen.

Austausch-Links verwalten

Bestehende Austausch-Links können über Extras > Verwaltung geteilter Links verwaltet werden. Im darauf angezeigten Fenster gibt es Optionen, um bestehende Austausch-Links zu bearbeiten, zu teilen oder zu entfernen.

Hinweis:

  1. Sie können bis zu 1000 geteilte Links erstellen.
  2. Im Status-Bereich jedes Austausch-Links wird Folgendes angezeigt:
    • Gültig: Der Austausch-Link funktioniert einwandfrei.
    • Abgelaufen: Die Gültigkeitsdauer des Austausch-Links ist abgelaufen.
    • Ungültig: Die geteilte Datei oder der geteilte Ordner ist nicht mehr vorhanden oder Sie besitzen keine ordnungsgemäße Berechtigung für den Zugriff auf die Datei oder den Ordner.
    • Noch nicht wirksam: Der Austausch-Link ist noch nicht aktiv.
