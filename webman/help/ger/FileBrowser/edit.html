Fotos bearbeiten

Mit der File Station können Sie Fotos mit webbasierten externen Fotobearbeitungsprogrammen z. B. Pixlr Editor oder Pixlr Express bearbeiten.

So bearbeiten Sie ein Foto mit einem webbasierten Fotobearbeitungsprogramm:

  1. Suchen Sie den Ordner, in dem sich das Foto befindet.
  2. Klicken Sie mit der rechten Maustaste auf das Foto, und wählen Sie In Pixlr Editor bearbeiten oder In Pixlr Express bearbeiten.
      Hinweis:
    • Pixlr Editor unterstützt die folgenden Formate: jpg, gif, png, psd, pxd (Pixlr-Schichtbild)
    • Pixlr Express unterstützt die folgenden Formate: jpg, gif, png
  3. Der ausgewählte Foto-Editor wird in einem neuen Browser-Fenster geöffnet. Das Foto wird sofort angezeigt, und Sie können es mit den zur Verfügung gestellten Werkzeugen bearbeiten.
  4. Abschließend speichern Sie das fertig bearbeitete Foto wie folgt.
    • In Pixlr Express: Klicken Sie auf Fertig, um das Foto im ursprünglichen Format an seinem ursprünglichen Speicherort auf DiskStation zu speichern.
    • In Pixlr Editor: Wählen Sie Datei > Speichern und dann den Speicherort (dies kann der ursprüngliche Speicherort Ihres Fotos auf DiskStation sein, Ihr Computer oder ein soziales Netzwerk) im Feld auf der linken Seite aus. Wählen Sie anschließend ggf. einen neuen Namen oder ein neues Format, und klicken Sie dann auf OK.

Hinweis: Vergewissern Sie sich vor der Bearbeitung Ihrer Fotos mit einem der webbasierten Fotobearbeitungsprogramme, dass Ihr DiskStation erfolgreich eine externe IP-Adresse abgerufen hat (unter Hauptmenü > Systemsteuerung > Externer Zugriff > DDNS oder Hauptmenü > Systemsteuerung > Netzwerk > Netzwerk-Schnittstelle > PPPoE, abhängig von der Netzwerkkonfiguration). Wenn sich Ihr DiskStation hinter einem Router befindet, achten Sie bitte darauf, dass Sie die Portweiterleitungsregel für DSM unter Hauptmenü > Systemsteuerung > Externer Zugriff > Routerkonfiguration eingerichtet haben. Wenn Sie immer noch Schwierigkeiten beim Öffnen der Fotobearbeitungsprogramme haben, geben Sie den Router-Port für DSM unter Hauptmenü > Systemsteuerung > Externer Zugriff > Erweitert an.
