Remote-Ordner bereitstellen

Sie können Remote-Ordner von Remote-Servern (z. B. Netzwerkcomputer oder einem anderen DiskStation) bereitstellen, um mit der File Station leicht nach verfügbaren Netzwerkressourcen zu suchen.

So stellen Sie einen Remote-Ordner bereit:

  1. Klicken Sie auf Extras > Remote-Ordner anhängen.
  2. Geben Sie "\\Remote_Server\Remote_Folder" in das Feld Ordner ein, um den Pfad des Remote-Ordners anzugeben. Zum Beispiel "\\192.168.0.2\share" oder "\\chad\cheeseburger\pictures".
  3. Wenn vom Remote-Server Zugriffsprivilegien zum Remote-Ordner verlangt werden, geben Sie Ihre Anmeldeinformationen in die Felder Kontoname und Passwort ein.
  4. Klicken Sie auf Durchsuchen, um einen leeren Zielordner auf Ihrem DiskStation für die Bereitstellung des Remote-Ordners auszuwählen oder zu erstellen.
  5. Aktivieren Sie Beim Start automatisch bereitstellen, wenn Sie möchten, dass DiskStation den Remote-Ordner bei jedem Systemstart bzw. Neustart bereitstellt.
  6. Klicken Sie auf Bereitstellen. Sie können anschließend den Zielordner, in dem der Remote-Ordner bereitsteht, durchsuchen und die Inhalte des Remote-Ordners verwalten.

So heben Sie die Bereitstellung eines Remote-Ordners auf:

  1. Wählen Sie den Remote-Ordner aus, den Sie trennen möchten, und klicken Sie auf Extras > Trennen.
  2. Sie können auch einfach auf den Remote-Ordner rechtsklicken und Trennen auswählen.

Hinweis:

  • Die maximale Anzahl von gleichzeitigen Remote-Ordnern ist 100.
  • Sie dürfen nur Remote-Ordner eines Remote-Servers bereitstellen, welcher das CIFS-Protokoll unterstützt.
