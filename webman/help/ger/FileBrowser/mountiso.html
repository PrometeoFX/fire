Virtuelle Laufwerke bereitstellen

Sie können virtuelle Laufwerke bereitstellen, um mit der File Station auf Inhalte von Datenträger-Abbildern zuzugreifen.

So stellen Sie ein virtuelles Laufwerk bereit:

  1. Wählen Sie ein Datenträger-Abbild aus und klicken Sie auf Extras > Virtuelles Laufwerk anhängen.
  2. Klicken Sie auf Durchsuchen, um auf DiskStation einen leeren Zielordner für die Bereitstellung des virtuellen Laufwerks auszuwählen oder zu erstellen.
  3. Aktivieren Sie Beim Start automatisch anhängen, falls Ihre DiskStation das virtuelle Laufwerk bei jedem Systemstart bzw. Neustart anhängen soll. Klicken Sie auf Anhängen, um fortzufahren. Sie können anschließend den Zielordner, in dem das virtuelle Laufwerk bereitsteht, durchsuchen und auf die Inhalte des Datenträger-Abbilds zugreifen.

So heben Sie die Bereitstellung eines virtuellen Laufwerks auf:

  1. Rechtsklicken Sie auf den Zielordner, in dem das virtuelle Laufwerk angehängt ist, und wählen Sie Trennen aus.
    Hinweis:
  1. Unterstütztes Datenträger-Abbildformat: ".iso" Datei unter Verwendung des Dateisystems ISO 9660 und UDF.
  2. Die maximale Anzahl von gleichzeitigen virtuellen Laufwerken beträgt 16.
