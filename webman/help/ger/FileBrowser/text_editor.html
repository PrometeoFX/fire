Text-Editor

Text-Editor ist ein sehr vielseitiger Text- und Code-Editor, mit dem Sie schnell auf Textdateien oder Codes auf Ihrem DiskStation direkt von einem Webbrowser aus zugreifen und sie bearbeiten können.

Dateiverwaltung

So werden neue Dateien erstellt:

Klicken Sie auf Datei und wählen Sie Neu aus. Eine neue leere Registerkarte mit dem Titel New Document.txt erscheint.

So werden Dateien vom Text-Editor aus geöffnet:

Klicken Sie einfach auf Datei, und wählen Sie Öffnen aus. Ein Dialog wird angezeigt, in dem Sie eine Datei mit einer bestimmten Zeichenkodierung auswählen können, die in einer Registerkarte geöffnet werden soll. Wenn die Datei bereits geöffnet ist, zeigt der Text-Editor die Datei in ihrem aktuellen Zustand.

So werden Dateien von File Station aus geöffnet:

Rechtsklicken Sie in File Station einfach auf die ausgewählte Datei, und klicken Sie dann im Kontextmenü auf Mit Text-Editor öffnen. Wenn die Datei bereits geöffnet ist, zeigt der Text-Editor die Datei in ihrem aktuellen Zustand. Eine weitere Möglichkeit, eine Datei zu öffnen ist, eine oder mehrere Dateien von File Station mit dem Cursor in den Text-Editor zu ziehen.


Hinweis:

Mit Text-Editor können Sie keine Datei öffnen, deren Größe 10 MB überschreitet.

So werden Dateien gespeichert:

Klicken Sie einfach auf Datei, und wählen Sie Speichern. Wenn es eine vorhandene Datei ist, wird sie mit dem aktuellen Inhalt überschrieben. Wenn Sie alle geöffneten Dateien auf einmal speichern möchten, gehen Sie zu Datei > Alle speichern. Wenn die Datei noch keinen Namen besitzt (eine neue Datei), oder Sie Datei > Speichern unter ausgewählt haben, öffnet sich ein Dialogfenster, um die Datei zu benennen. Sie können dann den Speicherort, den Dateinamen und die Codierung wählen, um die Datei zu speichern.

So werden Dateien geschlossen:

Klicken Sie einfach auf Datei, und wählen Sie Schließen. Für eine weitere Möglichkeit zum Schließen einer Datei klicken Sie auf die Schaltfläche „Schließen“ auf der jeweiligen Registerkarte. Wenn noch nicht gespeicherte Daten in einer Datei vorhanden sind, wird der Text-Editor Sie immer fragen, ob die Datei ohne die Änderungen gespeichert werden soll.

Hinweis:

Sie können Dateien mit einem bestimmten Zeichensatz speichern, um bidirektionale Sprachen zu unterstützen. Außerdem kann auch eine Codierung beim Öffnen einer Datei angegeben werden, damit Text-Editor die Datei korrekt anzeigt.

Bearbeiten

  • Standardvorgänge
    Die Bearbeitungssteuerung unterstützt die einfachen Befehle „Rückgängig“, „Wiederholen“ und „Alle auswählen“. Diese Vorgänge sind in den meisten Editoren zu finden und sind sehr ähnlich. Diese Vorgänge können durch Auswahl des entsprechenden Menüpunkts im Menü Bearbeiten ausgeführt werden. Zusätzlich sind die Standard-Befehle Ausschneiden, Kopieren und Einfügen über das Kontextmenü des Browsers verfügbar.
  • Einzug
    Das Einrücken von Text ist ein häufiger Vorgang beim Bearbeiten von Quellcodes. Das Einrücken bezieht sich auf das Einfügen von zusätzlichen Leerzeichen am Anfang einer Zeile, um Textblöcke zu trennen und eine Hierarchie hinzuzufügen. Normalerweise wird durch Drücken der Tabulatortaste ein Tabulator an der aktuellen Position eingefügt. Falls allerdings ein Text ausgewählt ist, fügt das Drücken der Tabulatortaste zusätzliche Leerzeichen am Anfang jeder Zeile des markierten Textes ein, sodass ganze Textblöcke auf einmal eingerückt werden können. Durch gleichzeitiges Drücken der Umschalttaste und der Tabulatortaste wird der Einzug auf einmal entfernt.
  • Überschreiben
    Überschreiben kann durch Drücken der Taste „Einfg“ (Einfügen) aktiviert und deaktiviert werden. Dies bedeutet, dass Zeichen an der Cursorposition ersetzt werden, anstatt sie einzufügen. Die angezeigte Cursor-Form ändert sich im Überschreiben-Modus entsprechend.
  • Spalten-Bearbeitungsmodus
    Eine besondere Art der Bearbeitung von Dateien ist die Bearbeitung im Spaltenmodus. Dies bedeutet, dass Text in Spalten ausgewählt werden kann, anstatt in Zeilen. Um eine Spaltenmodus-Auswahl zu starten, drücken Sie bei der Auswahl von Text die Alt-Taste und bewegen Sie den Cursor nach Bedarf nach oben oder unten. Wenn der Text in diesem sogenannten Spaltenmodus ausgewählt wurde, wird beim Eingeben jede ausgewählte Spalte von Zeichen zum eingegebenen Zeichen geändert.

Suchen

  • Einfache Suche
    Sie können Text wie in den meisten Programmen mit dem Suchen-Dialog suchen. Dieser Dialog öffnet sich unter Suche > Suchen. Dieses Dialogfenster wird immer über dem Text angezeigt, und der Text kann nicht bearbeitet werden, solange es geöffnet ist. Geben Sie in das Suchfeld den gesuchten Text ein.
  • Ersetzen
    Sie können Text wie in den meisten Programmen mit dem Ersetzen-Dialog suchen und ersetzen. Dieser Dialog öffnet sich unter Suche > Ersetzen. Alle ersetzen ersetzt alle Einträge der abgefragten Suchbegriffe durch den Ersatztext.
  • Groß-/Kleinschreibung
    Sie erzwingt, dass die Suchergebnisse genau die gleiche Schreibweise wie die Abfrage aufweisen. Wenn dies nicht markiert ist, wird die Groß-/Kleinschreibung bei der Suche nicht beachtet.
  • Nur ganzes Wort
    Es werden nur Suchergebnisse angezeigt, wenn das Ergebnis aus einem ganzen Wort und nicht nur einem Teil davon besteht. Wenn Sie z. B. das Wort „Bei“ mit Nur ganzes Wort suchen, wird nur „Bei" gefunden, und nicht „Beispiel“.
  • Regulärer Ausdruck
    Die Abfrage wird wie ein regulärer Ausdruck behandelt.

Was ist ein regulärer Ausdruck?

Ein regulärer Ausdruck ist eine Folge von Zeichen, die für eine leistungsstarke, zeitsparende Suchtechnik ein Suchmuster bildet. Eine sehr einfache Verwendung eines regulären Ausdrucks wäre, ein im Text-Editor auf zwei Arten geschriebenes Wort zu finden, z. B. Ha[un]s.

Stil-Konfigurator

Der Stil-Konfigurator wird in erster Linie verwendet, um das Aussehen des Textes im Texteditor zu steuern.
Die Drop-down-Liste unten rechts enthält alle unterstützten Programmiersprachen, deren Stil geändert werden kann, um Ihnen dabei zu helfen, eine geeignete Programmierumgebung zu erstellen.

Voreinstellungen

  • Unsichtbare Zeichen anzeigen
    Wenn dies markiert ist, werden Leerzeichen, Tabulatoren, Zeilenvorschub- oder Zeilenende-Zeichen angezeigt.
  • Einrückungslinien anzeigen
    Wenn dies markiert ist, werden Einrückungslinien angezeigt. Dies ist nützlich für das Ausrichten von Codes in der Codiersprache.
  • Aktuelle Zeile markieren
    Wenn dies markiert ist, wird die Zeile, bei der sich die Einfügemarke befindet, mit einer speziellen Hintergrundfarbe gekennzeichnet.
  • Ausgewähltes Wort markieren
    Wenn dies markiert ist, werden automatisch alle Wörter (nicht teilweise) in derselben Textdatei markiert, die mit dem ausgewählten Wort übereinstimmen.
  • Zeilennummern anzeigen
    Wenn dies markiert ist, werden die Zeilennummern angezeigt.
  • Codefaltung aktivieren
    Wenn dies markiert ist, können bestimmte Zeilen des Codes, basierend auf bestimmten Merkmalen, verborgen werden. Diese Funktion eignet sich gut für strukturierte Sprachen wie C++ oder XML. Wenn Offenlegungs-Widgets automatisch ausblenden markiert ist, werden bestimmte Codezeilen automatisch gefaltet. Sie können auch festlegen, dass nur die Start-Klammer oder sowohl Start- und End-Klammer bei den Offenlegungs-Widgets markiert werden.
  • Zeilenumbruch
    Bestimmen Sie, ob der Text am Fensterrand oder bei einer bestimmten Spaltennummer umgebrochen wird, ohne einen harten Zeilenumbruch einzufügen.
  • Schließende Zeichen automatisch hinzufügen
    Wenn dies markiert ist, werden End-Klammern (Zeichen) automatisch zu den Codes hinzugefügt.
  • Leerzeichen anstelle von Tabulatoren einfügen
    Wenn dies markiert ist, wird durch Drücken der Tabulatortaste ein Leerzeichen anstatt eines Tabulators eingefügt. Tabulaborbreite kann durch die Anzahl von Zeichen unter Voreinstellungen > Schriftart und Farbe festgelegt werden.
  • Standardkodierung
    Die Standardkodierung wird verwendet, um Dateien mit einem bestimmten Zeichensatz zu speichern, damit bidirektionale Sprachen unterstützt werden. Außerdem kann auch eine Codierung beim Öffnen einer Datei angegeben werden, damit Text-Editor die Datei korrekt anzeigt.
  • Schriftart und Farbe
    Tabulatorbreite, Schriftgröße und Thema können nach Ihren persönlichen Vorlieben angepasst werden. Tabulaborbreite wird nach Anzahl von Leerzeichen definiert.

Unterstützte Dateiformate

Text-Editor unterstützt nur einfache Textdateien. Text-Dateien mit umfangreicher Formatierung wie *.doc und *.pdf werden nicht unterstützt.
