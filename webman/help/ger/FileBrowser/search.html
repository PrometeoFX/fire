Dateien oder Ordner suchen

Der Filter von File Station bietet Ihnen Optionen zur Suche und Anzeige von Dateien oder Ordnern, die Ihren Kriterien entsprechen. Sie können Dateien oder Ordner im aktuellen Ordner filtern, oder eine erweiterte Suche ausführen, die sämtliche Unterordner abdeckt, um genauere Suchergebnisse zu erzählen.

So suchen Sie nach Dateien oder Ordnern im aktuellen Ordner:

  1. Wechseln Sie mit der File Station zu dem Ordner, der die Inhalte enthält, die Sie durchsuchen möchten.
  2. Geben Sie im Feld Filter oben rechts einen Suchbegriff ein, dann werden die Suchergebnisse im Hauptbereich des Fensters angezeigt.

So führen Sie eine erweiterte Suche durch:

  1. Wechseln Sie mit der File Station zu dem Ordner, der die Inhalte enthält, die Sie durchsuchen möchten.
  2. Klicken Sie auf das Lupensymbol neben dem Feld Filter oben rechts, um das erweiterte Suchmenü anzuzeigen.
  3. Geben Sie Informationen ein oder treffen Sie eine Auswahl in den Dropdown-Menüs für die folgenden Suchkriterien:
    • Schlüsselwort: Geben Sie ein Schlüsselwort ein, um nach Dateien oder Ordnern mit übereinstimmenden Namen zu suchen.

      Hinweis:

      • Sie können Platzhalter [*] (d. h. kein oder mehrere Zeichen) und [?] (d. h. ein beliebiges Zeichen) in den Suchbegriffen verwenden. Beispielsweise kann [a*n] für [aktion], [alben] oder [an] stehen.
      • Sie können Kommas (,) verwenden, um mehrere Schlüsselwörter zu trennen. [a,tion] kann z. B. für [action], [a function], [function added] oder [nation] stehen.
      • Sie können ein Anführungszeichen ["] an den Anfang und das Ende von Schlüsselwörtern stellen, um nach exakten Übereinstimmungen zu suchen. Z. B. kann [„a typ“] für [a type], aber nicht für [typical] stehen.
    • Speicherort: Wählen Sie aus dem Dropdown-Menü die Einstellung des Bereichs und der Tiefe der erweiterten Suche aus. Sie können auch den Suchumfang anpassen, indem Sie andere gemeinsame Ordner in der Liste auswählen.
    • Dateityp: Wählen Sie Erweiterung, um den Suchbereich auf bestimmte Dateitypen zu begrenzen, oder wählen Sie Beliebiger Ordner, um nur nach Ordnern zu suchen.
    • Datum: Wählen Sie Änderungsdatum, Erstellungsdatum oder Zuletzt zugegriffen und wählen Sie anschließend ein Datum aus dem Kalender, um nach Dateien mit den entsprechenden Daten zu suchen.
    • Größe (MB): Wählen Sie Gleich, Ist mehr als oder Ist weniger als und geben Sie die Größe für die Suche nach Dateien mit übereinstimmenden Größen ein. (Wenn Sie Gleich wählen, werden die Suchergebnisse Dateien mit ungefähr derselben Größe wie angegeben enthalten.)
    • Besitzer/Gruppe: Wählen Sie Besitzer oder Gruppe und treffen Sie eine Auswahl im Dropdown-Menü, um nach Dateien oder Ordnern zu suchen, die zum ausgewählten Besitzer oder zur ausgewählten Gruppe gehören.

      Hinweis:

      • Für die erweiterte Suche müssen Sie Speicherort und mindestens ein weiteres Suchkriterium angeben.
      • Wenn ein ISO-Ordner oder ein Remote-Ordner an einen indizierten gemeinsamen Ordner angehängt ist, können Sie über die erweiterte Suche die darin enthaltenen Dateien nicht finden.
    • Klicken Sie auf Suche, um mit der Suche zu beginnen, oder klicken Sie auf Zurücksetzen und geben Sie ein anderes Suchkriterium an. Nachdem die Suche durchgeführt wurde, werden übereinstimmende Dateien im Hauptbereich der File Station aufgelistet.
