Berichtsprofile verwalten

Sie können geplante Aufgaben erstellen und ändern, um Berichte über die Volume-Auslastung auf dem Speicher-Analysator > Berichtsprofil zu erzeugen.

Erstellen eines Berichtsvorgangs:

  1. Klicken Sie auf Erstellen, um den Assistenten zu öffnen.
  2. Legen Sie einen Namen für die neue Berichtsaufgabe fest, und geben Sie die E-Mail-Adresse ein, an die Sie den Bericht senden möchten.
  3. Wählen Sie einen oder mehrere Berichtstypen aus, die Sie erstellen möchten.
  4. Führen Sie eine der folgenden Schritte aus, um eine Uhrzeit für die Erstellung von Berichten festzulegen:
    • Aktivieren Sie Berichte nach einem Zeitplan erstellen und geben Sie die Uhrzeit an, wenn Sie möchten, dass das System die Volume-Auslastung in regelmäßigen Abständen analysiert.
    • Aktivieren Sie Berichte jetzt erstellen, wenn Sie jetzt einen Berichtsvorgang manuell starten möchten.
  5. Falls notwendig, geben Sie bestimmte Benutzer oder gemeinsame Ordner an, welche eingefügt werden sollten, um die Berichte einzugrenzen. (Beachten Sie folgendes: Wenn Sie verschlüsselte Berichte und gemeinsame Ordner festlegen und erstellen möchten, sollten Sie sicherstellen, dass die verschlüsselten Dateien angehängt werden, bevor der Berichtsvorgang ausgeführt wird.)
  6. Klicken Sie auf Übernehmen, um die Einstellungen zu speichern.

Hinweis:

  • Drücken Sie nach jeder E-Mail-Adresse auf die Eingabe/Zurück-Taste, um mehrere E-Mail-Adressen zu trennen.
  • Drücken Sie nach jedem Benutzer auf die Eingabe/Zurück-Taste, wenn Sie Benutzer angeben, deren Dateien individuell nach Dateigruppen aufgelistet werden.
  • Der Bericht Dateien nach Dateigruppe listet die folgenden Dateitypen auf:
    Audio, Disk-Image-Dateien, Dokumente, Ausführbare Dateien, Bilder, Videos, Webseiten, gezippte Dateien, Sonstiges
  • Der Abschnitt Große Dateien des Berichts listet die 500 größten Dateien auf der Verwendungsberichtsseite, oder die 200 größten Dateien auf der vollständigen Berichtsseite auf.

Bearbeiten eines Berichtsvorgangs:

  1. Wählen Sie einen Berichtsvorgang aus der Liste aus und klicken Sie auf Bearbeiten.
  2. Klicken Sie auf die einzelnen Registerkarten, um Einstellungen für diese Aufgabe zu ändern.

Löschen eines Berichtsvorgangs:

Wählen Sie einen Berichtsvorgang aus der Liste aus, und klicken Sie auf Löschen.

Historische Berichte anzeigen:

  1. Wählen Sie einen Berichtsvorgang aus der Liste aus, und klicken Sie auf Bericht > Historische Berichte anzeigen.
  2. Klicken Sie im angezeigten Fenster auf einen Eintrag, um den zu diesem Zeitpunkt erstellten Bericht anzuzeigen.

Berichte sofort erstellen:

Wählen Sie einen Berichtsvorgang aus der Liste aus, klicken Sie auf Bericht > Berichte jetzt erstellen, um Berichte auf Abruf zu erstellen.

Ändern, wo die Berichte gespeichert werden:

  1. Klicken Sie auf Einstellungen.
  2. Wählen Sie einen neuen Zielordner, um den Ort zu ändern, an dem die Berichte gespeichert werden.

Den Volume-Auslastungsverlauf nach einem regelmäßigen Zeitplan erfassen:

Die Daten der Volume-Auslastung werden standardmäßig jedes Mal bei der Erstellung eines Volume-Auslastungsberichts und beim Systemstart erfasst. Zusätzlich zu den obigen Fällen, können Sie einen neuen Zeitplan auch einrichten, um die Daten des Auslastungsverlaufs zu erfassen. Hierzu klicken Sie auf Einstellungen, und markieren Sie Volume-Auslastungsverlauf nach einem Zeitplan sammeln, und legen Sie einen Zeitpunkt fest.

Hinweis:

Der zu diesem festgelegten Zeitpunkt erfasste Verlauf dient hauptsächlich zur Bestimmung der Auslastungstrends im Abschnitt Volume-Auslastung. Wenn diese Option aktiviert ist, wird ein Liniendiagramm mit der Anzeige der Trends und zeitlichen Veränderungen für die Volume-Auslastung angezeigt; falls nicht, erscheint ein Balkendiagramm, das stattdessen den aktuellen Auslastungsprozentsatz anzeigt.

Analysieren eines Berichtsvorgangs:

Doppelklicken Sie auf den gewünschten Berichtsvorgangs oder klicken Sie einfach auf Diesen Bericht analysieren (das >-Symbol) neben dem gewünschten Berichtsvorgang, um seine Analyseseite zu öffnen. In der Datei Nutzung und Berichte erhalten Sie weitere Informationen.
