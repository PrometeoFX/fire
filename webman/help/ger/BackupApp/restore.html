Wiederherstellen

Gehen Sie zu Datensicherung & Replikation > Wiederherstellen, um die gesicherten Daten wiederherzustellen.

Datensicherungsaufgabe wiederherstellen:

  1. Wählen Sie eine Datensicherungsaufgabe, und klicken Sie dann auf Wiederherstellen. Wählen Sie Wiederherstellen > Gesamte Aufgabe aus, wenn die Aufgabe die Sicherung mehrerer Versionen umfasst.
  2. Sie können aufgefordert werden, die Systemkonfiguration, verschiedene Versionen von gesicherten Daten oder andere Optionen auszuwählen. Dies ist davon abhängig, ob es sich um eine Datensicherungsaufgabe, eine LUN-Sicherungsaufgabe*, ein Datensicherungsziel für einzelne Versionen (d. h. freigegebener Ordner) oder ein Datensicherungsziel für mehrere Versionen (d. h. Volume) handelt.
  3. Folgen Sie dem Assistenten, um das Setup abzuschließen.
  1. Wählen Sie eine Datensicherungsaufgabe, und klicken Sie dann auf Wiederherstellen.
  2. Sie können aufgefordert werden, die Systemkonfiguration, verschiedene Versionen von gesicherten Daten oder andere Optionen auszuwählen. Dies hängt davon ab, um welche Art von Datensicherungsaufgabe es sich handelt.
  3. Folgen Sie dem Assistenten, um das Setup abzuschließen.

Hinweis:

  • Die Wiederherstellung für bestimmte Anwendungen wird unterstützt. Ob ein bestimmtes Paket die Wiederherstellung unterstützt und welche Einschränkungen dabei gelten, können Sie in der Hilfe zu den einzelnen Anwendungen nachlesen.
  • Wenn Sie ein Paket wiederherstellen, das jedoch nicht installiert ist, installiert das System automatisch dieselbe oder eine aktuelle Version der Anwendung. Wenn die Anwendung installiert, deren Version aber älter als die gesicherte ist, wird sie vom System automatisch auf die ursprünglich gesicherte Version aktualisiert.
  • Anwendungen, die in DSM 5.1 oder höher gesichert wurden, können in DSM 5.0 oder niedriger nicht wiederhergestellt werden.

Einzelne Datei oder einzelnen Ordner für Aufgabe zur Sicherung mehrerer Versionen wiederherstellen:

  1. Starten Sie den Dateiversions-Browser. Hierzu können Sie nach einer der nachstehenden Methoden vorgehen:
    • Klicken Sie auf Wiederherstellen und wählen Sie Einzelne Dateien/Ordner aus dem Dropdown-Menü aus.
    • Gehen Sie zu Datensicherungsziel und erweitern Sie Ihr ausgewähltes Datensicherungsziel. Klicken Sie auf das Symbol Informationen zur Datensicherungsversion. Wählen Sie im Fenster, das eingeblendet wird, die Sicherungsversion aus, die Sie wiederherstellen möchten, und klicken Sie auf das Symbol Dateiversions-Browser.
  2. Die Versionen der gewünschten Datei oder des gewünschten Ordners können Sie über das Zeitachsen- oder das Kalendersymbol am unteren Bildschirmrand durchsuchen.
  3. Wählen Sie die Version aus, die Sie verwalten möchten, und klicken Sie auf Wiederherstellen.
  4. Alternativ k��nnen Sie eine Datei oder einen Ordner zu einem anderen Speicherort kopieren oder über die Schaltfläche Kopieren nach... oder Herunterladen herunterladen.
  5. Sie können auch auf die Datei oder den Ordner rechtsklicken und diese Optionen aus dem Dropdown-Menü auswählen.

Nach einer bestimmten Version suchen:

  1. Klicken Sie auf den Pfeil in der Filterleiste oben rechts.
  2. Im Dropdown-Menü können Sie Dateien nach den folgenden Kriterien filtern:
    • Schlüsselwort: Geben Sie den Namen einer Datei oder eines Ordners ein. Sie können Platzhalter [*] (d. h. ein oder mehrere Zeichen) und [?] (d. h. ein beliebiges Zeichen) in den Schlüsselwörtern verwenden. Beispielsweise kann [a*n] für [aktion], [alben] oder [an] stehen.
    • Dateityp: Aus dem Dropdown-Menü können Sie aus Beliebig, Beliebiger Ordner und Beliebige Datei auswählen, um Ihre Suche einzugrenzen.
    • Größe: Wählen Sie Beliebig aus oder wählen Sie Entspricht, Ist mehr als, Ist weniger als aus und geben Sie einen Wert ein.
    • Datumsbereich: Wählen Sie Heute, Gestern, Letzte Woche, Letzter Monat oder Anpassen aus, um ein Start- und Enddatum einzugeben, um nach Versionen innerhalb eines bestimmten Bereichs zu suchen.
  3. Klicken Sie auf Filtern, um Ihre Filter anzuwenden.
  4. Klicken Sie auf die Schaltfläche Reset, um Ihre gefilterte Suche zurückzusetzen.

Hinweis:

  • Die Funktion "Dateiversions-Browser" ist nur bei Aufgaben zum Sichern mehrerer Versionen verfügbar.
  • In DSM 5.1 oder niedriger hat der Dateiversions-Browser einen begrenzten Funktionsumfang.
  • Die Systemkonfiguration und Anwendungen können nur über Wiederherstellen > Gesamte Aufgabe wiederhergestellt werden.
  • Metadaten werden von den Funktionen Kopieren nach... und Download ignoriert.

Datensicherungsaufgaben von anderen Datensicherungszielen wiederherstellen:

  1. Klicken Sie auf Wiederherstellen aus....
  2. Wählen Sie eine der folgenden Methoden aus, um die Datensicherungsaufgabe zu suchen, und klicken Sie auf Weiter:
    • Lokale Wiederherstellung: Wählen Sie diese Option aus, um Daten von einem lokalen freigegebenen Ordner oder einer externen Festplatte aus wiederherzustellen.
    • Netzwerkwiederherstellung: Wählen Sie diese Option aus, um Daten von einer anderen DiskStation oder einem rsync-kompatiblen Server aus wiederherzustellen.
    • Wiederherstellung von öffentlichem Cloud Service: Wählen Sie diese Option aus, um Daten von einem öffentlichen Cloud Server wiederherzustellen.
  3. Wählen Sie bei Aufforderung die Quelle aus, wo sich die Datensicherungsaufgabe befindet. Wählen Sie anschließend das Ziel aus, an dem die Daten wiederhergestellt werden sollen.
  4. Folgen Sie dem Assistenten, um den Vorgang abzuschließen.

Hinweis:

  • Embedded DataStation kann freigegebene Ordner nur auf externen ext3/ext4-Geräten wiederherstellen.
  • Embedded DataStation unterstützt nicht die Anwendungswiederherstellung ohne ein externes ext3/ext4-Gerät.
