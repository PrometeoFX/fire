Edit Photos

With File Station, you can edit photos with web-based external photo editors, such as Pixlr Editor or Pixlr Express.

To edit a photo with an web-based photo editor:

  1. Browse to the folder where the photo is located.
  2. Right-click the photo and choose Edit in Pixlr Editor or Edit in Pixlr Express.
      Note:
    • Pixlr Editor supports the following formats: jpg, gif, png, psd, pxd (Pixlr layered image)
    • Pixlr Express supports the following formats: jpg, gif, png
  3. The chosen photo editor will be opened in a new browser window. You can see your photo in a moment, and then edit it with the provided tools.
  4. After you finish editing your photo, do the following to save your photo.
    • For Pixlr Express: Click Done to save your photo in its original format at its original location on DiskStation.
    • For Pixlr Editor: Choose File > Save, choose the location (which could be the photo's original location on DiskStation, your computer, or other social networks) to save it from the left panel, decide a new name or format according to your preferences, and then click OK.

Note: To edit your photos with any of the web-based photo editors, make sure your DiskStation has successfully retrieved an external IP address (at Main Menu > Control Panel > External Access > DDNS or Main Menu > Control Panel > Network > Network Interface > PPPoE, depending on your network configuration). If your DiskStation is located behind a router, make sure you have specified the port forwarding rule for DSM at Main Menu > Control Panel > External Access > Router Configuration. If you still have trouble opening the photo editors, you might need to specify the router port for DSM at Main Menu > Control Panel > External Acess > Advanced.
