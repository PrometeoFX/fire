Hot Spare*

Hot spare disks are standby hard disks that can repair a degraded volume/Disk Group/iSCSI LUN by automatically replacing a failed disk. Hot spare disks can be globally assigned to protect any volume/Disk Group/iSCSI LUN within your DiskStation, as long as the volume/Disk Group/iSCSI LUN matches the following criteria:

  • The RAID type of the volume/Disk Group/iSCSI LUN must be one with data protection (i.e. RAID 1, RAID 5, RAID 6, RAID 10).
  • The hot spare disk size must be equal to or larger than the size of the smallest disk in the volume/Disk Group/iSCSI LUN.

To set up hot spare disks:

  1. Click Manage.
  2. A dialog appears, displaying the hot spare status of each volume/Disk Group/iSCSI LUN, as well as available disks. Volume/Disk Group/iSCSI LUN status includes the following:
    • Assigned: The volume/Disk Group/iSCSI LUN is protected by at least one assigned hot spare disk.
    • Available: There are available disks that can protect the volume/Disk Group/iSCSI LUN, but no hot spare has been assigned yet.
    • Not available: There are no available hot spare disks that can protect the volume/Disk Group/iSCSI LUN.
  3. The Available Disks sections displays hard disks that can be assigned as hot spare disks. To assign a hot spare disk, check the box next to the disk you wish to assign as a hot spare.
  4. Click Apply to save settings.

Note: When a hard disk is assigned as a hot spare disk, all data on disk is erased. Before assigning a hot spare disk, make sure the disk has no important data on it.

*Only available on certain models.
