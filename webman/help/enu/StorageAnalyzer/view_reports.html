View Usage and Reports

With Storage Analyzer, you can not only monitor the overall volume usage trends, but also have a glance at detailed storage analysis generated from scheduled tasks, so as to learn more about your storage space and to have better management of your DiskStation.

Volume Usage

In the Volume Usage section, you'll find a line graph displaying the volume usage trends and changes over time, or a bar chart indicating the current usage percentage if you have not enabled the volume usage history option (at Report Profiles > Settings > Collect volume usage history according to a schedule).

To view volume usage, do any of the following:

  • Hover your mouse over the line in the line graph to view the used, available, and total capacity of the volume at the date indicated.
  • Click on the 1 and 2 buttons on the top-right corner to switch between two display modes (line graph and list) according to your preference. The line graph allows you to have a quick look at the overall trend over a short period of time, while the list offers the most recently collected data.

Note:

If you have more than 10 volumes on your DiskStation, only the top ten volumes with the highest usage will be shown.

Usage Reports

In addition to current volume usage, you can also view various types of usage reports generated from previous tasks, or easily find the desired report that meets the specified criteria or date in just a few simple clicks. Analysis reports are classified into different types, including the quota usage of each folder or file type, and the usage of files according to their modified or accessed time.

To view usage reports:

  1. In the Report Profile section, select a task from the profile list and then double-click the task, or simply click Analyze this report task (the > icon) next to the desired task to view its collected data and analysis. (Please note tasks that have never been performed cannot be selected.)
  2. Click the individual tabs on the left panel of the report page to view corresponding analysis:
    • Quota Usage: Displays the quota occupied by each shared folder.
    • User Quota: Displays the file count and size owned by each user in every single shared folder.
    • Type: Displays the quota of each file type in every single shared folder.
    • Duplicates: Displays duplicated files found in the system for the convenience of organizing or deleting unnecessary copies. Please note that the greater the maximum number of duplicated files you set, the longer it will take to process.
    • Size: Displays up to 500 files in descending order of their size.
    • Most Recently Modified Files: Displays up to 500 files in descending order of their last modified time.
    • Least Recently Accessed Files: Displays up to 500 files in ascending order of their last accessed time.
  3. If necessary, specify criteria from the drop-down menu to filter reports and to find the information you need more efficiently.
  4. Click on any dates on the timeline slider at the bottom to view reports generated on the selected date.
  5. Alternatively, you can click View complete report at the top-right corner if you prefer viewing full analysis in one single page.
