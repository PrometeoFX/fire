Sync files between eCore Server and your computer using Cloud Station

Overview

Cloud Station for eCore Server is a file-syncing application that lets you easily synchronize files on your eCore Server with other devices, such as computers or mobile devices (with DS cloud). Once Cloud Station has been installed on both your eCore Server and computer, files stored in a specified folder on your computer will be automatically synced to your eCore Server.

1. Install Cloud Station

  1. Go to Package Center, find Cloud Station, and then click Install.

2. Enable Cloud Station

  1. Launch Cloud Station.
  2. If you have not enabled the user home feature, you'll be prompted to do so. Click OK.
  3. You'll be redirected to the user settings page. Enable user homes and click Apply.
  4. Now go back to Cloud Station and you'll be prompted to enable it. Click Yes.
  5. Go to Privileges and specify which users will be able to use Cloud Station. Click Save.

3. Install and enable client

  1. Go to Cloud Station > Overview and click the Computers button.
  2. Click the button to download Cloud Station for your operating system. (You can also download Cloud Station from the Synology Download Center.)
  3. Launch the installer on your computer and follow the onscreen instructions.
  4. After installation, run Cloud Station on your computer. Click Start Now.
  5. Enter the address for the eCore Server running Cloud Station (or QuickConnect ID), username, and password. You can click the search icon on the right to search for IPs within your LAN. Click Next.
  6. Select how you want to set up your task (here we'll select Quick Setup), then click Next.
  7. Click Done to complete setup of your Cloud Station folder.
  8. Click Open it now to open your Cloud Station folder.
  9. You can find the Cloud Station icon in your system tray.
  10. Double click the Cloud Station icon to open the tray menu. Here you can view the progress and status of your files. Click Main app to launch Cloud Station.
  11. Upon launch, you will be shown the some tips on how to use Cloud Station. Click the right arrow to see more tips, or click Go to online tutorials for more information. If you don't need the tips, click Don't show this again.
  12. Now you can manage your sync tasks in Cloud Station.
