Application Portal

With Application Portal, you can configure the connection settings of various applications, allowing you to access and run these applications in independent browser tabs or windows.

For example, you could open File Station by entering http://:diskstation:5000/file/ into your web browser. Available connection settings include custom aliases, HTTP, and HTTPS ports. To find these settings, go to Control Panel > Application Portal.

Note:

Application Portal allows you edit access settings for various applications, such as Audio Station, Download Station, Surveillance Station, Video Station, File Station, as well as others.

Customize Alias

With a custom alias, an application like File Station could be opened by entering an address like http://:diskstation:5000/file/ in your web browser.

To enable a custom alias:

  1. Select an application.
  2. Click Edit.
  3. In the window that appears, tick Enable customized alias.
  4. Specify a custom alias.

Note:

  • You can tick the checkbox Redirect from and use a shorter, easy-to-remember alias to access the selected application.
  • Following are the respective default aliases of the applications, Audio Station, Download Station, Surveillance Station, Video Station, File Station: audio, download, cam, video, file.
  • Alias names cannot be the same as those reserved for system, browsers, or used by other applications, and their number of characters must be limited within 2 to 20.
  • Alias names can consist of mixed case letters, numeric characters, and two special characters (-), (_), but these two characters can only be used for the other characters except the first and last ones.

Custom HTTP or HTTPS Ports

With custom HTTP or HTTPS ports, an application can be opened by entering an address like http://diskstation:7000 or http://diskstation:7001.

To enable a custom HTTP or HTTPS port:

  1. Select an application.
  2. Click Edit.
  3. In the window that appears, tick Enable customized port (HTTP) or Enable customized port (HTTPS).
  4. Specify a custom port number.

Note:

  1. The range of port number can be from 1 to 65535, but the following ports should be excluded because they are reserved for system use: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule default ports: 4662 (TCP), 4672(UDP), and FTP default port range: the actual range may vary depending on different models.
