DSM Settings

At Control Panel > Network > DSM Settings, you can change the port number that clients use to access DSM. When HTTPS is enabled, connections will be encrypted with SSL/TLS.

To change the default port number:

  1. Enter the port number in the HTTP or HTTPS field.
  2. Click Apply.

To Enable HTTPS connections:

  1. Tick Enable HTTPS connection.
  2. If you would like to redirect clients to the HTTPS port (5001) when trying to access DSM through the HTTP port (5000), tick the Automatically redirect HTTP connections to HTTPS box.
  3. Click Apply.

Ports used for HTTPS connection:

  • DSM: The default port number is 5001, so you can securely access DSM by entering the following address: https://server-hostname:5001/
  • Web Station and Photo Station: The default port number is 443, so you can securely access Web Station and Photo Station by entering the following address: https://server-hostname/

To make the DiskStation discoverable via Windows network discovery:

  1. Tick Enable Windows network discovery.
  2. Click Apply.
