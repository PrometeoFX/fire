编辑照片

File Station 可让您通过网页界面的外部照片编辑器（如 Pixlr Editor 或 Pixlr Express 等）来编辑照片。

若要使用网页界面的照片编辑器编辑照片：

  1. 请浏览至照片所在的文件夹。
  2. 右键单击该照片，并选择在 Pixlr Editor 编辑或在 Pixlr Express 编辑。
      注：
    • Pixlr Editor 支持下列格式：jpg、gif、png、psd、pxd（Pixlr 分层图片）
    • Pixlr Express 支持下列格式：jpg、gif、png
  3. 所选照片编辑器会在新的浏览器窗口中打开。请稍待片刻，便可以看到您的照片，然后使用屏幕上提供的工具进行编辑。
  4. 照片编辑完成后，进行下列操作来保存照片。
    • 对于 Pixlr Express：单击完成来将照片以原始文件格式保存到 DiskStation 上的原始位置。
    • 对于 Pixlr Editor：选择文件 > 保存，从左侧面板选择照片的存储位置（您可以选择将照片保存到 DiskStation 上的原始位置、您的计算机或其他社交网络），并按照您的喜好决定新名称或文件格式，然后单击确定。

注：若要使用网页界面照片编辑器来编辑照片，请确认您的 DiskStation 已成功检索到外部 IP 地址（位于主菜单 > 控制面板 > 外部访问 > DDNS 或主菜单 > 控制面板 > 网络 > 网络接口 > PPPoE，取决于您的网络配置）。如果您的 DiskStation 位于路由器后方，请确认您已在主菜单 > 控制面板 > 外部访问 > 路由器配置中为 DSM 指定端口转发规则。如果您打开照片编辑器时仍遇到问题，则可能需要在主菜单 > 控制面板 > 外部访问 > 高级中指定 DSM 的路由器端口。
