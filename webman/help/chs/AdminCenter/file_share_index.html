创建文件索引

为共享文件夹中的内容创建索引，可辅助您运行文件搜索时更快速地找到文件或文件夹。

为共享文件夹中的文件创建索引：

  1. 选择您要创建索引的共享文件夹。
  2. 单击编辑，然后单击文件索引选项卡。
  3. 勾选启用文件索引。
  4. 单击确定。

为文件重建索引

在一般情况下，您不需要为文件重建索引。然而，如果文件夹中的文件符合搜索条件，却没有出现在文件的搜索结果里，您也许需要重建索引来重新创建文件的索引结构。

若要为共享文件夹中的文件重建索引：

  1. 选择您要重建索引的共享文件夹。
  2. 单击编辑，然后单击文件索引选项卡。
  3. 单击重建索引。
  4. 过程结束之后，系统会显示已索引的文件及子文件夹数量。

注：File Station 不支持远程文件夹或外接 USB 盘上的文件索引。
