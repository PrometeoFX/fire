DHCP Server

DiskStation 用作 DHCP 服务器并将动态 IP 地址指派给局域网中的 DHCP 客户端。

若要启用 DHCP 服务器：

  1. 请进入控制面板 > 网络 > 网络接口。
  2. 选择网络接口（例如：局域网）并单击编辑。
  3. 进入 DHCP Server 选项卡并勾选启用 DHCP Server。
  4. 在地址租用时间栏内输入一个值为指派给 DHCP 客户端的 IP 地址设定 DHCP 租用时间（以分钟为单位）。
  5. 在主 DNS（必填）以及从 DNS（可选）栏中为 DHCP 客户端输入 DNS 地址。
  6. 在域名（可选）栏中为 DHCP 服务器设置域名。
  7. 单击确定。

若要添加子网：

  1. 请单击添加然后指定下列栏的信息：
    • 启用：启用或禁用子网。
    • 开始地址：子网的首个 IP 地址。
    • 最后地址：子网的最后一个 IP 地址。
    • 网络掩码：子网掩码。
    • 网关：子网的网关。
  2. 单击确定。

注：

  • 最多可添加 32 个子网。

DHCP 客户端

若要查看 DHCP 客户端列表及其网络配置 (如 MAC 地址、IP 地址、主机名以及剩余的 DHCP 租用时间)，请进入 DHCP 客户端选项卡。

DHCP 保留

如果您要让客户端在 DHCP 租用延期内始终收到相同的 IP 地址，可以将该客户端添加到地址保留列表中。

若要将客户端添加到地址保留列表：

  • 单击添加。
  • 指定客户端的 MAC 地址、您要指派给该客户端的 IP 地址以及该客户端主机名。
  • 单击确定来保存设置。
