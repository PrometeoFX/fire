应用程序门户

通过应用程序门户，您可配置各种应用程序的连接设置，让您在独立浏览器选项卡或窗口中访问和运行这些应用程序。

例如，在网页浏览器中输入 http://:diskstation:5000/file/ 可打开 File Station。可用的连接设置包括自定义别名、HTTP 端口和 HTTPS 端口。要找到这些设置，请进入控制面板 > 应用程序门户。

注：

应用程序门户可让您编辑各种应用程序的访问设置，如 Audio Station、Download Station、Surveillance Station、Video Station、File Station 等。

自定义别名

通过自定义别名，在网页浏览器中输入 http://:diskstation:5000/file/ 那样的地址即可打开 File Station 那样的应用程序。

若要启用自定义别名：

  1. 请选择一个应用程序。
  2. 单击编辑。
  3. 在显示的窗口中，勾选启用自定义别名。
  4. 指定自定义别名。

注：

  • 您可以勾选重定向自复选框并使用简单好记的别名来访问所选的应用程序。
  • 以下为 Audio Station、Download Station、Surveillance Station、Video Station 和 File Station 等应用程序各自默认的别名：audio、download、cam、video、file。
  • 别名不能与系统和浏览器保留的或其他应用程序使用的相同，且其字符数必须在 2 到 20 个字符之间。
  • 别名可以包括大小写、数字和两个特殊字符（-）和（_），但是这两个字符不能用作为首字符和尾字符。

自定义 HTTP 或 HTTPS 端口

通过自定义 HTTP 或 HTTPS 端口，输入 http://diskstation:7000 或 http://diskstation:7001 等地址即可开应用程序。

若要启用自定义 HTTP 或 HTTPS 端口：

  1. 请选择一个应用程序。
  2. 单击编辑。
  3. 在显示的窗口中，勾选启用自定义端口（HTTP） 或启用自定义端口（HTTPS）。
  4. 指定自定义端口号。

注：

  1. 端口号范围为 1 到 65535，但是不含以下端口，因为它们已经为系统保留：20、21、22、23、25、80、110、137、138、139、143、199、443、445、515、543、548、587、873、993、995、3306、3689、5000、5001、5005、5006、5335、5432、6881、8080、7000、7001、8081、9997、9998、9999、50001、50002、eMule 默认端口：4662（TCP）、4672（UDP） 和 FTP 默认端口范围：实际范围视不同的型号而定。
