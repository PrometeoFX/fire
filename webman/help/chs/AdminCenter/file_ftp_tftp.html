TFTP/PXE

控制面板 > 文件服务 > TFTP/PXE 中的选项可让您配置 TFTP 和 PXE 设置。

TFTP

TFTP（Trivial File Transfer Protocol）是一个简易的文件传输协议，它一般用于在无需验证的情况下传输配置或启动文件。启用 TFTP 服务后，DiskStation 就能将启动加载器文件传送到客户端设备。

若要修改 TFTP 设置：

  1. 请勾选启用 TFTP 服务的复选框。
  2. 于 TFTP 根文件夹栏中，指定 TFTP 客户端在 DiskStation 上可访问的文件夹。
  3. 您可单击高级设置按钮来查看并修改以下设置：
    • 启用 TFTP 文件传输日志：此选项允许您追踪通过 TFTP 传输的数据。
    • TFTP 客户端权限：此选项允许您为 TFTP 客户端选择读取和写入权限。当您选择只读时，TFTP 客户端仅能查看根文件夹的内容。而选择可写时，TFTP 客户端就可修改根文件夹的内容。
    • 允许的客户端：选择允许通过 TFTP 连接的客户端。您可选择所有连接让所有的客户端来进行连接；或者指定 IP 地址的范围来限制可连接的客户端。
  4. 修改设置后，点击应用来保存设置。

使用 DiskStation 来为 PXE 设置 DHCP 服务

除了 TFTP，DiskStation 还可提供 DHCP 服务，来执行一体化的 PXE 设置。预启动执行环境（PXE）是一个网络环境，客户端可使用网络接口而非数据存储设备（如硬盘）来启动其设备。当执行 PXE 后，启动加载器可存储在 DiskStation，而客户端设备可使用启动加载器来通过网络进行启动。

请确认您已安装 Synology 的 DHCP Server 套件，DiskStation 才能提供 DHCP 服务。

若要为 PXE 设置 DHCP 服务：

  1. 请勾选为 PXE 在该服务器上设置 DHCP 服务的复选框。如果您尚未在 DiskStation 中安装 DHCP Server 套件，系统将提示您安装此套件。
  2. 从 TFTP 根文件夹选择启动加载器文件，然后指定其它 DHCP 相关设置。
  3. 单击应用来保存设置。
