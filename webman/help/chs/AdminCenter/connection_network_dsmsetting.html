DSM 设置

在控制面板 > 网络 > DSM 设置中，您可以更改客户端用于访问 DSM 的端口号。启用 HTTPS 后，将会使用 SSL/TLS 来加密连接。

若要更改默认端口号：

  1. 请在 HTTP 或 HTTPS 栏中输入端口号。
  2. 单击应用。

若要启用 HTTPS 连接：

  1. 请勾选启用 HTTPS 连接。
  2. 如果您要在尝试通过 HTTP 端口（5000）访问 DSM 时将客户端重新导向到 HTTPS 端口（5001），请勾选将 HTTP 连接自动重新导向到 HTTPS 选项。
  3. 单击应用。

HTTPS 连接使用的端口：

  • DSM：默认的端口号为 5001，因此您可通过输入以下地址安全访问 DSM：https://server-hostname:5001/
  • Web Station 和 Photo Station：默认的端口号为 443，因此您可通过输入下列地址以安全地访问 Web Station 和 Photo Station：https://server-hostname/

若要通过 Windows 网络发现让 DiskStation 可被发现：

  1. 请勾选启用 Windows 网络发现。
  2. 单击应用。
