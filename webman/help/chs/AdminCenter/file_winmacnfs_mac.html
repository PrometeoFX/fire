Mac 文件服务

在控制面板 > 文件服务 > Win/Mac/NFS 中，您可以启用 Mac 文件服务，使得 DiskStation 成为一台 Mac 文件服务器（AFP 服务器）。

若要启用 Mac 文件服务：

  1. 请勾选启用 Mac 文件服务，使得 DiskStation 成为一台 Mac 文件服务器（AFP 服务器）。
  2. 如需要，设置 Time Machine 备份文件夹。（详情请参阅以下部分。）
  3. 单击应用。

关于 Time Machine 备份文件夹：

若要将数据从 Time Machine 备份至 DiskStation，请从下拉菜单中选择一个共享文件夹。

注：系统服务所使用的共享文件夹无法用在 Time Machine 上。

高级设置

应用默认的 UNIX 权限：

启动此选项在上传或创建文件和文件夹时应用默认的 UNIX 权限。所应用的权限将与 UNIX umask 命令所应用的权限相同。启用此选项后，文件的 UNIX 权限为 644，而文件夹为 755。禁用此选项后，文件的 UNIX 权限为 666，而文件夹为 777。默认 umask 值为 022。

注：启动此选项可能会导致不同协议之间发生权限不一致的问题。为避免不一致问题，建议您保持禁用此选项。

在断开连接后立即释放资源：

立即启用此选项可在 AFP 服务器连接意外断开连接时释放资源。如果您在使用 Time Machine 时收到“卷在使用”错误信息，请启用此选项。

若要启用 Bonjour 打印机共享：

勾选启用 Bonjour 打印机共享来允许 Mac 客户端浏览并找到 DiskStation 上连接的打印机。
