还原

进入备份和复制 > 还原可还原备份数据。

若要还原备份任务：

  1. 选择备份任务，然后单击还原。如果任务是多版本备份，请选择 还原 > 整个任务。
  2. 可能会提示您选择系统配置、不同版本的备份数据等。这取决于它是数据备份任务、LUN 备份任务*、单个版本备份目的地（即共享文件夹）还是多重版本备份目的地（即卷）。
  3. 按向导完成设置。
  1. 选择备份任务，然后单击还原。
  2. 可能会提示您选择系统配置、不同版本的备份数据等。它取决于备份任务的类别。
  3. 按向导完成设置。

注：

  • 支持某些应用程序的还原。若要查看特定套件是否支持还原功能及其限制，请参见各应用程序的帮助。
  • 还原套件时，如果该套件尚未安装，系统会自动安装相同的或最新版本的应用程序。如果该套件已安装，但其比备份版本旧，系统会自动升级到原备份版本。
  • 在 DSM 5.1 或以上版本中备份的应用程序无法在 DSM 5.0 或以下版本中还原。

注：

  • 完成还原任务后，备份 LUN 将被还原为所选目的地卷中的文件类 LUN。*
  • 支持某些应用程序的还原。若要查看特定套件是否支持还原功能及其限制，请参见各应用程序的帮助。
  • 还原套件时，如果该套件尚未安装，系统会自动安装相同的或最新版本的应用程序。如果该套件已安装，但其比备份版本旧，系统会自动升级到原备份版本。
  • 在 DSM 5.1 或以上版本中备份的应用程序无法在 DSM 5.0 或以下版本中还原。

若要还原多版本备份任务的单个文件或文件夹：

  1. 启用文件版本浏览器。您可以用下列任一方式操作：
    • 单击还原并从下拉菜单中选择具体文件/文件夹。
    • 请进入备份目的地并扩展所选的备份目的地。单击备份版本信息图标。在出现的窗口中，选择您要还原的备份版本，并单击文件版本浏览器图标。
  2. 您可使用屏幕底部的时间线或日历图标来浏览所需的文件或文件夹版本。
  3. 选择您要管理的版本，然后单击还原。
  4. 或者，您可将文件或文件夹复制到另一个位置，或使用复制到... 或下载按钮来下载。
  5. 您还可右键单击文件或文件夹并从下拉菜单选择这些选项。

若要搜索特定版本：

  1. 单击右上角过滤器中的箭头。
  2. 在下拉菜单中，您可以按以下选项过滤文件:
    • 关键字：输入任何文件或文件夹的名称您可在关键字中可使用统配符 [*] (亦即零个或多个字符) 和 [?] (亦即任意随机字符)。例如，[a*n] 可以代表 [action]、[attention] 或 [an]。
    • 文件类型：在下拉菜单中，您可从任何、任何文件夹或任何文件进行选择以缩小搜索的范围。
    • 大小：选择任何、或选择等于、大于、小于，然后输入一个值。
    • 日期范围：从今天、昨天、上周或上个月中进行选择，或在自定义中输入起始与结束日期以在特定范围内寻找版本。
  3. 若要应用过滤器，单击过滤器。
  4. 若要重新设置过滤搜索，单击重置按钮。

注：

  • 只有多版本备份任务才提供“文件版本浏览器”功能。
  • 文件版本浏览器在 DSM 5.1 或以下版本中只有部分功能。
  • 系统配置和应用程序只能通过还原 > 整个任务来还原。
  • 元数据将不包括在复制到... 和下载操作中。

若要从其他备份目的地还原备份任务：

  1. 单击还原自...。
  2. 选择下列选项之一可找到备份任务，然后单击下一步：
    • 本地还原：选择此选项从本地共享文件夹或外接硬盘中还原数据。
    • 网络还原：选择此选项从另一台 DiskStation 或 rsync 兼容服务器中还原数据。
    • 公共云服务还原：选择此选项可从公共云服务器中还原数据。
  3. 出现提示后，选择保留备份任务的来源，然后选择要还原数据存放的目的地。
  4. 按照向导指示完成操作。

注：

  • 内置的 DataStation 仅将共享文件夹还原到 ext3/ext 外接设备。
  • Embedded DataStation 不支持无 ext3/ext 外接设备的应用程序还原功能。
