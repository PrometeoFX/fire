Hoe met Cloud Station een back-up maken van de gegevens op mijn pc

Overzicht

Cloud Station is meer dan alleen gesofisticeerde platformonafhankelijke synchronisatiesoftware. Gebruikers kunnen hem ook gebruiken als ze op een meer doeltreffende wijze een back-up van hun persoonlijke computer willen maken. Er zijn een aantal voordelen verbonden aan het gebruik van Cloud Station als back-upinstrument. Real-timereplicatie zorgt in de eerste plaats voor een betere RPO, een kortere periode waarin gegevens van uw pc definitief verloren kunnen gaan ten gevolge van een ernstig defect. Dat gaat niet alleen onmiddellijk, het helpt u ook bandbreedte besparen in vergelijking met een traditionele back-up die telkens grote hoeveelheden bandbreedte opslokt. U kunt bestanden in eenzelfde map die u niet wilt back-uppen gemakkelijk uitfilteren door bestandsgrootte en -type in te stellen op uw pc-client. Cloud Station kan bovendien tot 32 geschiedenisversies van een enkel bestand bijhouden. Zo kunt u met Cloud Station vanaf de server (NAS) een back-up terugzetten naar een pc-client.

1. Cloud Station instellen op eCore Server

  1. Meld u bij DSM aan met een account van de groep administrator.
  2. Start Cloud Station op.
  3. Als u de functie gebruiker basismap nog niet hebt ingeschakeld, wordt u gevraagd om dit te doen. Klik op OK.
  4. U wordt omgeleid naar de pagina met de gebruikersinstellingen. Schakel de gebruiker basismapinstelling in en klik op Toepassen.
  5. Ga terug naar Cloud Station en u wordt gevraagd om het in te schakelen. Klik op Ja.
    6. Opmerking: QuickConnect is een relayservice waarmee u van buiten uw lokaal netwerk gemakkelijk een verbinding kunt maken met uw eCore Server. Als u uw bestanden wilt synchroniseren wanneer u onderweg bent, raden we u aan om QuickConnect te installeren. Ga hiervoor naar Configuratiescherm > QuickConnect.
  6. Ga naar Rechten en geef op welk gebruikers Cloud Station kunnen gebruiken. Klik op Opslaan.
  7. Installeer een gedeelde map op uw eCore Server voor back-updoeleinden. Ga hiervoor naar Configuratiescherm > Gedeelde map en klik op Maken.
  8. Geef uw map een naam en voer de overeenkomstige details in. Klik op OK. We zullen de map bijvoorbeeld Backup noemen.
  9. Geef op het tabblad Machtigingen op welke gebruikers toegang zullen krijgen tot de map. We zullen bij wijze van voorbeeld admin lezen/schrijven-machtigingen voor de map toekennen. Klik op OK.
  10. Ga naar Cloud Station > Instellingen > Delen en selecteer de map Back-up en klik op Inschakelen.

2. Cloud Station op uw Windows-pc installeren

Volg deze stappen om Cloud Station op uw pc of Mac te installeren

2,1 Voor pc

  1. Ga naar Cloud Station > Overzicht en klik op de knop Download.
  2. Klik op de knop om Cloud Station voor Windows-pc te downloaden. (U kunt Cloud Station ook downloaden vanaf Synology Downloadcenter.)
  3. Start het installatieprogramma op uw computer en volg de instructies om Cloud Station te installeren.
  4. Voer na de installatie Cloud Station uit op uw computer. Klik op Nu starten.
  5. Voer het adres in voor het eCore Server waarop Cloud Station (of QuickConnect ID) wordt uitgevoerd, samen met uw gebruikersnaam en wachtwoord. (U kunt ook rechtsklikken op het zoekpictogram en Cloud Station zoekt automatisch naar andere eCore Server in uw LAN). Klik op Volgende.
    6. Opmerking: zorg ervoor dat de ingevoerde account gemachtigd is in Cloud Station > Rechten op het eCore Server.
  6. Selecteer hoeu de taak wilt instellen (in dit voorbeeld selecteren we Geavanceerde instelling aangezien we een map speciaal voor back-up willen instellen). Klik op Volgende.
  7. Selecteer een map op uw computer om deze te synchroniseren met de back-upmap die u eerder op uw eCore Server hebt aangemaakt. Klik op Volgende.
  8. Klik op Voltooid om de installatie te voltooien.

Een andere taak maken:

  1. Ga naar Cloud Station > Instellingen en schakel een gedeelde map in voor deze nieuwe taak. Mijn voorbeeld is photo.​ Zorg ervoor dat u lees/schrijfrechten hebt voor photo.
  2. Klik in Cloud Station op uw pc op Maken.
  3. ​Selecteer een eCore Server voor de nieuwe taak. In dit voorbeeld selecteren we Aangesloten eCore Server aangezien we een andere taak voor dezelfde eCore Server willen maken. Klik op Volgende.​
  4. Klik op Geavanceerde instelling en volg de overige instructies van de wizard om de instelling te voltooien.

3. Gegevens ophalen van Cloud Station

Als u per ongeluk een bestand verwijdert of overschrijft, kunt u dat terughalen met Cloud Station.

4.1 Om een vorige versie van een op uw pc gesynchroniseerd bestand te downloaden

  1. Klik op het Cloud Station-pictogram op uw systeembalk en klik op het mappictogram.
  2. Rechtsklik op het geselecteerde bestand en selecteer Synology Cloud Station > Vorige versie doorbladeren.
  3. Zoek de versie die u wilt downloaden en klik op het pictogram Downloaden.
  4. Voer de bestandsnaam in en selecteer het doel om het bestand op te slaan.

4.1 Om een vorige versie van een op een Mac gesynchroniseerd bestand te downloaden

  1. Klik op het Cloud Station-pictogram op uw systeembalk en klik op het mappictogram.
  2. Rechtsklik op het geselecteerde bestand en selecteer Synology Cloud Station > Vorige versie doorbladeren.
  3. Zoek de versie die u wilt downloaden en klik op het pictogram Downloaden.
  4. Voer de bestandsnaam in en selecteer het doel om het bestand op te slaan.

4.3 Om een vorige versie van een van een op Cloud Station gesynchroniseerd bestand terug te zetten

  1. Ga naar Cloud Station > Versiegeschiedenis en selecteer de eerder aangemaakte map Back-up.
  2. Selecteer het bestand dat u wilt terughalen en klik op Actie > Vorige versies doorbladeren.
  3. Selecteer de versie van het bestand dat u wilt downloaden en klik op Downloaden. Als u zeker weet dat u de geselecteerde versie wilt terugzetten, klikt u op Terugzetten waarna uw huidige versie overschreven zal worden.

4. Belangrijke opmerkingen

  1. Houd eerdere versies altijd bij voor het geval een bestand per ongeluk wordt verwijderd of overschreven.
  2. In Algemene instellingen kunt u configureren of een bestand van uw NAS zal worden teruggehaald of verwijderd als u een bestand op uw pc verwijdert.
