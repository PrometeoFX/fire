DSM Ayarları

Denetim Masası > Ağ > DSM Ayarları seçeneğinden istemcilerin DSM'ye erişmek için kullandığı bağlantı noktası numarasını değiştirebilirsiniz. HTTPS etkinleştirildiğinde, bağlantılar SSL/TLS ile şifrelenir.

Varsayılan bağlantı noktası numarasını değiştirmek için:

  1. HTTP veya HTTPS alanına bağlantı noktası numarasını girin.
  2. Uygula'ya tıklayın.

HTTPS bağlantılarını etkinleştirmek için:

  1. HTTPS bağlantısını etkinleştir seçeneğini işaretleyin.
  2. DSM'ye HTTP bağlantı noktası (5000) yoluyla erişmeye çalışırken istemcileri HTTPS bağlantı noktasına (5001) yeniden yönlendirmek istiyorsanız, HTTP bağlantılarını otomatik olarak HTTPS'e yeniden yönlendir kutusunu işaretleyin.
  3. Uygula'ya tıklayın.

HTTPS bağlantısı için kullanılan bağlantı noktaları:

  • DSM: Varsayılan bağlantı noktası numarası 5001 olduğundan, aşağıdaki adresi girerek DSM'ye güvenli bir şekilde erişebilirsiniz: https://server-hostname:5001/
  • Web Station ve Photo Station: Varsayılan bağlantı noktası numarası 443 olduğundan, aşağıdaki adresi girerek Web Station ve Photo Station'a güvenli bir şekilde erişebilirsiniz: https://server-hostname/

DiskStation sunucusunu Windows ağ bulucu yoluyla bulunur hale getirmek için:

  1. Windows ağ bulucuyu etkinleştir öğesini işaretleyin.
  2. Uygula'ya tıklayın.
