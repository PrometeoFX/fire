eCore Server ile Cloud Station kullanan bilgisayar arasında dosyaların senkronize edilmesi

Genel Bakış

eCore Server'a yönelik Cloud Station, eCore Server sunucunuzdaki dosyaları bilgisayarlar veya mobil cihazlar gibi diğer cihazlarla kolayca senkronize etmenizi sağlayan bir dosya senkronize etme uygulamasıdır (DS cloud ile). Cloud Station hem eCore Server hem de bilgisayarınızı kurulduğunda, bilgisayarınızdaki ilgili klasörde saklanan dosyalar otomatik olarak eCore Server sunucunuzla senkronize edilecektir.

1. Cloud Station Kurun

  1. Paket Merkezi'ne gidin, Cloud Station'u bulun ve ardından Kur'a tıklayın.

2. Cloud Station’ı etkinleştirin

  1. Cloud Station'ı başlatın.
  2. Kullanıcı ana sayfa özelliğini etkinleştirmediyseniz, etkinleştirmeniz istenecektir. Tamam'a tıklayın.
  3. Kullanıcı ayarları sayfasına yönlendirileceksiniz. Kullanıcı ana sayfalarını etkinleştirerek Uygula'ya tıklayın.
  4. Ardından Cloud Station'a geri dönün, etkinleştirmeniz istenecek. Evet'e tıklayın.
  5. Ayrıcalıklar bölümüne giderek hangi kullanıcıların Cloud Station'ı kullanabileceklerini belirleyin. Kaydet'e tıklayın.

3. İstemciyi kurun ve etkinleştirin

  1. Cloud Station > Genel Bakış bölümüne giderek Bilgisayarlar düğmesine tıklayın.
  2. İşletim sisteminiz için Cloud Station indirmek üzere düğmeye tıklayın. (Cloud Station'ı Synology İndirme Merkezi'nden de indirebilirsiniz.)
  3. Kurulum programını bilgisayarınızda başlatın ve ekrandaki talimatları izleyin.
  4. Yüklemeden sonra Cloud Station'ı bilgisayarınızda çalıştırın. Şimdi Başlat'a tıklayın.
  5. Cloud Station çalıştıran eCore Server'ın adresini (veya QuickConnect ID'sini), kullanıcı adını ve parolayı girin. LAN dahilindeki IP'leri aramak için sağdaki arama simgesine tıklayabilirsiniz. İleri'ye tıklayın.
  6. Görevinizi nasıl kurmak istediğinizi seçin (burada Hızlı Kurulum'u seçeceğiz), ardından İleri'ye tıklayın.
  7. Cloud Station klasörünüzün kurulumunu tamamlamak için Bitir'e tıklayın.
  8. Cloud Station klasörünüzü açmak için Şimdi aç üzerine tıklayın.
  9. Sistem tepsisinde Cloud Station simgesini bulabilirsiniz.
  10. Tepsi menüsünü açmak için Cloud Station simgesine çift tıklayın. Burada dosyalarınızın ilerlemesini ve durumunu görüntüleyebilirsiniz. Cloud Station'u başlatmak için Ana uygulama'ya tıklayın.
  11. Başlatma sonrasında, Cloud Station'u nasıl kullanacağınıza dair bazı ipuçları gösterilecektir. Daha fazla ipucu görmek için sağ ok üzerine tıklayın veya daha fazla bilgi için Çevrimiçi eğitimlere git öğesine tıklayın. İpuçlarına ihtiyacınız yoksa, Bunu tekrar gösterme üzerine tıklayın.
  12. Şimdi Cloud Station'daki senkronizasyon görevlerinizi yönetebilirsiniz.
