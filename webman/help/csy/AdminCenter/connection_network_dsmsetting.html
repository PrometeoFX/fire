Nastavení DSM

V části Ovládací panel > Síť > Nastavení DSM můžete změnit číslo portu, které klienti využívají pro přístup k systému DSM. Po povolení protokolu HTTPS jsou připojení šifrována technologií SSL/TLS.

Postup změny výchozího čísla portu:

  1. Číslo portu zadejte do pole HTTP nebo HTTPS.
  2. Klikněte na možnost Použít.

Postup povolení připojení HTTPS:

  1. Zaškrtněte možnost Povolit připojení HTTPS.
  2. Chcete-li klienty při přístupu k systému DSM prostřednictvím portu HTTP (5000) přesměrovat na port HTTP (5001), zaškrtněte políčko Automaticky přesměrovat připojení HTTP na HTTPS.
  3. Klikněte na možnost Použít.

Porty používané pro připojení HTTPS:

  • DSM: Výchozí číslo portu je 5001, zabezpečený přístup k systému DSM získáte zadáním následující adresy: https://název hostitele-serveru:5001/
  • Web Station a Photo Station: Výchozí číslo portu je 443, zabezpečený přístup ke službám Web Station a Photo Station získáte zadáním následující adresy: https://název hostitele-serveru/

Postup nastavení zjistitelnosti zařízení DiskStation přes zjišťování sítě Windows:

  1. Zaškrtněte možnost Povolit zjišťování sítě Windows.
  2. Klikněte na možnost Použít.
