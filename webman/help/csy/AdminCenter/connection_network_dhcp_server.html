DHCP Server

Funkce server DHCP vytvoří z vašeho zařízení DiskStation server DHCP který může v rámci vaší místní sítě přiřazovat klientským zařízením DHCP dynamické IP adresy.

Postup povolení serveru DHCP:

  1. Přejděte do části Ovládací panel > Síť > Síťové rozhraní.
  2. Vyberte síťové rozhraní (např. LAN) a klikněte na možnost Upravit.
  3. Přejděte na kartu DHCP Server a zaškrtněte možnost Povolit DHCP Server.
  4. Zadáním hodnoty v poli Doba propůjčení adresy nastavíte dobu propůjčení DHCP (v jednotkách minut) pro IP adresy přidělené klientským zařízením DHCP.
  5. Zadejte adresu DNS klientských zařízení v kolonce Primární DNS (povinný údaj) a Sekundární DNS (volitelný údaj).
  6. Nastavte název domény serveru DHCP v kolonce Název domény (volitelný údaj).
  7. Klikněte na možnost OK.

Postup přidání podsítě:

  1. Klikněte na možnost Přidat a zadejte do následujících sloupců údaje:
    • Povolit: Povolení a zakázání podsítě.
    • Začátek: První IP adresa podsítě.
    • Konec: Poslední IP adresa podsítě.
    • Maska sítě: Maska podsítě.
    • Brána: Brána podsítě.
  2. Klikněte na možnost OK.

Poznámka:

  • Můžete přidat maximálně 32 podsítí.

Klienti DHCP

Chcete-li zobrazit seznam klientů DHCP a příslušných síťových konfigurací (např. MAC adresu, IP adresu, název hostitele a dobu expirace propůjčení DHCP), můžete přejít na kartu Klienti DHCP.

Rezervace DHCP

Pokud si chcete, aby během obnovení propůjčení DHCP byla klientskému zařízení přidělována stále stejná IP adresa, můžete klientské zařízení přidat na seznam rezervovaných adres.

Postup přidání klienta do seznamu rezervovaných adres:

  • Klikněte na možnost Přidat.
  • Zadejte MAC adresu, IP adresu, kterou chcete klientu přiřadit, a název hostitele klienta.
  • Nastavení uložíte kliknutím na tlačítko OK.
