Portál aplikací

Díky portálu aplikací můžete nakonfigurovat nastavení připojení nejrůznějších aplikací a nastavit přístup a spouštění těchto aplikací v samostatných kartách nebo oknech prohlížeče.

File Station můžete otevřít například zadáním adresy http://:diskstation:5000/file/ do webového prohlížeče. Mezi dostupná nastavení připojení patří přizpůsobení aliasů portálu a portů HTTP a HTTPS. Tato nastavení najdete v části Ovládací panel > Portál aplikací.

Poznámka:

Portál aplikací umožňuje úpravu nastavení přístupu pro nejrůznější aplikace, jako je Audio Station, Download Station, Surveillance Station, Video Station, File Station a další.

Přizpůsobení aliasů

Díky vlastnímu aliasu lze aplikace jako File Station otevřít zadáním adresy http://:diskstation:5000/file/ do webového prohlížeče.

Postup povolení vlastního aliasu:

  1. Vyberte aplikaci.
  2. Klikněte na možnost Upravit.
  3. V zobrazeném okně zaškrtněte Povolit vlastní alias.
  4. Zadejte vlastní alias.

Poznámka:

  • Zaškrtnutím políčka Přesměrovat od a vytvořením kratšího a jednoduchého aliasu vytvoříte přístup k vybrané aplikaci.
  • Následuje seznam výchozích aliasů zmíněných čtyř aplikací Audio Station, Download Station, Surveillance Station, Video Station, File Station: audio, download, cam, video, file.
  • Nelze používat názvy aliasů určené pro systém, prohlížeče nebo ostatní aplikace a název aliasu musí obsahovat min. 2 a max. 20 znaků.
  • Názvy aliasů mohou obsahovat různá písmena psané velkými i malými písmeny, číselné znaky a dva speciální znaky (-), (_). Tyto znaky však mohou být používány pro ostatní znaky, s výjimkou prvního a posledního znaku.

Vlastní porty HTTP nebo HTTPS

Díky vlastním portům HTTP nebo HTTPS lze aplikaci otevřít zadáním adresy, jako je http://diskstation:7000 nebo http://diskstation:7001.

Postup povolení vlastního portu HTTP nebo HTTPS:

  1. Vyberte aplikaci.
  2. Klikněte na možnost Upravit.
  3. V zobrazeném okně zaškrtněte možnost Povolit vlastní port (HTTP) nebo Povolit vlastní port (HTTPS).
  4. Zadejte číslo vlastního portu.

Poznámka:

  1. Hodnoty portu mohou být v rozsahu 1 až 65535, nelze však použít tyto porty, které jsou vyhrazeny pro systémové použití: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, výchozí porty pro eMule: 4662 (TCP), 4672(UDP) a výchozí rozsah portů pro FTP: konkrétní rozsah se může u různých modelů lišit.
