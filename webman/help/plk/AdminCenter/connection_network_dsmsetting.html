Ustawienia DSM

Po przejściu do strony Panel sterowania > Sieć > Ustawienia DSM można zmienić numer portu, którego używają klienci w celu uzyskania dostępu do DSM. Kiedy obsługa połączeń HTTPS jest włączona, połączenia są szyfrowane przy użyciu mechanizmu SSL/TLS.

Aby zmienić domyślny numer portu:

  1. Wprowadź numer portu w polu HTTP lub HTTPS.
  2. Kliknij przycisk Zastosuj.

Aby włączyć połączenia HTTPS:

  1. Zaznacz pole wyboru Włącz połączenie HTTPS.
  2. Jeżeli chcesz przekierowywać klientów na port HTTPS (5001), kiedy próbują uzyskać dostęp do DSM za pośrednictwem portu HTTP (5000), zaznacz pole wyboru Automatycznie przekieruj połączenia HTTP na HTTPS.
  3. Kliknij przycisk Zastosuj.

Porty używane do połączenia HTTPS:

  • DSM: Domyślny numer portu to 5001, dlatego można bezpiecznie uzyskać dostęp do systemu DSM, wprowadzając następujący adres: https://nazwa hosta-serwera:5001/
  • Web Station i Photo Station: Domyślny numer portu to 443, dlatego można bezpiecznie uzyskać dostęp do Web Station i Photo Station, wprowadzając następujący adres: https://nazwa_hosta-serwera/

Aby umożliwić wykrywanie serwera DiskStation przy użyciu funkcji odnajdowania sieci systemu Windows:

  1. Włącz odnajdowanie sieci systemu Windows.
  2. Kliknij przycisk Zastosuj.
