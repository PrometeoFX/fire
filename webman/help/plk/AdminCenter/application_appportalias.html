Portal aplikacji

Portal aplikacji umożliwia konfigurację połączeń różnych aplikacji. Dzięki temu można uzyskiwać do nich dostęp i uruchamiać je w osobnych kartach lub oknach przeglądarki.

Można na przykład otworzyć aplikację File Station, wprowadzając w przeglądarce internetowej adres http://:diskstation:5000/file/. Dostępne ustawienia połączeń obejmują niestandardowe aliasy oraz porty HTTP i HTTPS. Aby znaleźć te ustawienia, przejdź do pozycji Panel sterowania > Portal aplikacji.

Uwaga:

Portal aplikacji umożliwia edycję ustawień dostępu różnych aplikacji, takich jak Audio Station, Download Station, Surveillance Station, Video Station, File Station i inne.

Dostosowywanie aliasu

Przy użyciu niestandardowego aliasu można otworzyć aplikację, na przykład File Station, wprowadzając w przeglądarce internetowej adres taki jak http://:diskstation:5000/file/.

Aby włączyć niestandardowy alias:

  1. Wybierz aplikację.
  2. Kliknij przycisk Edytuj.
  3. W wyświetlonym oknie zaznacz pole wyboru Włącz dostosowany alias.
  4. Określ dostosowany alias.

Uwaga:

  • Aby uzyskiwać dostęp do wybranej aplikacji za pomocą krótszego, łatwego do zapamiętania aliasu, można zaznaczyć pole wyboru Przekieruj z.
  • Domyślne aliasy aplikacji Audio Station, Download Station, Surveillance Station, Video Station i File Station to odpowiednio: audio, download, cam, video, file.
  • Nazwy aliasów nie mogą być takie same jak nazwy zarezerwowane dla systemu i przeglądarek lub używane przez inne aplikacje, a liczba ich znaków musi się zawierać w przedziale od 2 do 20.
  • Nazwy aliasów mogą się składać z liter o różnej wielkości, cyfr i dwóch znaków specjalnych (-), (_), ale znaków tych nie można używać jako pierwszych ani ostatnich.

Niestandardowe porty HTTP i HTTPS

Przy użyciu niestandardowych portów HTTP i HTTPS można otworzyć aplikację, wprowadzając adres taki jak http://diskstation:7000 lub http://diskstation:7001.

Aby włączyć niestandardowy port HTTP lub HTTPS:

  1. Wybierz aplikację.
  2. Kliknij przycisk Edytuj.
  3. W wyświetlonym oknie zaznacz pole wyboru Włącz dostosowany port (HTTP) lub Włącz dostosowany port (HTTPS).
  4. Określ niestandardowy numer portu.

Uwaga:

  1. Numery portów mogą zawierać się w zakresie od 1 do 65 535, ale następujące porty należy pominąć, ponieważ są zarezerwowane na potrzeby systemu: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, domyślne porty eMule: 4662 (TCP), 4672 (UDP) i domyślny zakres portów FTP: zakres może zależeć od modelu.
