DHCP Server

Serwer DiskStation może pełnić funkcję serwera DHCP i dynamicznie przypisywać adresy IP do klientów DHCP w sieci lokalnej.

Aby włączyć pakiet DHCP Server:

  1. Przejdź do pozycji Panel sterowania > Sieć > Interfejs sieciowy.
  2. Wybierz interfejs sieciowy (np. LAN), a następnie kliknij Edytuj.
  3. Przejdź do karty DHCP Server i zaznacz pole wyboru Włącz DHCP Server.
  4. Wprowadź wartość w polu Czas dzierżawy adresu, aby ustawić czas dzierżawy (w minutach) adresu IP na serwerze DHCP przypisanego klientom DHCP.
  5. Wprowadź adres DNS klientów DHCP w polach Podstawowy serwer DNS (wymagane) i Dodatkowy serwer DNS (opcjonalne).
  6. Ustaw nazwę domeny dla Serwera DHCP w polu Nazwa domeny (opcjonalne).
  7. Kliknij przycisk OK.

Aby dodać podsieć:

  1. Kliknij Dodaj, a następnie podaj informacje w następujących kolumnach:
    • Włącz: włącza lub wyłącza podsieć.
    • Początek: pierwszy adres IP podsieci.
    • Koniec: ostatni adres IP podsieci.
    • Maska sieci: maska podsieci.
    • Brama: brama podsieci.
  2. Kliknij przycisk OK.

Uwaga:

  • Dodać można maksymalnie 32 podsieci.

Klienci DHCP

Aby wyświetlić listę klientów DHCP i ich konfiguracje sieciowe (np. adresy MAC, adresy IP, nazwy hosta oraz ilość czasu pozostałą do zakończenia dzierżawy DHCP), można przejść do karty Klienci DHCP.

Rezerwacja DHCP

Jeżeli chcesz, aby klient zawsze otrzymywał ten sam adres IP podczas odnawiania dzierżawy DHCP, możesz dodać tego klienta do listy rezerwacji adresów.

Aby dodać klienta do listy rezerwacji adresów:

  • Kliknij przycisk Dodaj.
  • Określ adres MAC klienta, adres IP do przypisania, a następnie nazwę hosta klienta.
  • Kliknij przycisk OK, aby zapisać ustawienia.
