Montowanie folderów zdalnych

Za pomocą aplikacji File Station można montować zdalne foldery ze zdalnych serwerów (takich jak komputery w sieci lub inne urządzenia DiskStation) w celu łatwej nawigacji po dostępnych zasobach sieciowych.

Aby zamontować folder zdalny:

  1. Kliknij Narzędzia > Montuj folder zdalny.
  2. Wprowadź „\\Serwer_zdalny\Folder_zdalny” w polu Folder, aby określić ścieżkę folderu zdalnego. Na przykład „\\192.168.0.2\share” lub „\\chad\cheeseburger\pictures”.
  3. Jeśli serwer zdalny wymaga uprawnień dostępu do folderu zdalnego, wprowadź poświadczenia użytkownika zdalnego serwera w polach Nazwa konta i Hasło.
  4. Kliknij Przeglądaj, aby wybrać lub utworzyć pusty folder docelowy w urządzeniu DiskStation w celu zamontowania folderu zdalnego.
  5. Zaznacz opcję Zamontuj automatycznie przy uruchamianiu, aby serwer DiskStation montował ten folder zdalny przy każdym uruchomieniu lub ponownym uruchomieniu systemu.
  6. Kliknij Montuj. Teraz można przejść do folderu docelowego, w którym zamontowany został folder zdalny, oraz zarządzać zawartością folderów zdalnych.

Aby odmontować folder zdalny:

  1. Wybierz folder zdalny, który ma być odmontowany, a następnie kliknij Narzędzia > Odmontuj.
  2. Ewentualnie kliknij prawym przyciskiem myszy folder zdalny, a następnie wybierz opcję Odmontuj.

Uwaga:

  • Maksymalna liczba równoczesnych folderów zdalnych to 100.
  • Foldery zdalne można montować wyłącznie z serwera zdalnego obsługującego protokół CIFS.
