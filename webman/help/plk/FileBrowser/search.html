Wyszukiwanie plików i folderów

Funkcja Filtruj aplikacji File Station umożliwia wyszukiwanie oraz wyświetlanie plików i folderów spełniających określone kryteria. Można filtrować pliki lub foldery w bieżącym folderze, a także przeprowadzać bardziej zaawansowane wyszukiwanie obejmujące wszystkie podfoldery, co umożliwia uzyskanie bardziej precyzyjnych wyników.

Aby wyszukać pliki lub foldery w bieżącym folderze:

  1. Przejdź za pomocą aplikacji File Station do folderu, który ma być przeszukany.
  2. W polu Filtruj w prawym górnym rogu wprowadź słowo kluczowe. Wyniki filtrowania zostaną wyświetlone w głównym obszarze okna.

Aby przeprowadzić wyszukiwanie zaawansowane:

  1. Przejdź za pomocą aplikacji File Station do folderu, który ma być przeszukany.
  2. Kliknij ikonę lupy obok pola Filtruj w prawym górnym rogu, aby wyświetlić menu zaawansowanego wyszukiwania.
  3. Wprowadź odpowiednie informacje lub z list rozwijanych wybierz następujące kryteria wyszukiwania:
    • Słowo kluczowe: wprowadź słowo kluczowe, aby wyszukać pliki lub foldery o pasujących nazwach.

      Uwaga:

      • W słowach kluczowych można używać symboli wieloznacznych [*] (zastępuje zero lub więcej znaków) i [?] (zastępuje jeden dowolny znak). Na przykład [z*e] może oznaczać [zdarzenie], [zdanie] lub [ze].
      • Do rozdzielenia słów kluczowych można użyć przecinków (,). Na przykład [z,nia] może oznaczać [zdania], [z zadania], [zadania zakończone] lub [stycznia].
      • Na początku i na końcu słowa kluczowego można dodać cudzysłów ["], aby wyszukać dokładne dopasowania. Na przykład ["w dni"] może oznaczać [w dniu], ale nie może oznaczać [dni w styczniu].
    • Lokalizacja: z listy rozwijanej wybierz zakres i głębokość wyszukiwania zaawansowanego. Zakres wyszukiwania można także dostosować, wybierając z listy dowolne pozostałe foldery udostępnione.
    • Typ pliku: wybierz pozycję Rozszerzenie, aby ograniczyć zakres wyszukiwania do plików określonego typu, lub wybierz pozycję Dowolny folder, aby wyszukać tylko foldery.
    • Data: wybierz pozycję Data modyfikacji, Data utworzenia i Ostatni dostęp, a następnie z kalendarza wybierz żądaną datę, aby wyszukać pliki o pasujących datach.
    • Rozmiar (MB): wybierz pozycję równa się, jest większe niż i jest mniejsze niż, a następnie wprowadź wielkość, aby wyszukać pliki o pasujących wielkościach (po wybraniu pozycji równa się w wynikach wyszukiwania będą uwzględniane pliki o wielkości zbliżonej do podanej wartości).
    • Właściciel/grupa: wybierz pozycję Właściciel lub Grupa, a następnie z listy rozwijanej wybierz pozycję, aby wyszukać pliki lub foldery należące do wybranego właściciela lub grupy.

      Uwaga:

      • Aby wykonać wyszukiwanie zaawansowane, trzeba określić kryterium Lokalizacja i przynajmniej jedno inne kryterium.
      • Jeżeli w indeksowanym folderze udostępnionym zamontowany jest folder ISO lub folder zdalny, nie można wyszukiwać plików w tym folderze za pomocą funkcji wyszukiwania zaawansowanego.
    • Kliknij przycisk Szukaj, aby rozpocząć wyszukiwanie, lub przycisk Resetuj i określ ponownie kryteria wyszukiwania. Po ukończeniu wyszukiwania pasujące pliki zostaną wyświetlone w głównym obszarze aplikacji File Station.
