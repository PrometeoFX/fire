TFTP/PXE

Las opciones en Panel de control > Servicios de archivos > TFTP/PXE le permiten configurar los parámetros de TFTP y PXE.

TFTP

El Protocolo trivial de transferencia de archivos (TFTP) es un sencillo protocolo de transferencia de archivos que se suele usar para transferir los archivos de configuración o arranque en situaciones en las que no es necesaria ninguna autenticación. Al habilitar el servicio TFTP, DiskStation podrá transferir cargadores de arranque a dispositivos cliente.

Para modificar la configuración de TFTP:

  1. Marque la casilla Habilitar servicio TFTP.
  2. En el campo Carpeta root TFTP, indique a que carpeta de DiskStation pueden acceder los clientes TFTP.
  3. Puede hacer clic en el botón Configuración avanzada para ver y modificar las siguientes opciones de configuración:
    • Habilitar registro de transferencia de archivos TFTP: esta opción le permite realizar un seguimiento de los datos transferidos por TFTP.
    • Permiso de cliente TFTP: seleccione los permisos de lectura y escritura para clientes TFTP. Si se selecciona la opción Solo lectura, los clientes TFTP solo podrán ver el contenido de la carpeta root. Si se selecciona la opción Grabable, los clientes TFTP podrán modificar el contenido de la carpeta root.
    • Clientes permitidos: elija qué clientes tendrán permiso para conectarse a través de TFTP. Puede permitir que todos los clientes se conecten seleccionando Todas las conexiones; o bien puede limitar qué clientes pueden conectarse indicando un intervalo de direcciones IP.
  4. Tras modificar la configuración, haga clic en Aplicar para guardar.

Configurar el servicio DHCP para PXE con su DiskStation

Además de TFTP, DiskStation también puede ofrecer un servicio DHCP para implementar una configuración PXE todo en uno. El Entorno de ejecución previo al arranque (PXE) es un entorno de red que permite que los dispositivos cliente arranquen utilizando una interfaz de red en lugar de un dispositivo de almacenamiento de datos (p. ej., discos duros). Al implementar PXE, se puede almacenar un cargador de arranque en DiskStation y los dispositivos cliente pueden arrancar a través de la red utilizando el cargador de arranque.

Para ofrecer servicios DHCP con DiskStation, asegúrese de que ha instalado el paquete de DHCP Server de Synology.

Para configurar el servicio DHCP para PXE:

  1. Active la casilla Configurar servicio DHCP en este servidor para PXE. Se le pedirá que instale el paquete de DHCP Server si aún no está instalado en su DiskStation.
  2. Seleccione un archivo de cargador de arranque de la carpeta root TFTP y especifique otras opciones de configuración relacionadas con DHCP.
  3. Haga clic en Aplicar para guardar la configuración.
