Configuración de DSM

En Panel de control > Red > Configuración DSM, puede cambiar el número de puerto que los clientes utilizan para acceder a DSM. Cuando se habilita HTTPS, las conexiones se cifrará con SSL/TLS.

Para cambiar el número de puerto predeterminado:

  1. Escriba el número de puerto en el campo HTTP o HTTPS.
  2. Haga clic en Aplicar.

Para habilitar las conexiones HTTPS:

  1. Marque Habilitar conexión HTTPS.
  2. Si desea redirigir clientes al puerto HTTPS (5001) cuando intenten acceder a DSM con el puerto HTTP (5000), marque la casilla Redirigir automáticamente las conexiones HTTP a HTTPS.
  3. Haga clic en Aplicar.

Puertos usados para la conexión HTTPS:

  • DSM: El número de puerto predeterminado es 5001, de manera que puede acceder a DSM introduciendo la siguiente dirección: https://server-hostname:5001/
  • Web Station y Photo Station: El número de puerto predeterminado es 443, de manera que puede acceder a Web Station y Photo Station introduciendo la siguiente dirección: https://server-hostname/

Para hacer que DiskStation se pueda detectar a través de la detección de redes de Windows:

  1. Marque Activar la detección de redes de Windows.
  2. Haga clic en Aplicar.
