Portal de aplicaciones

Con el Portal de aplicaciones puede configurar la configuración de conexión de distintas aplicaciones, lo que le permitirá acceder y ejecutar dichas aplicaciones en pestañas o ventanas independientes del explorador.

Por ejemplo, podría abrir File Station introduciendo http://:diskstation:5000/file/ en su explorador web. Los ajustes de conexión disponibles incluyen alias personalizados, puertos HTTP y HTTPS. Para encontrar estas configuraciones, vaya a Panel de control > Portal de aplicaciones.

Nota:

El Portal de aplicaciones le permite editar configuraciones de acceso para distintas aplicaciones, como Audio Station, Download Station, Surveillance Station, Video Station, File Station y otras.

Personalizar alias

Con un alias personalizado, una aplicación como File Station se podría abrir introduciendo una dirección como http://:diskstation:5000/file/ en su explorador web.

Para habilitar un alias personalizado:

  1. Seleccione una aplicación.
  2. Haga clic en Editar.
  3. En la ventana que aparecerá, marque Habilitar alias personalizado.
  4. Especifique un alias personalizado.

Nota:

  • Puede marcar la casilla de selección Redirigir desde y utilizar un alias más corto y más fácil de recordar para acceder a la aplicación seleccionada.
  • A continuación se detallan los respectivos alias predeterminados de las aplicaciones, Audio Station, Download Station, Surveillance Station, Video Station, File Station: audio, download, cam, video, file.
  • Los nombres de alias son los mismos que los reservados para el sistema, los exploradores o los utilizados por otras aplicaciones, y su número de caracteres debe limitarse entre 2 y 20.
  • Los nombres de alias pueden estar formados por mayúsculas y minúsculas mezcladas, caracteres numéricos y dos caracteres especiales (-), (_), pero estos dos caracteres solo pueden utilizarse para los demás caracteres excepto el primero y el último.

Puertos HTTP o HTTPS personalizados

Con puertos HTTP o HTTPS personalizados, una aplicación se puede abrir introduciendo una dirección como http://diskstation:7000 o http://diskstation:7001.

Para habilitar un puerto HTTP o HTTPS personalizado:

  1. Seleccione una aplicación.
  2. Haga clic en Editar.
  3. En la ventana que aparecerá, marque Habilitar puerto personalizado (HTTP) o Habilitar puerto personalizado (HTTPS).
  4. Especifique un número de puesto personalizado.

Nota:

  1. El rango de número de puerto va del 1 al 65535, pero los puertos siguientes deben excluirse porque están reservados para uso del sistema: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, puertos predeterminados de eMule: 4662 (TCP), 4672(UDP), e intervalo de puertos predeterminado para FTP: el intervalo concreto puede variar dependiendo de los distintos modelos.
