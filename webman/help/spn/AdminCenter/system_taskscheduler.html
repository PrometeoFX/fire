Programador de tareas

El Programador de tareas le permite programar y ejecutar servicios o scripts a horas predefinidas.

Tareas compatibles

Se pueden crear tareas para hacer lo siguiente:

  • Script definido por usuario: Ejecute scripts o comandos definidos por el usuario.
  • Papelera de reciclaje: Vaciar la papelera de reciclaje de ciertas carpetas compartidas específicas. Además, la configuración de la Política de retención le permitirá controlar los archivos que se eliminarán permanentemente al vaciar la papelera de reciclaje. Por ejemplo, los archivos se pueden eliminar o conservar de acuerdo con su tamaño, hora de eliminación o formato de archivo.
  • Control de pitido: Cree el pitido de DiskStation.
  • Servicio: Inicie o detenga los servicios de red, como el Servicio de archivos Windows, FTP, Bloqueo automático, etc.

Administrar tareas

La página principal del Programador de tareas muestra todas las tareas existentes y ofrece opciones para crear, editar o ejecutar tareas.

Para crear una tarea:

  1. Haga clic en Crear y seleccione el tipo de tarea que desea programar.
  2. Cuando aparezca la ventana Crear tarea, modifique las siguientes opciones en la ficha General:
    • Configuración general: Introduzca un nombre de tarea, seleccione un usuario y habilite o deshabilite la tarea.
    • Otra configuración: En esta ficha se pueden modificar opciones adicionales específicas de la tarea (solo para determinadas tareas). Por ejemplo, cuando cree una tarea para vaciar una papelera de reciclaje, tendrá que seleccionar la papelera de reciclaje de una carpeta compartida determinada.
  3. En la pestaña Programa, modifique las siguientes opciones:
    • Fecha: Indique cuándo ejecutar la tarea. Las tareas se pueden programar para que se ejecuten determinados días de la semana (por ejemplo, diariamente, los fines de semana o todos los domingos, etc.) o en determinadas fechas (p. ej., 21/10/2015).
    • Veces: Indique cuántas veces se debe ejecutar la tarea cada día, así como la primera y la última hora de ejecución.
  4. Modifique las opciones específicas de la tarea en la ficha Configuración de tarea (solo para ciertas tareas).
  5. Haga clic en OK para terminar

Para editar la configuración de la tarea:

  1. Seleccione una tarea existente.
  2. Haga clic en Editar.
  3. Aparece una ventana en la que puede modificar la configuración de la tarea.

Para eliminar una tarea:

  1. Seleccione una tarea existente.
  2. Haga clic en Eliminar.

Para habilitar o deshabilitar una tarea:

  1. En la página principal del Programador de tareas, active o desactive la casilla de verificación adecuada bajo Habilitada para habilitar o deshabilitar una tarea.
  2. Haga clic en Guardar.

Para ejecutar manualmente una tarea:

  1. Seleccione una o varias tareas existentes.
  2. Haga clic en Ejecutar.

Limitaciones del Programador de tareas:

  • Los nombres de las tareas pueden incluir 512 caracteres visibles.
  • Número máximo de tareas: 1024.
  • Longitud máxima de la ruta del script o de la línea de comando: 4095.
  • La duración del pitido se puede ajustar entre 1 y 600 segundos.
