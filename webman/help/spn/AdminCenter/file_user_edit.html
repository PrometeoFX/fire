Editar un usuario

Busque la cuenta de usuario (escribiendo el nombre del usuario en el campo Buscar), haga doble clic en ella o haga clic en Editar y edite sus datos de cuenta.

    Nota:
  1. Si deshabilita una cuenta, dejará de funcionar.
  2. La cuota debe ser un número entero mayor o igual a 0 y menor que la capacidad del disco o volumen. Un valor de cuota de 0 significa que la Cuota efectiva estarán determinados por el Límite de grupo.
  3. Si hay más de un volumen/carpeta compartida en el servidor, puede configurar una cuota para el usuario en cada volumen/carpeta compartida.
  4. La información de correo electrónico en la página "Notificación" debe configurarse correctamente para que el sistema notifique al usuario por correo electrónico una vez creada o modificada la información de cuenta.
  5. Al ajustar un límite de velocidad para un usuario, verá unos números debajo de Resultado si ya hay un límite de velocidad de grupo aplicado a ese usuario.
  6. El límite de velocidad asignado a un usuario no se incluirá en el límite de velocidad del grupo al que pertenece ese usuario.
