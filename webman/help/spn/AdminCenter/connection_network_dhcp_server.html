DHCP Server

Su DiskStation puede asumir el papel de DHCP server y asigna direcciones IP dinámicas a clientes DHCP dentro de su red de área local.

Para habilitar DHCP Server:

  1. Vaya al Panel de control > Red > Interfaz de red.
  2. Seleccione un interfaz de red (por ejemplo LAN) y haga clic en Editar.
  3. Vaya a la pestaña DHCP Server y marque Habilitar DHCP Server.
  4. Introduzca un valor en el campo Tiempo de uso de dirección para ajustar el tiempo de uso de DHCP (en minutos) para direcciones IP asignadas a clientes DHCP.
  5. Introduzca la dirección DNS para clientes DHCP en los campos DNS primario (obligatorio) y DNS secundario (opcional).
  6. Establezca el nombre del dominio para DHCP Server en el campo Nombre de dominio (opcional).
  7. Haga clic en OK.

Para agregar una subred:

  1. Haga clic en Agregar y luego especifique la información en las siguientes columnas:
    • Habilitar: habilite o deshabilite la subred.
    • Inicio: la primera dirección IP de la subred.
    • Fin: la última dirección IP de la subred.
    • Máscara de subred: la máscara de subred.
    • Pasarela: la pasarela de la subred.
  2. Haga clic en OK.

Nota:

  • Solo puede agregar hasta 32 subredes.

Clientes DHCP

Para ver una lista de clientes DHCP y sus configuraciones de red (por ejemplo, dirección MAC, dirección IP, nombre de host y la cantidad de tiempo restante hasta la caducidad del uso de DHCP), puede ir a la pestaña Clientes DHCP.

Reserva de DHCP

Si desea que un cliente reciba siempre la misma dirección IP al renovar el uso de DHCP, puede agregar el cliente a la lista de direcciones reservadas.

Para agregar un cliente a la lista de direcciones reservadas:

  • Haga clic en Agregar.
  • Especifique la dirección MAC del cliente, la dirección IP que quiere asignar al cliente y el nombre de host del cliente.
  • Haga clic en OK para guardar su configuración.
