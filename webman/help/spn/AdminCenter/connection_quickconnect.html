QuickConnect

QuickConnect permite a las aplicaciones cliente conectarse a su DiskStation a través de Internet sin necesidad de configurar reglas de reenvío de puerto. QuickConnect también funciona con paquetes desarrollados por Synology, como Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Note Station, CMS, Cloud Station y aplicaciones móviles.

Para habilitar QuickConnect:

  1. Vaya a Panel de control > QuickConnect.
  2. Marque Habilitar QuickConnect.
  3. Introduzca la información de su Cuenta eCore existente o registre una cuenta nueva.
  4. Indique un nuevo QuickConnect ID. Asegúrese de que sea fácil de recordar para poder acceder a su DiskStation en cualquier momento.
  5. Haga clic en Aplicar.

Para habilitar/deshabilitar QuickConnect para paquetes específicos:

  1. Vaya a Panel de control > QuickConnect.
  2. Haga clic en Configuración avanzada.
  3. Elija los paquetes para los que desea habilitar QuickConnect.
  4. Haga clic en Aplicar.

Nota:

  • Un QuickConnect ID personalizado puede incluir letras, números y guiones ("-"), y debe empezar con una letra.
  • Para un mejor rendimiento de QuickConnect, se recomienda ir a Panel de control > Acceso externo > Configuración del enrutador para configurar el reenvío de puerto para cada servicio:
    • DSM: 5000 (HTTP); 5001 (HTTPS)
    • Photo Station: 80 (HTTP); 443 (HTTPS)
    • Cloud Station: 6690

Solución de problemas

Si ve el mensaje Ha ocurrido un error de red, significa que el servicio QuickConnect se ha detenido debido a errores de red. Compruebe lo siguiente:

  • Su DiskStation está conectado a una red activa (en Panel de control > Red > Interfaz de red).
  • El servidor DNS y la configuración de pasarela predeterminada están configurados correctamente (en Panel de control > Red > General).
  • Asegúrese de que DiskStation puede acceder a Internet utilizando los siguientes puertos de destino: 80, 443, 8888. 

Cuando se solucionen los errores de red, el servicio QuickConnect debería reanudarse en unos minutos.
