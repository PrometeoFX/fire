Copia de seguridad de configuración

Vaya a Panel de control > Actualizar y restaurar para realizar inmediatamente copias de seguridad de las configuraciones del sistema a un equipo local. Esto le permitirá guardar diversas configuraciones (por ejemplo, grupos de usuarios, configuraciones de servicios de archivos, etc.) y restaurar dichas configuraciones más adelante. Las configuraciones del sistema se guardan como archivos con formato .dss.

Para restaurar la configuración del sistema:

    Se copiarán las siguientes configuraciones:

    • Usuarios, grupos y carpetas compartidas
    • Grupo de trabajo, Dominio y LDAP
    • Uso compartido de archivos (por ejemplo, servicio de archivos de Windows, servicio de archivos de Mac, servicio NFS, FTP y WebDAV) y servicios de copias de seguridad (por ejemplo, copia de seguridad de red, sincronización de carpetas compartidas y el servicio Time Backup)
    • Otros (por ejemplo, Inicio de usuario, Configuración de contraseña, Programador de tareas, Configuración del informe de uso de disco)

    • Otros (por ejemplo, Inicio de usuario, Configuración de contraseña, Programador de tareas, Configuración del informe de uso de disco)

    Nota:

    • Servicios de copia de seguridad incluye parámetros de servicios básicos, a la vez que excluyen información detallada acerca de tareas de copias de seguridad y destinos, etc.
    • Inicio de usuario incluye parámetros de servicios básicos, a la vez que excluye la carpeta compartida homes. Vaya a Copias de seguridad y replicación para realizar copias de seguridad y restaurar la carpeta compartida homes y sus datos.
    • Programador de tareas incluye tareas que se pueden crear en Programador de tareas mientras que excluye tareas generadas por otras aplicaciones (por ejemplo, Copias de seguridad y replicación e Informe de uso de disco, etc.)
    • Configuración del informe de uso de disco incluye tareas de preparación del informe, mientras que excluye los informes generados previamente.
    • Si los nombres de los usuarios o grupos restaurados ahora están reservados solo para el uso del sistema, se restaurarán como los otros nombres.
