Editor de texto

El Editor de texto es un editor de texto y códigos bastante versátil, que le permite acceder y editar rápidamente archivos de texto o códigos en su DiskStation directamente desde un explorador web.

Administración de archivos

Para crear archivos nuevos:

Haga clic en Archivo y elija Nuevo. Aparecerá una pestaña nueva con el título New Document.txt.

Para abrir archivos desde el Editor de texto:

Sencillamente haga clic en Archivo y elija Abrir. Aparecerá un cuadro de diálogo y podrá seleccionar un archivo con una codificación específica de caracteres que se debe abrir en una pestaña. Si el archivo ya estaba abierto, el Editor de texto le mostrará ese archivo en su estado actual.

Para abrir archivos desde File Station:

En File Station, sencillamente haga clic con el botón secundario en el archivo seleccionado y a continuación haga clic en Abrir con el Editor de texto del menú de contexto. Si el archivo ya estaba abierto, el Editor de texto le mostrará ese archivo en su estado actual. Otra manera de abrir un archivo es arrastrar uno o varios archivos desde File Station hasta el Editor de texto con el cursor.


Nota:

El editor de texto no puede abrir un archivo que supera los 10MB.

Para guardar archivos:

Sencillamente haga clic en Archivo y elija Guardar. Si es un archivo existente, se sobrescribirá con los contenidos actuales. Si desea guardar simultáneamente todos los archivos abiertos, puede ir a Archivo > Guardar todo. Si el archivo todavía no tiene nombre (un archivo nuevo), o si ha seleccionado Archivo > Guardar como, aparecerá un cuadro de diálogo que le pedirá un nombre para el archivo. En ese momento podrá elegir una ubicación, el nombre del archivo y la codificación para guardar el archivo.

Para cerrar archivos:

Sencillamente haga clic en Archivo y elija Cerrar. Otra manera de cerrar un archivo es haciendo clic en el botón de cerrar en cada pestaña. Si tiene datos que no se han guardado en el archivo, el Editor de texto siempre le preguntará si desea cerrar el archivo sin guardar dichos cambios.

Nota:

Puede guardar archivos con una codificación de caracteres especial para mantener la compatibilidad de idiomas bidireccionales. También puede especificar una codificación al abrir un archivo, de manera que el Editor de texto lo muestre correctamente.

Edición

  • Operaciones estándar
    El control de edición es compatible con instrucciones sencillas de Deshacer, Rehacer y Seleccionar todo. Estas operaciones se pueden encontrar en la mayoría de editores y funcionan de manera muy similar. Puede realizar estas operaciones seleccionando el elemento del menú correspondiente en el menú de Editar. Además, las opciones de cortar, copiar y pegar están disponibles de manera predeterminada a través del menú de contexto del explorador.
  • Sangría
    La sangría es una operación habitual en textos cuando se trabaja con la edición de código fuente. La sangría es la adición de espacios blancos adicionales al principio de una línea para separar bloques de texto, añadiendo una jerarquía. Normalmente, durante la edición, si pulsa la tecla del tabulador se insertará un carácter de tabulación en la ubicación actual. Sin embargo, si se selecciona texto, el pulsar la tecla de tabulación se añadirán espacios blancos al principio de cada línea del texto seleccionado, permitiéndole realizar una sangría para bloques enteros de texto simultáneamente. Si pulsa Mayus. y Tab simultáneamente se eliminará la sangría.
  • Sobrescribir
    Puede habilitar y deshabilitar la sobrescritura pulsando la tecla de Insertar. O sea, en lugar de insertar caracteres en el cursos, los sustituye. El cursos mostrado cambiará de forma de acuerdo con el estado de sobrescritura.
  • Edición en modo columna
    Una manera especial de editar archivos es la edición en modo columna. Esto significa que el texto se puede seleccionar en columnas, en lugar de en filas. Para iniciar la selección del modo columna, pulse la tecla Alt al seleccionar texto y mueva el cursos hacia abajo y hacia arriba según sea necesario. Cuando se ha seleccionado texto en el modo columna, al escribir se cambiará cada columna de caracteres seleccionada por el carácter seleccionado.

Búsqueda

  • Búsqueda normal
    Puede buscar texto como en la mayoría de programas, utilizando el diálogo de Búsqueda. Puede abrir este diálogo seleccionando Buscar > Encontrar. Este diálogo siempre aparecerá sobre el texto y no podrá continuar editando el texto si está abierto. En el campo de búsqueda, escriba el texto que desea encontrar.
  • Reemplazar
    Puede buscar y reemplazar texto, como en la mayoría de programas, con el diálogo de Reemplazar. Puede abrir este diálogo seleccionando Buscar > Reemplazar. Reemplazar todo reemplazará todas las ocasiones en que se repita el texto buscado por el nuevo texto.
  • Distingue mayúsculas y minúsculas
    Las coincidencias deben tener las mismas mayúsculas y minúsculas que el texto de consulta. Si no está marcado, la búsqueda no distinguirá entre mayúsculas y minúsculas.
  • Solo palabras completas
    Buscará solo coincidencias de palabras completas y no de partes de las mismas. Por ejemplo, si está buscando "nota" con Solo palabras completas, entonces solo encontrará "nota" en lugar de "notable".
  • Expresión habitual
    Se consultará se tomará como una expresión habitual.

Â¿Qué es una expresión habitual?

Una expresión habitual es una secuencia de caracteres que forman un patrón de búsqueda para técnicas potentes de ahorro de tiempo. Un uso muy sencillo de una expresión habitual sería identificar la misma palabra escrita de dos maneras distintas en un editor de texto, por ejemplo M[ea]rry.

Configurador de estilo

El configurador de estilo se utiliza principalmente para controlar el aspecto del texto en el Editor de texto.
La lista desplegable de la parte inferior derecha contiene todo el lenguaje de programación compatible que se puede utilizar como estilo para ayudarle a construir un entorno de codificación adecuado.

Preferencias

  • Mostrar caracteres invisibles
    Si se marca, se mostrarán los caracteres de espacio, tabulación, línea nueva o retorno.
  • Mostrar guías de sangría
    Si se marca, se mostrarán las guías de sangría. Resulta útil para alinear códigos en lenguaje de codificación.
  • Resaltar la línea actual
    Si se marca, la línea en la que se encuentra el acento circunflejo se resaltará con un color de fondo especial.
  • Resaltar la palabra seleccionada
    Si se marca, todas las palabras (no parciales) dentro del mismo texto que coincidan con la palabra seleccionada, se resaltarán automáticamente.
  • Mostrar números de línea
    Si se marca, se mostrarán los números de las líneas.
  • Habilitar el plegamiento de códigos
    Si se marca, ciertas líneas de códigos se pueden ocultar en base a ciertas características. Esta función es muy útil para lenguajes estructurados, como C++ o XML. Cuando se marca Ocultar widgets de revelación automáticamente, ciertas líneas de códigos se plegarán automáticamente. También puede elegir marcar solo el paréntesis de apertura o ambos, el de apertura y el de cierre, con los widgets de revelación.
  • Ajustar palabras
    Especifica si su texto se ajustará hasta el borde de la ventana o hasta un número de columna específica, sin insertar un retorno.
  • Reemplazar caracteres de cierre automáticamente
    Si se marca, los paréntesis de cierre (caracteres) se reemplazarán automáticamente en sus códigos.
  • Insertar espacios en lugar de tabulaciones
    Si se marca, al pulsar la tecla de tabulación se insertará un carácter de espacio en lugar de uno de tabulación. Ancho de tabulación se puede definir por el número de caracteres en Preferencias > Fuente y color.
  • Codificación predeterminada
    La codificación predeterminada se utiliza para guardar archivos con una codificación de caracteres específica compatible con lenguajes bidireccionales. También puede especificar una codificación al abrir un archivo, de manera que el Editor de texto lo muestre correctamente.
  • Fuente y color
    Ancho de tabulación, Tamaño de fuente y Tema se pueden personalizar de acuerdo con su preferencia personal. Ancho de tabulación está definido por el número de caracteres de espacio.

Formatos de archivo compatibles

El Editor de texto es solo compatible con archivos de solo texto. Archivos de texto con formato rico como *.doc y *.pdf no serán compatibles.
