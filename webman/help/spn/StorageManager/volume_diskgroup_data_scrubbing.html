Depuración de datos*

La depuración de datos es una característica de mantenimiento de datos que inspecciona volúmenes o grupos de discos y repara cualquier inconsistencia que detecte en los datos. Esta función se puede usar con volúmenes o grupos de discos que implementen SHR (compuestos por tres o más discos), RAID 5 o RAID 6.

Le recomendamos que realice regularmente una depuración de datos para mantener la consistencia de los datos y evitar perder cualquier dato crítico en el caso de que falle una unidad de disco duro.

Para realizar una depuración de datos:

  1. Abra el Administrador de almacenamiento.
  2. Vaya a Volumen o Grupo de discos
  3. Seleccione el volumen o grupo de discos en el que desea realizar una depuración de datos.
  4. Haga clic en Administrar
  5. Seleccione Depuración de datos.

Nota:

  • La depuración de datos solo se puede realizar cuando el estado del volumen o grupo de discos sea Normal.
  • No apague el sistema hasta que finalice la depuración de datos. Si el sistema se apaga antes de que finalice el proceso, se debe volver a iniciar la depuración de datos desde el principio.
