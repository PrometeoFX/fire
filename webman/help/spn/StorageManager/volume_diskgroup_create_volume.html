Crear un volumen

Este articulo explica cómo crear un volumen en su DiskStation. Para crear un volumen, puede elegir uno de los siguientes métodos:

  • Rápido: Esta es la mejor opción para principiantes. Elija este método si desea optimizar automáticamente la capacidad de almacenamiento y el rendimiento, sin tener que preocuparse por los aspectos técnicos, como los niveles de RAID.
  • Personalizado: Para usuarios avanzados que deseen controlar los aspectos más avanzados de la creación de volúmenes, como los tipos de RAID y la asignación de capacidad. Esta opción es compatible con varios tipos de RAID, le permite crear volúmenes individuales o múltiples, y le ofrece opciones para signar cantidades específicas de capacidad para cada volumen (solo al crear volúmenes múltiples en RAID).

Â¿Volúmenes únicos o múltiples en RAID?

Al crear un volumen con el modo personalizado, puede elegir uno de las siguientes opciones:

  • Volumen único en RAID
    • Crea un volumen único con toda la capacidad disponible de las unidades de disco duro seleccionadas.
    • Ofrece una mejor velocidad y rendimiento.
    • Admite varios tipos de RAID.
  • Volúmenes múltiples en RAID
    • Le permite crear volúmenes múltiples en un grupo de discos.
    • Permite la asignación de determinadas cantidades de capacidad para cada volumen.
    • Ofrece una mayor flexibilidad de administración del almacenamiento.
    • Admite varios tipos de RAID.

Antes de comenzar:

  • Antes de comenzar, asegúrese de que no hay datos importantes en las unidades de disco duro. Todos los datos existentes se eliminarán durante el proceso de creación.
  • Vaya ala página HDD/SSD y asegúrese de que el estado de cada unidad de disco duro es Normal o No inicializado.

Para crear un volumen:

  1. Abra el Administrador de almacenamiento.
  2. Vaya a Volumen y haga clic en Crear.
  3. Verá el asistente de creación de volúmenes. Seleccione Rápido o Personalizado según sus necesidades. A continuación siga los pasos del asistente para finalizar la creación de volúmenes.
