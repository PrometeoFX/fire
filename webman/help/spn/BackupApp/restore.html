Restaurar

Vaya a Copia de seguridad y replicación > Restaurar para restaurar datos de copias de seguridad.

Para restaurar una tarea de copia de seguridad:

  1. Seleccione una tarea de copia de seguridad y a continuación haga clic en Restaurar. Si la tarea es un copia de seguridad de varias versiones, seleccione Restaurar > Tarea completa.
  2. Tal vez se le pida que seleccione la configuración del sistema, diferentes versiones de datos de copias de seguridad o más. Dependerá de si se trata de una tarea de copia de seguridad, una tarea de copia de seguridad de LUN*, destino de copia de seguridad de una sola versión (o sea, carpeta compartida) o destino de copia de seguridad de varias versiones (o sea, volumen).
  3. Siga el asistente para completar la configuración.

Nota:

  • La restauración en algunas aplicaciones es compatible. Para comprobar si un paquete específico es compatible con la restauración, además de sus limitaciones, consulte la sección de ayuda de cada aplicación.
  • Cuando restaure un paquete, si el paquete no está instalado, el sistema instalará automáticamente la misma versión, o una posterior, que la de la aplicación. Si la aplicación está instalada pero la versión es anterior a la que se ha utilizado para realizar la copia de seguridad, el sistema la actualizará automáticamente a la versión original de la que se hizo la copia de seguridad.
  • Aplicaciones de las que se haya realizado copia de seguridad en DSM 5.1 o versiones posteriores no se pueden restaurar en DSM 5.0 o versiones anteriores.

Para restaurar un archivo o carpeta individual con tareas de copia de seguridad de varias versiones:

  1. Inicie el Explorador de versiones de archivos. Puede hacer esto con uno de los métodos siguientes:
    • Haga clic en Restaurar y seleccione Archivos/carpetas individuales del menú desplegable.
    • Vaya a Destino de la copia de seguridad y amplíe el destino de copia de seguridad seleccionado. Haga clic en el icono de Información de la versión de copia de seguridad. En la ventana que aparece selecciona la versión de la copia de seguridad que desea restaurar y haga clic en el icono de Explorador de versiones de archivo.
  2. Puede explorar las versiones del archivo o carpeta que desee con el icono de línea temporal o calendario en la parte inferior de la pantalla.
  3. Seleccione la versión que desea administrar y haga clic en Restaurar.
  4. Alternativamente puede copiar un archivo o carpeta a otra ubicación o descargarlos con el botón Copiar a... o Descargar.
  5. También puede hacer clic con el botón secundario en el archivo o carpeta y seleccionar estas opciones del menú desplegable.

Para buscar una versión específica:

  1. Haga clic en la flecha de la barra de filtro de la esquina superior derecha.
  2. En el menú desplegable puede filtrar archivos por:
    • Palabra clave: introduzca el nombre de cualquier archivo o carpeta. Puede utilizar comodines [*] (para representar uno o varios caracteres) y [?] (para representar un carácter aleatorio) en las palabras clave. Por ejemplo, [a*n] puede representar [acción], [atención], o [an].
    • Tipo de archivo: en el menú desplegable puede elegir entre Cualquiera, Cualquier carpeta o Cualquier archivo para ajustar la búsqueda.
    • Tamaño: elija Cualquiera o seleccione igual a, es mayor que, es menor que y a continuación introduzca un valor.
    • Intervalo de fechas: elija entre Hoy, Ayer, Última semana, Último mes o Personalizar para introducir una fecha de inicio y una fecha de finalización para encontrar versiones dentro de un intervalo de fechas específico.
  3. Para aplicar los filtros haga clic en Filtrar.
  4. Para restablecer la búsqueda filtrada, haga clic en el botón de Restablecer.

Nota:

  • Solo las tareas de copias de seguridad de varias versiones ofrecen la función de Explorador de versiones de archivos.
  • El Explorador de versiones de archivos solo tiene funciones parciales en DSM 5.1 o versiones inferiores.
  • La configuración del sistema y las aplicaciones solo se pueden restaurar a través de Restaurar > Tarea completa.
  • Los metadatos no se incluirán en las acciones de Copiar a... y Descargar.

Para restaurar tareas de copia de seguridad desde otro destino de copia de seguridad:

  1. Haga clic en Restaurar desde....
  2. Elija cualquiera de las siguientes opciones para encontrar una tarea de copia de seguridad y haga clic en Siguiente:
    • Restauración local: elija esta opción para restaurar datos desde una carpeta compartida local o disco externo.
    • Restauración de red: elija esta opción para restaurar datos de otro DiskStation o de un servidor compatible con rsync.
    • Restauración del servicio de nube pública: elija esta opción para restaurar datos desde un servidor de nube pública.
  3. Cuando se le pida, seleccione el origen donde se conserva la tarea de copia de seguridad y a continuación seleccione el destino para restaurar los datos.
  4. Siga las instrucciones del asistente para completar el proceso.

Nota:

  • Embedded DataStation solo puede restaurar carpetas compartidas a dispositivos externos ext3/ext4.
  • Embedded DataStation no es compatible con la restauración para aplicaciones sin un dispositivo externo ext3/ext4.
