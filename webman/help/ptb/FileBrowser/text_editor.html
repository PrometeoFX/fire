Editor de Texto

O Editor de Texto é um editor de texto e código altamente versátil, permitindo acessar rapidamente e editar arquivos de texto ou códigos em seu DiskStation direto de um navegador da Web.

Gerenciamento de arquivos

Para criar arquivos novos:

Clique em Arquivo e escolha Novo. Uma nova guia vazia aparecerá com o título Novo Documento.txt.

Para abrir arquivos do Editor de Texto:

Simplesmente clique em Arquivo e escolha Abrir. Uma caixa de diálogo aparecerá e você pode selecionar um arquivo com a codificação de caractere específica e necessária para abrir em uma guia. Se o arquivo já estiver aberto, o Editor de Texto mostrará que o arquivo está em seu estado atual.

Para abrir arquivos do File Station:

No File Station, simplesmente clique com o botão direito no arquivo selecionado e, em seguida, clique em Abrir com Editor de Texto no menu de contexto. Se o arquivo já estiver aberto, o Editor de Texto mostrará que o arquivo está em seu estado atual. Outra forma de abrir um arquivo é arrastar um ou vários arquivos do File Station para o Editor de Texto usando o cursor.


Observação:

O Editor de Texto não pode abrir arquivos que excedam 10 MB.

Para salvar arquivos:

Simplesmente clique em Arquivo e escolha Salvar. Se for um arquivo existente, ele será substituído pelo conteúdo atual. Se deseja salvar todos os arquivos que abriu de uma só vez, pode ir para Arquivo > Salvar todos. Se o arquivo ainda não tiver nome (um arquivo novo), ou se você selecionou Arquivo > Salvar como, uma caixa de diálogo será exibida, solicitando o nome do arquivo. Você pode escolher o local, o nome do arquivo e a codificação para salvar o arquivo.

Para fechar os arquivos:

Simplesmente clique em Arquivo e escolha Fechar. Outra forma de fechar um arquivo é clicando no botão fechar em cada guia. Se houver dados não salvos em um arquivo, o Editor de Texto sempre perguntará se você deseja fechar sem salvar as alterações.

Observação:

Você pode salvar arquivos com a codificação de caractere específica para suportar idiomas bidirecionais. Também pode especificar uma codificação ao abrir um arquivo, assim o Editor de Texto abrirá o arquivo corretamente.

Editar

  • Operações padrão
    O controle de edição suporta Desfazer, Refazer e Selecionar todos. Essas operações podem ser encontradas na maior parte dos editores e funcionam de maneira muito parecida. Você pode executar essas operações selecionando o item de menu correspondente no menu Editar. Além disso, por padrão as operações Cortar, Copiar e Colar estão disponíveis por meio do menu de contexto do navegador.
  • Recuo
    Recuar o texto é uma operação comum quando se trata de edição de código fonte. O recuo é a adição de espaço em branco extra no começo da linha para separar blocos de texto, adicionando uma hierarquia. Normalmente ao editar, pressionando a tecla Tab será inserido um caractere de tabulação no local atual. Entretanto, se algum texto for selecionado, pressionando a tecla Tab será adicionado espaço em branco no início de cada linha que possui um texto selecionado, permitindo que você recue blocos inteiros de texto de uma só vez. Pressionando Shift e Tab ao mesmo tempo removerá o recuo.
  • Sobrescrever
    Você pode ativar ou desativar esta função pressionando a tecla Insert. Ou seja, em vez de inserir caracteres na posição do cursor, você os substitui. O cursor exibido mudará de formato de acordo com o status da substituição.
  • Edição de modo de coluna
    Uma maneira especial de editar arquivos é a edição no modo de coluna. Isso significa que o texto pode ser selecionado em colunas, em vez de linhas. Para iniciar uma seleção no modo de coluna, solte a tecla Alt quando selecionar o texto, movendo o cursor para baixo ou para cima, conforme necessário. Quando o texto é selecionado no chamado modo de coluna, a digitação alterará cada coluna de caracteres selecionada para o caractere digitado.

Pesquisando

  • Pesquisa normal
    Você pode pesquisar texto como em muitos programas, usando a caixa de diálogo Localizar. Você pode abrir essa caixa de diálogo selecionando Pesquisar > Localizar. Essa caixa de diálogo sempre fica na parte superior da página, e não é possível continuar editando o texto se ela estiver aberta. No campo de pesquisa, digite o texto que deseja localizar.
  • Substituir
    Você pode localizar e substituir texto como em muitos programas, usando a caixa de diálogo Substituir. Você pode abrir essa caixa de diálogo selecionando Pesquisar > Substituir. Substituir tudo substituirá todas as ocorrências da consulta pelo texto de substituição.
  • Diferenciar maiúsculas de minúsculas
    Reforça que as dicas têm exatamente a mesma capitalização que a consulta. Se não estiver marcada, a pesquisa não diferencia maiúsculas de minúsculas.
  • Apenas palavras inteiras
    Resultará em uma dica somente se corresponder à palavra inteira, e não a uma parte dela. Por exemplo, se você procurar por "exem" com a opção Apenas palavras inteiras, então localizará "exem" somente, em vez de "exemplo".
  • Expressão regular
    A consulta será tratada como uma expressão regular.

O que é uma expressão regular?

Uma expressão regular é uma sequência de caracteres que forma um padrão de pesquisa para uma técnica de economia de tempo poderosa. Um uso muito simples de uma expressão regular seria localizar a mesma palavra com duas ortografias diferentes em um editor de texto, por exemplo N[ós]oz.

Preferências

  • Exibir caracteres invisíveis
    Se marcada, os caracteres de espaços, tabulações, novas linhas ou retornos de linha serão exibidos.
  • Exibir guias de indentação
    Se marcada, as guias de indentação serão exibidas. Isso é útil para alinhar códigos na linguagem de codificação.
  • Destacar linha atual
    Se marcada, a linha em que o cursor está será apresentada com uma cor de plano de fundo especial.
  • Destacar palavra selecionada
    Se marcada, todas as palavras (não parte delas) dentro do mesmo arquivo de texto que correspondem à palavra selecionada serão automaticamente destacadas.
  • Exibir números de linha
    Se marcada, os números de linha serão exibidos.
  • Ativar dobra de código
    Se marcada, certas linhas de seus códigos podem ficar ocultas, de acordo com determinadas peculiaridades. Esse recurso funciona bem para linguagens estruturadas, como C++ ou XML. Quando Esconder expansores de visualização automaticamente estiver marcada, certas linhas de códigos serão dobradas automaticamente. Você também pode escolher marcar somente colchetes iniciais ou ambos, de início e fechamento, com os expansores de visualização.
  • Ajustar Palavra
    Especifica se seu texto é ajustado ao limite da janela ou em um número de coluna específico, sem inserir um retorno de linha.
  • Anexar automaticamente caracteres de fechamento
    Se marcada, colchetes de fechamento (caracteres) serão anexados automaticamente em seus códigos.
  • Inserir espaços em vez de tabulações
    Se marcada, ao pressionar a tecla Tab serão adicionados caracteres de espaço em vez de um caractere de tabulação. Largura da aba pode ser definida pelo número de caracteres em Preferências > Fonte e cor.
  • Codificação padrão
    A codificação padrão é usada para salvar arquivos com codificação de caractere específica para suportar idiomas bidirecionais. Também pode especificar uma codificação ao abrir um arquivo, assim o Editor de Texto abrirá o arquivo corretamente.
  • Fonte e Cor
    Largura da aba, Tamanho da fonte e Tema podem ser personalizados de acordo com sua preferência pessoal. Largura da aba é definida pelo número de caracteres de espaço.

Formatos de arquivo suportados

O Editor de Texto suporta somente arquivos de texto simples. Os arquivos de texto com formatação RTF como *.doc e *.pdf não serão suportados.
