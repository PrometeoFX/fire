Editar fotos

Com o File Station você pode editar fotos com editores de foto externos baseados na Internet como o Pixlr Editor ou o Pixlr Express.

Para editar uma foto com um editor de foto baseado na Internet:

  1. Navegue até a pasta onde a foto está localizada.
  2. Clique com o botão direito na foto e escolha Editar no Pixlr Editor ou Editar no Pixlr Express.
      Observação:
    • O Pixlr Editor suporta os seguintes formatos: jpg, gif, png, psd, PXD (imagem em camadas Pixlr)
    • O Pixlr Express suporta os seguintes formatos: jpg, gif, png
  3. O editor de fotos selecionado será aberto em uma nova janela do navegador. Em alguns instantes você verá sua foto e poderá editá-la com as ferramentas oferecidas.
  4. Após terminar a edição da foto, faça o seguinte para salvá-la.
    • Para o Pixlr Express: Clique em Concluído para salvar a foto no formato original na localização original no DiskStation.
    • Para o Pixlr Editor: Selecione Arquivo > Salvar, escolha a localização (que pode ser a localização original da foto no DiskStation, seu computador ou outras redes sociais) para salvá-la no painel esquerdo, escolha o novo nome ou formato conforme suas preferências e clique em OK.

Observação: Para editar suas fotos com qualquer um dos editores de fotos com base na Web, verifique se o seu DiskStation recuperou com êxito um endereço IP externo (em Menu principal > Painel de Controle > Acesso Externo > DDNS ou Menu principal > Painel de Controle > Rede > Interface de Rede > PPPoE dependendo de suas configurações de rede). Se o DiskStation estiver localizado atrás de um roteador, verifique se foi especificada uma regra de encaminhamento de portas para o DSM em Menu principal > Painel de controle > Acesso Externo > Configuração do Roteador. Caso ainda tenha problemas para abrir os editores de fotos, você pode ter de especificar a porta do roteador para o DSM em Menu principal > Painel de controle > Acesso Externo > Avançado.
