Montar pastas remotas

Você pode montar pastas remotas de servidores remotos (como computadores de rede ou outro DiskStation) para navegar facilmente pelos recursos disponíveis da rede com o File Station.

Para montar uma pasta remota:

  1. Clique em Ferramentas > Montar pasta remota.
  2. Digite "\\Remote_Server\Remote_Folder" no campo Pasta para especificar o caminho da pasta remota. Por exemplo, "\\192.168.0.2\share" ou "\\chad\cheeseburger\pictures".
  3. Se privilégio para a pasta remota for exigido pelo servidor remoto, digite suas credenciais de usuário do servidor remoto nos campos Nome da conta e Senha.
  4. Clique em Procurar para selecionar ou criar uma pasta de destino vazia em seu DiskStation para montar a pasta remota.
  5. Marque Montar automaticamente ao iniciar se desejar que o DiskStation monte essa pasta remota a cada inicialização ou reinicialização do sistema.
  6. Clique em Montar. Agora você pode navegar até a pasta de destino em que a pasta remota está montada e acessar o conteúdo das pastas remotas.

Para desmontar uma pasta remota:

  1. Selecione a pasta remota que você deseja desmontar e clique em Ferramentas > Desmontar.
  2. Como alternativa, basta clicar com o botão direito na pasta remota e selecionar Desmontar.

Observação:

  • O número máximo de pastas remotas concomitantes é de 100.
  • Você só tem permissão para montar pastas remotas de um servidor remoto que suporte o protocolo CIFS.
