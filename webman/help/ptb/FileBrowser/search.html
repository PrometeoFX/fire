Pesquisar arquivos ou pastas

O Filtro do File Station fornece opções para pesquisar e exibir arquivos ou pastas que atendam seus critérios. Você pode filtrar através de arquivos ou pastas na pasta atual ou executar a pesquisa avançada que cobre todas as subpastas para obter resultados de pesquisa mais refinados.

Para pesquisar os arquivos ou pastas na pasta atual:

  1. Use o File Station para ir até a pasta cujo conteúdo deseja pesquisar.
  2. No campo Filtro no canto superior direito, digite uma palavra-chave e veja o resultado filtrado na área principal da janela.

Para executar uma pesquisa avançada:

  1. Use o File Station para ir até a pasta cujo conteúdo deseja pesquisar.
  2. Clique na lupa ao lado do campo Filtro no canto superior direito para revelar o menu de pesquisa avançado.
  3. Digite as informações ou escolha nos menus suspensos os seguintes critérios de pesquisa:
    • Palavra-chave: Digite uma palavra-chave para pesquisar arquivos ou pastas com nomes correspondentes.

      Nota:

      • Você pode usar curingas [*] (ou seja, zero ou mais caracteres) e [?] (um caractere aleatório) em suas palavras-chave. Por exemplo, [a*n] pode representar [action], [attention], ou [an].
      • Você pode usar vírgulas (,) para separar várias palavras-chave. Por exemplo, [a,tion] pode representar [action], [a function], [function added] ou [nation].
      • Você pode adicionar aspas duplas ["] no início e no final da palavra-chave para pesquisar correspondências exatas. Por exemplo, ["a typ"] pode representar [a type], mas não pode representar [typical].
    • Localização: Selecione do menu suspenso para definir a faixa e a profundidade da pesquisa avançada. Você também pode personalizar o escopo da pesquisa selecionando outras pastas compartilhadas listadas.
    • Tipo de Arquivo: Selecione a Extensão para limitar o alcance da pesquisa a tipos específicos de arquivos, ou selecione Qualquer Pasta para pesquisar apenas por pastas.
    • Data: Selecione Data de Modificação, Data de Criação ou Último Acesso e selecione a data no calendário para pesquisar arquivos com datas correspondentes.
    • Tamanho (MB): Selecione igual a, maior que e menor que e digite um valor para pesquisar arquivos com tamanhos correspondentes. (Ao selecionar igual a, os resultados da pesquisa incluirão arquivos com aproximadamente o mesmo tamanho especificado.)
    • Proprietário/Grupo: Selecione Proprietário ou Grupo e selecione no menu suspenso para pesquisar arquivos ou pastas que pertençam ao proprietário ou grupo selecionado.

      Nota:

      • Para realizar a pesquisa avançada, é necessário especificar Localização e pelo menos um outro critério de pesquisa.
      • Se uma pasta ISO ou uma pasta remota não estiver montada na pasta compartilhada indexada, você não conseguirá encontrar os arquivos dentro dela através da pesquisa avançada.
    • Clique em Pesquisar para iniciar a pesquisa ou clique em Redefinir e especifique novamente o critério de pesquisa. Quando a pesquisa estiver concluída, os arquivos correspondentes serão listados na área principal do File Station.
