Montar unidades virtuais

Você pode montar unidades virtuais para acessar o conteúdo das imagens do disco com o File Station.

Para montar uma unidade virtual:

  1. Selecione uma imagem de disco e clique em Ferramentas > Montar unidade virtual.
  2. Clique em Procurar para selecionar ou criar uma pasta de destino vazia em seu DiskStation para montar a unidade virtual.
  3. Marque Montar automaticamente ao iniciar caso queira que o DiskStation monte a unidade virtual a cada inicialização ou reinicialização do sistema. Clique em Montar para continuar. Agora você pode navegar até a pasta de destino em que a unidade virtual está montada e acessar o conteúdo da imagem do disco.

Para desmontar uma unidade virtual:

  1. Clique com o botão direito na pasta de destino em que a unidade virtual está montada e selecione Desmontar.
    Observação:
  1. Formato de imagem de disco suportado: Arquivo ".iso" usando o sistema de arquivos ISO 9660 e UDF.
  2. O número máximo de unidades virtuais concomitantes é 16.
