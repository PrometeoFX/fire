Gerenciar Configurações do File Station

Para gerenciar as configurações do File Station, abra o File Station e clique no botão Configurações.

Geral

Aqui você pode habilitar os registros do File Station para gravar as atividades de todos os usuários. As atividades de usuário incluem Criar pasta, Carregar, Baixar, Excluir, Renomear, Mover, Copiar e Propriedade definida.

Para ativar o registro do File Station:

  1. Se desejar gravar as atividades de todos os usuários, marque Habilitar registro do File Station.
  2. Clique em OK.

Para ver o registro do File Station:

  1. Clique em Visualizar Registros, o que o levará ao Log Center.
  2. Clique na guia Transferência de arquivo.

Para sobrescrever automaticamente os arquivos com o carregamento arrastar e soltar ou atalhos do teclado:

Marque Sempre sobrescrever arquivos quando eu arrastá-los e soltá-los no navegador de Internet ou colá-los com atalhos do teclado se desejar sobrescrever automaticamente os arquivos com o mesmo nome quando arrastar e soltar arquivos de sua área de trabalho do computador no navegador de Internet ou colar arquivos com atalhos do teclado dentro do File Station.

Para pesquisar e gerenciar arquivos do computador local com o File Station:

  1. Marque Procurar arquivos do computador local com o File Station.
  2. Clique em OK.

Aplicar Permissões UNIX Padrão

Marque Configurações avançadas e marque essa opção para aplicar as permissões UNIX padrão ao fazer upload ou criar arquivos e pastas. As permissões aplicadas são as mesmas das permissões aplicadas pelo comando UNIX umask. Quando essa opção está habilitada, as permissões UNIX são 644 para arquivos e 755 para pastas. Quando essa opção está desabilitada, as permissões UNIX são 666 para arquivos e 777 para pastas. O valor umask padrão é 022.

Observação:

  • Para pastas compartilhadas com a ACL do Windows habilitada (todos os compartilhamentos, exceto "foto" e compartilhamentos em unidades externas), execute o comando chmod no cliente Linux ou FTP para alterar os tipos de permissão de arquivos e pastas da ACL do Windows para UNIX.
  • Habilitar essa opção pode causar problemas de permissão inconsistente entre os diferentes protocolos. Para evitar inconsistências, sugerimos deixar essa opção desabilitada.

Montagem/Conexões

Por padrão, todos os usuários podem montar pastas remotas ou unidades virtuais. Aqui você pode limitar o acesso à pasta remota ou o gerenciamento de unidades virtuais apenas para administradores.

Observação:

Usuários não administradores podem montar apenas pastas remotas em sua pasta home.

Para permitir que diferentes usuários montem pastas remotas ou unidades virtuais:

  1. Selecione Todos os usuários ou Administradores na seção Pasta Remota ou Unidade Virtual.
  2. Clique em OK.

Para permitir que diferentes usuários montem servidores e serviços na nuvem:

  1. Selecione Todos os usuários, Administradores ou Usuário específico.
  2. Selecionar Usuário específico abrirá uma janela permitindo configurar as permissões de montagem para diferentes usuários. Para permitir o acesso de um usuário específico, verifique se a caixa está marcada na coluna Ativado. Para desativar o acesso de um usuário específico, verifique se a caixa está desmarcada na coluna Ativado. Quando estive concluído, clique em OK para fechar essa janela.

    3. Observação:

    • Os usuários que pertencem ao grupo administrators podem usar este recurso, mas não podem fazer alterações. Usuários conectados como guest não podem nem usar este recurso, nem fazer quaisquer alterações. As permissões para todos os outros usuários podem ser configuradas aqui.
  3. Clique em OK.

Links compartilhados

Por padrão, todos os usuários podem compartilhar links de arquivos. Aqui você pode limitar o acesso ao gerenciamento de links compartilhados apenas para administradores.

Para permitir que diferentes usuários compartilhes links de arquivo:

  1. Selecione Todos os usuários, Administradores ou Usuários específicos.
  2. Clique em OK.

Para abrir arquivos HTML como texto sem formatação:

  1. Marque a caixa Sempre abrir arquivos HTML usando o formato de texto sem formatação. Se essa opção estiver ativada, os arquivos HTML (ou seja: arquivos HTM, HTML, JSP ou XHTML) serão exibidos usando o formato de texto sem formatação quando os usuários os abrirem por um link de compartilhamento de arquivos. Essa opção ajuda a evitar que pessoas mal intencionadas roubem sessões de login de usuários usando scripts incorporados em arquivos HTML.
  2. Clique em OK.

Limite de velocidade

Você pode habilitar um limite de velocidade para controlar a largura de banda máxima usada pelos usuários e grupos que transferem os dados via File Station.

Para ativar o limite de velocidade:

  1. Marque a caixa Ativar limite de velocidade ou Ativar limite de velocidade de acordo com uma programação se deseja implementar um limite de velocidade apenas em momentos específicos. Esta é uma configuração para todo o servidor e anulará as configurações de usuários e grupos individuais. Escolha Nenhum limite de velocidade para desativar as configurações de limite de velocidade de todos os usuários e grupos.
  2. Clique em Configurações do limite de velocidade para modificar as configurações para usuários ou grupos individuais. Para cada usuário e grupo, você pode selecionar uma das seguintes opções:
    • Aplicar configurações de grupo: As configurações do limite de velocidade serão iguais às do grupo ao qual o usuário pertence.
    • Não há limite de velocidade: Nenhum limite de velocidade será aplicado quando os membros do grupo transferirem arquivos com o serviço.
    • Constante: Um limite de velocidade fixo e constante será aplicado ao usuário ou membros do grupo. Os limites de velocidade de upload e download podem ser especificados nos campos à esquerda.
    • Variável: Dois limites de velocidade diferentes podem ser especificados e aplicados ao usuário ou aos membros de grupo, de acordo com uma programação. Clique no botão Personalizar para modificar as configurações de limite de velocidade e também uma programação.
  3. Clique em Aplicar.

Observação:

  • Para obter mais detalhes sobre a dinâmica do limite de velocidade do grupo, clique aqui.
  • Ao configurar o limite de velocidade para um usuário, a seção Resultado exibe o limite de velocidade real do usuário de acordo com as configurações individuais dele e as do grupo. Por exemplo, se o usuário "Achilles" tiver um limite de download individual de 100 KB/s, mas pertence a um grupo com um limite de 500 KB/s, a seção Resultados mostrará "0/500".

Segurança

Aqui é possível encontrar as configurações de segurança abaixo.

Para abrir arquivos HTML como texto sem formatação:

  1. Marque a caixa Sempre abrir arquivos HTML usando o formato de texto sem formatação. Se essa opção estiver habilitada, arquivos HTML (ou seja: arquivos HTM, HTML, JSP ou XHTML) serão exibidos usando o formato de texto sem formatação quando forem abertos no File Station. Essa opção ajuda a evitar que pessoas mal intencionadas roubem sessões de login de usuários usando scripts incorporados em arquivos HTML.

    2. Observação:

    • Essa não é uma configuração global. Cada usuário pode escolher se quer ou não habilitar essa opção de acordo com suas necessidades individuais.
  2. Clique em OK.
