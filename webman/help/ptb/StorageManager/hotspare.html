Hot Spare*

Os discos hot spare são discos rígidos de reserva que podem reparar um volume/Grupo de discos/iSCSI LUN danificado substituindo um disco com defeito automaticamente. Os discos hot spare podem ser atribuídos globalmente para proteger qualquer volume/Grupo de discos/iSCSI LUN dentro do seu DiskStation, desde que o volume/Grupo de discos/iSCSI LUN coincida com os seguintes critérios:

  • O tipo RAID do grupo volume/Grupo de discos/iSCSI LUN deve ser um com proteção de dados (isto é, RAID 1, RAID 5, RAID 6, RAID 10).
  • O tamanho do disco hot spare deve ser maior ou igual ao tamanho do menor disco do volume/Grupo de discos/iSCSI LUN.

Para configurar discos hot spare:

  1. Clique em Gerenciar.
  2. Aparece uma caixa de diálogo exibindo o status do hot spare de cada volume/Grupo de discos/iSCSI LUN, bem como os discos disponíveis. O status de volume/Grupo de disco/iSCSI LUN inclui o seguinte:
    • Atribuído: O volume/Grupo de discos/iSCSI LUN é protegido por pelo menos um disco hot spare atribuído.
    • Disponível: Existem discos disponíveis que podem proteger o volume/Grupo de discos/iSCSI LUN, mas nenhum hot spare foi atribuído ainda.
    • Não disponível: Não há discos hot spare disponíveis que possam proteger o volume/Grupo de discos/iSCSI LUN.
  3. As seções Discos disponíveis exibem discos rígidos que podem ser atribuídos como discos hot spare. Para atribuir um disco hot spare, marque a caixa ao lado do disco que deseja atribuir como hot spare.
  4. Clique em Aplicar para salvar as configurações.

Nota: Quando um disco rígido é atribuído como um disco hot spare, todos os dados do disco são apagados. Antes de designar um disco hot spare, certifique-se de que ele não tenha dados importantes.

____
*Disponível apenas em certos modelos.
