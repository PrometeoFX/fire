Depuração de Dados*

A depuração de dados é o recurso de manutenção que inspeciona os volumes ou grupos de discos e repara qualquer inconsistência de dados detectada. Essa função pode ser usada com volumes ou grupos de discos que utilizem o SHR (composto de três ou mais discos), RAID 5 ou RAID 6.

Recomendamos a execução regular da depuração de dados para manter a consistência dos dados e evitar perda de quaisquer dados críticos em caso de falha no disco rígido.

Para executar a depuração de dados:

  1. Abra o Gerenciador de armazenamento.
  2. Vá para Volume ou Grupo de Discos
  3. Selecione o volume ou grupo de discos no qual executar a depuração de dados.
  4. Clique em Gerenciar
  5. Selecione Depuração de dados.

Observação:

  • A depuração de dados pode ser executada apenas quando o status do volume ou grupo de discos for Normal.
  • Não desligue o sistema antes que a depuração de dados esteja concluída. Se o sistema for desligado antes da conclusão, a depuração de dados deve ser reiniciada a partir do início.
