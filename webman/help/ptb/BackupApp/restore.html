Restaurar

Vá para Backup e replicação > Restaurar para restaurar os dados de backup.

Para restaurar uma tarefa de backup:

  1. Selecione uma tarefa de backup e clique em Restaurar. Se a tarefa for um backup multiversão, selecione Restaurar > Tarefa inteira.
  2. Você pode ser solicitado a selecionar a configuração do sistema, as diversas versões de dados de backup ou outras. Isso depende se a tarefa for de backup de dados, tarefa de backup do LUN*, destino de backup de uma única versão (ou seja, pasta compartilhada) ou destino de backup de diversas versões (ou seja, volume).
  3. Siga o assistente para concluir a configuração.

  2. Você pode ser solicitado a selecionar a configuração do sistema, as diversas versões de dados de backup ou outras. Depende do tipo de tarefa de backup.
  3. Siga o assistente para concluir a configuração.

Nota:

  • A restauração de determinados aplicativos é suportada. Para saber se um pacote específico suporta a restauração e conhecer suas limitações, consulte a ajuda de cada aplicativo.
  • Durante a restauração de um pacote, se o pacote não estiver instalado, o sistema instalará automaticamente a versão mais recente ou a mesma versão do aplicativo. Se o aplicativo estiver instalado, mas for de uma versão mais antiga que a versão de backup, o sistema irá atualizá-lo automaticamente para a versão de backup original.
  • Os aplicativos de backup do DSM 5.1 ou superior não podem ser restaurados no DSM 5.0 ou inferior.

Para restaurar um único arquivo ou pasta para tarefa de backup multiversão:

  1. Abra a Versão do navegador de arquivos. Você pode fazer isso com um dos métodos abaixo:
    • Clique em Restaurar e selecione Arquivos/pastas individuais do menu suspenso.
    • Vá para Destino de backup e expanda o destino de backup selecionado. Clique no ícone Informações de versão do backup. Na janela que aparece, selecione a versão do backup que você deseja restaurar e clique no ícone Versão do navegador de arquivos.
  2. Você pode procurar as versões de um arquivo ou pasta que desejar usando a linha do tempo ou o ícone do calendário na parte inferior da tela.
  3. Selecione a versão que deseja gerenciar e clique em Restaurar.
  4. Como alternativa, você pode copiar um arquivo ou pasta para outro local ou baixá-lo usando o botão Copiar para... ou Baixar.
  5. Você também pode clicar com o botão direito no arquivo ou pasta e selecionar essas opções do menu suspenso.

Para pesquisar uma versão específica:

  1. Clique na seta na barra de filtros no canto superior direito.
  2. No menu suspenso, você pode filtrar os arquivos por:
    • Palavra-chave: digite o nome de qualquer arquivo ou pasta. Você pode usar curingas [*] (representando um ou mais caracteres) e [?] (representando um caractere aleatório) em suas palavras-chave. Por exemplo, [a*n] pode representar [action], [attention], ou [an].
    • Tipo de arquivo: no menu suspenso você pode selecionar de Qualquer, Qualquer pasta ou Qualquer arquivo para refinar sua pesquisa.
    • Tamanho: selecione Qualquer, ou selecione igual a, maior que, menor que e digite um valor.
    • Intervalo de datas: selecione entre Hoje, Ontem, Semana passada ou Mês passado, ou Personalizar para digitar uma data de início e final a fim de localizar registros dentro de um intervalo específico.
  3. Para aplicar seus filtros, clique em Filtro.
  4. Para restabelecer sua pesquisa filtrada, clique no botão Redefinir.

Nota:

  • Apenas as tarefas de backup multiversão fornecem a função Versão do navegador de arquivos.
  • A versão do navegador de arquivos te apenas funções parciais no DSM 5.1 ou inferior.
  • A configuração e os aplicativos do sistema podem ser restaurados apenas através de Restaurar > Tarefa inteira.
  • Os metadados não serão incluídos nas ações Copiar para... e Baixar.

Para restaurar tarefas de backup de outro destino de backup:

  1. Clique em Restaurar de....
  2. Selecione um dos itens a seguir para localizar a tarefa de backup e clique em Próximo:
    • Restauração local: Selecione esta opção para restaurar os dados de uma pasta compartilhada local ou disco externo.
    • Restauração de rede: Selecione esta opção para restaurar os dados de outro DiskStation ou de um servidor compatível de rsync.
    • Restauração do serviço de nuvem pública: Selecione essa opção para restaurar os dados de um servidor de nuvem pública.
  3. Quando solicitado, selecione a origem onde a tarefa de backup é mantida e selecione o destino para restaurar os dados.
  4. Siga o assistente para concluir.

Nota:

  • O Embedded DataStation só pode restaurar pastas compartilhadas para dispositivos externos ext3/ext4.
  • O Embedded DataStation não suporta a restauração de aplicativos sem um dispositivo externo ext3/ext4.
