Gerenciar perfis de relatório

Você pode criar e modificar tarefas agendadas para gerar relatórios sobre o uso do volume em Analisador de Armazenamento > Perfil de Relatório.

Para criar uma tarefa de relatório:

  1. Clique em Criar para abrir o assistente.
  2. Especifique um nome para a nova tarefa de relatório e digite o endereço de e-mail para o qual deseja enviar o relatório.
  3. Selecione um ou vários tipos de relatório que deseja gerar.
  4. Realize uma das seguintes ações para especificar um horário para gerar relatórios:
    • Marque Gerar relatórios de acordo com um agendamento e especifique o horário se desejar que o sistema analise o uso do volume periodicamente.
    • Marque Gerar relatórios agora se desejar iniciar manualmente uma tarefa de relatório agora.
  5. Se necessário, especifique determinados usuários ou pastas compartilhadas a serem incluídas para limitar os relatórios. (Observe que se você desejar especificar e gerar relatórios em pastas compartilhadas que estejam criptografadas, certifique-se de que os compartilhamentos criptografados foram montados antes de a tarefa de relatório ser executada.)
  6. Clique em Aplicar para salvar as configurações.

Observação:

  • Pressione Enter/Return após cada endereço de e-mail para separar vários endereços de e-mail.
  • Pressione Enter/Return após cada usuário ao especificar usuários cujos arquivos serão listados individualmente por grupos de arquivo.
  • O relatório Arquivos por grupo de arquivo listará os seguintes tipos de arquivo:
    Áudio, arquivos de imagem de disco, documentos, arquivos executáveis, imagens, vídeos, páginas da Internet, arquivos compactados e outros.
  • A seção Arquivos grandes do relatório lista os 500 arquivos maiores na página do relatório de uso ou os 200 arquivos maiores na página completa do relatório.

Para editar uma tarefa de relatório:

  1. Selecione uma tarefa de relatório na lista e clique em Editar.
  2. Clique nas guias individuais para modificar as configurações dessa tarefa.

Para excluir uma tarefa de relatório:

Selecione uma tarefa de relatório na lista e clique em Excluir.

Para visualizar relatórios do histórico:

  1. Selecione uma tarefa de relatório na lista e clique em Relatório > Exibir relatórios do histórico.
  2. Na janela que aparece, clique em uma entrada para visualizar o relatório gerado no horário especificado.

Para gerar relatórios imediatamente:

Selecione uma tarefa de relatório na lista e clique em Relatório > Gerar relatórios agora para gerar relatórios sob demanda.

Para alterar onde os relatórios são salvos:

  1. Clique em Configurações.
  2. Selecione uma nova pasta de destino para alterar o local onde os relatórios são salvos.

Para coletar o histórico de uso do volume em um cronograma regular:

Por padrão, os dados de uso do volume são reunidos sempre que o relatório Uso do Volume é gerado e na inicialização do sistema. Além dos casos acima, você pode optar por um novo agendamento para reunir o histórico de uso. Para isso, clique em Configurações e marque Coletar histórico de uso do volume de acordo com um agendamento e agende um horário.

Observação:

O histórico coletado no horário agendado aqui será usado principalmente para a finalidade especificada e como fonte das tendências de uso na seção Uso do Volume. Quando essa opção for habilitada, você verá um gráfico de linhas exibindo as tendências e alterações de uso ao longo do tempo; caso não seja habilitada, um gráfico de barras indicando a porcentagem de uso atual será exibido em vez disso.

Para analisar uma tarefa de relatório:

Clique duas vezes na tarefa de relatório desejada ou apenas clique em Analisar essa tarefa de relatório (ícone > ) próximo à tarefa de relatório desejada para abrir sua página de análise. Consulte Visualizar Uso e Relatórios para obter mais informações.
