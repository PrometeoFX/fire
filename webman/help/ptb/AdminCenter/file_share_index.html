Arquivos de indexação

A indexação do conteúdo de uma pasta compartilhada pode ajudá-lo a encontrar arquivos ou pastas mais rapidamente durante a pesquisa de arquivos.

Para indexar arquivos em uma pasta compartilhada:

  1. Selecione a pasta compartilhada que quer indexar.
  2. Clique em Editar e em seguida clique na guia Indexação de arquivos.
  3. Marque Habilitar indexação de arquivos.
  4. Clique em OK.

Reindexar Arquivos

A reindexação de arquivos não é necessária sob circunstâncias normais. Entretanto, se os arquivos em uma pasta correspondem aos critérios de pesquisa mas de alguma forma parecem estar ausentes durante a pesquisa de arquivos, você pode executar a reindexação para reconstruir a estrutura de indexação dos arquivos.

Para reindexar arquivos em uma pasta compartilhada:

  1. Selecione a pasta compartilhada que quer reindexar.
  2. Clique em Editar e em seguida clique na guia Indexação de arquivos.
  3. Clique em Reindexar.
  4. Quando o processo estiver concluído, o sistema mostrará quantos arquivos e sub pastas foram indexados.

Observação: A indexação de arquivos em pastas remotas ou unidades USB externas não é suportada pelo File Station.
