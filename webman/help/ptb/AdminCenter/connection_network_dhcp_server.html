DHCP Server

O DiskStation pode assumir a função de servidor DHCP e atribuir endereços IP dinâmicos para clientes DHCP dentro de sua rede de área local.

Para habilitar o servidor DHCP:

  1. Vá até Painel de Controle > Rede > Interface de Rede.
  2. Selecione uma interface de rede (como LAN) e clique em Editar.
  3. Vá para a guia DHCP Server e marque Habilitar DHCP Server.
  4. Digite um valor no campo Tempo de concessão do endereço para definir o tempo de concessão do DHCP (em minutos) para os endereços IP atribuídos aos clientes DHCP.
  5. Digite o endereço DNS de seus clientes DHCP nos campos DNS primário (obrigatório) e DNS secundário (opcional).
  6. Defina o nome do domínio do servidor DHCP no campo Nome de domínio (opcional).
  7. Clique em OK.

Para adicionar uma sub rede:

  1. Clique em Adicionar e especifique as informações nas seguintes colunas:
    • Habilitar: Habilita ou desabilita a sub rede.
    • Iniciar: O primeiro endereço IP da sub rede.
    • Fim: O último endereço IP da sub rede.
    • Máscara de rede: A máscara de sub rede.
    • Gateway: O gateway da sub rede.
  2. Clique em OK.

Observação:

  • Só é possível adicionar até 32 subredes.

Clientes DHCP

Para exibir uma liste de clientes DHCP e suas configurações de rede, (como endereço MAC, endereço IP, nome de host e o tempo restante antes que a concessão do DHCP expire), vá até a aba Clientes DHCP.

Reserva de DHCP

Se você quiser que um cliente receba sempre o mesmo endereço IP durante a renovação da concessão do DHCP, você pode adicionar o cliente à lista de reserva do endereço.

Para adicionar um cliente à lista de reserva de endereço:

  • Clique em Adicionar.
  • Especifique o endereço MAC do cliente, o endereço IP que você deseja atribuir ao cliente e o nome de host do cliente.
  • Clique em OK para salvar suas configurações.
