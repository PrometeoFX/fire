TFTP/PXE

As opções em Painel de Controle > Serviços de Arquivo > TFTP / PXE permitem que você configure as configurações de TFTP e PXE.

TFTP

O protocolo de transferência de arquivo trivial (TFTP) é um protocolo de transferência de arquivo simples, usado geralmente para transferir arquivos de configuração ou reinicialização em situações onde a autenticação não é necessária. A habilitação do serviço TFTP permite que o DiskStation transfira carregadores de inicialização para os dispositivos clientes.

Para modificar as configurações de TFTP:

  1. Marque a caixa de seleção Habilitar serviço TFTP.
  2. No campo pasta TFTP raiz, especifique qual pasta do DiskStation pode ser acessada pelos clientes TFTP.
  3. Você pode clicar no botão Configurações avançadas para visualizar e modificar as seguintes configurações:
    • Habilitar registro de transferência de arquivos por TFTP: Essa opção permite que você mantenha o registro dos dados transferidos via TFTP.
    • Permissão de cliente TFTP: Selecione as permissões de leitura e gravação para clientes TFTP. Quando Somente leitura estiver selecionado, os clientes TFTP poderão visualizar apenas conteúdos da pasta raiz. Quando Gravável estiver selecionado, os clientes TFTP poderão modificar conteúdos da pasta raiz.
    • Clientes permitidos: Escolha quais clientes têm permissão para se conectar via TFTP. Você pode permitir que todos os clientes se conectem selecionando Todas as conexões ou limitar quais clientes podem se conectar especificando uma faixa de endereços IP.
  4. Após modificar as configurações, clique em Aplicar para salvar.

Configure o serviço DHCP para PXE com seu DiskStation

Além do TFTP, o DiskStation também pode fornecer serviço DHCP para implementar uma configuração PXE tudo em um. O ambiente de execução pré-inicialização (PXE) é um ambiente de rede que permite que os dispositivos clientes sejam inicializados usando uma interface de rede em vez de um dispositivo de armazenagem de dados (por exemplo, discos rígidos). Ao implementar o PXE, um carregador de inicialização pode ser armazenado no DiskStation e os dispositivos do cliente podem ser inicializados através da rede usando-se o carregador de inicialização.

Para fornecer serviços DHCP com o DiskStation, certifique-se de ter o pacote do DHCP Server da Synology instalado.

Para configurar o serviço DHCP para o PXE:

  1. Marque a caixa Configuração do serviço DHCP nesse servidor para PXE. O sistema pedirá a instalação do pacote DHCP Server se ele ainda não estiver instalado em seu DiskStation.
  2. Selecione um arquivo carregador de inicialização da pasta raiz do TFTP e especifique as outras configurações relacionadas ao DHCP.
  3. Clique em Aplicar para salvar as configurações.
