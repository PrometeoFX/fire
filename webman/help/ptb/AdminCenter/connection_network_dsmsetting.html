Configurações do DSM

Em Painel de Controle > Rede > Configurações do DSM, você pode alterar o número da porta que os clientes usam para acessar o DSM. Quando o HTTPS está ativado, as conexões serão criptografadas com SSL/TLS.

Para mudar o número de porta padrão:

  1. Digite o número da porta no campo HTTP ou HTTPS.
  2. Clique em Aplicar.

Para ativar as conexões HTTPS:

  1. Marque Habilitar conexão HTTPS.
  2. Se você gostaria de redirecionar os clientes para a porta HTTPS (5001) ao tentar acessar o DSM através da porta HTTP (5000), marque a caixa Redirecionar automaticamente conexões HTTP para HTTPS.
  3. Clique em Aplicar.

Portas usadas para a conexão HTTPS:

  • DSM: O número da porta padrão é 5001, portanto é possível acessar o DSM de forma segura digitando o seguinte endereço: https://server-hostname:5001/
  • Web Station e Photo Station: O número da porta padrão é 443, portanto é possível acessar o Web Station e o Photo Station de forma segura digitando o seguinte endereço: https://server-hostname/

Para tornar possível a descoberta do DiskStation pela descoberta de rede do Windows:

  1. Marque Ativar a descoberta de redes do Windows.
  2. Clique em Aplicar.
