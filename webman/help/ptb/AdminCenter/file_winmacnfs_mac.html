Serviço de arquivos Mac

Em Painel de Controle > Serviços de arquivos > Win/Mac/NFS, você pode ativar o serviço de arquivos Mac para tornar DiskStation um servidor de arquivos Mac (servidor AFP).

Para ativar o serviço de arquivos Mac:

  1. Marque Ativar serviço de arquivos Mac para tornar o DiskStation um servidor de arquivos Mac (servidor AFP).
  2. Configure a pasta de backup Time Machine se necessário. (Consulte a seção abaixo para mais informações.)
  3. Clique em Aplicar.

Sobre a pasta de backup do Time Machine:

Para fazer o backup de dados do Time Machine para o DiskStation, selecione uma pasta compartilhada no menu suspenso.

Observação: As pastas compartilhadas que estão sendo usadas por outros serviços não estarão disponíveis para o Time Machine.

Configurações avançadas

Aplique as permissões UNIX padrão:

Ativar essa opção aplica as permissões UNIX padrão ao carregar ou criar arquivos e pastas. As permissões aplicadas são as mesmas das permissões aplicadas pelo comando UNIX umask. Quando essa opção está habilitada, as permissões UNIX são 644 para arquivos e 755 para pastas. Quando essa opção está desabilitada, as permissões UNIX são 666 para arquivos e 777 para pastas. O valor umask padrão é 022.

Observação: Ativar essa opção pode causar problemas de permissão inconsistente entre os diferentes protocolos. Para evitar inconsistências, sugerimos deixar essa opção desabilitada.

Liberar recursos imediatamente após a desconexão:

A habilitação dessa opção libera recursos imediatamente quando a conexão do servidor AFP é desconectada inesperadamente. Se você receber mensagens de erro "Volume em uso" ao usar o Time Machine, habilite essa opção.

Para ativar o Bonjour Printer Broadcast:

Marque Ativar Bonjour Printer Broadcast para que os clientes Mac pesquisem e localizem a impressora conectada ao DiskStation.
