写真を編集

File Station では、Web ベースの外部フォト エディタである Pixlr Editor や Pixlr Express を使用して写真を編集することができます。

Web ベースのフォト エディタでフォトを編集するには:

  1. 写真が保存されているフォルダを検索します。
  2. 写真を右クリックして、[Pixlr Editor で編集] または [Pixlr Express で編集] を選択します。
      注：
    • Pixlr Editor は以下のフォーマットに対応しています：jpg、gif、png、psd、pxd (Pixlr レイヤー画像)
    • Pixlr Express は以下のフォーマットに対応しています：jpg、gif、png
  3. 選択したフォト エディタが新しいブラウザウィンドウに開きます。すぐに写真が表示され、提供されているツールで編集することができます。
  4. 写真の編集を終了したら、次のように写真を保存します。
    • Pixlr Express の場合:[完了] をクリックして、DiskStation の元の場所にその元の形式で写真を保存します。
    • Pixlr Editor の場合:[ファイル] > [保存] の順に選択し、左のパネルから保存する場所を選択し (DiskStation、コンピュータ、またはその他ソーシャルネットワークの写真の元の場所)、設定に応じて新しい名前や形式を決定して [OK] をクリックします。

注：Web ベースのフォト エディタでフォトを編集するには、DiskStation が正常に外部 IP アドレスを取得していることを確認します ([メイン メニュー] > [コントロール パネル] > [外部アクセス] > [DDNS] の順に選択するか、[メイン メニュー] > [コントロール パネル] > [ネットワーク] > [ネットワーク インターフェイス] > [PPPoE] の順に選択します。ネットワーク設定により異なります)。DiskStation がルーターの後ろに配置されている場合は、DSM のポート転送ルールを [メイン メニュー] > [コントロール パネル] > [外部アクセス] > [ルーターの設定] の順に選択して設定する必要があります。写真エディタを開くことができない場合は、[メイン メニュー] > [コントロール パネル] > [外部アクセス] > [詳細設定] の順に選択して DSM のルーター ポートを指定する必要があります。
