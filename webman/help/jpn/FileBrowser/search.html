ファイルまたはフォルダの検索

File Station のフィルタを使用すると、指定した条件に合ったファイルかフォルダだけを検索し、表示します。現在のフォルダ内のファイルやフォルダをフィルタにかけることができます。あるいは、高度な検索を行うと、すべてのサブフォルダもカバーしてより正確な結果を得ることができます。

現在のフォルダのファイルやフォルダを検索する：

  1. File Station を使用して検索するコンテンツを含んでいるフォルダに移動します。
  2. 右上隅の [フィルタ] 欄にキーワードを入力すると、ウィンドウのメインエリアにフィルタ化された結果が表示されます。

詳細検索を実施する：

  1. File Station を使用して検索するコンテンツを含んでいるフォルダに移動します。
  2. 右上にある［フィルタ］欄の隣の拡大鏡をクリックすると、詳細検索メニューが開きます。
  3. 以下の検索条件に情報を入力するか、ドロップダウンメニューから選択してください。
    • キーワード：キーワードを入力して一致する名前のファイルやフォルダを検索します。

      注：

      • キーワードにワイルドカード [*]（ゼロかその他の文字を意味します）や [?]（1個のランダムな文字を意味します）を使用することができます。例えば、[a*n] は [action]、[attention]、または [an] などを代表できます。
      • 複数のキーワードを分ける場合は、カンマ（,）をお使いください。例えば、[a,tion] は、[action]、[a function]、[function added]、またはnationを表すことができます。
      • キーワードの先頭と最後に二重引用符 ["] を追加して完全一致の検索を行うことができます。例えば、["a typ"] は、[a type] を表すことができますが、[typical] を表すことはできません。
    • 位置：高度な検索の対象範囲と深さをドロップダウン メニューから選択します。その他の共有フォルダを選択すると、検索範囲をカスタマイズすることもできます。
    • ファイルタイプ：特定のファイルタイプに絞り込む場合は［拡張子］を選択します。フォルダだけを検索する場合は、［すべてのフォルダ］を選択します。
    • 日付：[変更日]、[作成日]、または [最後にアクセスした日時] を選択し、カレンダーから検索する日付を選択します。
    • サイズ (MB)：[等しい]、[より大きい]、および [より小さい] から選択して、サイズを入力して一致するサイズのファイルを検索します（同等を選択すると、指定されたサイズとほぼ同じサイズのファイルが検索結果に含まれます）。
    • 所有者/グループ：[所有者] または [グループ] を選択して、ドロップダウンメニューから選択して選択した所有者やグループに属するファイルやフォルダを検索します。

      注：

      • 高度な検索を行うには、位置と最低1個は別の検索条件を指定する必要があります。
      • ISO フォルダまたはリモート フォルダがインデックス付きの共有フォルダにマウントされている場合は、高度な検索を使ってこの中にあるファイルを見ることはできません。
    • [検索] をクリックして検索を開始するか、[リセット] をクリックして再度検索条件を指定します。検索が完了すると、一致したファイルが File Station のメインエリアに表示されます。
