ドキュメント、画像、動画を表示

File Station では、文書、ビデオ、写真を簡単に表示できます。

アイテムを表示する方法を変更するには：

メイン エリアにアイテムを表示する方法を、リスト ビューとアイコン ビューから選択できます。ファイルをリスト表示するには、右上にある [リスト ビュー] ボタンをクリックします。ファイルにアイコンとテキストを付けて表示するには、[アイコン ビュー] ボタンをクリックします。

次の条件を満たしている場合は、アイコン ビューで写真のサムネイルが表示されます。

  • 写真が [コントロール パネル] > [メディア ライブラリ] > [インデックス付きフォルダ] を選択してリストしたインデックス付きフォルダにあり、DiskStation にすでに Photo Station またはメディア サーバーがインストールされている場合。
  • デジタル カメラが作成した EXIF エントリにサムネイル データが保管されている場合。

Google docs で文書を表示する。

開きたい文書を右クリックして [Google docs で表示] を選択します。

サポートされる形式：

Google docs ビューアは以下のフォーマットに対応しています。doc、docx、odt、ppt、pps、xlsx、pdf、tif

注：

Google docs で文書を見るには、インターネット上で DiskStation デバイスが完全にアクセス可能かどうかを確認してください。DiskStation がルーターの後ろに配置されている場合は、DSM のポート転送ルールを [コントロール パネル] > [外部アクセス] > [ルーターの設定] の順に選択して設定する必要があります。DSM のデフォルトのポート番号は 5000 (HTTP) と 5001 (HTTPS) です。Google docs でドキュメントを表示できない場合は、[コントロール パネル] > [外部アクセス] > [詳細設定] の順に選択して DSM のルーター ポートを指定する必要があります。

Office Online で文書を表示する：

開きたい文書を右クリックして [Office Online で見る] を選択します。

Office Viewer を活用して DiskStation デバイスにホストされている Microsoft Office ドキュメントを表示すると、コンピュータに Office がインストールされていなくても、Office 機能を使ってウェブ ブラウザでこれらのドキュメントにアクセスすることができます。

サポートされる形式：

Office ビューアは以下のフォーマットに対応しています。

  • Word：docx、docm、dotm、dotx
  • Excel：xlsx、xlsb、xls、xlsm
  • PowerPoint：pptx、ppsx、ppt、pps、pptm、potm、ppam、potx、ppsm

注：

  • Office Online でドキュメントを開くには、QuickConnect を有効にしてください。まだこれが有効になっていない場合は、administrators グループに属するユーザーとして DSM にログインし、[コントロール パネル] > [QuickConnect] を選択します。
  • Word および PowerPoint ドキュメントは 10 MB 以下でなければなりません。Excel は 5 MB でなければなりません。
  • ドキュメントのファイル名には、文字と数字しか使用できません。

写真をプレビューする

  1. 写真を右クリックし、[プレビュー] を選択して写真ビューアウィンドウを開きます。
  2. マウスを写真ビューアの下の上に置いてサムネイルのスライドバーを表示すると、操作がしやすくなります。

サポートされる形式：

写真ビューアは以下のフォーマットに対応しています。

  • 「photo」共有フォルダ：jpg、jpeg、jpe、bmp、gif、png、tiff、tif、arw、srf、sr2、dcr、k25、kdc、cr2、crw、nef、mrw、ptx、pef、raf、3fr、erf、mos、orf、rw2、raw
  • その他のフォルダ：jpeg、png

サウンド ファイルの再生（プレビュー）：

  1. サウンド ファイルをダブルクリックすると、それらがオーディオ プレーヤーで開きます。代わりに、サウンド ファイルを右クリックして [再生] を選択することで、サウンド ファイルを再生することもできます。*

サポートされる形式：

オーディオプレーヤーは以下のフォーマットに対応しています。

  • mp3、m4a、m4b、aac、ogg、wav、flac、ape、aiff、aif、wma

ビデオを再生する：

  1. ビデオを右クリックして [再生] を選択し、ビデオプレーヤーウィンドウを開きます。
  2. すぐにビデオの再生が開始します。再生されない場合は、ローカルコンピュータに VLC Player (Windows) がインストールされているか確認してください。

    3. 注：

    • VLC プレーヤーのインストール時に Mozilla プラグインと ActiveX プラグインが選択されているか確認してください。
    • VLC は以下のブラウザで使用できます。
      Windows：Internet Explorer、Firefox、Google Chrome
    • VLC がコンピュータにインストールされていて、使用できるウェブブラウザで DSM に接続する場合、ビデオプレーヤーは次のフォーマットをサポートします。avi、dat、mkv、vob、asf、wmv、mov、qt、divx、flv、swf、rm、rmvb、mpg、mpeg、mpeg1、m1v、mpeg2、m2v、mp2、dvr-ms、m2ts、mts、m2t、ts、tp、trp、mpeg4、m4v、mp4、3g2、3gp、3gp2、webm、mpe、xvid
    • VLC がローカルコンピュータにインストールされていないと、ビデオプレーヤーは HTML5 がサポートするビデオを再生します。各ウェブブラウザでサポートされている HTML5 ビデオについては、ここ から参照してください。
    • HTTPS プロトコルで DSM に接続する場合は、信頼できる証明書（[コントロール パネル] > [セキュリティ] > [証明書]）をインポートして File Station でビデオを再生できるようにする必要があります。
    • VLC は HTTPS ストリーミングに対応していません。
  3. ビデオを同じフォルダに同じ名前のサブタイトルファイルがある場合、またはビデオに組み込みサブタイトルが含まれている場合は、ビデオ再生時にサブタイトルを見ることができます。別のサブタイトルを使用する場合は、ドロップダウンメニューから選択します。

____
* 特定モデルのみ。
