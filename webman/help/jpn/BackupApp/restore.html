復元

[バックアップと複製] > [復元] を選択して、バックアップ データを復元します。

バックアップ タスクを復元する:

  1. バックアップ タスクを選択し、[復元] をクリックします。マルチバージョンのバックアップ タスクの場合は、[復元] > [タスク全体] を選択します。
  2. システム設定、バックアップ データの別のバージョン、その他を選択するよう要求される場合があります。要求内容は、データのバックアップなのか、LUN バックアップなのか、シングル バージョンのバックアップ先なのか（共有フォルダなど）、あるいは複数バージョンのバックアップ先なのか（ボリュームなど）によって異なります。
  3. ウィザードに従って設定を終了します。
  1. バックアップ タスクを選択し、[復元] をクリックします。
  2. システム設定、バックアップ データの別のバージョン、その他を選択するよう要求される場合があります。バックアップ タスクの種類によります。
  3. ウィザードに従って設定を終了します。

注意：

  • 特定アプリケーションの復元に対応しています。特定のパッケージが復元、および制限に対応しているかどうかを確認するには、各アプリケーションのヘルプを参照してください。
  • パッケージを復元するときパッケージがインストールされていなければ、システムは自動的にそのアプリケーションの同じバージョンか、それよりも新しいバージョンをインストールします。アプリケーションがインストールされていても、バックアップしたアプリケーションよりもバージョンが古い場合は、システムがバックアップしたバージョンと同じバージョンに自動的にアップグレードします。
  • DSM 5.1 以上でバックアップされたアプリケーションは、DSM 5.0 以下では復元できません。

注意：

  • 復元が完了すると、バックアップ LUN は選択したボリュームにファイルベースの LUN に復元されます。
  • 特定アプリケーションの復元に対応しています。特定のパッケージが復元、および制限に対応しているかどうかを確認するには、各アプリケーションのヘルプを参照してください。
  • パッケージを復元するときパッケージがインストールされていなければ、システムは自動的にそのアプリケーションの同じバージョンか、それよりも新しいバージョンをインストールします。アプリケーションがインストールされていても、バックアップしたアプリケーションよりもバージョンが古い場合は、システムがバックアップしたバージョンと同じバージョンに自動的にアップグレードします。
  • DSM 5.1 以上でバックアップされたアプリケーションは、DSM 5.0 以下では復元できません。

マルチバージョン バックアップタスクで1個のファイルまたはフォルダを復元する：

  1. ファイルバージョン ブラウザ を起動します。これは、以下のいずれかの方法で実行できます。
    • [復元] をクリックし、ドロップダウンメニューから [各ファイル/フォルダ] を選択します。
    • バックアップ先を選択し、選択したバックアップ先を拡張してください。[バックアップ バージョン情報] アイコンをクリックします。開いた画面で復元したいバックアップ バージョンを選択し、[ファイルバージョン ブラウザ] アイコンをクリックします。
  2. 画面の下にあるタイムラインやカレンダー アイコンを使うと、ファイルやフォルダのバージョンを参照することができます。
  3. 管理するバージョンを選択し、[復元] をクリックします。
  4. 代わりに、ファイルやフォルダを別の場所にコピーするか、[コピー先...] ボタンか [ダウンロード] ボタンを使ってダウンロードすることもできます。
  5. また、ファイルやフォルダを右クリックして、ドロップダウン メニューからオプションを選択することができます。

特定のバージョンを検索する：

  1. 右上にあるフィルタバーの矢印をクリックします。
  2. ドロップ ダウン メニューでファイルを絞り込むことができます。
    • キーワード： ファイルやフォルダの名前を入力します。キーワードにワイルドカード [*] （1文字以上を示します）と [?] を使用できます（1個のランダムな文字を示します）。たとえば、[a*n] は [action]、[attention]、[an] などを代表します。
    • ファイル形式: ドロップダウン メニューから どれでも、任意のフォルダ、任意のファイル の中から選択し、検索を絞り込んでいきます。
    • サイズ：どれでもを選択するか、または同等、以上、以下を選択して、値を入力します。
    • 期間範囲： 今日、昨日、先週、先月を選択するか、カスタマイズを選択して、開始日と終了日を入力して特定の範囲内でのバージョンを検索します。
  3. フィルタを適用するには、[フィルタ] をクリックします。
  4. 絞り込んだ検索をリセットするには、[リセット] ボタンをクリックします。

注意：

  • ファイルバージョン ブラウザ機能は、マルチバージョン バックアップタスクにしか使用できません。
  • DSM 5.1 以前のバージョンでは、ファイルバージョン ブラウザの一部しか提供されていません。
  • システム構成とアプリケーションは、[復元] > [タスク全体] でしか復元できません。
  • [コピー先...] および [ダウンロード] アクションにはメタデータは含まれません。

別のバックアップ先からバックアップ タスクを復元する：

  1. [復元元...] をクリックします。
  2. バックアップ タスクの場所を次の中から選択し、[次へ] をクリックします。
    • ローカル復元：ローカル共有フォルダまたは外部ディスクからデータを復元させます。
    • ネットワークの復元:別の DiskStation または再同期対応のサーバーからデータを復元します。
    • パブリッククラウド サービスの復元:パブリッククラウド サーバーからデータを復元します。
  3. 要求されたら、バックアップ タスクがあるソースを選択し、データの復元先を選択します。
  4. ウィザードに従って手続きを完了してください。

注意：

  • 埋め込み DataStation は、ext3/ext4 外部デバイスに共有フォルダしか復元しません。
  • 埋め込み DataStation は、ext3/ext4 外部デバイスがないアプリケーションの復元には対応していません。
