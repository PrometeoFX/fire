復元

このページでは、システム管理者が復元を行います。ユーザーが共有フォルダのスナップショットにアクセスできるようにするには、各共有フォルダに対して [スナップショットを検出できるようにする] を有効にすると、ユーザーは File Station や Windows File Explorer からスナップショットにアクセスできるようになります。

共有フォルダ

共有フォルダの各ファイル/フォルダを復元する：

  1. 復元したい共有フォルダを選択します。
  2. [復元] をクリックします。
  3. 復元したいスナップショットのバージョンを選択します。
  4. [参照] をクリックし、復元したいファイル/フォルダを復元先にコピーします。

共有フォルダを復元するには：

  1. 復元したい共有フォルダを選択します。
  2. [復元] をクリックします。
  3. 復元したいスナップショットのバージョンを選択します。
  4. [操作] をクリックし、ドロップダウン リストから [その場で復元] を選択します。
  5. 確認メッセージが表示されます。[はい] をクリックして操作を確定します。

新しい共有フォルダにスナップショットをクローンする：

  1. クローンしたい共有フォルダを選択します。
  2. [復元] をクリックします。
  3. 新しい共有フォルダにクローンしたいスナップショットのバージョンを選択します。
  4. [操作] をクリックし、ドロップダウン リストから [新しい名前でクローン] を選択します。
  5. [コントロール パネル] > [共有フォルダ] > [作成] を選択すると、[クローン] ページが開きます。クローンした共有フォルダに名前を付けて、[OK] をクリックします。

注：

  • 共有フォルダ「homes」は復元できません。
  • クローンした共有フォルダを暗号化することはできません。
  • ファイルを転送中にスナップショットを撮影すると、データの整合性が失われます。
  • 暗号化した共有ファイルについての制約：
    • マウントした暗号化済みの共有フォルダでは、どのタイプの復元も実行できません。
    • 暗号化した共有フォルダでは、[新しい名前でクローン] は使用できません。
    • アンマウントした暗号化済みの共有フォルダは、その他で復元 を使って前に撮影したスナップショットから復元することはできません。

iSCSI LUN

iSCSI LUN を修復する：

  1. 復元したい iSCSI LUN を選択します。
  2. [復元] をクリックします。
  3. 復元したいスナップショットのバージョンを選択します。
  4. [操作] をクリックし、ドロップダウン リストから [その場で復元] を選択します。
  5. 確認メッセージが表示されます。[はい] をクリックして操作を確定します。

新しい iSCSI LUN にスナップショットをクローンする：

  1. クローンしたい iSCSI LUN を選択します。
  2. [復元] をクリックします。
  3. 新しい iSCSI LUN にクローンしたいスナップショットのバージョンを選択します。
  4. [操作] をクリックし、ドロップダウン リストから [新しい名前でクローン] を選択します。
  5. [ストレージ マネージャ] > [iSCSI LUN] を選択すると、[クローン] ページが開きます。クローンした iSCSI LUN に名前を付けて、[OK] をクリックします。

注：

  • ファイルを転送中にスナップショットを撮影すると、データの整合性が失われます。
