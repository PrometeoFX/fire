Cloud Station で PC/Mac のデータをバックアップするには

概要

Cloud Station はプラットフォームをまたぐ同期ソフトウェアだけではなく、パソコンのスマートなバックアップ方法としても機能します。Cloud Station をバックアップ ツールとして使用すると、いくつかの利点があります。まず、リアルタイムの複製が RPO（目標復旧地点）を短くするため、大規模な事故が発生しても PC から複製できない部分の期間が短期間分ですみます。また、帯域を大量に使用してきた従来のバックアップ方式とは異なり、帯域も大幅に節約できます。PC クライアントでファイル サイズやファイル形式を設定すると、同一フォルダ内にある、バックアップしないファイルも簡単に振り分けることができます。Cloud Station では、1 個のファイルにつき最大で 32 個のバージョン履歴を保持することができ、サーバー（NAS）側から PC クライアントに復元することができます。

1. eCore Server に Cloud Station を設定する

  1. administrator に属するアカウントで DSM にログインします。
  2. Cloud Station を起動します。
  3. ユーザーホーム機能を有効にしていない場合は、有効にするよう要求されます。[OK] をクリックします。
  4. ユーザー設定ページに転送されます。ユーザーホームを有効にして、[適用] をクリックします。
  5. Cloud Station に戻ると、これを有効にするよう要求されます。[はい] をクリックしてください。
    6. 注：QuickConnect は、ローカル ネットワーク外から eCore Server に簡単に接続できるリレーサービスです。外出先からファイルを同期したい場合は、[コントロール パネル] > [QuickConnect] を選択して、QuickConnect を設定するようお薦めします。
  6. [権限] を選択し、Cloud Station を使用できるユーザーを指定します。[保存] をクリックします。
  7. バックアップの目的で、eCore Server に共有フォルダを設定します。これを実行するには、[コントロール パネル] > [共有フォルダ] を選択した後、[作成] をクリックします。
  8. フォルダに名前を付けて、詳細を入力します。[OK] をクリックします。例えば、フォルダに「バックアップ」という名前を付けます。
  9. [許可] タブで、フォルダにアクセスできるユーザーを指定します。例えば、フォルダに admin 読み/書き権限を与えることができます。[OK] をクリックします。
  10. [Cloud Station] > [設定] > [共有] を選択し、[バックアップ] フォルダを選択した後で [有効] をクリックします。

2. Windows PC に Cloud Station を設定する

Mac または PC に Cloud Station を設定するには次の手順にしたがってください

2.1 PC

  1. [Cloud Station] > [概要] を選択し、[ダウンロード] ボタンをクリックします。
  2. このボタンをクリックすると、Windows PC 用の Cloud Station がダウンロードされます（Cloud Station は Synology ダウンロードセンターからもダウンロードできます）。
  3. コンピュータでインストーラを実行し、画面の指示に従って Cloud Station をインストールしてください
  4. インストール後は、コンピュータで Cloud Station を実行します。[今すぐスタート] をクリックします。
  5. Cloud Station を起動している eCore Server のアドレス（または QuickConnect ID）、ユーザー名、パスワードを入力します。（右側の検索アイコンをクリックすると、Cloud Station が LAN 内にある別の eCore Server を検索します）。[次へ] をクリックします。
    6. 注： eCore Server で [Cloud Station] > [権限] を選択し、入力したアカウントに権限があるか確認してください。
  6. タスクを設定する方法を選択します（バックアップ専用のフォルダを設定しますので、ここでは [詳細設定] を選択します）。[次へ] をクリックします。
  7. eCore Server で先に作成しておいた「バックアップ」フォルダとの同期を維持するフォルダをコンピュータ上で選択します。[次へ] をクリックします。
  8. [完了] をクリックすると、セットアップが完了します。

別のタスクを作成する：

  1. [Cloud Station] > [設定] を選択し、この新しいタスクのための共有フォルダを有効にします。例えば、「写真」です。​ 「写真」に読み書き権があるか確認してください。
  2. PC で Cloud Station に戻り、[作成] をクリックします。
  3. ​新しいタスクを実行する eCore Server を選択します。同じ eCore Server に別のタスクを作成しますので、ここでは [接続した eCore Server] を選択します。[次へ] をクリックします。​
  4. [詳細設定] を選択し、ウィザードに従って設定を続行してください。

3. Cloud Station からデータを取得する

誤ってファイルを削除したり、上書きしたりしてしまっても、Cloud Station で簡単に復元することができます。

4.1 同期したファイルの以前のバージョンを PC にダウンロードする

  1. システムトレイで [Cloud Station] アイコンをクリックし、フォルダ アイコンをクリックします。
  2. 選択したファイルを右クリックして、[Synology Cloud Station] > [以前のバージョンを参照] を選択します。
  3. ダウンロードしたいバージョンを探し、ダウンロード アイコンをクリックします。
  4. ファイル名を入力し、ファイルの保存先を選択します。

4.1 同期したファイルの以前のバージョンを Mac にダウンロードする

  1. システムトレイで [Cloud Station] アイコンをクリックし、フォルダ アイコンをクリックします。
  2. 選択したファイルを右クリックして、[Synology Cloud Station] > [以前のバージョンを参照] を選択します。
  3. ダウンロードしたいバージョンを探し、ダウンロード アイコンをクリックします。
  4. ファイル名を入力し、ファイルの保存先を選択します。

4.3 同期したファイルの前のバージョンを Cloud Station からダウンロードまたは復元する

  1. [Cloud Station] > [バージョン履歴] を選択し、作成した「バックアップ」フォルダを選択します。
  2. 復元したいファイルを選択し、[操作] > [以前のバージョンを参照] をクリックします。
  3. ダウンロードしたいファイルのバージョンを選択し、[ダウンロード] をクリックします。または、選択したバージョンを復元したい場合は、[復元] をクリックすると、現在のバージョンに上書きされます。

4. 重要な注意

  1. 誤ってファイルを削除したり、上書きしたりしたときのために、バージョン履歴は常に維持してください。
  2. [グローバル設定] で、PC でファイルを削除したときに、NAS から取り戻すか、または NAS で削除するかを設定することができます。
