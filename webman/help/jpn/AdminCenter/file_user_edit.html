ユーザーの編集

ユーザーアカウントを検索し（[検索]フィールドにユーザー名を入力することによって）、その名前をダブルクリックするか、[編集] をクリックしてユーザーアカウントのデータを編集します。

    注：
  1. アカウントを無効にすると、アカウントの機能が停止します。
  2. 割り当て量の値は 0 より大きい、または等しいか、またはディスク/ボリューム容量より小さい整数でなければなりません。0 にせっていすると、効率的な割り当て] がグループ制限によって決定されます。
  3. サーバーにボリューム/共有フォルダが1つ以上ある場合は、そのユーザーに各ボリューム/共有フォルダで容量を割り当てることができます。
  4. アカウント情報が作成または変更された後にシステムがユーザーに電子メールで通知できるよう、「通知」 ページで正しい電子メール情報を設定する必要があります。
  5. ユーザーに対して速度制限を設ける場合、そのユーザーにすでにグループ速度制限が設けられている場合は、[結果] にその制限値が表示されます。
  6. ユーザーに割り当てた速度制限は、そのユーザーが属するグループの速度制限には含まれません。
