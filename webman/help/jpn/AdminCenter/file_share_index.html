インデックスファイル

共有フォルダのコンテンツにインデックスを付けると、ファイルやフォルダをさらに速く検索することができます。

共有フォルダのスファイルにインデックスを付ける：

  1. インデックスを付ける共有フォルダを選択します。
  2. [編集] をクリックし、[ファイルインデックス処理] タブをクリックします。
  3. [ファイルインデックス処理を有効にする] を選択します。
  4. [OK] をクリックします。

ファイルの再インデックス

ファイルの再作成は通常の状況では必要ありません。但し、フォルダのファイルが検索条件に一致しているが、検索中に紛失エラーが表示された場合は、ファイルのインデックス構造を再構築するために再インデックスを実行する必要があります。

共有フォルダのファイルを再作成する

  1. インデックスを付ける共有フォルダを選択します。
  2. [編集] をクリックし、[ファイルインデックス処理] タブをクリックします。
  3. ［再インデックス］ をクリックします。
  4. 処理が終了すると、インデックスが付けられたファイル数とサブフォルダ数が表示されます。

注：File Station は、リモート フォルダまたは外部 USB ドライブでのファイル インデックスには対応していません。
