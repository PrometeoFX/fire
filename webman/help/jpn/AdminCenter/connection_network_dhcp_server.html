DHCP Server

DiskStation が DHCP サーバーの役割を果たすと仮定し、ダイナミック IP アドレスを LAN 内の DHCP クライアントに割り当てます。

DHCP サーバーを有効にする：

  1. [コントロール パネル] > [ネットワーク] > [ネットワーク インターフェイス] の順に選択します。
  2. ネットワーク インターフェイス（LAN など）を選択し、[編集] をクリックします。
  3. [DHCP Server] タブを開き、[DHCP Server を有効にする] にチェックマークを付けてください。
  4. [アドレス リース時間] 欄に値を入力して、DHCP クライアントに割り当てられた IP アドレスに DHCP リースタイム（分で）を設定します。
  5. [プライマリ DNS]（必須）および [セカンダリ DNS]（オプション）欄に、DHCP クライアントの DNS アドレスを入力します。
  6. [ドメイン名]（オプション）欄で、DHCP サーバーのドメイン名を設定します。
  7. [OK] をクリックします。

サブネットを追加する：

  1. 追加をクリックして、次のカラムに情報を指定します。
    • 有効：サブネットを有効または無効にします。
    • 開始：サブネットの最初の IP アドレスです。
    • 終了：サブネットの最後の IP アドレスです。
    • Netmask：サブネット マスクです。
    • ゲートウェイ：サブネットのゲートウェイです。
  2. [OK] をクリックします。

注：

  • 最大で 32 個のサブネットを追加できます。

DHCP クライアント

DHCP クライアントのリストと、それらのネットワーク設定（MAC アドレス、IP アドレス、ホスト名、DHCP リース期限が切れるまでの残り時間など）を見るには、[DHCP クライアント] タブを開きます。

DHCP 予約

DHCP リースの更新中にも、クライアントがいつも同じ IP アドレスを使用できるようにしたい場合は、そのクライアントをアドレス予約リストに追加することができます。

アドレス予約リストにクライアントを追加する：

  • [追加] をクリックします。
  • クライアントの MAC アドレス、クライアントに割り当てたい IP アドレス、クライアントのホスト名を指定します。
  • OKをクリックして、設定を保存します。
