DSM 設定

[コントロール パネル] > [ネットワーク] > [DSM 設定] を選択すると、クライアントが DSM にアクセスするために使うポート番号を変更できます。HTTPS を有効にすると、接続は SSL/TLS で暗号化されます。

デフォルトのポート番号を変更する：

  1. HTTP または HTTPS フィールドにポート番号を入力します。
  2. [適用] をクリックします。

HTTPS 接続を有効にする：

  1. [HTTPS 接続を有効にする] を選択します。
  2. HTTP ポート (5000) を介して DSM にアクセスするとき、クライアントを HTTPS ポート (5001) に転送するには、[HTTP 接続を HTTPS に自動リダイレクトする] にチェックマークを付けます。
  3. [適用] をクリックします。

HTTPS 接続で使用するポート：

  • DSM：デフォルトのポート番号は 5001 ですので、次のアドレスを入力すると DSM に安全にアクセスできます。https://server-hostname:5001/
  • Web Station と Photo Station：デフォルトのポート番号は 443 ですので、次のアドレスを入力すると Web Station と Photo Station に安全にアクセスできます。https://server-hostname/

Windows のネットワーク探索方法で DiskStation を検出するには：

  1. [Windows ネットワーク探索を有効にする] にチェックマークを付けます。
  2. [適用] をクリックします。
