Indexer des fichier

L'indexation du contenu d'un dossier partagé peut vous aider à trouver des fichiers ou dossiers plus rapidement pendant la recherche de fichiers.

Pour indexer les fichiers dans un dossier partagé :

  1. Sélectionnez le dossier partagé que vous souhaitez indexer.
  2. Cliquez sur Modifier puis cliquez sur l’onglet Indexation des fichiers.
  3. Cochez Activer l’indexation des fichiers.
  4. Cliquez sur OK.

Ré indexation des fichiers

La réindexation de fichiers n'est pas requise en temps normal. Cependant, si les fichiers d'un dossier correspondent au critère de recherche, mais qu'ils n'apparaissent finalement pas pendant la recherche de fichiers, il vous faudra peut-être effectuer une réindexation pour reconstruire la structure d'index des fichiers.

Pour réindexer des fichiers dans un dossier partagé :

  1. Sélectionnez le dossier partagé que vous souhaitez réindexer.
  2. Cliquez sur Modifier puis cliquez sur l’onglet Indexation des fichiers.
  3. Cliquez sur Réindexer.
  4. Lorsque le processus est terminé, le système affiche le nombre de fichiers et sous-dossiers indexés.

Remarque : L'indexation des fichiers sur les dossiers distants ou les lecteurs USB externes n'est pas pris en charge par File Station.
