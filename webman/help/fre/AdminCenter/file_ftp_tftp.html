TFTP/PXE

Les options dans Panneau de configuration > Services fichier > TFTP/PXE vous permettent de configurer les paramètres TFTP et PXE.

TFTP

Le protocole TFTP (Trivial File Transfer Protocol) est un protocole de transfert de fichiers simple, qui est généralement utilisé pour transférer des configurations ou des fichiers de démarrage, dans les situations où l'authentification n'est pas requise. L'activation du service TFTP permet au DiskStation de transférer des chargeurs de démarrage aux appareils clients.

Pour modifier les paramètres TFTP :

  1. Cochez la case marquée Activer le service TFTP.
  2. Dans le champ Dossier root TFTP, spécifiez quel dossier sur le DiskStation peut être accédé par les clients TFTP.
  3. Vous pouvez cliquer sur le bouton Paramètres avancés pour afficher et modifier les paramètres suivants :
    • Activer le journal des transferts de fichiers TFTP : Cette option vous permet de garder une trace des données transférées via TFTP.
    • Autorisation client TFTP : Sélectionnez les autorisations de lecture et d'écriture pour les clients TFTP. Quand Lecture seule est sélectionnée, les clients TFTP peuvent uniquement afficher le contenu du dossier root. Quand Inscriptible est sélectionné, les clients TFTP peuvent modifier le contenu du dossier root.
    • Clients autorisés : Choisissez quels clients sont autorisés à se connecter via TFTP. Vous pouvez permettre à tous les clients de se connecter en sélectionnant Toutes les connexions ; ou vous pouvez limiter les clients qui peuvent se connecter en spécifiant une plage d'adresses IP.
  4. Après avoir modifié les paramètres, cliquez sur Appliquerpour enregistrer.

Configurer un service DHCP pour PXE avec votre DiskStation

En plus du TFTP, le DiskStation peuvent également fournir le service DHCP pour implémenter une configuration PXE tout-en-un. PXE (Pre-Boot Execution Environment) est un environnement réseau qui permet aux machines clientes de démarrer grâce à une interface réseau au lieu d'un périphérique de stockage de données (par ex. les disques durs). Lors de l'implémentation de PXE, un chargeur de démarrage peut être stocké sur le DiskStation, et les appareils clients peuvent démarrer sur le réseau en utilisant le chargeur de démarrage.

Afin de fournir des services DHCP avec le DiskStation, veuillez vous assurer d'avoir installé le paquet DHCP Server de Synology.

Pour configurer le service DHCP pour PXE :

  1. Cochez la case marquée Configurer le service DHCP sur ce serveur pour PXE. Vous serez invité à installer le paquet DHCP Server s'il n'est pas déjà installé sur votre DiskStation.
  2. Sélectionnez un fichier de chargeur de démarrage depuis le dossier root TFTP et spécifiez les paramètres en rapport avec DHCP.
  3. Cliquez sur Appliquer pour enregistrer les paramètres.
