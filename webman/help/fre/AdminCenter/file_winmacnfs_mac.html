Service de fichiers Mac

Dans Panneau de configuration > Services de fichier > Win/Mac/NFS, vous pouvez activer le service fichier Mac pour que votre DiskStation devienne un serveur fichier Mac (serveur AFP).

Pour activer le service de fichier Mac :

  1. Cochez la case Activer le service de fichiers Mac pour que DiskStation soit un serveur de fichier Mac (serveur AFP).
  2. Paramétrez un dossier retour Time Machine si nécessaire. (Reportez-vous à la section ci-dessous pour plus d’informations.)
  3. Cliquez sur Appliquer.

À propos du dossier de sauvegarde de Time Machine :

Pour sauvegarder des données depuis Time Machine vers DiskStation, sélectionnez une dossier partagé depuis le menu déroulant.

Remarque : Les dossiers partagés qui sont utilisés par d'autres services ne seront pas disponibles pour Time Machine.

Paramètres avancés

Application des permissions UNIX par défaut :

L'activation de cette option applique les permissions UNIX par défaut lors du chargement ou de la création de fichiers et dossiers. Les autorisations appliquées sont les mêmes que les autorisations appliquées par la commande UNIX umask. Lorsque cette option est activée, les permissions UNIX sont de 644 pour les fichiers et de 755 pour les dossiers. Lorsque cette option est désactivée, les permissions UNIX sont de 666 pour les fichiers et de 777 pour les dossiers. La valeur umask par défaut est 022.

Remarque : L'activation de cette option pourrait provoquer des problèmes d'incohérence de permission entre différents protocoles. Afin d'éviter les incohérences, nous suggérons de laisser cette option désactivée.

Libérer les ressources aussitôt après la déconnexion :

L'activation de cette option permet de libérer immédiatement les ressources lorsque la connexion au serveur AFP est brusquement interrompue. Si vous recevez un message d'erreur « Volume utilisé » lors de l'utilisation de Time Machine, désactivez cette option.

Pour activer la diffusion imprimante Bonjour :

Cochez Activer la diffusion imprimante Bonjour pour permettre aux clients Mac de rechercher et localiser l’imprimante connectée au DiskStation.
