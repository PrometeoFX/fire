DHCP Server

Votre DiskStation peut assumer le rôle d'un serveur DHCP et attribuer des adresses IP dynamiques aux clients DHCP au sein de votre réseau local.

Pour activer DHCP Server :

  1. Accédez dans Panneau de configuration > Réseau > Interface réseau.
  2. Sélectionnez une interface réseau (p. ex. réseau local) et cliquez sur Modifier.
  3. Allez dans l’onglet DHCP Server et cochez Activer DHCP Server.
  4. Entrez une valeur dans le champ Durée de bail de l'adresse pour définir la durée du bail DHCP (en minutes) pour les adresses IP affectées aux clients DHCP.
  5. Entrez les adresses DNS pour vos clients DHCP dans les champs DNS principal (requis) et DNS secondaire (optionnel).
  6. Définissez le nom de domaine pour DHCP Server dans le champ Nom de domaine (optionnel).
  7. Cliquez sur OK.

Pour ajouter un sous-réseau :

  1. Cliquez sur Ajouter, puis spécifiez les informations dans les colonnes suivantes :
    • Activer: Activez ou désactivez le sous-réseau.
    • Démarrer: La première adresse IP du sous-réseau.
    • Fin: La dernière adresse IP du sous-réseau.
    • Masque de sous réseau: Le masque de sous réseau.
    • Passerelle: La passerelle du sous-réseau.
  2. Cliquez sur OK.

Remarque :

  • Vous ne pouvez pas ajouter plus de 32 sous-réseaux.

Clients DHCP

Pour voir une liste des clients DHCP et leurs configuration de réseau (p. ex. l'adresse MAC, l'adresse IP, le nom d'hôte, et la quantité de temps restante avant que le bail DHCP n'expire), vous pouvez aller dans l'onglet Clients DHCP.

Réservation DHCP

Si vous voulez qu'un client reçoive toujours la même adresse IP lors du renouvellement du bail DHCP, vous pouvez ajouter le client à la liste de réservation d'adresse.

Pour ajouter un client à la liste de réservation d'adresses :

  • Cliquez sur Ajouter.
  • Spécifiez l'adresse MAC du client, l'adresse IP que vous désirez attribuer au client et le nom d'hôte du client.
  • Cliquez sur OK pour enregistrer vos paramètres.
