Paramètres de DSM

Au Panneau de configuration > Réseau > Paramètres avancés, vous pouvez modifier le nombre de ports que les clients utilisent pour accéder à DSM. Lorsque HTTPS est activé, les connexions seront cryptées avec SSL/TLS.

Pour modifier le numéro de port par défaut :

  1. Saisissez le numéro de port dans le champ HTTP ou HTTPS.
  2. Cliquez sur Appliquer.

Pour activer les connexions HTTPS :

  1. Cochez Activer la connexion HTTPS.
  2. Si vous désirez rediriger le clients vers le port HTTPS (5001) en essayant d'accéder à DSM à travers le port HTTP (5000), cochez la case Rediriger automatiquement les connexions HTTP vers HTTPS
  3. Cliquez sur Appliquer.

Ports utilisés pour la connexion HTTPS :

  • DSM : Le numéro de port par défaut est 5001, de sorte que vous pouvez accéder à DSM de façon sécurisée en saisissant l'adresse suivante : https://server-hostname:5001/
  • Web Station et Photo Station : Le numéro de port par défaut est 443, de sorte que vous pouvez accéder à Web Station et Photo Station de façon sécurisée en saisissant l'adresse suivante : https://server-hostname/

Pour rendre le DiskStation visible par la découverte sur réseau Windows :

  1. Cochez Activer Windows network discovery.
  2. Cliquez sur Appliquer.
