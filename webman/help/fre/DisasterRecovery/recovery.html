Récupération

Cette page permet aux administrateurs système de procéder à une récupération. Pour permettre aux utilisateurs d'accéder aux instantanés des dossiers partagés, activezRendre l'instantané visible pour chaque dossier partagé afin que les utilisateurs puissent accéder aux instantanés via File Station ou Windows File Explorer.

Dossier partagé

Pour restaurer des fichiers/dossiers individuels dans un dossier partagé :

  1. Sélectionnez un dossier partagé que vous souhaitez restaurer.
  2. Cliquez sur Récupérer.
  3. Sélectionnez la version d'un instantané que vous souhaitez restaurer.
  4. Cliquez sur Parcourir et copiez les fichiers/dossier que vous souhaitez restaurer dans la destination.

Pour restaurer un dossier partagé :

  1. Sélectionnez un dossier partagé que vous souhaitez restaurer.
  2. Cliquez sur Récupérer.
  3. Sélectionnez la version d'un instantané que vous souhaitez restaurer.
  4. Cliquez sur Action, et sélectionnez Restaurer en place dans la liste déroulante.
  5. Un message de confirmation s'affiche. Cliquez sur Oui pour confirmer l'action.

Pour cloner un instantané dans un nouveau dossier partagé :

  1. Sélectionnez un dossier partagé que vous souhaitez cloner.
  2. Cliquez sur Récupérer.
  3. Sélectionnez la version d'un instantané que vous souhaitez cloner dans un nouveau dossier partagé.
  4. Cliquez sur Action et sélectionnez Cloner sous un nouveau nom dans la liste déroulante.
  5. Vous êtes redirigé vers la page Cloner de Panneau de configuration > Dossier partagé > Créer. Saisissez le nom du dossier partagé cloné et cliquez sur OK pour terminer.

Remarque :

  • Il est impossible de restaurer le dossier partagé « homes ».
  • Les dossiers partagés clonés ne peuvent pas être chiffrés.
  • La cohérence des données n'est pas garantie si un instantané est pris alors qu'un transfert de fichiers est encore en cours.
  • Limitations relatives aux dossiers partagés chiffrés :
    • Aucun type de récupération n'est disponible pour des dossiers partagés chiffrés montés.
    • Cloner sous un nouveau nom n'est pas disponible pour des dossiers partagés chiffrés.
    • Il est possible de restaurer des dossiers partagés chiffrés non montés à partir d'un instantané pris précédemment à l'aide de l'option Restaurer en place.

iSCSI LUN

Pour restaurer un iSCSI LUN :

  1. Sélectionnez un iSCSI LUN que vous souhaitez restaurer.
  2. Cliquez sur Récupérer.
  3. Sélectionnez la version d'un instantané que vous souhaitez restaurer.
  4. Cliquez sur Action, et sélectionnez Restaurer en place dans la liste déroulante.
  5. Un message de confirmation s'affiche. Cliquez sur Oui pour confirmer l'action.

Pour cloner un instantané dans un nouvel iSCSI LUN :

  1. Sélectionnez l'iSCSI LUN que vous souhaitez cloner.
  2. Cliquez sur Récupérer.
  3. Sélectionnez la version d'un instantané que vous souhaitez cloner dans un nouvel iSCSI LUN.
  4. Cliquez sur Action et sélectionnez Cloner sous un nouveau nom dans la liste déroulante.
  5. Vous êtes redirigé vers la page Cloner de Gestionnaire de stockage > iSCSI LUN. Saisissez le nom de l'iSCSI LUN cloné et cliquez sur OK pour terminer.

Remarque :

  • La cohérence des données n'est pas garantie si un instantané est pris alors qu'un transfert de fichiers est encore en cours.
