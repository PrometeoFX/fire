Modification des photos

Avec File Station, vous pouvez modifier des photos à l'aide des éditeurs de photos externes en ligne, comme Pixlr Editor ou Pixlr Express.

Pour modifier une photo avec un éditeur de photos en ligne :

  1. Recherchez le dossier dans lequel se trouve la photo.
  2. Faites un clic droit sur la photo et sélectionnez Modifier dans Pixlr Editor ou Modifier dans Pixlr Express.
      Remarque :
    • Pixlr Editor prend en charge les formats suivants : jpg, gif, png, psd, pxd (images en couches Pixlr)
    • Pixlr Express prend en charge les formats suivants : jpg, gif, png
  3. L'éditeur de photo sélectionné s'ouvrira dans une nouvelle fenêtre du navigateur. Vous pouvez voir votre photo en un instant puis la modifier à l'aide des outils fournis.
  4. Une fois la modification de votre photo terminée, procédez comme suit pour l'enregistrer.
    • Pour Pixlr Express : Cliquez sur Terminépour enregistrer votre photo dans son format et à son emplacement d'origine sur DiskStation.
    • Pour Pixlr Editor : Sélectionnez Fichier > Sauvegarder, sélectionnez l'emplacement (pouvant être l'emplacement d'origine de la photo sur DiskStation, sur votre ordinateur, ou autres réseaux sociaux) pour l'enregistrer depuis le panneau gauche, attribuez un nouveau nom ou choisissez un nouveau format selon vos préférences puis cliquez sur OK.

Remarque : Afin d'éditer vos photos sur n'importe quel éditeur de photos en ligne, assurez-vous que l'adresse IP externe de DiskStation ait bien été récupérée (dans Menu principal > Panneau de configuration > Accès externe > DDNS ou Menu principal > Panneau de configuration > Réseau > Interface réseau > PPPoE, selon la configuration de votre réseau). Si votre DiskStation dépend d'un routeur, assurez-vous d'avoir bien spécifié la règle du port de transmission de DSM dans Menu principal > Panneau de configuration > Accès externe > Configuration du routeur. Si des problèmes persistent pour ouvrir les éditeurs de photo, il vous faudra peut-être spécifier le port du routeur pour DSM dans Menu principal > Panneau de configuration > Accès externe > Avancé.
