Éditeur de texte

L'éditeur de texte est un éditeur de texte et de code hautement polyvalent, qui vous permet d'accéder et de modifier rapidement des fichiers ou des codes de texte sur votre DiskStation depuis un navigateur Web.

Gestion de fichier

Pour créer de nouveaux fichiers :

Cliquez sur Fichier et choisissez Nouveau. Un nouvel onglet vide apparaitra avec le titre Nouveau Document.txt.

Pour ouvrir les fichiers depuis l'éditeur de texte :

Cliquez simplement sur Fichier et choisissez Ouvrir. Une boîte de dialogue apparaîtra et vous pourrez sélectionner un fichier avec un codage de caractère spécifique qui doit être ouvert dans un onglet. SI le fichier a déjà été ouvert, l'éditeur de texte vous montrera ce fichier dans son état actuel.

Pour ouvrir les fichier depuis File Station :

Dans File Station, cliquez-droit simplement sur le fichier sélectionné et cliquez ensuite sur Ouvrir avec éditeur de texte depuis le menu de contexte. SI le fichier a déjà été ouvert, l'éditeur de texte vous montrera ce fichier dans son état actuel. L'autre façon d'ouvrir un fichier est de le ou les faire glisser depuis File Station vers l'éditeur de texte en utilisant le curseur.


Remarque :

L'éditeur de texte ne peut pas ouvrir un fichier dépassant 10 Mo.

Pour sauvegarder les fichiers :

Cliquez simplement sur Fichier et choisissez Sauvegarder. Si c'est un fichier existant, il doit être effacé avec les contenus actuels. Si vous voulez sauvegarder tous les fichiers que vous avez ouverts simultanément, vous pouvez aller dans Fichier > Sauvegarder tout Si le fichier n'a pas encore de nom (un nouveau fichier), ou vous sélectionnez Fichier > Sauvegarder en tant que, une boîte de dialogue s'ouvrira, vous demandant le nom du fichier. Vous pouvez alors choisir l'emplacement, le nom du fichier, et le codage pour sauvegarder le fichier.

Pour fermer les fichiers :

Cliquez simplement sur Fichier et choisissez Fermer. L'autre façon de fermer un fichier est de cliquer sur le bouton fermer sur chaque onglet. Si vous avez des données non sauvegardées dans un fichier, l'éditeur de texte vous demandera si vous voulez le fermer sans sauvegarder les modifications.

Remarque :

Vous pouvez également sauvegarder les fichiers avec un codage de caractère spécifique pour prendre en charge les langues bidirectionnelles. Vous pouvez également spécifier un codage lors de l'ouverture d'un fichier afin que l'éditeur de texte affiche correctement le fichier.

Modification

  • Opérations standards
    Le contrôle de modification prend en charge Annuler, Rétablir et Sélectionner tout. Ces opérations peuvent être trouvées dans la plupart des éditeurs et effectuées dans des façons très semblables. Vous pouvez effectuer ces opérations en sélectionnant l'élément du menu correspondant dans le menu Modifier. De plus, les options Couper, Copier et Coller par défaut sont disponibles dans le menu de contexte du navigateur.
  • Indentation
    Le texte indenté est une opération commune lors du traitement avec la modification du code source. L'indentation est l'addition de l'espacement supplémentaire au début d'une ligne vers des blocs de texte séparés, ajoutant une hiérarchie. Normalement lors de la modification, une tabulation sera insérée sur l'emplacement actuel en appuyant sur la touche Tab. Cependant, si du texte est sélectionné, appuyer sur la touche Tab ajoutera un espacement supplémentaire au début de chaque ligne qui a du texte sélectionné, vous permettant d'indenter des blocs de texte entiers en même temps. L'indentation sera retiré en appuyant sur Shift et Tab en même temps.
  • Remplacer
    Vous pouvez activer et désactiver le remplacement en appuyant sur la touche Insert. Par ce fait, au lieu d'insérer des caractères au curseur, vous les remplacez. Le curseur affiché modifiera la forme avec selon l'état de remplacement.
  • Modification mode colonne
    Une façon spéciale de modification des fichiers est la modification du mode colonne. Ce qui signifie que le texte peut être être sélectionné dans des colonnes plutôt qu'en lignes. Pour lancer une sélection en mode colonne, relâchez la touche Alt lors de la sélection du texte, en déplaçant le curseur de haut en bas autant de fois que nécessaire. Quand un texte est sélectionné dans ce mode appelé colonne, la frappe changera chaque colonne de caractères sélectionnée dans le caractère typé.

Recherche

  • Recherche normale
    Vous pouvez rechercher du texte comme dans la plupart des programmes utilisant la boîte de dialogue Trouver. Vous pouvez ouvrir cette boîte de dialogue en sélectionnant Recherche > Trouver. Cette boîte de dialogue restera toujours en haut du texte et vous ne pourrez pas garder sur modification le texte si elle est ouverte. Dans le champs de recherche, tapez le texte que vous voulez trouver.
  • Remplacer
    Vous pouvez trouver et remplacer le texte comme dans la plupart des programmes utilisant la boîte de dialogue Remplacer. Vuos pouvez ouvrir cette boîte de dialogue en sélectionnant Recherche > Remplacer. Remplacer tout remplacera toutes les évènements de la demande avec le texte de remplacement.
  • Sensible à la casse
    Il applique les frappes qui ont exactement la même casse que la demande. Si la case n'est pas cochée, la recherche n'est pas sensible à la casse.
  • Mot entier uniquement
    Le résultat sera dans une frappe uniquement si la correspondance consiste en un mot entier, non une partie de ce mot. Par exemple, sir vous recherchez le mot « examen » Mot entier uniquement alors vous trouverez « examen » uniquement au lieu de « exemple ».
  • Expression régulière
    La demande sera traitée comme une expression régulière.

Qu'est-ce qu'une expression régulière?

Une expression régulière est une séquence de caractères qui forme un modèle de recherche pour une technique complète de sauvegarde de temps. Une utilisation très simple d'une expression régulière serait de situer le même mot épelé de deux façons différentes dans un éditeur de texte, par exemple M[ea]rry.

Configurateur de style

Le configurateur de style est principalement utilisé pour contrôler l'apparence du texte dans l'éditeur de texte.
La liste déroulant inférieure droite contient toutes les langues programmables prises en charge qui peuvent être stylées pour vous aider à former un environnement de codage adapté.

Préférences

  • Afficher les caractères invisibles
    Si cochée, l'espace, la tabulation ou les caractères de retour seront affichés.
  • Afficher les guides d'indention
    Si cochée, les guides d'indention seront affichés. C'est utile pour les codes d'alignement dans le langage codé.
  • Surligner la ligne actuelle
    SI cochée, la ligne sur laquelle le curseur clignotant est présent sera mise en valeur avec une couleur de fond spéciale.
  • Surligner le mot sélectionné
    Si coché, tous les mots (non partiel) dans le même fichier de texte correspondant au mot sélectionné seront automatiquement soulignés.
  • Afficher les numéros de ligne
    Si cochée, les numéros de ligne seront affichés.
  • Activer le pliage de code
    Si cochée, certaines lignes de vos codes seront cachées sur la base de certains traits. Ce paramètre fonctionne bien pour les langues structurées telles que C++ ou XML. Quand Masquer les widgets automatiquement est cochée, certaines lignes seront automatiquement masquées. Vous pouvez également choisir de marquer uniquement le guillemet d'ouverture ou les guillemets d'ouverture et de fermeture avec les widgets.
  • Retour de mots
    Spécifiez si votre texte est revenu au bord de la fenêtre ou dans un numéro de colonne spécifique sans l'insertion d'un retour à la ligne.
  • Ajouter les caractères de fin automatiquement
    Si cochée, les crochets de fermeture (caractères) seront automatiquement ajoutés dans vos codes.
  • Insérer des espaces au lieu d'onglets
    Si cochée, les caractères d'espace seront insérés en appuyant sur la touche Tab au lieu d'une tabulation. Largeur d'onglet peut être définie par le nombre de caractères dans Préférences > Police et couleur.
  • Codage par défaut
    Le codage par défaut est utilisé pour sauvegarder les fichiers avec un codage de caractère spécifique pour prendre en charge les langues bidirectionnelles. Vous pouvez également spécifier un codage lors de l'ouverture d'un fichier afin que l'éditeur de texte affiche correctement le fichier.
  • Police et couleur
    Largeur d'onglet, Taille de police et Personnalisation peuvent être personnalisés selon votre préférence personnelle. La Largeur d'onglet est définie par le nombre des caractères d'espace.

Formats de fichier pris en charge

L'éditeur de texte prend en charge les fichiers en texte simple. Les fichiers en texte avec un format de texte enrichie comme *.doc et *.pdf ne seront pas pris en charge.
