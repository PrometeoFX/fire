Monter les dossiers distants

Vous pouvez monter les dossiers distants depuis les serveurs distants (comme les ordinateurs en réseau ou une autre DiskStation) pour naviguer dans les ressources réseau disponibles avec File Station.

Pour monter un dossier distant :

  1. Cliquez sur Outils > Monter le dossier distant.
  2. Entrez « \\Remote_Server\Remote_Folder » dans le champ Dossier pour spécifier le chemin du dossier distant. Par exemple, « \\192.168.0.2\share » ou « \\chad\cheeseburger\pictures ».
  3. Si le privilège d'accès au dossier distant est requis par le serveur distant, entrez vos identifiants utilisateur du serveur distant dans les champs Nom de compte et Mot de passe.
  4. Cliquez sur Parcourir pour sélectionner ou créer un dossier de destination vide sur votre DiskStation pour monter le dossier distant.
  5. Cochez Monter automatiquement au démarrage si vous souhaitez que DiskStation monte ce dossier distant à chaque démarrage ou redémarrage du système.
  6. Cliquez sur Montage. Maintenant, vous pouvez naviguer vers le dossier de destination où le dossier distant est monté, et gérer le contenu des dossiers distants.

Pour démonter un dossier distant :

  1. Sélectionnez le dossier distant que vous souhaitez démonter et cliquez sur Outils > Démonter.
  2. Vous pouvez également tout simplement cliquer sur le bouton droit sur le dossier distant et sélectionner Démonter.

Remarque :

  • Le nombre maximum de dossiers distants simultanés est de 100.
  • Vous êtes seulement autorisé à monter les dossiers distants depuis un serveur distant qui prend en charge le protocole CIFS.
