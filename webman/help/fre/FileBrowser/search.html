Rechercher des fichiers ou dossiers

Le Filtre de File Station offre des options pour rechercher et afficher les fichiers ou dossiers qui répondent à vos critères. Vous pouvez filtrer les fichiers ou dossiers dans le dossier actuel, ou effectuer une recherche avancée qui couvre tous les sous-dossiers pour des résultats plus détaillés.

Pour rechercher des fichiers ou dossiers dans le dossier actuel :

  1. Utilisez File Station pour aller au dossier contenant le contenu que vous souhaitez rechercher.
  2. Dans le champ Filtre dans le coin supérieur droit, saisissez un mot clé et consultez le résultat filtré dans la zone principale de la fenêtre.

Pour effectuer une recherche avancée :

  1. Utilisez File Station pour aller au dossier contenant le contenu que vous souhaitez rechercher.
  2. Cliquez sur l'icône de la loupe à côté du champ Filtre dans le coin supérieur droit pour faire apparaître le panneau de recherche avancée.
  3. Saisissez les informations ou choisissez dans les critères de recherche suivants des menus déroulants :
    • Mot-clé : Saisissez un mot clé pour rechercher les fichiers ou dossiers avec des noms correspondants.

      Remarque :

      • Vous pouvez utiliser des caractères de remplacement [*] (signifie zéro ou plus de caractères) et [?] (signifie un caractère quelconque) dans vos mots-clés. Par exemple, [a*n] peut représenter [action], [attention], ou [an].
      • Vous pouvez utiliser des virgules (,) pour séparer des mots-clés multiples. Par exemple, [a tion] peut signifier [action], [autre fonction], [fonction ajoutée], ou nation.
      • Vous pouvez ajouter des guillemets ["] au début et à la fin du mot-clé pour rechercher les correspondances exactes. Par exemple, ["a typ"] peut représenter [a type], mais ne peut pas représenter [typical].
    • Emplacement : Sélectionnez un élément du menu déroulant pour définir l'étendue et la profondeur de la recherche avancée. Vous pouvez également personnaliser le champ de recherche en sélectionnant d'autres dossiers partagés listés.
    • Type de fichier : Choisissez Extension pour limiter le champ de recherche à des types de fichier spécifiques, ou choisissez Tous les dossiers pour rechercher des dossiers uniquement.
    • Date : Choisissez entre Date de modification, Date de création et Dernier accès, puis choisissez une date dans le calendrier pour rechercher les fichiers avec des dates correspondantes.
    • Dimension (MB) : Choisissez parmi est égale à, est plus grande que, et est inférieure à, puis entrez une taille pour rechercher les fichiers ayant la taille correspondante. (Quand vous choisissez équivaut à, les résultats de recherche incluront les fichiers d'environ la même taille spécifiée.)
    • Propriétaire/groupe : Choisissez Propriétaire ou Groupe, puis choisissez dans le menu déroulant pour rechercher des fichiers ou des dossiers appartenant au propriétaire ou groupe sélectionné.

      Remarque :

      • Pour effectuer une recherche avancée, vous devez spécifier l'Emplacement et au moins un autre critère de recherche.
      • Si un dossier ISO ou un dossier distant est monté dans un dossier partagé indexé, il est impossible de trouver les fichiers qui se trouvent à l'intérieur via une recherche avancée.
    • Cliquez sur Recherche pour commencer la Recherche, ou cliquez sur Réinitialiser et spécifiez à nouveau le critère de recherche. Lorsque la recherche est terminée, les fichiers correspondants seront listés dans la zone principale de File Station.
