Autres

Vous pouvez personnaliser les autres options du compte personnel dans Options > Autres

Pour personnaliser d'autres options personnelles :

  1. Choisissez l’une des options suivantes :
    • Ne pas afficher la boîte de dialogue de confirmation à la fermeture de DSM :Ignorer la boîte de confirmation lorsque vous fermez la fenêtre du navigateur :
    • Reprendre DSM à mon statut de déconnexion précédent lorsque je me connecte : Se souvenir des applications ou des paquets ouverts et les restaurer la prochaine fois que vous vous connectez ou que vous rafraîchissez la fenêtre de navigation.
    • Activer la prévisualisation instantanée sur la barre des tâches :Affichez un aperçu rapide d’une application ouverte quand vous passez sur son icône d’application sur la barre des tâches :
    • Activer les notifications du navigateur sur le bureau : Afficher les notifications depuis votre navigateur web sur le bureau DSM. Les notifications du navigateur sur le bureau sont prises en charge par Google Chrome et Safari seulement. Selon la version de votre navigateur Web, la page des paramètres de notification peut être différente. Veuillez aller à la page des paramètres de votre navigateur pour gérer les paramètres des notifications.
  2. Cliquez sur OK.

Remarque :

  • La prévisualisation instantanée n'est pas prise en charge par Internet Explorer.
