Restaurer

Allez dans Sauvegarde et réplication > restaurer pour restaurer les données.

Pour restaurer une tâche de sauvegarde :

  1. Sélectionnez une tâche de sauvegarde puis cliquez sur Restaurer. Si la tâche est une sauvegarde multi-versions, sélectionnez Restaurer > Tâche entière.
  2. Vous serez peut-être invité à sélectionner une configuration système, différentes versions des données de sauvegarde, ou davantage. Cela dépend s'il s'agit d'une tâche de sauvegarde de données, d'une tâche de sauvegarde de LUN*, d'une destination de sauvegarde à version unique (c.-à-d. un dossier partagé), ou d'une destination de sauvegarde à version multiple (c.-à-d. le volume).
  3. Suivez l'assistant pour terminer la configuration.

Remarque :

  • La restauration pour certaines applications est prise en charge. Pour voir si un paquet spécifique prend en charge la restauration, ainsi que ses limitations, veuillez consulter l'aide de chaque application.
  • Lors de la restauration d'un paquet, si le paquet n'est pas installé, le système installe automatiquement la même version, ou la version la plus récente de l'application. Si l'application est installée, mais qu'il s'agit d'une version plus ancienne que celle sauvegardée, le système l'améliore automatiquement avec la version originale sauvegardée.
  • Les applications sauvegardées dans DSM 5.1 ou une version ultérieure ne peuvent pas être restaurées dans DSM 5.0 ou inférieur.

Pour restaurer un fichier ou un dossier unique pour une tâche de sauvegarde multi-versions :

  1. Lancez Navigateur de versions de fichiers. Pour ce faire, utilisez l'une des méthodes suivantes :
    • Cliquez sur Restaurer et sélectionnez Fichiers/dossiers individuels dans le menu déroulant.
    • Accédez à Destination de la sauvegarde et développez la destination de sauvegarde que vous avez sélectionnée. Cliquez sur l'icône Information version de la sauvegarde. Dans la fenêtre qui s'affiche, sélectionnez la version de sauvegarde que vous souhaitez restaurer et cliquez sur l'icône Navigateur de versions de fichiers.
  2. Vous pouvez parcourir les versions d'un fichier ou d'un dossier de votre choix en utilisant le tableau chronologique ou l'icône du calendrier en bas de l'écran.
  3. Sélectionnez la version que vous souhaitez gérer et cliquez sur Restaurer.
  4. Vous pouvez également copier un fichier ou un dossier à un autre emplacement ou le télécharger à l'aide du bouton Copie vers... ou Télécharger .
  5. Vous pouvez également cliquer avec le bouton droit de la souris sur le fichier ou le dossier et sélectionner ces options à partir du menu déroulant.

Pour rechercher une version spécifique :

  1. Cliquez sur la flèche dans la barre de filtre en haut à droite.
  2. Dans le menu déroulant, vous pouvez filtrer les fichiers par :
    • Mot-clé : saisissez le nom d'un fichier ou d'un dossier Vous pouvez utiliser des caractères de remplacement [*] (représente un ou plusieurs caractères) et [?] (représente un caractère quelconque) dans vos mots-clés. Par exemple, [a*n] peut représenter [action], [attention], ou [an].
    • Type de fichier : dans le menu déroulant, vous pouvez choisir Tout, Tous les dossiers ou Tous les fichiers pour affiner votre recherche.
    • Taille : choisissez Tout ou sélectionnez équivaut à, est plus grand que, est inférieur à , puis saisissez une valeur.
    • Plage de date : choisissez Aujourd'hui, Hier, La semaine dernière, Le mois dernier ou Personnaliser pour saisir une date de début et une date de fin afin de rechercher des versions dans une plage spécifique.
  3. Pour appliquer vos filtres, cliquez sur Filtre.
  4. Pour réinitialiser la recherche filtrée, cliquez sur le bouton Réinitialiser.

Remarque :

  • Seules les tâches de sauvegarde multi-version proposent la fonction Navigateur de versions de fichiers.
  • Navigateur de versions de fichiers possède seulement des fonctions partielles dans DSM 5.1 et les versions antérieures.
  • La configuration système et les applications ne peuvent être restaurées que via Restaurer > Tâche entière.
  • Les métadonnées ne sont pas incluses dans les actions Copie vers... et Télécharger.

Pour restaurer les tâches de sauvegarde depuis une autre destination de sauvegarde :

  1. Cliquez sur Restaurer depuis....
  2. Choisissez l'une des méthodes suivantes pour localiser la tâche de sauvegarde, puis cliquez sur Suivant :
    • Restauration locale : Choisissez cette option pour restaurer les données depuis un dossier partagé local ou un disque externe.
    • Restauration réseau : Choisissez cette option pour restaurer les données depuis un autre DiskStation ou depuis un serveur compatible rsync.
    • Restauration du service cloud public Choisissez cette option pour restaurer les données depuis un serveur de cloud public.
  3. Lorsque vous y êtes invité, sélectionnez la source où la tâche de sauvegarde est conservée, puis sélectionnez la destination pour restaurer les données.
  4. Suivez l'assistant pour terminer la configuration.

Remarque :

  • Embedded DataStation peut uniquement restaurer les dossiers partagés vers des périphériques externes ext3/ext4.
  • Embedded DataStation ne prend pas en charge la restauration des applications sans appareil externe ext3/ext4.
