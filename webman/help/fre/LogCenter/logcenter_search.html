Recherche des journaux

Sur la page Centre des journaux > Recherche de journaux, vous pouvez voir, rechercher et exporter des journaux.

Pour rechercher des journaux :

  1. Choisissez d'afficher les journaux du système local ou les journaux d'autres services en sélectionnant l'onglet Locaux ou Provenant d'autres serveurs.
  2. Si nécessaire, choisissez d'afficher Derniers journaux ou ouvrez une archive de journal existante en sélectionnant Ouvrir archive journal.
  3. Dans le menu déroulant de recherche, vous pouvez spécifier les critères suivants selon la situation :
    • Mot-clé : Saisissez un mot clé pour rechercher les messages de journal contenant le mot clé correspondant.

      Remarque :

      Vous pouvez utiliser des modèles d'expressions régulières comme [*] (ce qui signifie un nombre quelconque de caractères aléatoires) ou [?] (ce qui signifie 1 caractère aléatoire) dans le champ du mot-clé lors de la recherche. Par exemple, [a*n] peut représenter [action], [attention], ou [an].

    • Plage de date : Recherche les journaux selon la période spécifiée.
    • Niveau : Recherche les journaux selon le niveau spécifié.
    • Nom d'hôte : Recherche les journaux selon le nom d'hôte spécifié.
    • Programme : Recherche les journaux selon le nom de programme spécifié.
    • Catégorie : Recherche les journaux selon le nom de catégorie spécifié.
  4. Cliquez sur Recherche pour rechercher les journaux correspondant aux critères spécifiés.

Pour exporter des journaux :

  1. Cliquez sur la flèche à côté du bouton Export.
  2. Choisissez HTML ou CSV pour télécharger les journaux à l'aide du format désiré.

Remarque :

  • Le cryptage des caractères des fichiers journaux exportés CSV est UTF-8. Vous devez ouvrir le fichier avec un programme qui prend en charge le cryptage UTF-8.
