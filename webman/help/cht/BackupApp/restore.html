還原

前往備份 & 同步 > 還原來還原已備份的資料。

若要還原備份任務：

  1. 選擇備份任務，然後按一下還原。 多重版本備份，請點選還原 > 任務還原 。
  2. 您可能會需要選擇系統設定檔、不同版本的備份資料等等。 實際情況將視其為資料備份任務、LUN 備份任務*、單一版本備份目的地 (即：共用資料夾) 或多重版本備份目的地 (即：儲存空間) 而定。
  3. 依照設定精靈的指示完成設定。

  2. 您可能會需要選擇系統設定檔、不同版本的備份資料等等。 實際情況將視其為資料備份任務類型而定。
  3. 依照設定精靈的指示完成設定。

注意：

  • 系統支援備份特定的套件。 關於套件是否支援備份，以及備份的具體內容與限制，請參閱應用程式內各套件說明。
  • 還原套件時若沒有安裝該套件，系統會自動為您安裝同一版本或最新版本的套件。 若已有安裝套件，且比當初備份的版本舊，系統會自動為您升級到當初備份的版本。
  • DSM 5.1 以上備份的應用程式，無法在 DSM 5.0 及 DSM 5.0 以下的系統中還原。

  • 還原完畢後，系統會將所備份的 LUN 還原成檔案類型的 LUN，並存放在您選擇的目的地儲存空間裡。*

  • DSM 5.1 以上備份的應用程式，無法在 DSM 5.0 及 DSM 5.0 以下的系統中還原。

若要還原多重版本備份任務裡的單一檔案或資料夾：

  1. 開啟檔案版本瀏覽器 。 您可以使用以下任一個方式：
    • 點一下還原並從下拉選單選擇檔案/資料夾還原。
    • 前往備份目的地並展開您所選的備份目的地。 點一下備份版本資訊圖示。 在視窗裡選擇您想要還原的被分版本並點一下檔案版本瀏覽器圖示。
  2. 您可以使用月曆圖示或時間線來瀏覽不同版本的檔案或資料夾。
  3. 選擇您要管理的版本，然後按一下還原。
  4. 您也能使用複製到或下載來複製或下載您的檔案。
  5. 您也能在資料夾或檔案上點擊右鍵並從下拉式選單選擇此選項。

若要搜尋特定版本：

  1. 點一下右上角搜尋欄位裡的箭頭。
  2. 在下拉選單您可以用以下的條件來過濾檔案：
    • 關鍵字：輸入任何檔案/資料夾名稱。您可以在關鍵字中使用萬用字元 [*] (表示零個以上的字元) 以及 [?] (表示一個隨機字元)。例如，[a*n] 可以代表 [action]、[attention] 或 [an]。
    • 檔案類型：選擇任何、任何資料夾或任何檔案來限制搜尋範圍。
    • 大小：選擇任何、等於、大於或小於並輸入數值。
    • 日期區間：從今天，昨天，上週，上個月選擇您想要的日期區間，或選擇自訂並輸入開始和結束日期來搜尋特定日期區間裡的版本。
  3. 若要套用您的過濾設定，點一下過濾。
  4. 若您想重新設定您的搜尋條件，點一下重置。

注意：

  • 只有多版本備份任務提供備份檔案瀏覽器功能。
  • 備份目的地為 DSM 5.1 以下版本僅提供備份檔案瀏覽器部分功能。
  • 系統設定檔和應用程式僅能由還原>任務還原還原。
  • 檔案屬性將不會包含在複製到或下載操作。

若要從其它備份目的地還原備份任務：

  1. 按一下還原至...。
  2. 選擇以下任一項目來找到備份任務，並按一下下一步：
    • 本地還原：選擇此選項來從本地共用資料夾或外接硬碟還原資料。
    • 網路還原：選擇此選項來從另一台 DiskStation 或 rsync 相容伺服器還原資料。
    • 公有雲還原：選擇此選項來從公有雲伺服器還原資料。
  3. 當畫面上出現提示時，請選擇備份任務的來源，然後選擇要還原資料的目的地。
  4. 依照精靈的指示來完成操作。

注意：

  • Embedded DataStation 機種僅能還原共用資料夾至 ext3 或 ext4 外接裝置上。
  • 系統不支援 Embedded DataStation 機種在沒有 ext3 或 ext4 外接裝置的情況下還原應用程式。
