文字編輯器

文字編輯器是一款功能齊全的文字、程式碼編輯器，可讓您直接透過網頁瀏覽器快速存取並編輯 DiskStation 上的文字檔或程式碼。

檔案管理

若要建立新檔案：

按一下檔案並選擇開新檔案。 您會看到一個新的空白頁籤，其標題為新文件.txt。

若要透過文字編輯器開啟檔案：

按一下檔案並選擇開啟舊檔即可。 看見對話框後，請選擇要在頁籤中開啟的檔案及其編碼。 若檔案已被開啟，文字編輯器會顯示該檔案目前的狀態。

若要透過 File Station 開啟檔案：

在 File Station 中，對著檔案按右鍵，並從操作功能表中選擇以文字編輯器開啟。 若檔案已被開啟，文字編輯器會顯示該檔案目前的狀態。 另一種開啟檔案的方式為：使用滑鼠游標將一或多個檔案從 File Station 拖拉至文字編輯器中。


注意：

文字編輯器無法開啟超過 10 MB 大小的檔案。

若要儲存檔案：

按一下檔案並選擇儲存即可。 若檔案已存在，舊有的內容會被目前的內容所覆寫。 若您想一次儲存所有開啟的檔案，請前往檔案 > 全部儲存。 若您尚未命名此檔案 (全新的檔案)，或您選擇了檔案 > 另存新檔，則畫面上會出現對話框，請您輸入檔案名稱。 此時您可選擇要儲存檔案的位置、檔案名稱及編碼。

若要關閉檔案：

按一下檔案並選擇關閉即可。 另一種關閉檔案的方式為：按一下各頁籤上的關閉按鈕。 若檔案中有尚未儲存的資料，文字編輯器總是會詢問您是否要關閉檔案而不儲存任何變更。

注意：

您可將檔案儲存為特定編碼來支援雙向語言。 您也可在開啟檔案時就指定編碼，讓檔案能於文字編輯器中正確顯示。

編輯

  • 標準操作
    編輯功能支援簡易的復原、取消復原及全選。 這些操作項目與大多數編輯軟體的運作方式類似。 您可以在編輯選單中選擇相對應的項目來執行以上操作。 此外，在預設的情況下，您可在瀏覽器中點按右鍵，從操作功能表中找到剪下、複製及貼上選項。
  • 縮排
    編輯原始碼時，文字縮排是相當常用的操作。 「縮排」意即在某行文字起始處加入額外的空白，以便區分不同文字區塊並增加層級架構。 在編輯時，按下 Tab 鍵通常會在目前位置插入一個 tab 字元。 然而，當您選擇一段文字並按下 Tab 鍵時，所選的文字中每一行起始處均會插入額外的空白，讓您一次將整個文字區塊縮排。 同時按下 Shift 與 Tab 鍵會移除縮排。
  • 取代模式
    您可以按下 Insert 鍵來啟動或停用取代模式。 也就是說，於游標處打字時將不會插入新字元，而是取代現有字元。 您所看到的游標會依取代模式的狀態而改變形狀。
  • 欄位模式編輯
    欄位模式編輯為一種編輯檔案的特殊方式。 此功能表示您可以以「欄」為單位選取文字，而非以「行」。 若要啟用欄位模式選取，請按住 Alt 鍵並選取文字，再依需求將游標上下移動。 當您在此種欄位模式下選取文字後，所輸入的字元會在選取的所有欄位中進行變更。

搜尋

  • 一般搜尋
    您可以使用「尋找」對話框來搜尋文字，如同大多數軟體一樣。 您可以選擇搜尋 > 尋找來開啟此對話框。 此對話框將總是保持在文字上方，且在其開啟的狀態下您無法繼續編輯文字。 在搜尋欄位中，輸入您想尋找的文字。
  • 取代
    您可以使用「取代」對話框來尋找並取代文字，如同大多數軟體一樣。 您可以選擇搜尋 > 取代來開啟此對話框。 全部取代會將所有符合查詢項目的文字取代為您要替換的文字。
  • 區分大小寫
    搜尋目標的大小寫必須與查詢項目完全相同。 若未勾選此選項，則搜尋時不會區分大小寫。
  • 完全符合
    搜尋目標必須包含整個 (而非部分) 字詞，才會被視為符合項目。 例如，若您查詢「exam」並勾選完全符合，則您將只會尋找到「exam」而非「example」。
  • 正規表示式
    您的查詢會依正規表示式的方式處理。

什麼是正規表示式？

正規表示式是一組字元序列，可構成強大而省時的搜尋模式。 其中一種較簡單的應用方式為：在文字編輯器中找出兩種拼法不同的字，例如 M[ea]rry。

樣式設定器

樣式設定器主要功能為控制文字編輯器中的文字樣式。
右下角的下拉選單包含所有已支援的程式語言，可協助您建構適合的程式碼編寫環境。

偏好設定

  • 顯示隱藏字元
    若勾選此選項，則會顯示空白、tab、新行或換行字元。
  • 顯示縮排輔助線
    若勾選此選項，則會顯示縮排輔助線。 在程式語言中，此功能可協助您將程式碼對齊。
  • 標示所在行數
    若勾選此選項，則插入號所在的行數將以特別的底色標示。
  • 標示選取字詞
    若勾選此選項，則選取的字詞及該份文字檔中所有相符字詞 (非部分) 都會被自動標示出來。
  • 顯示行數
    若勾選此選項，則會顯示行數。
  • 啟動程式碼折疊
    若勾選此選項，則程式碼中特定行數會根據其特性而隱藏。 此功能適合用於結構性的語言，例如 C++ 或 XML。 當您勾選自動隱藏折疊符號，程式碼中特定行數會自動折疊。 您也可以選擇讓折疊符號僅標示開頭括弧，或同時標示開頭與結尾括弧。
  • 自動換行
    指定文字要在視窗邊界或特定欄數換行，不需強制換行 (hard return)。
  • 自動附加對應的結束符號
    若勾選此選項，則程式碼中會自動附加結束括弧 (字元)。
  • 按下 tab 鍵時插入空白字元
    若勾選此選項，則按下 Tab 鍵時會插入空白字元，而非 tab 字元。 您可於偏好設定 > 字體與色彩定義 Tab 字元寬度應為多少字元。
  • 預設編碼
    您可使用預設編碼將檔案儲存為特定字元編碼，藉以支援雙向語言。 您也可在開啟檔案時就指定編碼，讓檔案能於文字編輯器中正確顯示。
  • 字體與色彩
    您可根據個人偏好自訂 Tab 字元寬度、字體大小及佈景主題。 請以空白字元數定義 Tab 字元寬度。

支援的檔案格式

文字編輯器僅支援純文字檔案。 不支援 RTF 文字檔案如 *.doc 及 *.pdf。
