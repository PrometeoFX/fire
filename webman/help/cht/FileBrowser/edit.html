編輯相片

藉由 File Station，您可以使用網頁介面的外部相片編輯器 (例如 Pixlr Editor 或 Pixlr Express) 編輯相片。

若要使用網頁介面的相片編輯器編輯相片：

  1. 瀏覽至相片所在的資料夾。
  2. 以滑鼠右鍵點按相片，並選擇在 Pixlr Editor 編輯或在 Pixlr Express 編輯。
      注意：
    • Pixlr Editor 支援下列格式：jpg, gif, png, psd, pxd (分層 Pixlr 圖片)
    • Pixlr Express 支援下列格式：jpg, gif, png
  3. 所選相片編輯器會在新的瀏覽器視窗中開啟。稍待片刻，您便可以看到您的相片，然後使用畫面上提供的工具進行編輯。
  4. 相片編輯完成之後，執行下列操作來儲存相片。
    • 針對 Pixlr Express：按一下完成來將相片以原本的檔案格式儲存到 DiskStation 的原本位置。
    • 針對 Pixlr Editor：選擇檔案 > 儲存，從左側面板選擇相片的儲存位置 (您可以選擇將相片存到 DiskStation 的原本位置、您的電腦，或是其他社群網路)，並依照您的喜好決定新名稱或檔案格式，然後按一下確定。

注意：若要使用網頁介面的相片編輯器編輯相片，請確定 DiskStation 已成功擷取外部 IP 位址 (位於主選單 > 控制台 > 外部存取 > DDNS 或主選單 > 控制台 > 網路 > 網路介面 > PPPoE，視您的網路配置方式而定)。如果您的 DiskStation 位於路由器後方，請確定您已在主選單 > 控制台 > 外部存取 > 路由器配置指定 DSM 的連接埠轉送規則。如果您在開啟相片編輯器時仍然遇到問題，則可能需要在主選單 > 控制台 > 外部存取 > 進階設定指定 DSM 的路由器連接埠。
