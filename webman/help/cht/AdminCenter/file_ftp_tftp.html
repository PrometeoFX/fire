TFTP

TFTP (Trivial File Transfer Protocol) 是一種簡單的檔案傳輸通訊協定，一般用於傳輸設定或開機檔案且無須驗證的情況。啟動 TFTP 服務可讓 DiskStation 傳送開機載入器設定至用戶端裝置。

若要修改 TFTP 設定：

  1. 勾選啟動 TFTP 服務核取方塊。
  2. 在 TFTP 根目錄欄位，指定哪些 DiskStation 上的資料夾可以讓 TFTP 用戶端存取。
  3. 您可以按一下進階設定按鈕檢視並修改下列設定：
    • 啟動 TFTP 檔案傳輸日誌：此選項可讓您追蹤透過 TFTP 傳輸的資料。
    • TFTP 用戶端存取權限：為 TFTP 用戶端選擇唯讀及讀寫的存取權限。選擇唯讀的情況下，TFTP 用戶端只能檢視根目錄裡的內容。選擇可讀寫的情況下，TFTP 用戶端可以修改根目錄裡的內容。
    • 允許用戶端：選擇哪些用戶端可透過 TFTP 進行連線。您可以選擇所有連線，讓所有用戶端都能連線；也可以指定 IP 位址範圍來限定可以連線的用戶端。
  4. 設定修改完畢之後，按一下套用來儲存變更。

使用 DiskStation 來為 PXE 設定 DHCP 服務

DiskStation 除了 TFTP 之外，還可以提供 DHCP 服務讓您套用合而為一的 PXE 設定。PXE (Pre-Boot Execution Environment，「開機前執行環境」) 為一網路環境，可讓用戶端裝置透過網路介面來開機，無須使用資料儲存裝置 (例如：硬碟)。套用 PXE 時，您可以將開機載入器的檔案存放在 DiskStation，用戶端裝置可以使用載入器透過網路來開機。

若要提供 DHCP 服務一台 DiskStation，請確認您已安裝 Synology DHCP Server 套件。

若要為 PXE 設定 DHCP 服務：

  1. 勾選為 PXE 在此伺服器上設定 DHCP 服務。如果 DiskStation 未安裝 DHCP Server，系統會提示您安裝該套件。
  2. 從 TFTP 根目錄中選擇開機載入器檔案，並指定其他 DHCP 相關設定。
  3. 按一下套用來儲存設定。
