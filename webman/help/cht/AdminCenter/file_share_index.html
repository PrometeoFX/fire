建立檔案索引

為共用資料夾中的內容建立索引，可協助您執行檔案搜尋時更快速地找到檔案或資料夾。

若要為共用資料夾中的檔案建立索引：

  1. 選擇您要建立索引的共用資料夾。
  2. 按一下編輯，然後按一下檔案索引頁籤。
  3. 勾選啟動檔案索引。
  4. 按一下確定。

為檔案重建索引

在一般情況下，您不需要為檔案重建索引。然而，如果資料夾中的檔案符合搜尋條件，卻沒有出現在檔案的搜尋結果裡，您也許需要重建索引來重新建立檔案的索引結構。

若要為共用資料夾中的檔案重建索引：

  1. 選擇您要重建索引的共用資料夾。
  2. 按一下編輯，然後按一下檔案索引頁籤。
  3. 按一下重建索引。
  4. 過程結束之後，系統會顯示已索引的檔案及子資料夾數量。

注意：File Station 不支援遠端資料夾及外接 USB 硬碟的檔案索引功能。
