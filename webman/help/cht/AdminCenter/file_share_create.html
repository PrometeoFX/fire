建立共用資料夾

共用資料夾是 DiskStation 基本的目錄結構，可讓您儲存並管理檔案及資料夾。儲存檔案之前，您需先建立至少一個共用資料夾。

若要新增共用資料夾：

  1. 在「網路上的芳鄰」隱藏此共用資料夾：此選項可讓共用資料夾不在 Windows 檔案總管的「網路」下出現。勾選此選項並不會影響共用資料夾的存取權限。

    注意：

    讓共用資料夾在「網路上的芳鄰」隱藏並不會影響其存取權限。其存取權限仍是會如同您所指定。有存取權限的使用者仍可透過輸入「\\伺服器名稱\共用資料夾名稱」的方式來存取該共用資料夾。

  2. 對於不具權限的使用者隱藏子資料夾及檔案：啟動此選項後，如果不具讀取權限的使用者試圖透過 Windows 檔案分享通訊協定來存取共用資料夾，將無法看到共用資料夾中的資料夾或檔案。此選項可避免使用者連線至 DiskStation 之後，卻無法開啟大量檔案或子資料夾而產生的困惑。

    注意：

    • 僅 Windows 檔案分享通訊協定支援此功能。
    • 您可以以滑鼠右鍵點按 File Station 的資料夾及檔案來修改其權限設定。從下拉式選單中選擇屬性，並按一下權限頁籤。
    • 啟動對於不具權限的使用者隱藏子資料夾及檔案時，使用者仍可看到共用資料夾。只有共用資料夾中的子資料夾和檔案會被隱藏。
  3. 啟動資源回收筒：共用資料夾中的檔案被刪除後，會移至名為 #recycle 的資料夾。您可將資源回收筒設為只有屬於 administrators 群組的使用者可存取。
  4. 開啟快照瀏覽：您可以透過檔案服務從共用資料夾的 #snapshot 資料夾來存取快照中的資料夾和檔案。快照將以其拍攝的格林威治標準時間 (GMT) 命名。

    注意：

    • 使用者不具有寫入任何快照備份的權限。
    • 使用者可讀取快照備份的條件為該快照拍攝時，該使用者帳號已存在，並已具有該快照所拍攝之共用資料夾的讀取權限。
  5. 啟動共用資料夾空間配額：若要指定個別共用資料夾的最大空間限制，勾選啟動共用資料夾空間配額核取方塊，並在其後的欄位中輸入容量上限數值 (例如：10GB)。

    注意：

    • 只有在 Btrfs 檔案系統下，共用資料夾才能使用開啟快照瀏覽和啟動共用資料夾空間配額選項。
    • 所有使用者的 home 目錄使用空間總額不能超過「homes」共用資料夾的空間配額。
  6. 啟動進階資料完整性保護：若要使用 CRC32 checksum 和 copy-on-write 方法來保護共用資料夾，您可以在建立共用資料夾時啟動進階資料完整性保護。CRC32 checksum 應用於檢查資料是否與原先寫入時完全一致，並於偵測到損毀時通知使用者。Copy-on-write 可於系統發生不正常關機時，協助改善資料的一致性。

    注意：

    • 此選項僅適用於 Btrfs 檔案系統的儲存空間。
    • 您可以在建立每個共用資料夾時啟動 / 停用此選項。共用資料夾建立後，即無法再修改此選項。
    • 啟動此選項可能會影響系統效能。
  7. 加密此共用資料夾：若要加密共用資料夾，請執行以下操作：
    1. 輸入並確認加密金鑰。加密金鑰不可包含半形逗號 (,) 或等號 (=)。
    2. 視需求勾選開機時自動裝載來於系統啟動時自動裝載加密資料夾。若停用此選項，加密資料夾會在下次 DiskStation 重新開機時卸載。若要檢視或存取已卸載的共用資料夾，您需要匯入或輸入加密金鑰，藉此手動裝載此資料夾。
  8. 按一下確定來完成。

共用資料夾命名規則：

  • 共用資料夾名稱不分大小寫，可包含 1 至 32 個字元。但不可包含下列字元：[ ] ; : "" # $ % & ' ( ) * + , / : ;<= >?@ [ ] \ ^ ` { } | ~
  • 下列名稱已被保留，僅供系統使用︰.、..、global、home、homes、printers、public、satashare、surveillance、usbbackup、usbshare。
  • 共用資料夾名稱的第一個字不可為減號及空白，最後一個字不可為空白。
  • 共用資料夾描述有區分大小寫，最多可包含 64 個可顯示的 Unicode 字元。

為共用資料夾加密：

AES 256-bit 加密機制可以封鎖所有未經授權的存取嘗試來保護資料。共用資料夾一旦加密，沒有加密金鑰的使用者將無法存取或修改加密的資料，即使他們將硬碟從 DiskStation 取下並掛載至其它裝置，也一樣無法存取。

注意：

  • 共用資料夾僅能於建立時加密。
  • 加密共用資料夾中的檔案及資料夾名稱不可超過 143 個英文字元或 43 個亞洲字元。
  • 加密的共用資料夾無法透過 NFS 進行存取。
  • 如果您透過 DiskStation 上的重置按鈕來還原系統預設密碼，加密的共用資料夾將被卸載，自動裝載加密共用資料夾的選項也會被停用。
