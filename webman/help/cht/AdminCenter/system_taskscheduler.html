任務排程表

[任務排程表] 可讓您安排系統服務及指令碼於設定時段執行的時間。

支援任務

您可以新增任務來執行以下操作：

  • 使用者定義指令碼：執行使用者定義指令碼或命令。
  • 資源回收筒：清空特定共用資料夾的 [資源回收筒]。此外，檔案保留規則設定可讓您在清空 [資源回收筒] 時決定哪些檔案要永久刪除。例如，您可以根據檔案大小、刪除時間或檔案格式來決定是否刪除或保留。
  • 嗶聲控制：讓 DiskStation 發出嗶聲。
  • 系統服務：開啟或停止網路服務，列如 Windows 檔案服務、FTP、自動封鎖等等。
  • 使用者定義指令碼：執行使用者定義指令碼或命令。
  • 資源回收筒：清空特定共用資料夾的 [資源回收筒]。此外，檔案保留規則設定可讓您在清空 [資源回收筒] 時決定哪些檔案要永久刪除。例如，您可以根據檔案大小、刪除時間或檔案格式來決定是否刪除或保留。
  • 系統服務：開啟或停止網路服務，列如 Windows 檔案服務、FTP、自動封鎖等等。

管理任務

[任務排程表] 的主要頁面顯示了現有的全部任務，並且提供新增、編輯、執行任務的選項。

若要新增任務：

  1. 按一下新增並選擇您想要規劃排程的任務類型。
  2. 新增任務視窗出現後，請在一般頁籤修改以下設定：
    • 一般設定：輸入任務名稱、選擇使用者，並啟動或停用任務。
    • 其他設定：您可以在此頁籤修改其它和任務相關的設定 (限特定任務)。例如，新增清空資源回收筒任務時，您必須選擇某一共用資料夾的資源回收筒。
  3. 在排程頁籤，請修改下列設定：
    • 日期：指定執行任務的時間。您可以安排任務於一週內執行的天數 (如：每天、週末、或每星期日) 或特定的執行日期 (如：2015/10/21)。
    • 時間：指定任務每天執行的次數，以及開始和結束的時間。
  4. 在任務設定頁籤修改和任務相關的設定 (限特定任務)。
  5. 按一下確定來完成。

若要編輯任務設定：

  1. 選擇現有的任務。
  2. 按一下編輯。
  3. 在顯示的視窗中，修改任務設定。

若要刪除任務：

  1. 選擇現有的任務。
  2. 按一下刪除。

若要啟動或停用任務：

  1. 在任務排程表的主要頁面，勾選或取消勾選已啟動下方的核取方塊來啟動或停用任務。
  2. 按一下儲存。

若要手動執行任務：

  1. 選擇單一或多個現有任務。
  2. 按一下執行。

任務排程表限制：

  • 任務名稱可包含最多 512 個可顯示字元。
  • 最大任務數：1024。
  • 指令碼或命令列的長度上限：4095。
  • 嗶聲持續時間可介於 1 至 600 秒。
