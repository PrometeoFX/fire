DHCP Server

您的 DiskStation 可以成為 DHCP 伺服器，將動態 IP 位址指派給區域網路中的 DHCP 用戶端。

若要啟動 DHCP 伺服器：

  1. 前往控制台 > 網路 > 網路介面。
  2. 選擇網路介面 (例如 LAN) 並按一下編輯。
  3. 前往 DHCP Server 頁籤並勾選啟動 DHCP Server。
  4. 在位址租約時間欄位中輸入數值來為 DHCP 用戶端的 IP 位址設定 DHCP 租約時間 (以分鐘為單位)。
  5. 在主要網域名稱伺服器 (DNS) (必填) 及次要網域名稱伺服器 (DNS) (可留空) 欄位中為 DHCP 用戶端輸入 DNS 位址。
  6. 在網域名稱 (可留空) 欄位中為 DHCP 伺服器指定網域名稱。
  7. 按一下確定。

若要新增子網路：

  1. 按一下新增，然後指定下列欄位的相關資訊：
    • 啟動：啟動或停用子網路。
    • 開始位址：子網路的第一個 IP 位址。
    • 開始位址：子網路的最後一個 IP 位址。
    • 結束位址：子網路遮罩。
    • 預設閘道器 (gateway)：子網路的預設閘道器。
  2. 按一下確定。

注意：

  • 您最多只能新增 32 個子網路。

DHCP 用戶端

若要查看 DHCP 用戶端清單及它們的網路設定 (例如：網路實體位址、IP 位址、主機名稱及 DHCP 租約到期剩餘時間)，可以前往 DHCP 用戶端頁籤。

DHCP 保留

如果您要讓用戶端在每次 DHCP 租約更新時都能接收到相同的 IP 位址，則可以將該用戶端加到保留位址清單中。

若要將用戶端加到保留位址清單：

  • 按一下新增。
  • 指定用戶端的網路實體位址、您想指派給用戶端的 IP 位址，以及用戶端主機名稱。
  • 按一下確定來儲存設定。
