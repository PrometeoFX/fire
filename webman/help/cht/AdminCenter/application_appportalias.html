應用程式入口

透過應用程式入口，您可以編輯各種應用程式的連線設定，讓您可在獨立的瀏覽頁籤或視窗中存取並執行這些應用程式。

例如，您可以在網頁瀏覽器上輸入 http://:diskstation:5000/file/ 來直接開啟 File Station。可供使用的連線設定包含了自訂別名、HTTP 埠、HTTPS埠。若要使用此功能，請前往控制台 > 應用程式入口。

注意：

可透過應用程式入口編輯存取設定的應用程式有：Audio Station、Download Station、Surveillance Station、Video Station、File Station 等。

自訂別名

有了您自訂的別名，您便可以在網頁瀏覽器上輸入如 http://:diskstation:5000/file/ 的位址，輕鬆開啟 File Station 等應用程式。

若要啟動自訂別名：

  1. 選擇一個應用程式。
  2. 按一下編輯。
  3. 在顯示的視窗中，勾選啟動自訂別名。
  4. 自訂一個別名

注意：

  • 您可以勾選由此重新導向核取方塊，使用較簡單好記的別名來登入該應用程式。
  • 以下分別為 Audio Station、Download Station、Surveillance Station、Video Station 及 File Station 的預設別名：audio、download、cam、video、file。
  • 自訂的別名名稱不能和系統保留的名稱及瀏覽器相同，也不能是其他應用程式的名稱，名稱字數須限制在 2 到 20 個字元。
  • 別名名稱的組成可以是大小寫字母、數字、兩個特殊字元 - 和 _，但這兩個特殊字元只可出現在字首和字尾以外的其他字元。

自訂 HTTP 或 HTTPS 埠

有了您自訂的 HTTP 或 HTTPS 埠，您便可以在網頁瀏覽器上輸入如 http://diskstation:7000 或 http://diskstation:7001 的位址，輕鬆開啟應用程式。

若要啟動自訂的 HTTP 或 HTTPS 埠

  1. 選擇一個應用程式。
  2. 按一下編輯。
  3. 在顯示的視窗中，勾選啟動自訂連接埠 (HTTP) 或啟動自訂連接埠 (HTTPS)。
  4. 自訂一個連接埠編號

注意：

  1. 連接埠可為 1 到 65535 的任何數字，但不可為下列編號，因為它們已保留給系統使用：20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, eMule 預設埠：4662 (TCP), 4672　(UDP), 以及 FTP 預設埠範圍：實際範圍因機種而異。
