DSM 設定

您可於控制台 > 網路 > DSM 設定變更用戶端用來存取 DSM 的連接埠編號。HTTPS 一旦啟動，連線將會透過 SSL/TLS 機制進行加密。

若要更改默認連接埠編號：

  1. 在 HTTP 或 HTTPS 欄位中輸入連接埠編號。
  2. 按一下套用。

若要啟動 HTTPS 安全連線：

  1. 勾選啟動 HTTPS 安全連線。
  2. 如果您想將嘗試透過 HTTP 連接埠 (5000 埠) 存取 DSM 的用戶端重新導向至 HTTPS 連接埠 (5001 埠)，請勾選將 HTTP 連線自動導到 HTTPS。
  3. 按一下套用。

HTTPS 安全連線使用的連接埠：

  • DSM：預設的連接埠編號為 5001，因此您可以輸入以下位址來安全存取 DSM：https://伺服器主機名稱:5001/
  • Web Station 及 Photo Station：預設的連接埠編號為 443，因此您可以輸入以下位址來安全存取 Web Station 及 Photo Station：https://伺服器主機名稱/

若要透過 Windows 網路探索來搜尋 DiskStation：

  1. 勾選啟動 Windows 網路探索。
  2. 按一下套用。
