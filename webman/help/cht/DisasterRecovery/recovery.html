還原

系統管理員可於此頁面進行還原。若要允許使用者存取共用資料夾快照，請啟動每個共用資料夾的開啟快照瀏覽功能，使用者即可透過 File Station 或 Windows 檔案總管來存取快照。

共用資料夾

若要還原共用資料夾中的個別檔案 / 資料夾：

  1. 選擇您要還原的共用資料夾。
  2. 按一下還原。
  3. 選擇您要還原的快照版本。
  4. 按一下瀏覽，並複製您要還原的檔案 / 資料夾至目的地。

若要還原共用資料夾：

  1. 選擇您要還原的共用資料夾。
  2. 按一下還原。
  3. 選擇您要還原的快照版本。
  4. 按一下動作，並從下拉式選單中選擇依原始狀態還原。
  5. 您將會看到確認視窗。按一下是來確認動作。

若要複製快照到新的共用資料夾：

  1. 選擇您要複製的共用資料夾。
  2. 按一下還原。
  3. 選擇您要複製到新共用資料夾的快照版本。
  4. 按一下動作，並從下拉式選單中選擇複製為新名稱。
  5. 您將被重新導向至控制台 > 共用資料夾 > 新增的複製頁面。輸入複製的共用資料夾的名稱，並按一下確定來完成。

注意：

  • 共用資料夾「homes」無法還原。
  • 複製的新共用資料夾無法加密。
  • 若在檔案傳輸仍在進行時拍攝快照，將無法確保資料一致性。
  • 加密共用資料夾的限制：
    • 掛載的加密共用資料夾將無法以任一方式還原。
    • 加密共用資料夾無法複製為新名稱。
    • 卸載的加密共用資料夾可使用依原始狀態還原方式還原為先前拍攝快照的狀態。

iSCSI LUN

若要還原 iSCSI LUN：

  1. 選擇您要還原的 iSCSI LUN
  2. 按一下還原。
  3. 選擇您要還原的快照版本。
  4. 按一下動作，並從下拉式選單中選擇依原始狀態還原。
  5. 您將會看到確認視窗。按一下是來確認動作。

若要複製快照到新的 iSCSI LUN：

  1. 選擇您要複製的 iSCSI LUN。
  2. 按一下還原。
  3. 選擇您要複製到新 iSCSI LUN 的快照版本。
  4. 按一下動作，並從下拉式選單中選擇複製為新名稱。
  5. 您將被重新導向至儲存空間管理員 > iSCSI LUN的複製頁面。輸入複製的 iSCSI LUN 的名稱，並按一下確定來完成。

注意：

  • 若在檔案傳輸仍在進行時拍攝快照，將無法確保資料一致性。
