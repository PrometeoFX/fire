Editor de Texto

O Editor de Texto é um editor de texto e código altamente versátil, permitindo-lhe aceder e editar rapidamente ficheiros de texto ou códigos no seu DiskStation diretamente a partir de um navegador Web.

Gestão de Ficheiros

Para criar novos ficheiros:

Clique em Ficheiro e escolha Novo. Será apresentado um separador novo vazio com o título Novo Documento.txt.

Para abrir ficheiros a partir do Editor de Texto:

Basta clicar em Ficheiro e escolher Novo. Será apresentada uma caixa de diálogo, onde pode selecionar um ficheiro com uma codificação de caracteres específica que tenha de ser aberto num separador. Se o ficheiro já estiver aberto, o Editor de Texto mostrar-lhe-á esse ficheiro no seu estado atual.

Para abrir ficheiros a partir da File Station:

Na File Station, basta clicar com o botão direito do rato no ficheiro selecionado e, em seguida, clicar em Abrir com o Editor de Texto no menu de contexto. Se o ficheiro já estiver aberto, o Editor de Texto mostrar-lhe-á esse ficheiro no seu estado atual. Uma outra forma de abrir um ficheiro é arrastar um ou vários ficheiros da File Station para o Editor de Texto utilizando o cursor.


Nota:

O Editor de Texto não pode abrir ficheiros que excedam 10 MB.

Para guardar ficheiros:

Basta clicar em Ficheiro e escolher Guardar. Se for um ficheiro existente, será substituído pelos conteúdos atuais. Se pretender guardar todos os ficheiros abertos de uma só vez, pode aceder a Ficheiro > Guardar Tudo. Se o ficheiro ainda não tiver nome (um ficheiro novo) ou se tiver selecionado Ficheiro > Guardar Como, será apresentada uma caixa de diálogo a solicitar o nome do ficheiro. Em seguida, pode escolher a localização, o nome do ficheiro e a codificação a guardar para o ficheiro.

Para fechar ficheiros:

Basta clicar em Ficheiro e escolher Fechar. Uma outra forma de fechar um ficheiro é clicar no botão para fechar em cada separador. Se tiver dados não guardados num ficheiro, o Editor de Texto perguntar-lhe-á sempre se pretende fechar sem guardar essas alterações.

Nota:

Pode guardar ficheiros com uma codificação de caracteres específica para suportar idiomas bidirecionais. Também pode especificar uma codificação ao abrir um ficheiro para que seja apresentado corretamente pelo Editor de Texto.

Editar

  • Operações normais
    O controlo de edição suporta as operações simples de Anular, Refazer e Selecionar Tudo. Estas operações podem ser encontradas na maioria dos editores e funcionam de forma muito semelhante. Pode executar estas operações selecionando o item de menu correspondente no menu Editar. Além disso, por predefinição, as opções de Cortar, Copiar e Colar estão disponíveis através do menu de contexto do navegador.
  • Avanço
    O avanço de texto é uma operação comum na edição de código fonte. O avanço é o processo de adicionar um espaço em branco adicional no início de uma linha para separar blocos de texto, adicionando uma hierarquia. Geralmente, durante a edição, ao premir a tecla Tab, será inserido um caractere de tabulação na localização atual. Contudo, se estiver selecionado algum texto, premir a tecla Tab adicionará um espaço em branco suplementar no início de cada linha com texto selecionado, permitindo-lhe avançar blocos inteiros de texto de uma só vez. Ao premir Shift e Tab em simultâneo, o avanço será removido.
  • Substituir
    Pode ativar ou desativar a substituição premindo a tecla Insert (Inserir). Isto é, em vez de se inserir caracteres na posição do cursor, estes são substituídos. O cursor apresentado mudará de forma consoante o estado da substituição.
  • Edição em modo de coluna
    Uma forma especial de editar ficheiros é a edição em modo de coluna. Isto significa que o texto pode ser selecionado em colunas, em vez da seleção em linhas. Para iniciar uma seleção em modo de coluna, prima a tecla Alt ao selecionar o texto, movendo o cursor para baixo ou para cima, conforme necessário. Quando o texto for selecionado no chamado modo de coluna, a introdução irá alterar todas as colunas de caracteres selecionadas para o caractere introduzido.

Pesquisar

  • Pesquisa normal
    Pode pesquisar texto tal como na maioria dos programas através da caixa de diálogo Localizar. Pode abrir esta caixa de diálogo selecionando Pesquisar > Localizar. Esta caixa de diálogo permanecerá sempre sobre o texto e não será possível continuar a editar o texto enquanto a caixa de diálogo estiver aberta. No campo de pesquisa, escreva o texto que pretende localizar.
  • Substituir
    Pode localizar e substituir texto tal como na maioria dos programas através da caixa de diálogo Substituir. Pode abrir esta caixa de diálogo selecionando Pesquisar > Substituir. Substituir Tudo substituirá todas as ocorrências do pedido pelo texto de substituição.
  • Sensível a maiúsculas e minúsculas
    As ocorrências apresentadas serão exatamente iguais ao pedido no que se refere a maiúsculas e minúsculas. Se esta opção não estiver assinalada, a pesquisa não será sensível a maiúsculas e minúsculas.
  • Apenas palavra inteira
    Resultará em ocorrência apenas se a correspondência for uma palavra inteira e não uma parte de uma palavra. Por exemplo, se procurar a palavra "casa" com Apenas palavra inteira, irá localizar apenas "casa" e não "casamento".
  • Expressão regular
    O pedido será processado como uma expressão regular.

O que é uma expressão regular?

Uma expressão regular é uma sequência de caracteres que forma um padrão de pesquisa como técnica poderosa de poupança de tempo. Uma utilização muito simples de uma expressão regular seria localizar a mesma palavra escrita de duas formas diferentes num editor de texto, por exemplo, cota/quota.

Preferências

  • Mostrar caracteres invisíveis
    Se assinalado, serão mostrados caracteres de espaço, tabulação, nova linha ou quebra.
  • Mostrar guias de avanço
    Se assinalado, serão mostradas guias de avanço. Esta opção é útil para alinhar os códigos na linguagem de codificação.
  • Realçar linha atual
    Se assinalado, a linha em que se encontra o sinal de inserção será apresentada com uma cor de fundo especial.
  • Realçar palavra selecionada
    Se assinalado, todas as palavras (não parciais) dentro do mesmo ficheiro de texto que correspondam à palavra selecionada serão automaticamente realçadas.
  • Mostrar números das linhas
    Se assinalado, os números das linhas serão apresentados.
  • Ativar dobragem de código
    Se assinalado, certas linhas dos códigos podem ser ocultas com base em determinadas características. Esta funcionalidade funciona bem para linguagens estruturadas, tais como C++ ou XML. Se Ocultar widgets de divulgação automaticamente estiver selecionado, determinadas linhas de códigos serão dobradas automaticamente. Também pode optar por marcar apenas o parêntese de abertura ou ambos os parênteses de abertura e de fecho com os widgets de divulgação.
  • Moldagem do Texto
    Especifique se pretende que o texto seja moldado na margem da janela ou num determinado número de coluna personalizado sem inserir uma quebra de parágrafo.
  • Acrescentar caracteres de fecho automaticamente
    Se assinalado, os parênteses de fecho (caracteres) serão acrescentados automaticamente aos seus códigos.
  • Inserir espaços em vez de tabulações
    Se assinalado, ao premir a tecla Tab, serão inseridos espaços em vez de tabulações. A Largura da tabulação pode ser definida pelo número de caracteres em Preferências > Tipo de letra e cor.
  • Codificação predefinida
    A codificação predefinida é utilizada para guardar ficheiros com uma codificação de caracteres específica para suportar idiomas bidireccionais. Também pode especificar uma codificação ao abrir um ficheiro para que seja apresentado corretamente pelo Editor de Texto.
  • O Tipo de letra e cor
    Largura da tabulação, Tamanho do tipo de letra e Tema podem ser personalizados de acordo com as suas preferências pessoais. A Largura da tabulação é definida pelo número de espaços.

Formatos de Ficheiro Suportados

O Editor de Texto apenas suporta ficheiros de texto simples. Os ficheiros de texto com formatação avançada, como *.doc e *.pdf, não são suportados.
