Gerir Perfis de Relatório

Pode criar e alterar tarefas agendadas para gerar relatórios sobre a utilização do volume em Analisador de Armazenamento > Perfil de Relatório.

Para criar uma tarefa de relatório:

  1. Clique em Criar para abrir o assistente.
  2. Especifique um nome para a nova tarefa de relatório e introduza o endereço de e-mail para o qual pretende enviar o relatório.
  3. Selecione um ou vários tipos de relatório que pretende gerar.
  4. Execute uma das seguintes opções para determinar uma hora para gerar os relatórios:
    • Assinale a opção Gerar relatório de acordo com agendamento e especifique a hora se pretender que o sistema analise periodicamente a utilização do volume.
    • Assinale a opção Gerar relatórios agora se pretender dar início agora de forma manual a uma tarefa de relatório.
  5. Se necessário, especifique determinados utilizadores ou pastas partilhadas a incluir para reduzir os relatórios. (Note que se pretender especificar e gerar relatórios de pastas partilhadas que estejam encriptadas, é favor certificar-se que as partilhas encriptadas estão devidamente montadas antes de a tarefa de relatório ser desempenhada.)
  6. Clique em Aplicar para guardar as definições.

Nota:

  • Prima Enter/Return depois de cada endereço de e-mail para separar múltiplos endereços de e-mail.
  • Prima Enter/Return depois de cada utilizador, quando especificar utilizadores cujos ficheiros serão listados individualmente por grupos de ficheiros.
  • O relatório Ficheiros por Grupo de Ficheiros irá listar os seguintes tipos de ficheiros:
    Ficheiros áudio, ficheiros de imagem de disco, documentos, ficheiros executáveis, imagens, vídeos, páginas web, ficheiros comprimidos e outros
  • A secção dos Ficheiros Grandes do relatório lista os 500 ficheiros maiores na página do relatório de utilização ou os 200 ficheiros maiores na página do relatório completo.

Para editar uma tarefa de relatório:

  1. Selecione uma tarefa de relatório da lista e clique em Editar.
  2. Clique nos separadores individuais para alterar as definições da tarefa.

Para eliminar uma tarefa de relatório:

Selecione uma tarefa de relatório da lista e clique em Eliminar.

Para ver relatórios históricos:

  1. Selecione uma tarefa de relatório da lista e clique em Relatório > Ver relatórios históricos.
  2. Na janela que aparece, clique numa entrada para ver o relatório gerado na hora especificada.

Para gerar relatórios de imediato:

Selecione uma tarefa de relatório da lista e clique em Relatório > e escolha Gerar relatórios agora para gerar relatórios a pedido.

Para alterar onde são guardados os relatórios:

  1. Clique em Definições.
  2. Selecione uma nova pasta de destino para alterar a localização onde são guardados os relatórios.

Para reunir um histórico de utilização do volume, de forma regular:

Por predefinição, os dados de utilização do volume são reunidos sempre que é gerado um relatório de Utilização do Volume e no arranque do sistema. Para além desses casos, pode também optar por um novo horário para recolha do histórico de utilização. Para tal, clique em Definições, assinale Recolher histórico de utilização de volume de acordo com um agendamento e agende uma hora.

Nota:

O histórico recolhido na hora aqui agendada será essencialmente utilizado para efeitos e como fonte de tendências de utilização na secção Utilização de Volume. Sempre que esta opção estiver ativa, verá um gráfico de linhas com as tendências e alterações relativas à utilização do volume ao longo do tempo, caso contrário, será apresentado um gráfico de barras com a percentagem atual de utilização.

Para analisar uma tarefa de relatório:

Clique duas vezes na tarefa de relatório pretendida ou clique simplesmente em Analisar esta tarefa de relatório (no ícone >) junto da tarefa de relatório pretendida para abrir a respetiva página de análise. Consulte Ver Utilização e Relatórios para obter mais informações.
