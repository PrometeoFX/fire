Recuperação

Esta página permite aos administradores de sistema realizar uma recuperação. Para permitir que utilizadores acedam a instantâneos de pastas partilhadas, active Tornar visível instantâneo para cada pasta partilhada, para que os utilizadores possam aceder aos instantâneos através da File Station ou Explorador de Ficheiros do Windows.

Pasta Partilhada

Para restaurar ficheiros/pastas individuais numa pasta partilhada:

  1. Seleccione uma pasta partilhada que deseja restaurar.
  2. Clique em Recuperação.
  3. Seleccione uma versão de instantâneo da qual deseja restaurar.
  4. Clique em Percorrer, e copie os ficheiros/pastas que deseja restaurar para o destino.

Para restaurar uma pasta partilhada:

  1. Seleccione uma pasta partilhada que deseja restaurar.
  2. Clique em Recuperação.
  3. Seleccione uma versão de instantâneo da qual deseja restaurar.
  4. Clique em Acção, e seleccione Restaurar em substituição da lista de selecção.
  5. Verá uma mensagem de confirmação. Clique em Sim para confirmar a acção.

Para clonar um instantâneo para uma nova pasta partilhada:

  1. Seleccione uma pasta partilhada que deseja clonar.
  2. Clique em Recuperação.
  3. Seleccione uma versão de instantâneo que deseja clonar para uma nova pasta partilhada.
  4. Clique em Acção, e seleccione Clonar com um novo nome da lista de selecção.
  5. Será redireccionado para a página Clonar no Painel de Controlo > Pasta Partilhada > Criar. Introduza o nome da pasta partilhada clonada e clique em OK para terminar.

Nota:

  • Não é possível restaurar os "homes" da pasta partilhada.
  • A pasta partilhada clonada não pode estar encriptada.
  • Se for tirado um instantâneo quando está a decorrer uma transferência de quaisquer ficheiros, não pode ser garantida a consistência dos dados.
  • Limitações em pastas partilhadas encriptadas:
    • Nenhum tipo de recuperação está disponível para pastas partilhadas montadas e encriptadas.
    • Clonar com um novo nome não está disponível para pastas partilhadas encriptadas.
    • Pastas partilhadas encriptadas e desmontadas podem ser restauradas a partir de instantâneos tirados anteriormente, usando Restaurar em substituição.

iSCSI LUN

Para restaurar um iSCSI LUN:

  1. Selecione um iSCSI LUN que pretende restaurar.
  2. Clique em Recuperação.
  3. Seleccione uma versão de instantâneo da qual deseja restaurar.
  4. Clique em Acção, e seleccione Restaurar em substituição da lista de selecção.
  5. Verá uma mensagem de confirmação. Clique em Sim para confirmar a acção.

Para clonar um instantâneo para um novo iSCSI LUN:

  1. Selecione o iSCSI LUN que pretende clonar.
  2. Clique em Recuperação.
  3. Seleccione uma versão de instantâneo que deseja clonar para um novo iSCSI LUN.
  4. Clique em Acção, e seleccione Clonar com um novo nome da lista de selecção.
  5. Será redireccionado para a página Clonar no Gestor de Armazenamento > iSCSI LUN. Introduza o nome do iSCSI LUN clonado e clique em OK para terminar.

Nota:

  • Se for tirado um instantâneo quando está a decorrer uma transferência de quaisquer ficheiros, não pode ser garantida a consistência dos dados.
