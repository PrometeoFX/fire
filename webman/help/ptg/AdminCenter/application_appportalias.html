Portal de Aplicações

Com o Portal de Aplicações, pode configurar as definições de ligação de várias aplicações, permitindo que execute e aceda a estas aplicações em janelas ou separadores independentes do navegador.

Por exemplo, poderia abrir a File Station introduzindo http://:diskstation:5000/file/ no navegador web. As definições de ligação disponíveis incluem portas HTTP e HTTPS e aliases personalizados. Para encontrar estas definições, vá para Painel de Controlo > Portal de Aplicações.

Nota:

O Portal de Aplicações permite editar as definições de acesso para várias aplicações, tais como a Audio Station, a Download Station, a Surveillance Station, a Video Station, a File Station, entre outras.

Personalizar o Alias

Com um alias personalizado, poderia abrir uma aplicação como a File Station introduzindo um endereço tal como http://:diskstation:5000/file/ no navegador web.

Para activar um alias personalizado:

  1. Seleccione uma aplicação.
  2. Clique em Editar.
  3. Na janela que aparece, assinale Activar alias personalizado.
  4. Especifique um alias personalizado.

Nota:

  • Pode assinalar a caixa de verificação Redireccionar de e utilizar um alias mais pequeno e de fácil memorização para aceder à aplicação seleccionada.
  • Seguem-se os respectivos aliases predefinidos das aplicações, Audio Station, Download Station, Surveillance Station, Video Station, File Station: audio, download, cam, video, file.
  • Os nomes de alias não podem ser iguais aos reservados para o sistema, navegadores ou utilizados por outras aplicações e o respectivo número de caracteres tem de ser limitado entre 2 e 20.
  • Os nomes de alias podem conter letras com maiúsculas e minúsculas, números e dois caracteres especiais (-), (_) mas estes apenas podem ser utilizados para outros caracteres excepto o primeiro e o último.

Portas HTTP ou HTTPS Personalizadas

Com as portas HTTP ou HTTPS personalizadas, é possível abrir uma aplicação introduzindo um endereço tal como http://diskstation:7000 ou http://diskstation:7001.

Para activar uma porta HTTP ou HTTPS personalizada:

  1. Seleccione uma aplicação.
  2. Clique em Editar.
  3. Na janela que aparece, assinale Activar porta personalizada (HTTP) ou Activar porta personalizada (HTTPS).
  4. Especifique um número de porta personalizada.

Nota:

  1. O intervalo do número de porta pode ser entre 1 e 65535, mas as seguintes portas devem ser excluídas porque estão reservadas para utilização do sistema: 20, 21, 22, 23, 25, 80, 110, 137, 138, 139, 143, 199, 443, 445, 515, 543, 548, 587, 873, 993, 995, 3306, 3689, 5000, 5001, 5005, 5006, 5335, 5432, 6881, 8080, 7000, 7001, 8081, 9997, 9998, 9999, 50001, 50002, portas predefinidas do eMule: 4662 (TCP), 4672 (UDP) e intervalo de portas FTP predefinidas: o intervalo real poderá variar consoante os diferentes modelos.
