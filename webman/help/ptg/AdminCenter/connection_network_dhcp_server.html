DHCP Server

O seu DiskStation pode assumir a função de um servidor DHCP e atribui endereços IP dinâmicos a clientes DHCP na rede de área local.

Para ativar o servidor DHCP:

  1. Aceda a Painel de Controlo > Rede > Interface de Rede.
  2. Selecione uma interface de rede (por exemplo, LAN) e clique em Editar.
  3. Aceda ao separador DHCP Server e assinale Ativar DHCP Server.
  4. Introduza um valor no campo Tempo de concessão de endereço para definir o tempo de concessão DHCP (em minutos) para endereços IP atribuídos a clientes DHCP.
  5. Introduza o endereço DNS para clientes DHCP nos campos DNS Principal (necessário) e DNS Secundário (opcional).
  6. Defina o nome de domínio para o servidor DHCP no campo Nome de domínio (opcional).
  7. Clique em OK.

Para adicionar uma sub-rede:

  1. Clique em Adicionar e depois especifique as informações nas seguintes colunas:
    • Ativar: Ativa ou desativa a sub-rede.
    • Início: O primeiro endereço IP da sub-rede.
    • Fim: O último endereço IP da sub-rede.
    • Máscara de rede: A máscara de sub-rede.
    • Gateway: O gateway da sub-rede.
  2. Clique em OK.

Nota:

  • Só pode adicionar até 32 sub-redes.

Clientes DHCP

Para ver uma lista de clientes DHCP e respetivas configurações de rede (por exemplo, endereço MAC, endereço IP, nome de anfitrião e o tempo restante antes de a concessão DHCP expirar), pode aceder ao separador Clientes DHCP.

Reserva DHCP

Se pretende que um cliente receba sempre o mesmo endereço IP durante a renovação da concessão DHCP, pode adicionar o cliente à lista de reservas de endereços.

Para adicionar um cliente à lista de reserva de endereços:

  • Clique em Adicionar.
  • Especifique o endereço MAC do cliente, o endereço IP que pretende atribuir ao cliente e o nome de anfitrião do cliente.
  • Clique em OK para guardar as suas definições.
