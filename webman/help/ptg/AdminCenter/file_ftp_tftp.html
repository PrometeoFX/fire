TFTP/PXE

As opções em Painel de Controlo > Serviços de Ficheiros > TFTP/PXE permitem configurar as definições de TFTP e PXE.

TFTP

TFTP (Protocolo Trivial de Transferência de Ficheiros) é um simples protocolo de transferência de ficheiros, normalmente utilizado para transferir ficheiros de configuração ou arranque, em situações em que não é necessária autenticação. Ativar o serviço TFTP permite ao DiskStation transferir carregadores de arranque para dispositivos cliente.

Para modificar as definições TFTP:

  1. Assinale a caixa marcada Ativar serviço TFTP.
  2. No campo Pasta de raiz TFTP, especifique que pasta no DiskStation pode ser acedida por clientes TFTP.
  3. Pode clicar no botão Definições Avançadas para ver e modificar as seguintes definições:
    • Ativar o registo de transferência do ficheiro TFTP: Esta opção permite-lhe manter um registo dos dados transferidos por TFTP.
    • Permissão de cliente TFTP: Selecione as permissões leitura e escrita para clientes TFTP. Quando está selecionado Só de leitura, os clientes TFTP apenas podem ver o conteúdo da pasta de raiz. Quando está selecionado Gravável, os clientes TFTP podem modificar o conteúdo da pasta de raiz.
    • Clientes permitidos: Escolha que clientes podem ligar através de TFTP. Pode permitir todos os clientes selecionando Todas as ligações ou pode limitar quais os clientes que podem efetuar a ligação especificando um intervalo de endereços IP.
  4. Depois de modificar as definições, clique em Aplicar para guardar.

Configuração do serviço DHCP para PXE com o seu DiskStation

Para além de TFTP, o DiskStation pode também providenciar um serviço DHCP para implementar uma configuração tudo-em-um PXE. O Ambiente de Execução Pré-Arranque (PXE, Pre-boot Execution Environment) é um ambiente de rede que permite que dispositivos cliente arranquem utilizando uma interface de rede em vez de um dispositivo de armazenamento de dados (por exemplo, discos rígidos). Quando implementa o PXE, pode ser armazenado um carregador de arranque no DiskStation e os dispositivos cliente podem arrancar através da rede utilizando o carregador de arranque.

Para providenciar serviços de DHCP com o DiskStation, certifique-se de que instalou o pacote Synology DHCP Server.

Para configurar o serviço DHCP para PXE:

  1. Assinale a caixa marcada Configurar serviço DHCP neste servidor para PXE. Será solicitado que instale o pacote DHCP Server se ainda não estiver instalado no seu DiskStation.
  2. Selecione um ficheiro de carregador de arranque da pasta de raiz TFTP e especifique outras definições relativas a DHCP.
  3. Clique em Aplicar para guardar as definições.
