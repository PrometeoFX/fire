QuickConnect

QuickConnect permite que as aplicações cliente estabeleçam ligação a DiskStation através da Internet sem as complicadas configurações de regras de reencaminhamento de portas. O QuickConnect também pode funcionar com pacotes desenvolvidos pela Synology, como a Audio Station, Video Station, Download Station, Surveillance Station, Photo Station, File Station, Note Station, CMS, Cloud Station e aplicações móveis.

Para ativar o QuickConnect:

  1. Aceda a Painel de Controlo > QuickConnect.
  2. Selecione Ativar o QuickConnect:
  3. Introduza as informações da sua Conta Synology existente ou registe uma nova conta.
  4. Especifique um novo QuickConnect ID. Certifique-se de que é fácil de memorizar, para que possa aceder ao seu DiskStation em qualquer altura.
  5. Clique em Aplicar.

Para ativar/desativar o QuickConnect para pacotes específicos:

  1. Aceda a Painel de Controlo > QuickConnect.
  2. Clique em Definições Avançadas.
  3. Escolha os pacotes para os quais pretende ativar o QuickConnect.
  4. Clique em Aplicar.

Nota:

  • Um QuickConnect ID personalizado pode incluir letras, números e hífenes ("-") e deve começar com uma letra.
  • Para um melhor desempenho do QuickConnect, recomendamos que aceda a Painel de Controlo > Acesso Externo > Configuração do Router para configurar o reencaminhamento de portas para cada serviço:
    • DSM: 5000 (HTTP); 5001 (HTTPS)
    • Photo Station: 80 (HTTP); 443 (HTTPS)
    • Cloud Station：6690

Resolução de problemas

Se vir a mensagem Ocorreu um erro de rede, significa que o serviço QuickConnect parou devido a erros de rede. Verifique o seguinte:

  • Se o seu DiskStation está ligado a uma rede ativa (em Painel de Controlo > Rede > Interface de Rede).
  • Se o servidor DNS e predefinições de gateway estão devidamente configurados (em Painel de Controlo > Rede > Geral).
  • Certifique-se de que DiskStation pode aceder à Internet utilizando as seguintes portas de destino: 80, 443, 8888. 

Quando os erros de rede forem corrigidos, o serviço QuickConnect deverá retomar automaticamente após alguns minutos.
