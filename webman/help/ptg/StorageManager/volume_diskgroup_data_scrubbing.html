Análise de Dados*

A análise de dados é uma funcionalidade de manutenção de dados que inspecciona volumes ou grupos de discos e corrige quaisquer inconsistências de dados detectadas. Esta função pode ser usada com volumes ou grupos de discos que implementem SHR (com três ou mais discos) RAID 5 ou RAID 6.

Recomendamos a realização de uma análise de dados regular, para manter a consistência de dados e evitar perdas de dados críticos no caso de falha do disco rígido.

Para realizar a análise de dados:

  1. Abra o Gestor de Armazenamento.
  2. Aceda a Volume ou Grupo de Discos
  3. Seleccione o volume ou grupo de discos para realizar a análise de dados.
  4. Clique em Gerir
  5. Seleccione Análise de dados.

Nota:

  • A análise de dados apenas pode ser realizada quando o estado do volume ou grupo de discos for Normal.
  • Não desligue o sistema antes de terminar a análise de dados. Se o sistema for encerrado antes da conclusão, a análise de dados deverá ser reiniciada.
